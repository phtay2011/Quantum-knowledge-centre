Data Loader

Authorized user can use this screen to view status of deals being imported by AG Quantum - Data Loader Service. It also allows user to perform following actions on selected deals.

  • Delete - This action will be enabled / disabled based on ‘DELETE’ configuration key in Web.config. This configuration value can be set to Logical/Physical. Based on the value set in the configuration, the deals are either logically or physically deleted from the staging table. With logical delete it marks the deal’s status as “Deleted by user”.
  • Delete from Quantum - This action will be enabled / disabled based on ‘ALLOWQUANTUMDELETE’ key in Web.config. This option is configured to delete deals from AG Quantum. If this option is True and the status of deal is “Processed” and delete is allowed then by setting the DELETE configuration key to either Logical or Physical, the Delete from Quantum button would get enabled.
  • Reprocess - This action will be enabled / disabled based on ‘ALLOWREPROCESS’ key in Web.config. Setting this key to true will make the Reprocess button enabled. This depends upon other configuration settings as well.
    • The deal type should be listed in REPROCESSDEALTYPES configuration key.
    • The selected status in the lookup should be a valid status from the REPROCESSDEALSTATUS comma separated values.
  • Release - This action will be enabled / disabled based on ‘RELEASEDEALSTATUS’ key in Web.config. This action resets selected deals to status ‘1. Waiting to be Processed’
  • Refresh - This action is used to get the deals from database. This is used to check if other user has updated the same data and we are working on updated data. This action will get enabled on selection of a grid record.

Filters (default - you can add others)

  • Days - enter the days for which you want to view deals from the database. Any changes before this value will not appear in the report results.
  • Status - enter the status for which you want to view deals from the database.

WHAT DO YOU WANT TO DO?

  • View different deals - Click on the tabs to view different deals.
  • Select a different result View - Click on the current View heading and select the View in the drop-down list.
  • Learn about Filters and Options
  • Change the View or create a new one - Click the View Configure link, make your changes, and save them.

TIP: Resource Security settings do not apply on this screen.
