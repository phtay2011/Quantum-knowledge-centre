Partial dedesignation

Access the Partial Dedesignation form by selecting Dedesignate Partial in the page menu. Use this form to change the percent amount of the deal allocated to the hedge relationship.

Dedesignation Date - Specify the date from which you want the new allocation percentage to take effect. The default is Today's date.

Enter a date that is equal to or falls within the Start Date and End Date of the selected deal.

NOTE: You must enter a valid Dedesignation Date before the Dedesignation Details grid is populated with the linked deals in the hedge relationship.

Defer / Immediate - (only displayed for EXPOSURE or HYPOTHETICAL deals) This field specifies how you want to treat the release of either other comprehensive income (OCI) for a CASH FLOW hedge type, or Mark To Market (MTM) for a fair value hedge type.

The available options are as follows:

  • Amortize Release applies when Hedge Type is Fair Value and Hedge Dedesignation is AMORTIZED. The MTM is amortized from the dedesignation date to the maturity date of the hedge relationship.
  • Defer Release applies when Hedge Type is Fair Value and Hedge Dedesignation is DEFERRED. The MTM is retained in the Revaluation gain/loss accounts on the Balance Sheet until the maturity of the hedge relationship.
  • Amortize Release of OCI applies when Hedge Type is Cash Flow and Hedge Dedesignation is AMORTIZED. The release of OCI is Amortized.
  • Defer Release of OCI applies when Hedge Type is Cash Flow and Hedge Dedesignation is DEFERRED. The release of OCI is Deferred.
  • Immediate Release*applies when Hedge Type is Fair Value and Hedge Dedesignation is AMORTIZED.
  • Immediate Release of OCI* applies when Hedge Type is Cash Flow and Hedge Dedesignation is AMORTIZED.

    * Select Immediate Release options (INSTEAD of the Amortize or Defer Release options) if you want to specify one of the following immediate release options for the MTM (fair value hedge) or OCI (cash flow edge).

Release Account - (cash flow hedges only) select one of:

  • Release OCI to Hedge Gain Loss - to reverse the hedge accounting entries, reinstating the hedge instrument gain/loss that had previously been offset by the hedge relationship. This posts back to the Hedge Reclassification P&L account specified for the entity.
  • Reclassify OCI to Profit & Loss - to post to the OCI Reclassification (P&L) account specified for the entity, in order to match P&L recognition on the underlying hedged exposure.
  • Reclassify OCI to Balance Sheet - to post to the OCI Reclassification (B/S) account specified for the entity, as a basis adjustment for a hedged transaction resulting in a recognized asset or liability.

Release Amount - select one of:

  • Full (All Dedesignations) - to release the MTM/OCI related to all past and current dedesignation events to Profit & Loss. Enter the Percentage to release in the adjacent field.
  • Strategic (Current Dedesignation only) - to release MTM/OCI related to the current dedesignation only.
  • Past (Current and % of Past) - to release all MTM/OCI related to the current dedesignation event, and a percentage of MTM/OCI from past dedesignation events. Enter that Percentage to release in the adjacent field.

Percentage to Release - The percentage of MTM/OCI to be released.

Linked Deals Grid

NOTES:

  • You must enter a valid Dedesignation Date before the Dedesignation Details grid is populated with the linked deals in the hedge relationship.
  • Click the linked deal in the Dedesignation Details grid to edit the allocation details from the Edit Dedesignation Details form.

The following columns are available in the Linked Deal Designation grid.

Deal Number - Displays the deal number of the linked deal.

Transaction Type - Displays the transaction type of the deal.

Hedge / Exposure - Displays whether the deal is a Hedge or Exposure.

Current Allocation, New Allocation - Displays the current allocation amount and the new allocation amount respectively.

Curr %, New% - Displays the current allocation percentage and the new allocation percentage respectively.
