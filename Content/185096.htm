Configure Displayed Totals in Pivots

You can configure how the system calculates and displays grouped and aggregated totals in Pivots.

About Level Numbers

Main points:

  • By default, a pivot displays the Level 0 fields.
  • When there is a higher level configured, the pivot will display hyperlinks that can be used to drill down to that level - see Pivot Drill-Down summary and Examples.
  • Navigating to the next level will make all fields up to that level visible.
  • A breadcrumb trail appears at the top of the page from Level 1 onwards to show the context of the drill-down.
  • Click a higher level in the breadcrumb trail to return to that level.
  • You can use the assigned levels to split the pivot into separate views.

Rows - group rows by (or Columns - group columns by) settings

For each selected grouped field, optionally:

  • select the Sub Total option to display in-line subtotals per group;
  • select the End Total option to display end totals per group, together at the end of the pivot;
  • select Hidden where you want to group by a field value but not show the value itself - for example, when you want to group by an Analysis Code but hide the code value itself because the code description is displayed;
  • select the drill-down Level - 0, 1, 2, 3, and so on (see About Level Numbers above);
  • if the field is a number, amount, or date, select the display Format (see Format Examples below).

Values - aggregate on settings

Optionally select one or more fields you want to display aggregated values for, and for each of those fields:

  • in Aggregate By and Total By, select the type of aggregation and type of total to display - one of the following in each case (depending on the data type):
    • None - no display,
    • Count - the number of instances,
    • Distinct Count - the number of unique items rather than total number of items,
    • First - returns the first result matching the criteria,
    • Sum (Amount or Number only) - the arithmetic sum of the aggregated values,
    • Running Sum (Amount or Number only) - the accumulated sum of values aggregated so far,
    • Reverse Running Sum (Amount or Number only) - the accumulated sum of values aggregated in reverse order,
    • Max - the maximum value,
    • Min* - the minimum value,
    • Mean (Amount or Number only) - the arithmetic mean (the average), or
    • Mean (excluding zero) (Amount or Number in Analytics only) - the arithmetic mean or with any zero values excluded.
  • select the drill-down Level (see About Level Numbers above), and
  • if required, select the display Format (see Format Examples below).

Grand Total options

  • Select Show grand total row to display a grand total row at the end of the output that gives totals of all columns with totals specified.
  • Select Show grand total column to display a grand total column at the right of the output that gives totals of all rows with totals specified.
  • (Analytics only) Select Total on aggregated values to display a row showing totals of any aggregated values.
  • Select Split view by level 0 groups - to split the Pivot view, so that Level 0 will be included in the View selection list and the Pivot will be displayed at the next level.
  • Select Drill down to table view - to enable drill-down to a table view and in the Table View area, select the Columns and Sort items for that view.
