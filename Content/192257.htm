Deals that can use imported market values

Imported market values can be applied to certain deals where the revaluation method for the instrument used in the deal has been defined as Imported or Imported Separate (where there are two values imported) in Accounting Rules.

NOTE: The appropriate instruments must be set up. See Setting up the instrument.

The following deal types support imported market values:

  • Actual and Projected Cash Flows

    For example, an imported market value can be applied to a Forecasted Transaction or Firm Commitment that is part of a Hedge Relationship.

    A Forecasted transaction is one that is expected to occur, but for which there is no firm commitment. This means that when the event does occur, it will be at the prevailing market rates.

    A Firm commitment is an agreement with an unrelated party that is binding on both parties and usually legally enforceable. It has the following characteristics:

    The agreement specifies all significant terms, including the:

    • quantity to be exchanged,
    • timing of the transaction, and
    • Fixed price, expressed as a specified:
      • amount - in the Entity's functional currency or in a foreign currency,
      • interest rate, or...
      • effective yield.

    The agreement includes a disincentive for non-performance, significant enough to make performance probable.

  • FX Deals

    If the revaluation method is Imported or Imported Separate, an imported market value will be used to revalue the deal. How the book value is calculated depends on whether forward points are accrued/amortized, see Revaluing an FX Deal using an imported market value.

    FX Deals can be used to enter commodities in Quantum, if you set the Deal Rate to COMMODITY FWD.

  • Currency Options

    To revalue an exotic currency option using a revaluation method other than those already provided in Quantum, a market value can be imported.

    If a Currency Option is revalued on or after its maturity date, and no record has been imported for that date, the system assumes an imported market value of zero.

  • Interest Rate Swap Deals

    The revaluation method is defined as Imported Separate, indicating that two values will be imported to value the Pay and Receive legs of the Swap deal separately. The P & L is calculated on a per leg basis and then added to get the correct value for the deal.

  • Currency Swap Deals

    The revaluation method is defined as Imported Separate, indicating that two values are imported. The revaluation currency controls whether one or two postings occur.

  • Money Market Deals

    The revaluation method is defined as Imported.

  • Interest Rate Option Deals

    The revaluation method is defined as Imported.

  • Hypothetical deals

    You can import market values for the following Hypothetical derivative deals:

    • Currency Options (Vanilla),
    • FX Deals,
    • Futures and Exchange Traded Options (ETO),
    • Interest Rate Options (Caps, Floors and Collars), and..
    • Swaps(Currency and Interest Rate).
  • FRA Deals

    The revaluation method is defined as Imported.

  • Futures and ETO Deals

    The revaluation method is defined as Imported.

  • Repo Agreements

    The revaluation method is defined as Imported.
