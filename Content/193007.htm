Calculated Column enhancements - Quantum Report Views

When you are configuring a Quantum report View* and adding or editing a Calculated Column expression, you can now:

  • select an If or Join (in addition to the existing Abs and Round) formula,
  • select a date function (Date, Day, Month, Year, or Today), and/or
  • select a logical operator.

    TIP: Logical expressions that evaluate to a boolean value return a Yes or No value.

Additionally, you can use full Search functions when browsing for a field, function, or operator to include.

*NOTE: In Analytics queries, Calculated Columns are configured at the query level, within the selected Columns definition.
