Data Loader

Security Instrument

The Data Loader page now supports a way to import Security instruments and you can view the instrument records in Deal Viewer.

To make the SECINSTR tab visible in the deal viewer, configure the following keys in the Data Loader Web Configuration file SunGard\AvantGard Quantum\QuantumWeb\Areas\Processing\Views\AGSDeals\Web.config.

The REPROCESSDEALTYPES key contains the reprocess deal types.

The VISIBLEDEALTYPES key configures the deal types you want visible on the web page.

Allowed values are: ALL, AC, PC, FX, MM, AT, RA, CO, LC, IO, SWAP, FU, HR, GL, QUOTES, CPARTY, ANLAYTICSCOUNTERPARTYRATING, BLOOMBERGFACTORIMPORT, UN, FXPDE, DEALALLOCATION, SECINSTR.

  • Separate deal types by pipe characters (|) as shown above. For example, FX|MM shows only FX and MM tabs.
  • If you specify ALL, no other values are necessary.

SecDeal Monitor

A new component called SecDeal Monitor has been introduced in Data Loader, to perform following tasks:

  • Continuous polling of the Data Loader's Deals staging tables.

    Once it finds the required deal, it will create the Bloomberg’s request file to get the Instrument or Security Identifier details from Bloomberg for that deal.

  • During polling, finding if that required Instrument or Security Identifier is available.

    If available, it will update the status of those deals in Data Loader’s staging table so that the Loader component can pick them for processing.

For more details of how to configure this component, refer to the Data_Loader_Mappings_Guide.pdf and Data_Loader_Service_Installation_Instructions.pdf files, available in the Documentation folder.

Support for Exclamation Mark delimiters

The Data Loader parser now supports the use of an exclamation mark (!) delimiter in data file content.
