To import customized Interest Rate Options deals, you must:

  • Use the dm_data field.
  • Set the customised field to Y.
  • Supply deal map row items (dm_data entries) in the following order, with a pipe character (l) separating each entry:
    • Deal map line number
    • Action date
    • Effective date
    • Comment
    • Amount
    • Flag

Changing Deal Map Items (QXT)

All items in the deal map are associated with an Event Number. The Event Number uniquely identifies the Deal Map item and is automatically generated for cash flows and events in the life cycle of the deal.

When changing the deal map using a QXT file, the Event Number determines its treatment within the Deal Map; when the Event Number is set to 0 (zero) or not present, a new Deal Map item is created; when the Event Number matches an existing Deal Map item, the item is updated in the Deal Map.

IMPORTANT:

  • Retain the Event Number of an existing deal map item to ensure that Quantum does not treat it as a new Deal Map item.
  • Ensure that the date and number formats match those used for Quantum (for example: DD/MM/YYYY for dates and N,NNN.NN for amounts).
