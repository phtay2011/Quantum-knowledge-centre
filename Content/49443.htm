Amortized Cost Accrual Method (Money Market & Securities)

(Availability and usage restricted - see Supported Instruments and Exclusions below.)

Select this option to use the Effective Interest Rate method to calculate the Interest and Premium or Discount on a Money Market or Security deal where it must be recognized periodically until maturity.

If the deal includes a discount, this is subtracted from the face value of the deal to determine the net book value. If there is a premium, it is added. This book value is then multiplied by the effective interest rate to derive the interest expense or interest revenue to be recognized for the period.

Any difference in the calculated interest amount and the amount of cash interest that is paid is then used to reduce (amortize) the discount or premium. Because the discount or premium is now less, the book value changes and the interest calculation for the next period will be slightly different. The book value of the bond changes each time amortization is recorded, moving closer to the face value of the bond.

When you select this option, the following associated items are displayed:

  • Accrue issue fee (under Accrual Settings)
  • Apply Accounting calendar (under Amortized Cost Settings)
  • On Closeout or Unwind (under Amortized Cost Settings)

The system performs the following operations to calculate the total amount of Discount or Premium to amortize for each accrual period.

  1. Recalculates Book Value for the deal at the start of each period.

    Book Value = Face Value Â± (Premium|Discount) - (Premium|Discount Accruals)

    The Book Value amount changes each period, increasing for Discount, decreasing for Premium, gradually moving towards the Face Value.

  2. Applies the effective interest rate to the recalculated book value.

    Total Interest Expense|Income = (Book Value at Start of Period) * (Effective Interest Rate)

    The effective interest is the internal rate of return (IRR) calculated for the deal.

  3. Calculates the total amount to be amortized for the calculation period.

    Discount|Premium to Amortize = Total Interest Expense|Income - Accrued Coupon Payment

NOTE: These calculations are system-generated and only the final calculation amount to be amortized for each period is posted when the accrual process is run.

Supported Instruments

Money Market and Security instruments with the following Formula types can implement the AMORTIZED COST accrual method (where only Annuity formulas support Zero Coupons - See Exclusions below):

  • Annuity
  • Bond Periodic*
  • Bond Rolling*
  • Certificate of Deposit*
  • FRN-CGL*

Postings will also be accurately recalculated where any of the following deal actions occur:

  • Ratesets on Floating Rate Notes (FRNs)
  • Deal changes
  • Closeouts
  • Unwinds

*NOTE: If the accrual method is set to Amortized cost for:

  • ALL instruments - those instruments that do not specifically support this method will use STRAIGHT LINE.
  • Bond (Periodic or Rolling), Certificate of Deposit, or Forward Rate Note (FRN) instruments used by deals with Zero Coupons - a message will display at deal save giving you the choice of either changing the method to STRAIGHT LINE for the deal and saving, or canceling the save so you can change relevant deal settings (for example, Coupon).

Exclusions

Ad hoc amounts and ad hoc interest are not included in the Amortized Cost book value calculation used for Capital Premium / Discount allocation.

The Amortized Cost method is not supported for:

  • Bond (Periodic or Rolling), Certificate of Deposit, and Forward Rate Note (FRN) deals with Zero coupons (see Note above) or overnight indexed FRNs,
  • customized deals (level 2),
  • profiled deals,
  • indexed instruments - Capital Indexed Bonds, Indexed Annuities, Indexed Gilts, and Treasury Inflation-Protected Securities (TIPS),
  • deals with Ratesets in arrears, nor
  • deals with zero amount cash flows - for example, where No Coupon On Maturity is specified.
