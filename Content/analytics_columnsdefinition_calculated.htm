Calculated grid, Columns Definition page

Use the Calculated grid to add one or more report columns based on specified types of calculation on selected field values.

To add one or more new calculated columns:

  1. In the grid menu, select New.
  2. In the dialog box, click Add Column; enter an appropriate Actual Name for the new column; select the Return Type of the calculated result (Money, Char (text characters), Rate, Integer, or Date); then click the Add Expression link.

    The Expression Builder dialog box is displayed (with no contents) for you to define the calculation.

  3. Click the + button. A Browse window displays a list of Available Items (calculation item types), as follows.
    • Group By, Point In Time, Movement, Database, and Calculated items are columns already set up in the corresponding grids on this tab. These types do not appear if no columns have been set up in each case.
    • Constant items are parameters applicable to the query itself - for example, Query As At Date.
    • Function items are predefined formulas that are evaluated in a specific order - for example, Join, FX Rate.
    • Operator items are common arithmetic and Boolean operators.
  4. In the Browse window, select the type of item, then select the specific item you want in the expression. The text box now includes your added item.

    TIP: You can also use the keyboard to insert, edit, or delete parts of the expression.

  5. Position the cursor where the next calculation element is required (the default is at the end) and repeat Steps 3 and 4 to build up the required expression.
  6. Click Validate. If an error message is returned, edit the expression directly in the text box, or delete all or part of it and repeat the process from Step 3. If the expression is valid, proceed with the next step.
  7. Click Done.

    The Expression Builder dialog box closes and the Expression is shown in the Edit window against the column's Actual Name.

  8. If required, repeat Steps 2 through 7 to add another Calculated column.
  9. Click OK to close the Edit window.

To add, edit, or remove calculated columns:

  1. Click on a linked item in the grid.
  2. In the Edit dialog box, as and if required:
    • to add more columns - click Add Column, enter a new Actual Name, select the Return Type of the calculated result, click the Add Expression link and use the Expression Builder dialog box to define the calculation (as in Steps 3 through 7 of the previous procedure);
    • to delete a column - click the x button at the right of the column row;
    • to change a column's details - (as required) edit the Actual Name field directly, select a different Return Type, and/or click the Expression link and use the Expression Builder dialog box to edit the calculation (as in Steps 3 through 7 of the previous procedure);
  3. Click OK to close the Edit dialog box.
