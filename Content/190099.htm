Hedge Only the Intrinsic Value of Options

(First introduced in Release 6.5.1 SP, September 2017.)

In accordance with the IFRS 9 standard for accounting of net purchased options in a hedge relationship, Quantum recognizes the change in the time value of an option when only its intrinsic value is designated in a hedge relationship as follows:

  • For transaction related hedged items, the change in fair value is accumulated in a separate component of Equity and reclassified to profit and loss.
  • For time period related hedged items, the change in fair value is accumulated in a separate component of Equity and amortized on a rational basis over the term of the hedge relationship.

Background

Under the IAS 39 accounting standard, when an option is designated in its entirety (that is, at the Market Value) as the hedge of a non-option item, hedge effectiveness is determined by comparing the fair value of the entire option with the change of value of the hedged item. Changes to the time value portion of the fair value result in ineffectiveness and, in some cases, the entity could be forced to discontinue hedge accounting due to ineffectiveness resulting from these changes to the time value.

When only the intrinsic value of the option is designated in a hedge relationship, changes to the time value portion of the fair value have to be accounted in profit or loss, resulting in potentially significant profit or loss volatility.

Under the IFRS 9 accounting standard, an entity can continue to use the IAS 39 approach when designating the entire option; however, when only the intrinsic value is designated, the fair value change in the time value must be accumulated in Other Comprehensive Income (OCI) and recognized in profit and loss, either over the period of the hedge (for time period related hedges) or when the hedged transaction affects profit and loss (for transaction related hedges).

Implementation

Hedge Strategy

The Designated Risk field now includes a Currency Option - Intrinsic option to hedge only the intrinsic value of currency option risks (under IFRS 9). When selected, the time value is accounted in a separate component of equity.

NOTE: When selected, the following fields are hidden:

  • Hedge Dedesignation (value set as DEFERRED).
  • Enable Recognition Reclassifications (value set as False).
  • For Fair Value Hedges only, Posting Based On (value set as HEDGE).

Hedge Relationships

The Designated Risk field now includes a new Currency Option - Intrinsic option to hedge only the intrinsic value of currency option risks (under IFRS 9). When selected, the time value is accounted in a separate component of equity.

NOTE: When selected, the following fields are hidden:

  • Hedge Dedesignation (value set as DEFERRED).
  • Enable Recognition Reclassifications (value set as False).
  • For Fair Value Hedges only, Posting Based On (value set as HEDGE).
  • Perform Spot to Spot Adjustment

When Currency Option - Intrinsic is selected as the Designated Risk, you can select Amortize hedging costs (time value) to amortize the time value to Profit & Loss for time period related hedges.

Posting Rules

The following new fields have been added to the Hedge Relationship tab:

  • OCI Balance (Hedging Costs) - Holds the accumulated amount of the option's time value that is aligned to the hedged item.
  • Hedging Costs Reclass - The P&L account used when reclassifying the option's time value from OCI.
  • Deferred Tax (Hedging Costs) - Holds the tax calculated against the OCI Balance (Hedging Costs) account.

Implementation Notes

Hypothetical Derivative

It is only possible to account for the time value component of the option to the extent it relates to the hedged item. This is called the aligned time value. To measure the aligned time value Quantum requires a hypothetical deal to be added to the hedge relationship. The Hypothetical deal should represent an option that has critical terms that perfectly match the underlying hedged item.

Inception Rule

At inception of the Hedge Relationship the Time Value of the Hedging instruments is compared to the Time Value of the Hypothetical Derivatives (Aligned Time Value).

If Total Hedge(s) Time Value > Total of Hypothetical(s) Time Value, then:

  • the Fair Value change of the Hypothetical (s) Time Value is posted to OCI, and
  • the Difference between Hypothetical Change and Hedge Change is treated as an excluded component, and remains in P&L.

If Total Hedge (s) Time Value < Total of Hypothetical(s) Time Value, then

  • the amount posted to OCI is the lower of the following:
    • Fair Value change in Hedge Time Value
    • Fair Value change in Hypothetical (s) Time Value

    - and -

  • any remainder of the change in fair value of the hedge's Time Value shall be treated as an excluded component and remain in P&L.

Inception Valuations

If the date the Currency Option is designated into the hedge relationship coincides with the Option's Deal Date then an Inception valuation is calculated using the Dealt Terms of the Option or Hypothetical Derivative as follows:

  • the Market Value of the Option is set equal to the Option Premium, and
  • the Intrinsic Value of the Option is set equal to the difference between the Strike Rate and Spot Rate.

Exclusions and Restrictions

Deal Linking

Once a Currency Option - Intrinsic Hedge Relationship has been designated and postings have been started, it is currently not possible to add additional deals into the Hedge Relationship.
