Scenarios tab

(This tab appears only when an existing Scenarios Definition record is specified on the Properties tab for a Position or Cash Flow queries only.)

Scenarios can be used to apply a change in the curve.

The default settings are taken from the Scenarios Definition specified on the Properties tab. Any changes made on this tab are saved in that Definition record when the query configuration is saved.

To add a new Scenario to the current Definition - In the Scenarios grid menu, select New; in the New Scenario dialog box, enter the required fields; then click OK.

To add new Shifted Rates to a selected Scenario - In the Scenarios grid, select the required row; in the Shifted Rates area, click Add; in the browse list select one or more rates as required; then click OK. For each rate you selected, a new row is added to the Shifted Rates grid for that rate, showing the Rate Name, Rate Group, Rate Type, and Shift Type.

TIP: When you select a rate row, its points are displayed in the Rate Points grid on the Rates sub-tab - either by Instrument (for a Security rate) or by Term (for other rate types). A graphical view is shown on the Graph sub-tab.

To view and edit individual Rate Points for a Shifted Rate - In the Scenarios grid, select the required row; in the Shifted Rates grid, select the rate you want; in the Rate Points grid on the Rates sub-tab, select the point row in the Rate Points grid and in the grid menu, select Edit; in the dialog box as required, edit the Shift or Scenario value and if a Spectrum definition, also select the Hedge instrument; then click OK. To see the effect of your changes graphically, open the Graph sub-tab.

To linearly interpolate between two Rate Points - On the Rates sub-tab, click Interpolate; in the dialog box, select the Shift Column (Shift or Scenario), Start and End rate points, and Shift or Scenario values as required; then click OK.
