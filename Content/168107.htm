Exercising Interest Rate Options

You can exercise an Interest Rate Option deal provided it has not already been exercised or lapsed (as indicated by the Status value).

TIP: If the Results list does not have a Status column, use the view's Configure link and the Columns tab to add it.

To Exercise an Interest Rate option deal

  1. If the current result list does not show the deal you want, check the Filters values and that the selected view is either Exercise/Lapse or Default.

    TIP: Select the Exercise/Lapse view option to show only those options an Exercise is currently valid for.

  2. If necessary, adjust the Filter values and re-run the query.
  3. In the Result list, locate and select the deal entry.
  4. In the list menu, select Exercise.

    The Exercise dialog box opens. The Deal Number is displayed as a read-only value, as are the IO Instrument Type, Underlying Deal Number, Counterparty, Entity, Relationship, and Strategy, when applicable.

  5. Optionally edit the Ticket Number if required.
  6. Select another Dealer if required, or accept the default.
  7. If a Bond Option or Swaption, edit the Effective Date if required. See About the Effective Date below.
  8. Edit the Settlement Date if required. See About the Settlement Date below.
  9. Enter the market Reference Rate at the time of exercising the deal. this value is:
    • mandatory for Simple Interest Rate Options, and used with the Strike Rate of the cap to automatically calculate the Settlement amount, but
    • optional for Bond Options and Swaptions, and not used for valuation or analysis purposes.
  10. Optionally enter any text Comments for reference purposes.
  11. Click OK.

    The exercise operation is performed on the selected Interest Rate Option deal. In the results list, the entry's Status changes to Exercised, and:

    • if a Cash Exercise, a cash flow is created, or
    • if a Physical Exercise of a Simple Bond Option (Call or Put), a Security deal is created , or
    • if a Physical Exercise of a Swaption (Payer or Receiver), a Swap deal is created.

NOTE: When this action has been performed for a particular date, the action can only be undone by using the Rollback Deal function.

About the Effective Date

The Effective Date is used to calculate the individual settlement amount.

For Simple Interest Rate Options (Caps, Floors, Collars), the Effective Date is fixed and set to the start date of each optlet. This is based on both the Start Date and Exercise Frequency entered on the Interest Rate Options deal page.

For Bond Options (Calls, Puts) and Swaptions (Payers, Receivers):

  • The default Effective Date is the Expiry Date of the option (as entered on the Interest Rate Options deal page).
  • You can optionally enter an earlier date (when you want to exercise the option early). If you do, the system will update the following deal data:
    • Action Date and Effective Date (Deal Map tab)
    • Deal Date of the underlying deal (if PHYSICAL Settlement type)

About the Settlement Date

For Simple Interest Rate Options (Caps, Floors, Collars), this date defaults to either the optlet's Start Date (if the Discount Settlement formula* applies), or its End Date (if the Yield Settlement formula* applies).

*TIP: The formula used to calculate the default Settlement value is set on the Interest Rate Options Instrument page,

For Bond Options (Calls, Puts) and Swaptions (Payers, Receivers):

  • the default is the Effective Date, and
  • you can enter a later date, if required. If you do, the system will update the following deal data:
    • the Effective Date in the Deal Map (Deal Map tab, Interest Rate Option deal page), and
    • the Deal Date of the underlying deal (if Settlement type PHYSICAL)
