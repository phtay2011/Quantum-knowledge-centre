Commodity Swap Deal page

Use this page to create and maintain Commodity Swap transactions.

WHAT DO YOU WANT TO DO?

To list saved deals - Click the button. (Click the Configure link to change list properties.)

To add a new deal - Click New, fill in the data fields. Choosing the Instrument opens other fields. Fill in the Deal Details and Other Details tabs, add one or more periods, and click Save. Or Copy the current deal, change field values as required, and Save.

To add a period - In the Periods grid menu select New; in the dialog box enter the Start Date, End Date, Quantity (in the units indicated), Fix Date, and Payment Date; then click OK.

To add multiple regular periods (Fixed for Floating only) - click Generate Strip; in the dialog box edit the default settings if required; then click Generate.

To change a deal's details - Select the deal in the list; edit the relevant data fields on the required tabs; add or edit periods as required; and click Save.

To import or export commodity data - See Importing and Exporting Commodities Data.

  • Floating for Floating swaps have only one period, while Fixed for Floating swaps may have more, provided they don't overlap. Gaps between periods are allowed. Deals may be linked, and editing a period on this page edits all linked deals.
  • You cannot edit financial data if actions have been approved on the deal.

To add, view, or delete Notes and/or attached files - Use the page menu's NOTES functions. See Notes and Attachments.

To price the deal for a specific date and view applicable rates - use the Price and Rates sub-tabs to enter the Position Date, Rates Date, As At Date, and Commodity Rates curve, then click the Price button (or select Price in the page menu). The Market price displays. Use the Rates tab to see the applicable rate entries from the curve.

To update periods in bulk (Fixed for Floating only) - open the Deal Details tab and in the Periods grid menu, select Insert or Replace and edit directly or via a spreadsheet application (as with Bulk Replace and Insert for Deal Map Items).

To fix the price of the first unfixed period - In the page menu, select Price Fix; in the dialog box, enter the Price - per unit and the Conversion Rate (if required); then click Save.

To roll back the current deal to a previous deal image (when allowed) - In the page menu, select Rollback; in the Select Action dialog box, select the previous deal Image Number you want and click OK.

To reset the current deal (CAUTION: May necessitate deal reprocessing) - In the page menu, select Reset; and click Yes to confirm. See Resetting a Deal.

To delete the current deal (CAUTION: Takes immediate effect and cannot be undone - even if subject to Approval/Rejection) - In the page menu, select Delete; and click Yes to confirm. See Deleting a Deal.

To review changes to the current deal: In the page menu, select Audit History to view all the actions performed on a deal during its life so far.


For details of a tab or data item, click on it when the Help panel is displayed.
