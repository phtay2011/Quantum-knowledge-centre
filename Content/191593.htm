Pricing

The layout of Pricing pages is similar to that for equivalent Quantum Dealing pages.

The following example shows an Interest Rate Swap Pricing page, with the Details tab open.

NOTE: In the page header, both the Quantum Deal Number and the Analytics Deal Identifier appear, with the Quantum Deal Number appearing first (2367 in the example above).

The Details tab display (see above) is divided into three sections:

  • Common deal elements - always including Deal Number, Deal Identifier, Instrument and Deal Type;
  • Deal Details - always including at least tabs for Details, Deal, Schedule, and Analysis Codes; and
  • Pricing (and Calculation details) - always including at least a Price and Rates tab.

The following example shows the same Interest Rate Swap Pricing page with the Other Details tab open.

The Deal Number field label (highlighted below) acts as a link so that you can navigate directly to the related Quantum deal.

Limits and Pricing

A Check Limits function is available via the page menu (when a saved deal is displayed). This function opens the Check Limit dialog box, as shown in the following example.

*TIP: Limit Breach approval and notification requirements for queries are defined on the Breach Approval Rule page.

Other Features
