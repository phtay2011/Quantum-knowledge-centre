Time Profile Definition Procedures

Use the following procedures to create and maintain Time Profile Definition records and folders.

Managing Records and Folders

To display an existing folder or record - Click , navigate in the browse list to the item you want (or use the Search tool), and select it. The folder contents or record settings are displayed and the breadcrumb trail indicates the current location.

To add a new record (at the current location) - Click the New button and select the Mapping Profile type (General or VaR);; enter a new Name for the Definition record; optionally enter text Comments; select or enter the required settings*; then click Save.

TIP: You can also create a new record by copying the current one. In the page menu select Copy; enter a new Name; change the required settings*; then click Save.

To add a new folder at the current location - Click the New button and select Folder in the drop-down menu; enter a new Name for the folder; then click Save.

To change a Definition record's details - Locate the record as above, change settings* as required; then click Save.

To move a record or folder to a different location - Locate the record or folder and in the page menu select Move. In the Select Group to Move To dialog box, locate the target folder and select it. The new location becomes the current location.

To delete a record or folder not in use - Locate the record or folder and in the page menu select Delete. When prompted, click OK to confirm.

To view changes made to the Definition record - In the page menu select Audit History.

* Changing the current Definition record

TIP: Follow the links below for more details of each step.

On the Details tab, as required:

  1. Edit the Fixed Date Options area settings.
  2. Edit the Business Days Convention area settings.
  3. In the Time Profile area:
    1. Select the Definition Method (Frequency, Period, or Actual Dates) and
    2. In Grid Definition, select whether the Time Profile is generated based on Mid Point or End Point dates.
    3. If Definition Method is Frequency and Hedge Equivalents are to be specified (as for a Spectrum Analysis query), select the Apply hedge equivalents option. The Hedge Equivalents area then displays.
    4. In the Terms list, click Add Term to add a new entry or and enter the settings for that term as required. Or to remove an entry; click the x button at the right of the entry row.
  4. In the Hedge Equivalents area (if present):
    • Click Add Hedge to add a new row of parameters (Entity, Instrument, and Currency) defining a Hedge Equivalent corresponding to a row in the list of Terms in the Time Profile area.
    • The read-only Number value indicates the row sequence. These numbers populate the Hedge lookup in Time Profile area Term entries, enabling you to link entries in each list.
    • To remove an entry row, click the x button at the right of that row.
  5. Save your changes.

Previewing the Generated Dates

Use the Preview tab to list details of the Time Profile buckets your current Details tab settings will generate, as follows.

  1. Enter a Start Date (the default is the current date).
  2. Select the Revaluation Timing: Start of Day, Intra Day, or End of Day.
  3. Click Preview.

The Time Profile Dates grid lists, for each generated Time Profile bucket, the following data:

  • Time Profile - a description of the bucket,
  • Start - the start date,
  • Mid - the middle date,
  • End - the end date, and
  • Line - the sequence number of the bucket within the Time Profile settings (as listed on the Details tab).
