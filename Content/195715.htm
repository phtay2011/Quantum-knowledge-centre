Quantum Enterprise Solutions Suite

This section of the release notes covers the features that are new in the Enterprise Solutions Suite since the release of Version 6.7.2.

NOTE: This is the first time we have included the Enterprise Solutions Suite release notes in the main Quantum release notes.

AGS Table Lookup Editor

Computed columns support

This component has been enhanced to support calculated columns (as non-editable columns) in the AGS table lookup editor.

Deal Generator

Validation flag

A validation flag has been added in the component to verify whether the primary deal has been approved or not before creating/updating the mirror deal. If the user configures this functionality, then it is mandatory that the primary deal should be approved before creating/updating the mirror deal.

Deal Mirror Service

From now onwards, Commodity Swap deal mirroring is supported in the Deal Mirror Service.

Derivative Reporting Solution

Multiple repository reporting

We have added a feature to report trades to multiple repositories. The application will process trade for multiple repositories and generate multiple files for the same trade/deal for selected parsers in the format of the individual repository.

This feature allows the user to select multiple parsers and save them against a Report Format\Query.

Now multiple rows will appear on the Trade message screen for the same deal, and each row will display as per the selected parser.

Single period reporting

From now onwards, DRS will support single period reporting for Commodity Swaps. With this enhancement, two analysis codes have been added at commodity level; Commodity Base and Commodity Details.

Values available in Commodity Base:

  • Agriculture Product AG
  • Energy EN
  • Environmental EV
  • Exotic EX
  • Freights FR
  • Index IN
  • Metal ME
  • Other OT

Values available in Commodity Details:

  • Coal CO
  • Dairy DA
  • Dry DR
  • Electricity EL
  • Emissions EM
  • Forestry FO
  • Grains oilseeds GO
  • InterEnergy IE
  • Livestock LI
  • Natural gas NG
  • Non-precious NP
  • Oil Ol
  • Other OT
  • Precious PR
  • Seafood SF
  • Softs SO
  • Weather WE

For further information please refer to the product installation instructions.

New action types

The component has been enhanced to support these new action types: PRE-DELIVERY and EXTENSION.

From now onwards, the deals will be visible for these newly-added action types on the Trade to Report page.

Market Data Manager

MDM FTP - Commodity Swap support

The component has been enhanced to support the new rate type Commodity Swap in Analytics through FTP. Add the following configuration in the configuration file:

  • DataAdapterName="DataAdapterAnalyticsCommoditySwap"

MDM API - External Rate Feed Source

With this enhancement to the MDM API, the user will be able to import rates into Quantum from the External Rate Feed Source. This external rate feed source can be anything, such as, Database, MSMQ or Web Service.

All the data-specific validations will be implemented in the client-specific DLL. To implement this, raise a request with the ES Team for development activities. This request can be raised via a cluster team, with all required details in Jira.

To use this functionality, configure the following property in the configuration file:

  • RateFeedProvider="RateFeedProviderExternalSource"

Enterprise Solutions Data Management

In the Enterprise Data Management (ESDM) page, Audit log support has been introduced. All actions, such as, Insert, Update and Delete, will be saved into a database table for auditing purpose.
