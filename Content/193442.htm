Static & Rules menu (Analytics perspective)

This section covers core modules accessed via the Static & Rules menu in the Analytics perspective (shown below).

NOTE:

  • All these modules are also accessible from the Static & Rules menu in the Adaptiv perspective (which includes additional Adaptiv-only modules).
  • The Ratings Agency and Valuation Rule modules are also accessible via the Static menu in the Default perspective.
