Static menu (Default perspective)

This section covers core modules accessed via the Static menu in the Default perspective (shown below).

NOTE:

  • The Rate Name and Rate Type modules are also accessible via the Rates menu (Default perspective).
  • The Ratings Agency module is also accessible via the Static & Rules menu in the Analytics perspective.
