Coupon Type, Security Instrument

(NOT available with the ANNUITY or INDEXED ANNUITY formula; available with the YIELD PERIODIC and PRINCIPAL & INTEREST formulas only if Day Count is ACTUAL/ACTUAL; read-only and set to EXACT with the BOND ROLLING formula.)

Select the default coupon type for deals based on this instrument - one of the following:

  • EQUAL - to take the annual coupon amount and divide it equally by the number of coupon periods within the annual period, irrespective of the number of days within the coupon period.

    NOTE: With this coupon type, Calculation Date (under Business Day Conventions on the Dates tab) should be set to ACTUAL DATE.

  • EXACT - to base coupons on the actual number of days in the coupon period.

At deal entry time, the Quoted Yield Convention and Coupon Type fields must contain corresponding values to give accurate yield results . If you change either value for a deal, you must enter the corresponding other value to maintain accuracy. For example, if you have a semi-annual coupon frequency for this instrument, and the Exact Coupon Type is selected, then the Quoted Yield Convention should be SEMI-ANNUAL COMPOUNDED.

See Day Count Conventions for examples showing the calculation differences between the options.
