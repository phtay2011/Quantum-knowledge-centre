Inflation-linked Deals

Inflation-linked deal data can be added to the Analytics database using the External Portfolio File utility.

Required Price Factors

The following Price Factors are required for inflation-linked deal valuation.

Price Index

This Price Factor holds a list of historical inflation price index values that are announced periodically (for example, monthly). Each entry consists of a Date-Value pair, where Value corresponds to the index value and Date to the first day of the period to which the index value refers to. The date is expressed in Excel format (number of days since 30 December 1899).

In addition:

  • Next Publication Date specifies the date on which the next index value will be announced,
  • Last Period Start is the first day of period that the last provided index value refers to, and
  • Publication Period is the publication frequency.

Inflation Rate

The Inflation Rate curve contains the current inflation rates implied by the market for different tenors. The inflation rate curve is also required in Adaptiv Analytics to forecast future price index values.

The Inflation Rate Price Factor refers to the corresponding Price Index, Seasonal Price Index Factor, and Reference Name.

Seasonal Price Index

This Price Factor is needed to adjust inflation rates for seasonal variations.

Price Index Volatility

This Price Factor has the same structure as Interest Yield Volatility. It contains three dimensions: Moneyness, Tenor, and Expiry.

<Deal>
Object=InflationLinkedBond,Reference=2611-Test Price Index,
MtM=&lt;undefined&gt;,Currency=EUR,Discount_Rate=,
Index=Inflation EU HICPexTobb End of Day-HICP1 Revised Fixing,
Calendars=,Is_Forward_Deal=No,Buy_Sell=Buy,Repo_Rate=,Issuer=,
Survival_Probability=,Base_Reference_Date=03Aug2015,Issue_Date=31Jul2015,
Bond_Maturity_Date=15Feb2018,Coupon_Interval=6M,First_Coupon_Date=01Dec2015,
Penultimate_Coupon_Date=,Principal=1000,Amortisation=,Coupon_Rate=5%,
Coupon_Rate_Schedule=,Margin=0bp,Rate_Multiplier=1,Accrual_Day_Count=ACT_365,
Settlement_Date=31Jul2015,Investment_Horizon=,Price=1000,Price_Is_Clean=Yes,
Price_Is_Real=No,Recovery_Rate=
</Deal>

NOTES: The Reference and Index attribute formats are:

  • Reference=deal id - description

    ... where deal id must refer to an existing deal in Analytics, and
  • Index=Inflation Rate Name - Price Index Rate Name
