Accrued Interest calculation, 30/360

There is a separate logic for accruals in the Purchased Period. Accruals should refer to the Previous Coupon Date in the Purchased Period.

Accruals Logic

Where:

  • Start Date = Month1/Day1/Year1
  • End Date = Month2/Day2/Year2
  • Previous Coupon Date = Month3/Day3/Year3
  • Day Count Fraction = Number of Days/360
  • Number of Days = 360*(Year2 - Year1) + 30*(Month2 - Month1) + (D2 - D1)

Where Days1 is converted to D1 and Day2 is converted to D2 as follows:

If (Day1 = 31) and (Day3 = 30 or 31), set D1 = 30, otherwise set D1 = Day1

If (Day2 = 31) and (Day3 = 30 or 31), set D2 = 30, otherwise set D2 = Day2

Equal and Exact Coupon Calculation

For accrued interest calculations, Equal and Exact Coupon calculation results are the same.

In this example:

  • Previous Coupon Date is 1 Mar
  • Next Coupon Date is 1 Sep

Exact coupon calculation 31 Jul - 1 Sep:

5%*(30/360) = .416666667%, where:

  • 5% = Coupon interest rate
  • 30 = (Y2-Y1)*360+(M2-M1)*30+(D2-D1)
    = (0*360)+(9-7)*30+(1-31)

Where DAY1 is converted to D1 and DAY2 is converted to D2 as follows:

  • If(DAY1=31) and (DAY3=30 or 31), set D1=30, otherwise set D1=DAY1
  • If(DAY2=31) and (DAY3=30 or 31), then set D2=30, otherwise set D2=DAY2

 

  • Y1 = 2007
  • Y2 = 2007
  • M1 = 7 (Jul)
  • M2 = 9 (Sep)
  • D1 = 31
  • D2 = 1
  • D3 = 1
