Transfers on Unit Holdings

A Unit Transfer is a transaction where Units are transferred from one Holding in a Unit Security to another. The "source" holding decreases and the "destination" holding increases by the number of Units transferred.

A Unit Transfer creates a TRANSFER cash flow and stock flow on each holding, with the same Price, Deal Date/Time, and Settlement Date/Time. The number of Units is negative for the the source holding flows; positive for the source holding.

The source holding is the holding you initiated the Transfer from.

The destination holding need not yet exist, in which case you must enter further details that would otherwise be obtained from the destination holding (for example: Counterparty or Broker, Dividend Taxes, and Strategy).

Use the Unit Transfer page to enter or edit details of a Transfer of Units from a holding,

To create a new Transfer:

  1. On the Unit Holding page, open the holding you want.
  2. In the page menu, select Transfer.

    The Unit Transfer page opens in a new window.

  3. On the Details tab:
    • Enter or edit the To Entity, number of Units, Price, Dividend (Cum or Ex), Deal Date and time (Hours, Minutes, Seconds), and Settlement Date values.
    • Under Source Party to the Deal, select either the Broker or Counterparty handling the source holding (depending on the Trading Style of the source holding).
    • Under Destination Party to the Deal, select either the Broker or Counterparty handling the destination holding (depending on the Trading Style of the destination holding). If the destination holding does not yet exist, you must also specify the Corporate Action Counterparty.
  4. (Optional) On the Other Details tab:
    • Under Our References, select the Dealer, optionally enter or edit Ticket Number and Comments text, and select one r more Location values as and if required.
    • Under Their References, enter or edit EITHER Broker Reference and Broker Dealer, OR Counterparty Reference and Counterparty Dealer values (depending on the Trading Style of the destination holding).
  5. (Optional) On the Fees tab, enter details of any Source Fees and Destination Fees, as required.
  6. (Optional) On the Analysis Codes tab, enter or edit Source Analysis Code and Destination Analysis Code values, as and if required.
  7. Click Save.

    The Unit Split page window closes. Your Split action is applied and is displayed on the Corporate Actions tab of the Unit Holding page.

To edit the last Transfer for the current holding:

(Only possible when a Transfer has been made on the Holding.)

  1. On the Transfers tab, click Edit Last Transfer.

    The Unit Transfer page opens in a new window, displaying the entered details for the last Transfer.

  2. Edit the data items as required (see above).
  3. Click Save.

    The Unit Transfer page window closes. Your changes are applied and displayed on the Transfers tab for both the source and destination* holdings.

    TIP: If you changed the destination holding, the Transfer will be removed from the original destination holding and added to the new one.

To delete the last Transfer for the current holding:

(Only possible when a Transfer has been made on the Holding.)

  1. On the Transfers tab, click Delete Last Transfer.
  2. When prompted, click OK to confirm.

    The Transfer is deleted and no longer shows on the Transfers tab for both the source and destination holdings. All database changes caused by the Transfer are backed out.
