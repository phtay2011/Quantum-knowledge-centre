Details tab

This tab includes the following fields. Any default values are derived from the selected Instrument.

Buy/Sell - Select Buy or Sell.

For Analytics Repo transactions, defaults to Buy/Sell Back or Sell/Buy Back. If Borrow or Lend is selected a new Repo Instrument list box will be displayed.

Call/Put (not available if Style is American Digital) - select Call or Put.

Call Amount - Select the currency and enter the CALL amount of the transaction. If you enter the Strike Rate and either the CALL or PUT amount, Analytics will calculate the remaining amount. If you enter the both CALL and the PUT amount, Analytics will calculate the Strike Rate.

Strike - Enter the Strike price/rate.

Put Amount - Select the currency and enter PUT amount of the transaction. If you enter the Strike Rate and either the CALL or PUT amount, Analytics will calculate the remaining amount. If you enter both the CALL and the PUT amount, Analytics will calculate the Strike Rate.

Settlement - Select a settlement type: Cash or Physical.

Participating Rate (available when Type is Participating Forward; automatically calculated if both leg amounts are entered) - Optionally enter the percentage of the amount tradable at the Spot Rate - that is, the difference in the leg amounts as a proportion of the higher amount (the lower leg amounts are recalculated).

Deal Date - Enter the date of the transaction (defaults to the current date).

Expiry Date - Enter the date the option will expire.

Rebate Timing (applies when Style is an Out Barrier one, mandatory when Rebate is non-zero) - Select when the Rebate is to be paid: on either Breach or Expiry.

Monitor Frequency - Select how often the rate will be checked: Continuously, Daily, Weekly, or Monthly.

Barrier- Enter the single barrier rate that will determine, as the Strike Rate changes, whether an FX deal is activated (if an In) OR the option expires (if an Out).

Lower Barrier (applies when Style is a double Barrier one) - Enter the lower barrier rate that, if the Strike Rate drops below, will determine whether an FX deal is activated (if an In) OR the option expires (if an Out).

Upper Barrier (applies when Style is a double Barrier one) - Enter the upper barrier rate that, if the Strike Rate exceeds, will determine whether an FX deal is activated (if an In) OR the option expires (if an Out).

Barrier Validation - Select a validation method: Value or Actual.

Rebate (applies when Style is an Out Barrier one) - Enter the Rebate Points as a percentage (in terms of the Premium currency).

Premium Amount (automatically set if Put Amount and Premium Percent are entered) - Enter the premium amount (the Premium Percent is recalculated). The currency defaults to the Put Amount currency, but you may change it.

Premium Percent (automatically set if Put Amount and Premium Amount are entered) - Optionally enter the Premium percentage rate (the Premium Amount is recalculated).

Premium Date - Enter the premium payment date.

Value Date - Enter the date the transaction will be considered effective for reconciliation, accounting, bank account interest calculation, and Cash Management.

For more details of other items, click on an item when the Help panel is open.
