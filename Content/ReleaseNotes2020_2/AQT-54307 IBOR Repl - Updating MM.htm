Money Market and Securities - Overnight Index option

With the demise of LIBOR as a benchmark, the various jurisdictions have begun using their preferred benchmark rates. For GBP-denominated Floating Rate Note instruments, this is usually SONIA, using the newly created product "SONIA FRN".

In this second phase we've enhanced Quantum's Money Market and Security deal functionality to match that implemented for Interest Rate Swaps in the previous release.

On the Money Market and Security instrument pages' Details tab, when Formula Timing is YIELD PERIODIC or FRN - CGL PERIODIC, the Overnight Index check box now appears, as in the following example.

The new items and their settings therefore also appear in Money Market and Security deals based on such instruments.

Supported features

Analytics' Money Market and FRN instruments and pricing support the Overnight Index settings. See Analytics Overnight Indexed deal enhancements.

You can use an API to save a Money Market or Security deal that has the Overnight Index option selected.

You can use QXT to import and export Money Market and Security deals with the Overnight Index option selected.

For FRN - CGL PERIODIC deals, Rate Set Deal queries now support a Transaction Type of Money Market (SE), as shown below. The query returns the FRN overnight index rate-set items that have an Action Date within the range of the rate-set start and end date of the query.

Restrictions

Money Market and Securities deals using the Yield Periodic formula with the Overnight Index option selected automatically use the Yield (“As Earned”) Accrual Method. No other accrual methods are supported for deals with the Overnight Index option selected.

Upgrading from a prior release

The Overnight Index check box is disabled for existing instruments and deals.
