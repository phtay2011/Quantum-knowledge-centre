Flow Lifecycles, States, and Phases

The Flow Lifecycle value for a cash flow describes its current stage in the processing lifecycle within your system. This data item may be displayed as a grid column (for example on Dealing page Cash Flows tabs) and/or used as a filter in queries or views. As detailed in the table below, the system:

  • groups data by Flow Lifecycle Phase text values, each with an associated badge;
  • displays data on the Counterparty Instructions page in a badge colored to correspond to the applicable Flow Lifecycle Phase badge;
  • displays corresponding Flow Lifecycle State numeric values.

TIP: Look for the Flow Lifecycle, Flow Lifecycle State, Flow Lifecycle Phase text values, and/or the badge, in grid columns on Dealing page Cash Flows tabs.
