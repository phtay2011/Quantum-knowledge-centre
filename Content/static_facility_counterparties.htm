Counterparties tab (Facility)

(Not applicable to Groups.)

Use this tab's COUNTERPARTIES grid to record each Counterparty associated with the Facility, the role each plays, and the value of their participation. Use the read-only MOVEMENTS grid to view the history of changes made to Counterparties' details.

You can define more than one set of participating Counterparties, with each set having the same Active From date, and one row per Counterparty.

To add a new set of participants (from scratch) - In the Counterparties grid menu, select New; in the Initialize At dialog box, enter the fields listed below for the Principal Counterparty; then click OK. Your new entry is added as a new grid row. Click Save. To add further Counterparties, use the next procedure.

To assign part or all of a Counterparty's participation to a new Counterparty - In the Counterparties grid, select the existing Counterparty's row; in the grid menu, select Assign To; in the dialog box select the new Counterparty and enter either their Percentage of the total amount OR their Amount; enter other data as required; then click OK. Your new Counterparty is added as a new row and the other one's participation is reduced accordingly. Click Save.

To increase or decrease a Counterparty's participation - In the Counterparties grid, select the existing Counterparty's row; in the grid menu, select Increase By OR Decrease By as required; in the dialog box enter either the changed Percentage of the total amount OR the Amount; enter other data as required; then click OK. The Counterparty's participation is changed accordingly. Click Save.

To copy a full set of participating Counterparties - In the Counterparties grid select the first row of the set you want to copy; in the grid menu, select Copy All For Date; in the Copy From dialog box, enter a laterNew Active From Date and click OK. Your new set is added as new grid rows. Use the other grid menu functions to change participations, Click Save.

TIP: You cannot remove a Counterparty. Instead, reduce their participation to zero.
