Deal Details tab, Commodity Swap Deal

Use this tab to enter, edit, or view the key information for the deal, including the following components.

Key data fields (Counterparty, Entity, and so on) - for details, click on an item when the Help panel is open.

Periods grid

  • A Floating for Floating Swap type has only one period (one grid row), with both Near Side and Far Side values for the period.
  • A Fixed for Floating Swap type may have multiple periods (one or more grid rows), in date order. Periods must not overlap, but gaps between periods are allowed. You can change the quantity or pricing, and remove any periods.
  • To add a single period - click New (or select it in the grid menu); in the dialog box enter the Start Date, End Date, Quantity, and Fix Date, or override Payment Date; then click OK.
  • To generate a strip of multiple periods (Fixed for Floating only) - click Generate Strip (or select it in the grid menu); in the dialog box enter or edit the default Number of Periods, Quantity, Frequency, and Start Date Cycling settings (plus other settings that vary depending on the Start Date Cycling setting); then click OK.
  • To edit items in a period - in a new deal, click on an item in the period; make your changes in the Edit Period dialog box; then click OK.
  • To delete a period - select the grid row; in the grid menu select Delete; then click OK to confirm.
  • To edit or insert periods in bulk (Fixed for Floating only) - use the Bulk Insert or Replace grid menu function and edit directly or via a spreadsheet application (as with Bulk Replace and Insert for Deal Map Items).

To understand dates - The system automatically sets the Start Date from the first period's Start Date and automatically sets the Maturity Date from the last period's End Date. The "Fix Date" column lists the dates each period will be price fixed. The system creates the Payment Date based on the Payment Date Offset field and the "Fixed Date" column, OR you set the payment date manually by editing a period.

Price sub-tab

  • Enter the required dates and click the Price button to display the applicable rate curve.

Rates sub-tab

  • Select a Commodity Rate curve. Click either the Price button (on the Price tab), the rate date term definition for the selected rate curve on, or nearest to and preceding, and the Rate Date is displayed as a grid.
