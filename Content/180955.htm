Running a Query to Create a Data Version

When you run a query and create a new Data Version you must specify:

  • the Start Date of the first user-defined Horizon,
  • whether or not to include Worksheet Adjustments, and
  • whether or not to calculate concentration flows, and for which Horizons.

To run a query and create a new Data Version

  1. Select Run... from the page menu.
  2. Select the required options in the Run Query dialog box and click OK.

TIP: The Run Query process runs asynchronously, which enables you to perform other CashXplorer tasks while it runs. Notifications let you know when runs are completed. For more details, see CashXplorer Notifications.

The Run Query dialog box includes the following fields and check boxes:

First Horizon Start Date

Enter the Start Date for the Data Version. This date must be a valid business day and is based on your User Location's time zone (which defaults to the System date/time). The Saturday is a Business Day and Sunday is a Business Day option settings on the Properties tab influence what is considered a valid business day.

Calculate Concentration Flows

Select whether or not concentration flows are to be calculated for this Data Version.

  • Select None to ignore the concentration flows even if they are defined for any of the accounts.
  • Select First Horizon to calculate a concentration flow for each account that has cash concentration defined within the first user-defined Horizon only.
  • Select All Horizons to calculate concentration flows for each account that has cash concentration defined for all the user-defined Horizons.

The default selection is as defined in Additional Settings, Calculate Concentration Flows.

Refresh Additional Balances

Select this option to retrieve any additional balances, and include these in the New Data Version.

Additional Balances include the following.

Available and Float

Other 1

Available 1 Day float

Other 2

Available 2 Day Float

Value Dated

Collected

Internal Reconciled

Current Available

Internal Unreconciled

Ledger

 

NOTE: Retrieving Additional Balances will add extra processing time to the Data Version creation. Select this option only if you want to view Additional Balances, or if you want to use Value Dating.

By default, this option is selected.

Include Worksheet Adjustments

Select this option to include Worksheet Adjustments you have defined in this Data Version.

If you clear this option, the Worksheet Adjustments are not included in the Data Version. This enables you to view the original results before, perhaps, refreshing the Data Version with Worksheet Adjustments.

By default, this option is selected.
