Set Base Reset date options for Fees

In the Schedule grid (on the Schedule tab):

  1. Select the Base Reset row (the last) and in the grid menu select Edit (or click on any value in the row). In the Edit Schedule dialog box, do the following.
    1. Select a Frequency for fee base amount resets - one of the following:
      • Once (On the dealâ€™s Maturity Date),
      • Daily (overnight),
      • Weekly,
      • Monthly,
      • Quarterly,
      • Semi Annually,
      • Annually, or
      • Manual Schedule (Calculation-based schedule only; displays a Base Reset Schedule grid, enabling you to create virtually any Calculation and Payment schedule)
    2. Edit the First Date setting if required. This is the first Base Reset date, initially set to the Start Date. Changing this date will result in no fee Base Amount being calculated until that new First Date.

      TIP: The last calculation period automatically ends on the specified Maturity Date.

    3. If you selected a Frequency of Manual Schedule above, optionally select a different Business Day Convention (the rule to apply when the relevant date falls on a non-business day).
    4. Click OK to close the Edit window.

    Your Base Reset date settings are shown in the grid.

Now:

  • Check your other Schedule grid settings.
  • If you selected a Frequency of Manual Schedule above, enter the required dates in the Base Reset Schedule grid.
  • Use the View Schedule grid menu function to view and check the actual dates being scheduled.
  • Save your settings.

Manually Schedule Fee Base Reset dates

(Applicable only when the Schedule grid's Base Reset row has Frequency set to Manual Schedule.)

Use the Base Reset Schedule grid to manually schedule virtually any Base Reset dates you want to apply to the current Fee deal schedule.

NOTE: Base Reset Schedule dates are independent of the set of Calculation, Payment, and Invoice Dates. You can manually schedule either set of dates, both, or neither.

To enter schedule date ranges

  1. In the Base Reset Schedule grid menu, select Generate.

    The Generate Base Reset Dates dialog box opens.

  2. Select the Base Reset Frequency for the range:
    • Days,
    • Weeks,
    • Months,
    • Years,
    • Daily,
    • Weekly,
    • Monthly,
    • Quarterly,
    • Semi-Annually, or
    • Annually;
  3. If you selected Days, Weeks, Months, or Years, then in the adjacent field, enter the number of those time units between Base Reset dates.
  4. Set the Start Period Date and End Period Date to define the range, which must be within the Fee deal's effective date range (Start Date to Maturity Date).
  5. Click OK.

    The schedule of Reset dates is displayed.

  6. Repeat the procedure to define further dates. Any new ranges may supersede existing ranges, or parts of them, or may be distinct.

To change a single date in the schedule

  1. In the Base Reset Schedule grid, select the date row.
  2. In the grid menu, select Edit.

    In the Edit Base Reset Dates dialog box, enter the new Reset Date.

  3. Click OK.

To remove a single date from the schedule

  1. In the Base Reset Schedule grid, select the date row.
  2. In the grid menu, select Delete.
  3. Click OK to confirm.
