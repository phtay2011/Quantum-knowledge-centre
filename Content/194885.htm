Free Format MTn99 procedures

To create a new Free Format Message

  1. If you want to base the message on a currently displayed message, change the message category if required, and select that message in the grid.
  2. Select Create New Message.

    The New Message form opens. If you had selected an existing message in Step 1:

    • The Message Type is defaulted from that selected message. For example, if you had selected a 320 message, the output message type will be 399.
    • The values for Receiver, Sender, and Related Ref are also defaulted from your selected message.
  3. Change any of the defaulted values if required.
  4. In Message Text, enter the main text of the message, which will populate Tag 79 (Narrative). You can enter up to 35 lines of 50 characters each.
  5. In the XML Template lookup, select an XML Template for the Free Format message you want to create.
  6. In the Export Source lookup, select a source for the Free Format message you want to create. This is used to default the File Destination and File Name only.

    TIP: The lookup lists export data sources defined in External Method Sources.

  7. In the File Destination lookup, select a destination directory for the Free Format message you want to create.
  8. In the File Name field, enter an output file name for the Free Format message you want to create (or edit the default).
  9. Click Save.
  10. If you have specified a File Name that already exists, you will be informed of that, and prompted to either replace the existing file or not. Do one of the following:
    • Click No and return to Step 8 to specify a different file name.
    • Click Yes to replace the existing file, and continue.
  11. If validation fails (for example, the message content is invalid or you have specified an invalid File Destination), an error message displays. Click OK and return to the relevant step/s above as required.

    Otherwise:

    1. The message file is created in the destination directory.
    2. The message will be added to the list of sent messages displayed in Free Format Messages mode (subject to any filtering criteria).

To view previously created Free Format Messages

  1. In Message Workbench, select Free Format in the Message Category lookup.
  2. Optionally filter the displayed messages by setting Filter by, selecting or entering a filter value in the the adjacent calendar lookup or field, and clicking Refresh to update the display.
