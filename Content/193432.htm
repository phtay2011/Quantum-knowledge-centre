Configuration menu (Default perspective)

This section covers core modules accessed via the Configuration menu in the Default perspective (shown below).

MENU GROUP/
Module

Use to...

Unique ID (Area/Controller)

CONFIGURATION

Access Module

Define modules for user access to scheduled jobs, commands, or Crystal reports

configuration/accessmodule

Address Structure

Optionally configure system-wide structured address fields using predefined tags

configuration/addressstructure

Analysis Code

Maintain custom data items for categorizing and reporting on static data, instruments, or deals

configuration/analysis-code-maintenance

Bank Account Monitoring

Configure bank accounts to monitor End of Day balances or Import Statements to enable API calls to be made on that data

configuration/bank-account-monitoring

Confirmation Match Reply

Configure location (file or queue) and format of incoming responses from Confirmation Match Service

configuration/confirmation-match-reply

Crystal Mapping

Maintain tables for attaching standard reports or custom Crystal Report formats to Quantum applications

configuration/crystal

Data Transfer Location

Configure data streams into and out of Quantum as queues and files at various integration points

configuration/data-transfer-location

External Method Source

Maintain source information for External Methods

configuration/external-method-source

Job

Run, view, and edit existing Scheduler jobs (including job history and message logs)

configuration/scheduler

XML Data

View seed XML Template files; optionally create and edit non-seed files

static/ewtxmldt

SYSTEM

Logon Message

Specify messages displayed to users after logging on

configuration/logon-screen-settings

Logon Security

Specify system-wide security settings for log on, password expiry, and password values

configuration/security-settings

Password Filters

Specify site-specific password acceptance criteria or Windows password complexity

configuration/password-filter-settings

Data Security

Specify system-wide settings for Resource Security, Static Data Approval, and/or Security Change Approval

configuration/data-security-settings

Operational Settings

Specify system-wide application-related settings to apply throughout Quantum

configuration/workflow-settings

LDAP Authentication

Activate/ deactivate LDAP Authentication and specify authentication details

configuration/ldap-authentication-settings

SMTP Settings

Configure SMTP data store settings for system-generated e-mail messages

configuration/smtp-settings

USER

User

Maintain individual user records

static/user

Group

Maintain user groups

static/user-group

Unapproved User

View and approve/reject unapproved changes to User records (when Static Data Approval applies)

static/unapproved-user

Unapproved Group

View and approve/reject unapproved changes to User Group records (when Static Data Approval applies)

static/unapproved-user-group

Pending Resource Security

View and apply changes made to resource security for User and Group records

configuration/pending-resource-security

Applied Resource Security

View and optionally export currently assigned resource security permissions for Users and Groups

configuration/resource-security

Applied Module Security

View and optionally export currently assigned module access permissions for Users and Groups

configuration/applied-module-security

WORKFLOW

Workflow Action

Create a generic composite action for use in other Workflow modules where an action can be selected.

configuration/workflow-action

Dashboard

Create a composite page containing result sets of query-driven modules and other data.

configuration/dashboard

Workflow Event

Define actions to be performed when system events are triggered.

configuration/workflow-event

Workflow Function

Define functions that can be used in Expression Builder

configuration/workflow-function

Workflow Page

Modify user interfaces of existing modules (changing core page or creating a variant).

configuration/workflow-page

Workflow Parser

Create a Parser record defining the mapping of incoming text file data to Quantum data (referenced in a WorkflowIn Data Transfer Location record)

configuration/workflow-parser

Workflow Report

Maintain site-specific reporting pages (which can combine data from multiple data sources).

configuration/workflow-report

Workflow Template

Create output that contains user-specified textual information. Enterprise Workflow

configuration/workflow-template

LICENSING

License

View licensed features for your site

utility/licensing
