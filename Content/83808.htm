Face Value Accrual for TIPS

For a Treasury Inflated-Protected Securities (TIPS) deal, Face Value accrual postings are the only Consumer Price Index (CPI) entries posted into a separate account. These entries are the Face Value adjustments due to the CPI changes.

This CPI entry is a calculation of the difference between the CPI-adjusted Face Value and the original Face Value. The movement of value in this account is an unrealized gain or loss for the deal and it becomes realized Profit and Loss only when the deal is closed out or at the maturity date of the underlying instrument.

The accruals method is Straight Line. Using this method, the CPI index account will move at a fixed amount per day within the control points. The control points are calculated progressively over time when an index is updated.

A Profit and Loss account called Face Value Index Adjustment will be used to account for the movement for the Face Value. When the accrue Face Value Index Adjustment is set to accrue, the Face Value Index Adjustment will be posted to the Face Value Index Adjustment account (Profit and Loss) and the Money Market Face Value Investment account (Balance Sheet).

Diagram:

Deflation

When deflation has occurred, the owner of an Indexed Bond will always receive as a minimum the Face Value of the bond at maturity. In this case, the CPI falls below 100.

The reason for this is that throughout the life of the deal, the Capital will be accounted for as described above in Capital Premium/Discount Accrual for Indexed Bonds. The sum of the Face Value calculations will still equal the Repay Principal (the Repay Principal will have a value of less than 100).

In addition, at maturity, a new entry will be posted. This is a one off adjustment that brings the Face Value, and therefore the Repay Principal, back to 100.

NOTE Quantum considers deflation to have occurred when the CPI Indexation Multiplier falls below 100. Although it is possible to have deflation with the CPI Indexation Multiplier above 100, the description above, in this case, would not beapplicable.
