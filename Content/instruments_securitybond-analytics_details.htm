Details tab, Bond Instrument

This tab includes the following fields for entering default values to apply to deals based on this instrument.

Bond Type - select the type of Bond: Standard, Capital Indexed, or Index Linked Bond.

Price Type - select the price type of the traded bond: Yield, Capital Price, or Cash Price.

Coupon Rate - select the default Coupon Rate.

Frequency - select Daily, Weekly, Fortnightly, Monthly, Quarterly, Semi Annual, Annual, or [on] Maturity.

Ex Interest Period - specify the default Ex Interest period - EITHER:

  • in the drop-down list, select First Day of Month, OR...
  • enter a number in the left field and, in the drop-down list, select the time units (Business Days, Calendar Days, or Months) or select Preceding Month.

End of Month - select Yes or No to specify whether or not to move the next Payment Date to the last day of the applicable month.

For example, if End of Month is Yes, Frequency is Semi Annual, and First Payment Date is 30-Sep-2015, the next Payment Date will be 31-Mar-2016. If End of Month is No, the next Payment date will be 30-Mar-2016.

Amortizing - select Yes or No to specify whether or not the principal will be amortized/accreted.

Derived Amount - select which amount to derive from the other two: Capital Price, Accrued Interest, or Cash Price.

Final Period Pricing - If pricing is different in the final period, select if, by default, this takes effect from the penultimate Ex Date or the penultimate Coupon Date.
