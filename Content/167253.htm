Simple Interest Rate Option Instruments in Quantum

A Simple Interest Rate Option is a multi-period transaction, usually designed to hedge interest rate exposures. Simple Interest Rate Options in Quantum do not operate on an underlying Security or Swap instrument, and are always settled for cash. For more details, see Simple Interest Rate Option Deals in Quantum.

Default values for Simple Interest Rate Option deals are provided when an Interest Rate Option deal uses an instrument with the following settings.

Details area

Option Type

Select one of the following Simple Interest Rate Option types.

  • Cap - An agreement between the seller or provider of the cap and a borrower, to limit the borrower's floating interest rate to a specified level for a period of time.
  • Collar - A variation on a Cap agreement where the seller of the Collar agrees to limit the borrower's floating rate of interest to a band limited by a specified Cap rate and Floor rate. The transaction is two-sided, with the Cap side determining the buying or selling of the Collar. That is, buying a Collar comprises buying a Cap and selling a Floor; while Selling a Collar comprises selling a Cap and buying a Floor.
  • Floor - An agreement, between the seller or provider of the floor and an investor, guaranteeing that the investor's floating rate of return on investments will not fall below a specified level over an agreed period of time. Floors can be transacted alone but are typically associated with a Collar transaction.

Exercise Style

This setting determines when Interest Rate Option deals based on this instrument may be exercised or lapsed; either:

  • EUROPEAN - to exercise or lapse on the deal Expiry Date, or
  • AMERICAN - to exercise or lapse at any time up to the deal Expiry Date.

Settlement Type

Read-only and set to CASH for Simple Interest Rate Options, which can only be settled for cash.

Settlement Paid At

(Simple Interest Rate Options only.)

This setting determines the payment date of the Settlement Amount for Simple Interest Rate Option deals based on this instrument. Select:

  • EXERCISE - to indicate the start of an optlet period

    This setting will also default the Settlement formula to DISCOUNT. If exercised on the Exercise date the Discount formula will be used to calculate the Settlement amount to be paid. See Discount formulas.

  • MATURITY - to indicate the end of an optlet period

    This setting will also default the Settlement formula to YIELD.
    If exercised on Maturity the Yield formula will be used to calculate the Settlement amount to be paid. See Yield formulas.

NOTE: You can change the Settlement date from either the Maturity or Exercise date when you exercise the option - however, the default Settlement value will be calculated based on the Yield or Discount formula set on this form. See the Exercise topics under Processing Interest Rate Options.

Settlement Formula

(Simple Interest Rate Options only.)

This setting determines which underlying interest rate calculation method is used to calculate the settlement amount for deals based on this instrument. This setting is linked to the Settlement Paid At value. Select:

  • DISCOUNT - to default Settlement Paid At to EXERCISE, or
  • YIELD (interest) - to default Settlement Paid At to MATURITY.

Premium Paid In

This setting gives the default Premium Date value for Interest Rate Option deals based on this instrument. Select:

  • ADVANCE - to use the Deal Date, or
  • ARREARS - to use the Exercise Date.

Reference Margin

(Simple Interest Rate Options only.)

This optional value gives any default margin above a reference rate, for valuing Interest Rate Option deals based on this instrument.

Reference Basis

(Simple Interest Rate Options only.)

This optional setting gives the reference basis rate that will be used for valuing Interest Rate Option deals based on this instrument.

Show broker

Select this option to include the Broker field when entering a deal based on this instrument.

That Broker field can then be used to record a third party or intermediary Counterparty, where one is used to transact the deal.

Broker is required

(Available only when Show Broker is selected.)

Select this option to require users to select a Broker before saving deals based on this instrument.

Allow amortization

(Simple Interest Rate Options only.)

When selected, this option specifies that an amortization or accretion schedule is to be provided for Interest Rate Option deals based on this instrument.

TIP: This default setting may be overridden on individual deals.

By default, this option is cleared.

Dates area

Use these fields to record and view the timing of Premiums and Settlement amounts.

Rate Offset

The number of business days between the Rate Set date and the Start date of the Option period. For example, if the Start Date is 17-Oct-2016 (a Monday), a 2 day offset will give a Rate Set date of 13-Oct-2016 (the previous Thursday).

Day Count

The basis for calculating and accruing interest. For each option except USE CURRENCY:

  • the term before the slash indicates how the number of days in a period are calculated, and
  • the term on the right side indicates how the number of days in the year are calculated.

TIP: In Quantum the day count includes the start date but excludes the end date.

Non Business Day Rules area

(Mandatory for Simple Interest Rate Options only.)

Use the fields in this area to record and view Business Day rule settings for deals based on this instrument, for the following dates:

  • Interest Calculation start and end dates ("effective dates") for calculating the interest for any given time period,
  • Cash Flows dates,
  • Exercise/Lapse dates (which must fall on business days), and
  • Principal amount repayment dates when amortizing or accreting Notional Principal Amounts.

In each case, select one of the following adjustment conventions to use when the relevant date falls on a non business day (where valid):

  • FOLLOWING BUSINESS DAY
  • MODIFIED FOLLOWING BUSINESS DAY
  • PRECEDING BUSINESS DAY
  • ACTUAL DATE (Interest Calculation only)
  • CURRENCY DEFAULT (Cash Flows only)

Defaults area

The same fields apply to all Option types, and all are optional.

See Defaults Area Settings (all Interest Rate Option Types).
