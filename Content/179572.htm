Importing Deals that Belong to a Deal Set

The following deal types can be imported with a Deal Set.

  • Actual Cashflows (AC)
  • G/L Journals (GL)
  • Bank Account Transfers (AT)
  • Interest Rate Options (IO)
  • Currency Options (CO)
  • Projected Cashflows (PC)
  • Cross Currency Swaps (SC)
  • Repurchase Agreements (RA)
  • Forward Rate Agreements (FR)
  • Interest Rate Swaps (SI)
  • Futures Deals (FU)
  • Money Market Deals (SE)
  • Foreign Exchange Deals (FX)
    - see Note below

 

To import a deal belonging to a Deal Set

  • Add the following text to the deal importation:

    deal_set_id=deal set number

Importing example

The following example shows the QID-format importation of an Actual Cash flow that belongs to Deal Set Number 45.

EXEName=CFLOPROJ
*SET THE MODE TO ACTUAL
$DoMenu=moactual
ticket_no=DealSet01
ccy=AUD
amount=1000000
ccy2=NZD
rate=0.87
cflow_dt=10-Sep-96
cflocode=Analysis Code 1
comments=:A cashflow comment
cparty=ANZ
entity=Transactor - Level 2
bankacc=1000 1000 1000 1000
strategy=Strategy1
dealer=sqa_user1
analyse=INFLOWS|ENOUGH
analyse=OUTFLOWS|PLENTY
deal_set_id=45
$DoMenu=mfInsert

FX Swaps linked to Deal Sets

NOTE: When importing the two legs of an FX Swap Deal that is linked to a Deal Set, ensure that either:

  • The deal_set_id value is specified in only one leg,

    -or-
  • The deal_set_id value is the same in both legs.

    (If different values are specified in the two legs, the value from the last leg imported is used.)
