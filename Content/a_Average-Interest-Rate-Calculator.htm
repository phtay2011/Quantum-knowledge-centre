Average Interest Rate Calculator

Use the Average Interest Rate Calculator page to calculate the average interest rate for a specified yield curve and average period without affecting any actual pricing. This is a freestanding calculator.

Calculation Parameters area

Use this area to set up a calculation.

To perform a calculation

  1. In the Rate Name drop-down list, select a yield curve name.
  2. Select the Currency for the curve.
  3. Optionally, select a different As At Date. The default is the current date.
  4. Set the Start Date and the End Date for the calculation period. The system includes start and end dates in the average calculation.
  5. Select the Method.
  6. Optionally, set values that are different to the default values for all other parameters: Term, Term Type, Denominator, Margin, Suspension, Holidays List, Rounding, and Lag. When rounding percentages, remember to leave enough decimal places to express the value as a decimal and not as a percent.
  7. Click Calculate.

Calculated Rates area

After you click Calculate, the following results appear:

  • Calendar Days - the number of calendar days over which the average was calculated - that is, from Start Date to End Date;
  • Business Days - the number of business days from Start Date to End Date;
  • Avg Int Rate - the average interest rate calculated using the given parameters.

Learn how we calculate market values for IRS in Analytics.

Rates tab

This tab shows the list of all the Rate Dates and Rates used to calculate the Average Rate.

There are also columns for Days and Compounding.

  • Each value in the Days column is the number of calendar days in the period on which the rate is equal to the rate printed in the same row.
  • For each day, the Compounding column shows the compounding factor up to that date.

NOTE: A complete description of the input parameters is in the formulas section.

The last interest period's accrual date

Last Interest Period Accrual Date stores the latest interest period's accrual date for the coupon's period.

This date, together with End Date, dictates how the system uses Suspension to calculate the Average Interest Rate.

Last Interest Period Accrual Date = coupon end date – 1 calendar day

It is an optional field. Default = 0.
