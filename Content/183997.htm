Line and Column Combinations in Reporting Chart Views

(First introduced in Release 6.2.1 SP, May 2016.)

Enhancements have been made to the Chart View in Quantum Reporting which, among other things, enable you to combine line and column charts in certain cases.

Implementation

The following changes have been made to charts.

  • Multiple fields can be grouped at the same time. Each combination of the group values will become an axis point.
  • A Combo chart can display the data points as columns or markers.
  • The bars, columns, and markers can be sorted by the value of a different field. This field is either the first group by field or a field that is not grouped on.
  • When a Combo chart has a Sort By field specified, the markers are connected by a line.

    NOTE: All Sort By field values will be displayed as data points, and these data points will be grouped according to the Group By fields. This means that in some cases more data points will be displayed when a sort order is added.

Data Types

The available chart types depend on the data type of the chosen fields and whether only one field or multiple fields are chosen.

NOTE: Data types are consistent with those used for analysis codes.
