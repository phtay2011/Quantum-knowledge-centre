Adaptiv Query

Use this page to:

  • Run the current Query - Enter specific Filter and Option settings and click Run Query. The displayed options vary for each Query, depending on its configuration.

    TIP: Resource Security settings can restrict the deals returned.

  • Change the Query or create a new one based on it - Click the Query Configure link, make your changes, and save them.
  • Select a different Query of the same type - Click on the current Query heading and select the Query in the drop-down list.
  • Create a different type of Query - In the page menu, select New Base Valuation, New CVA, New Cash Flow, New Historical VaR, New Market Sensitivity, New Monte Carlo Var, or New PFE as required to go to the New Query page for that type of report, configure the query parameter Defaults and Properties including a unique Name; then click Save. The above functions are then available for this new report type.
  • Create an Adaptiv AAJ file containing the current run results - Click Download AAJ File, in the dialog box select the required options, then click OK.
  • Display previous run results - Click on the current query heading and select the run results in the drop-down list (under the relevant Query).
  • Select a different result View for the current query - Click on the current View heading and select the View in the lookup.
  • Change the current Query View or create a new one - Click the View Configure link, make your changes, and save them.
