Mature Deals (in bulk)

Use the Mature Deals page to mature Time Deposit Money Market deals in bulk. Time Deposit Money Market deals use instruments with Calculation Formula set to Yield, and Interest Timing set to On Maturity, On Statement, or Rolling.

  1. If required, select a different Query from the drop-down list next to the Run Query button.
  2. Enter your Filter criteria to define the deals you want to view; in the "(advanced options)" section, select the Mature Deal Defaults options if required; and click Run Query.

    TIP: To create new Filter criteria, click Configure, specify filter parameters (on the Defaults tab), name the new Query and set Access Permission to Public or Private (Properties tab), then Save.

    Deals meeting your filter criteria are listed.

  3. Do one of the following:
    • Review and optionally mature selected deals
    • Mature selected deals immediately, without review

    The selected deals are matured.

  4. (Optional) To view a summary log - in the page menu under LOG, select View.

To review and mature selected deals

  1. Select the deals in the result list.
  2. Select Review Mature Deal Details from the grid menu and in the dialog box select Ignore Settlement Checking to disable any warning messages resulting from renegotiating deals that have unactioned cash flows.
  3. Select Set Cash Flow Status to Unactioned to change to an unactioned cash flow on maturing.
  4. Click OK to confirm.
  5. The selected deals are matured.

To mature selected deals immediately, without review

  1. Select the deals in the result list.
  2. Select Run Query And Mature Deal from the Run Query drop-down button in the page header.
  3. The selected deals are matured.
