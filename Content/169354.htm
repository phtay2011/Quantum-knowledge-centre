Other Differences

Analytics Counterparties do not support the following features and functions of Quantum Counterparties:

  • Static Data Approval (and therefore the concept of unapproved and approved records)
  • Identifiers
  • Tax Numbers

Quantum Counterparties do not support the following features and functions of Analytics:

  • Ratings (by Ratings Agencies)
  • Exchange category Counterparties - These apply only to Unit Holding instruments and deals, and exchange-traded Security Identifiers. Futures Exchanges (in the Default perspective) are defined using the dedicated Futures Exchange page, NOT this Counterparty category (which Analytics does use for Futures Exchanges).
