Dealing and reporting enhancements

The following enhancements apply to dealing and reporting features.

Use Roll Deals page for repo agreements

You can now use the Processing > Roll Deals page for Repo Agreement (RA) transaction types, which means that you can roll the deals to a predefined Maturity Date (new field) or, by using the Defaults tab, to a new maturity date based off Basis Offset or Next Business Day (current functionality).

Regulatory reporting updated for Brexit

As a result of Brexit, reporting obligations of several entities (existing clients) could change overnight. We have set up the reporting system in a way that, when the change happens, clients can continue to report to the new jurisdiction seamlessly. In general, we'll update the rules screen in a way that accounts for Brexit so that on the basis of effective date field Quantum picks up the correct rule.

There are new fields:

  • Reporting Jurisdiction - text
  • Third Party LEI - text
  • Delegate to Entity- Yes / No
  • Is Entity Jurisdiction - Yes / No

The regulatory templates Regis and DTCC make use of the above fields to render the correct data in the outbound XML file.
