Set up the Sources area

Number/amount column formats

When configuring a custom report, two new formats are now available for columns: Amount and Number. This means the columns in your custom report can look more like your source columns.

If the data type for the column is Amount, choose one of the following as the Display As value:

  • Amount - the length varies with the Currency selected. If no Currency selected, then 2dp.
  • FX Rate - 8 dp
  • Interest Rate - 12 dp

If the data type for the column is Number, choose one of the following as the Display As value:

  • Number - an integer
  • Deal Number - this is the link to the deal

From the Currency field, pick the currency that has the format you want to use in the column. The Currency browse list only shows the currencies and the Reference and Text data types from the same data source.

Display As column

A Display As field has been added in the Columns area of the New Source or Edit Source dialog box.

When you set this to Deal Number, any numeric column can use it, the value is rounded down, and is displayed as a deal number link.

Add/Edit Source dialog box changes

When adding a new source, the Select Columns dialog box now shows Data Type.

When editing a source, the Edit Source dialog box now:

  • has an Add button at the top of the grid that enables you to add one or more items,
  • only shows Edit in the grid menu, and
  • has an X button at the right of grid rows, which you can click to delete that item.

Database view as data source

The system now supports views starting with qvw_datasource_ as a source of data:

If the systems finds these, then in the Edit Source or New Source dialog box, the Source browse list will include a folder called Database. All views that match the above criteria will be in the folder. When you select one view, the next screen allows you to select a field from that view (but not settings).

This applies to both SQL and Oracle databases.

Edit Settings dialog box for data sources

The Edit Setting dialog box has been simplified. In the Input field, select Filter or Value, then select the filter or value from the browse list.

When you edit or create a date-related setting, the items in the Operator drop down list have also been expanded to include Between. If you select Between, then From and To fields appear (as in the example below), enabling you to enter or edit the date range the operator applies to.

Join on any column

When joining data sources, you can now join on any column from each source.

In previous versions, only the columns displayed in the report could be joined on.
