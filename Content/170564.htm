Performing Netting Session Actions

To perform an action for a netting session

  1. On the Netting Session page, select the Pool Name and Session.

    TIP: You can use the search box in the list page to quickly find the record you need.

    The session is opened.

  2. Click to open the page menu and select the required action - when available for the current session status - as follows.
    • Advance Session - moves the session status to the next pool status - for example, from INPUT to REVIEW.

      See Netting Session States and Access.

    • Revert Session - moves the session status back to the previous pool status - for example, from REVIEW to INPUT.
    • Archive Session - moves the session data to the archive.
    • Copy Session - adds a new Netting Session to the pool by copying an existing one. See the procedure To add a new Netting Session by copying an existing one under Add a New Netting Session to a Pool.
    • Import Rates - imports and updates the foreign exchange rates used during the netting session. The updated rates can be viewed on the Rates tab - see Maintain FX Rates for a Session.
    • Calc Session - calculates netting summary results for the transactions in the session and displays these, and session statistics, on the Calc Summary and Statistics tabs, respectively.
    • Generate Settlements - calculates settlement results from the netting summary results, stores these in the mln_hss database table, and displays them in the Settlements area.
    • Create Cash Flows - creates Quantum Actual Cash Flow deals from the settlement results, excludingFailed records.

      TIP: You can also perform a Create Cash Flow action on selected records displayed in the Settlements area, where that action is available in the relevant tab's menu. See Viewing Settlement Cash Flows.

    • Create Cash Flows Incl/Failed - creates Quantum Actual Cash Flow deals from the calculated settlement results, includingFailed records.
    • Check Cash Flow Messages - displays a list of system messages relating to a Create Cash Flows action.
