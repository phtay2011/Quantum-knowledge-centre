CashXplorer and Intraday Matching

In addition to the availability of reconciled data for the previous day (Prior Day Matching), CashXplorer can also access provisionally matched data for the current day, through Intraday Matching.

Using the Intraday Matching functionality, CashXplorer can:

  • import Intraday data from the banks and other external sources,
  • validate the imported Intraday data,
  • Intraday-match (provisionally match) Quantum's projected cash position with the imported Intraday data,
  • view the data import status and Intraday Matching summary information,
  • automate the process of data import, Intraday Matching, and Data Version creation, and
  • accurately represent any zero balance cash flows.

Import Data

Using the available data import methods, CashXplorer has access to:

  • External Cash Flows (cash movements referenced in another system) and
  • Intraday Bank Flows

See Importing Bank Files for more information.

Validate, Filter, and Select Imported Intraday Data

It is crucial to the integrity of CashXplorer that imported bank files and external data files are accurate and appropriately formatted.

Intraday Matching includes a thorough validation of data imported in formats such as SWIFT and BAI.

You can modify Bank Importer's validation routines according to the quality of the imported intraday data and, where appropriate, switch dataload validation, uniqueness filtering, and Intraday loading options on and off.

Match Intraday Data

More accurate cash positions and cash flow forecasts can be made if current-day cash flows and projected cash flows can be provisionally matched (with unconfirmed bank data). Through Intraday Matching, CashXplorer has access to Intraday-matched bank flows and balances.

The Balances and Floats used as the starting position for the Intraday cash positions and cash flow forecasts can include or exclude unreconciled cash flows:

  • Current balance / last of the day (excluding the unreconciled cash flows of that day).
  • First cash flow of the day (including the unreconciled cash flows of that day).

To illustrate the accuracy of the starting position, CashXplorer can display unreconciled flows for the previous day as System-defined Horizons:

  • Unreconciled Bank Flows Prior to Start, and
  • Unreconciled External Cash Flows Prior to Start.

See Matching Cash Flows with Bank Flows.

Automate Import Data, Intraday Matching, and Data Version Creation

Bank Importer includes run-time parameters you can use to:

  • select the files you want to import (if you are importing Bank transaction data from ECHOS, no files are involved - see Importing Bank Data from ECHOS) and...
  • after a successful import - run the Intraday Matching process and automatically create a Worksheet Data Version.
