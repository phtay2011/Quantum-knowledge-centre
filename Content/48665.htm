Common Terms in these Formulas

Of the variables not previously defined, R is the rebate (applicable only for Out options) and H is the barrier level.

Greeks for the barrier options are computed using the approximation method that is described in the section for the Barone-Adesi/Whaley model. That is, each Greek letter is calculated by perturbing the appropriate input parameter.
