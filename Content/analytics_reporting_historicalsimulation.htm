Historical Simulation area

(Historical VaR reports only.)

Select one of the following method options (see also About Historical Simulation Methods):

  • Rank profit and loss - to calculate the Historical Simulation result as the difference between two simulated market values generated using standard valuation models, or
  • Use relative daily return - to rank the difference between market values on Position Date rates and simulated market values based on scenario rates.

Select one of the following timing options and specify the relevant settings in each case:

  • Define simulation period - to enter a Simulation Start Date and Simulation End Date, or
  • Set number of simulations - to enter the Number of Simulations that will determine the simulation period.

The following items appear only if Use relative daily return is selected:

  • Holding Period - the number of days to perturb in a HSVaR simulation;
  • Holding Period Type - one of the following:
    • Data - to calculate HSVaR based on a Sample date approach - for example, 11/1, 21/11, 31/11... simulations for a ten-day horizon, or
    • Rolling - to calculate HSVaR based on a Rolling date approach - for example, 11/1, 12/2, 13/3 14/4...simulations for a ten-day horizon; and
  • Convention - Business Days or Calendar Days (for example, a ten-day Business horizon is equivalent to a 14-day Calendar horizon).
