Facility Drawdowns

A Facility may allow drawdowns. For a borrower, these are taking out further advances or withdrawing further funds (for the lender, they are regarded as takedowns).

On the Facility page's Drawdown tab, use the two grids to record details of drawdowns under the Facility as follows:

  • DRAWDOWNS grid - Exposure Limits and other restrictions, and
  • SCHEDULES grid - scheduled drawdown cash flow amounts and dates where appropriate.

You can define more than one set (entry row) of limit data of each type, with different Active From dates. The entry settings active for the Deal will be applied when deals using the Facility are saved.

TIP: If specific drawdown limits apply, the system uses the Rate Type specified in Management Control to convert the deal amount to the specified currency if necessary.

Using the Grids

In the two grids:

  • each row represents one entry or set of field values, with a unique Active From date;
  • to add a new entry row - in the grid menu select New to display the New dialog box;
  • to delete an entry row - select the row and in the grid menu select Delete, then confirm;
  • to display the New dialog box,rows can be filtered on any value;
  • field names are given as column headings;
  • the displayed columns are configurable and may be sorted on;
  • the field values for each entry row are the cell values in the relevant columns, and
  • to edit a field value, click it to display the Edit dialog box.

In the New and Edit dialog boxes:

  • all available fields (see below) always appear, and
  • all fields are editable.

DRAWDOWNS grid Items

Active From - the date from which the entry row data becomes active.

Valid Currencies (New and Edit popups only) - the valid currencies for the drawdown: accept the default of ALL, or select one or more currencies.

Amount fields (optional) - If the Facility prescribes specific drawdown limits, record the amounts in the following Currency and Amount field pairs, as applicable.

  • Maximum Amount - the maximum total drawdown.
  • Minimum Amount - the minimum total drawdown.
  • Increment Amount - the incremental amount for each drawdown.
  • Maximum per Day - the maximum drawdown available on any one day.
  • Max Maturity Per Day - the maximum amount to have maturing on any one day.
  • Minimum Utilization - the minimum value of utilization at any time (not currently monitored, but may be reported by getting the daily utilization of the Facility).

Period Between Maturities (optional) - if the Facility specifies a minimum number of days between deal maturities, enter the number of days and select Business or Calendar days in the drop-down list.

Drawdown Notice (optional) - if the Facility specifies a minimum number of clear days' notice prior to drawdowns, enter the number of days and select Business or Calendar days in the drop-down list.

Redraw Notice (optional) - if the Facility specifies a minimum number of clear days' notice prior to a redrawing (subsequent drawdown) of funds, enter the number of days and select Business or Calendar days in the drop-down list.

Days Between Drawings (optional) - if the Facility specifies a minimum number of clear days between drawings, enter the number of days and select Business or Calendar days in the drop-down list.

Max Drawing Period (optional) - if the Facility specifies a maximum term for each transaction, enter the number of days and select Business or Calendar days in the drop-down list.

Min Drawing Period (optional) - if the Facility specifies a minimum term for each transaction, enter the number of days and select Business or Calendar days in the drop-down list.

Max No of Drawings (optional) - if the Facility specifies a maximum number of drawdowns, enter that number.

SCHEDULES grid items

(optional)

NOTE: These entries only list scheduled cash flows for deals using the Facility. They enable you to specify actual drawings that are to occur, and when the next deal after the Active From date is saved, these cash flows are then created as ONE deal.

Active From - the date from which the entry row data becomes active.

Date - the date on which a drawdown is to occur.

Currency - the currency of the cash flow.

Amount - the amount of each principal cash flow. The deal process automatically treats this entry as a drawdown, so enter each drawdown amount as a positive number. Only enter negative numbers if the deal is an investment.
