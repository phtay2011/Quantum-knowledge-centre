Recording Quotes Received

As each Counterparty responds to the Quote Request, record their bids in the Bids Received grid on the Deal details tab, by selecting New in the grid menu, or by copying an existing bid (by selecting it and the Copy grid menu function). Each grid row displays:

  • the name of the bidding Counterparty,
  • basic information about the potential deal,
  • the bid price and other details as required(see the TIPS below), and
  • the Speed of the Counterparty's response (V FAST, FAST, OK, SLOW, or NO BID).

When you have recorded enough Quotes to make a decision, review the Quotes and decide which ones you want to convert to deals.

Then proceed to the next step - see Converting Quotes to Deals.

TIPS:

  • To sort rows by a specific column, click that column's header. To reverse the sort sequence, click the column header again. (The default display sequence is in ascending order of the difference between the Spot and Contract Rates.)
  • Although most fields default from the Deal Details tab, you can still change these where necessary to record different Counterparty responses.
  • For FX Deals or FX Time Options where the selected instrument's Deal Rate is Spot & Forward, a Forward Points column is available. The system uses this value to display the difference between the Spot Rate and Contract Rate.
  • For FX Swaps, a Forward Points column is available. The system uses this value to display the difference between the Near Rate and Far Rate.
