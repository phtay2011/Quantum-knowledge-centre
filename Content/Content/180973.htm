Data Version History, Selected Accounts

Use the Accounts sub-tab to view details of each account resolved when the currently displayed Data Version was created.

The Accounts grid displays the following read-only details for each resolved account:

  • Account Number,
  • Name,
  • Currency, and
  • Requested OB Statement Date - the Statement Date requested for providing the worksheet's Opening Balance for the account.

    NOTE: If a requested Statement Date is NOT found and used, that is indicated by both:

    • a "missing statement" alert icon displayed to the left of the affected amount in the results view, and
    • an error message on the History tab's Messages sub-tab, indicating the account and which fallback balance was used instead, as in the following example for account CHIPS1:

      Requested Opening Balance for Start Date not found forCHIPS1. Using the Fallback Internal Unreconciled.

    (If the requested date IS found and used, that is also indicated on the Messages sub-tab.)

You can drill down to view more details for a listed account by selecting its grid row and View in the grid menu. The Account Details dialog box displays the following read-only values:

  • account identification details,
  • details of when the Data Version was last modified,
  • balances, and
  • cash concentration details.
