Filters area, G/L Interface Configure

Enter values and make option selections in these fields to reduce the result set.

  • As At Date - enter or select a date to include only entries created or changed up to and including that date.
  • Start Date and End Date - the start and end dates of the period you want to report on.

    TIP: These dates are filled automatically if the G/L Owner uses accounting periods, based on the Use Accounting Period selection.

  • Use Accounting Period (applicable when a General Ledger Owner is selected) - If Accounting Period records exist for the selected G/L Owner, select one from the drop-down list, and the system will set the Start and End date fields for that period. If the G/L Owner does not use accounting periods, this field will be left blank.
  • General ledger Owners - select one or more G/L owner entities or leave blank to cover all G/L owners.
  • Posting Source - select one or more posting sources (Accrual, Revaluation, CCY Translation, Deal, GLEOY, GL Journal), or ALL.
  • Include Cash Flows - select which un-interfaced G/L entries to include based on the associated cash flow status (if any*) or Counterparty Category*: All, Actioned, or ( External Bank Accounts only) Settlement Approved, Matched, or Reconciled.

    *NOTES: Provided they meet other filter criteria, G/L entries are always included in the results if they:

    • have NO associated cash flow, or...
    • have an associated cash flow with a Bank Internal Counterparty - EXCEPT when Actioned is selected here and that flow is unactioned.
