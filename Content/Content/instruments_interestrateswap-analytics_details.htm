Details tab, Interest Rate Swap Instrument

Use this page to create and maintain instrument details in Quantum. The first few fields of this tab are for the default values to apply to a deal when using this instrument. Click on an item to find out more.

Pay Type - select Bond, Vanilla Fixed, Vanilla Floating, Constant Maturity, In Arrears, or Quanto.

Receive Type - select Bond, Vanilla Fixed, Vanilla Floating, Constant Maturity, In Arrears, or Quanto.

NOTE: While you can alter the Pay Type and Receive Type in <Analytics_>, the system does not transfer this information to Quantum, and if someone edits Quantum, the changes overwrite <Analytics_>. To keep the settings in <Analytics_>, use the "newonly" key in the import XML file."

Principal Exchange - select the default for when to create deal principal cash flows: Both (for every change in principal), Start, Maturity, or None.

Frequency - select Daily, Weekly, Fortnightly, Monthly, Quarterly, Semi Annual, Annual, or [on] Maturity.

Cycle to Interest - select whether or not (where applicable*) the deal's rateset cycling frequency will be synchronized with the interest period frequency. If selected, this ensures that:

  • the final date of the last rateset period is the same as the end date of the interest period, and
  • the first rateset period of the next interest period will start on the start date of that interest period.

    *TIP: This option is supported only for Swap legs where:

    • Interest Calculation is YIELD,
    • Frequency is WEEKLY or FORTNIGHTLY, and
    • the type (Pay Type or Receive Type) is NOT Constant Maturity.

End of Month - select Yes or No to specify whether or not to move the next Payment Date to the last day of the applicable month.

For example, if End of Month is Yes, Frequency is Semi Annual, and First Payment Date is 30-Sep-2015, the next Payment Date will be 31-Mar-2016. If End of Month is No, the next Payment date will be 30-Mar-2016.

Amortizing - select Yes or No to specify whether or not the principal will be amortized/accreted.
