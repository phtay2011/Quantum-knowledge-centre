Configure View - Display Defaults area of Display tab, CashXplorer

Use this area to specify the following display settings for the current View.

Horizon (not available for Forecast Views) - select the Horizon for the Cash Position or Summary View.

Amounts Displayed In - select the currency for displaying amounts in the grid when this View is used. You can select:

  • Transaction CCY - the cash flow currency (this is the default);
  • Group CCY - the group currency to which the cash flow currency belongs, if any (for example, the Euro);
  • Base CCY - the accounting currency for the account's Entity, (as defined in the applicable Accounting Type record in each case); or...
  • Cross CCY - a currency other than the Base Currency, into which cash flows can be translated - provided that a Currency Pair record exists for the currency combination.

Apply filters - select this option to enable all filters on the grid, so that users without Maintenance (Mtn) access to Cash Xplorer Views cannot see restricted data by applying filters.

Exclude concentration flows - select this option to exclude concentration flows from the grid when it is first opened. By default, this option is not selected.

Hide system-defined horizons (Forecast Views only) - select or clear this option to respectively hide or show columns for system-defined horizon totals (for example, Previous Day Reconciled Cash Flows). When cleared, the actual horizon columns that will show, and their captions are as specified on the Columns tab for the Forecast View.

Hide items with no activity - select this option to hide items that have no flows, no alerts, or a zero balance.
