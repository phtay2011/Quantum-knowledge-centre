Counterparty Delivery Instruction Conversion page

Use this page to change or update the counterparty delivery instructions for cash flows without having to re-save deals.

NOTE: You cannot add new delivery instructions from this page. Use the Counterparty Bank Accounts or Counterparty Stock Accounts page instead.

WHAT DO YOU WANT TO DO?

Select a saved query - Click and select in the Browse list.

Create a new query - Click New; enter a Name; enter (or select) the Effective Date; select either Counterparty Instruction Change (to convert ALL cash flows for selected counterparties and accounts) or Specific Counterparty Instruction Change (to convert selected cash flows); enter Account Filters and Cash Flow Filters settings; then click Save.

NOTE: The Cash Flow Filters area is only available when Specific Counterparty Instruction Change is selected.

Change an existing query - Select the query, edit its settings as required (see above), then click Save.

Run the current query - Click Run Query. The Accounts grid lists the applicable counterparty accounts.

Specify conversion details for an account - In the Accounts grid row, click a linked value; in the Edit dialog box change the To Account settings as required, optionally select Apply defaults and/or Override adhoc settlements; then click OK.

Bulk change the Override adhoc settlements setting for selected accounts* - In the Accounts grid, click the check box at the left of each account* you want to change and select eitherOverride adhoc settlementsorDon'tOverride adhoc settlements in the grid menu, as required.

Bulk change the Apply defaults setting for selected accounts* - In the Accounts grid, click the check box at the left of each account* you want to change and select eitherApply DefaultsorDon't Apply Defaults in the grid menu, as required.

Convert selected accounts*, applying the current settings - Ensure a Name is entered; in the Accounts grid, click the box to the left of each account* you want to convert and select Convert in the page menu.

*TIP: To toggle selection of all grid rows, click the check mark in the column heading.

Reset the current Conversion settings - Select Reset in the page menu.

IMPORTANT: Resetting the conversion will reset which accounts the system targets and any To Account conversion details will be lost.
Quantum will prompt you with a warning to this effect and request confirmation before proceeding.
