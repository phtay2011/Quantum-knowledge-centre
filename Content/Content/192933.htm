Workflow Report Example 1

In this example, we join an: Analytics query to Quantum static data.

The data sources are the Analytics report BDR Position Test 1 and Quantum Counterparty static data.

  1. Enter header details

  2. Select the first data source: an Analytics query

    In the Sources grid menu, select New to open the New Source dialog box (All Data should be selected as below):

    In the Source field's Browse window, search on BDR:

    Select BDR Position Test 1. The New Source dialog box now shows Columns and Settings tabs.

  3. Add columns from the source

    On the Columns tab of the New Source dialog box, click Add and select the columns as below:

    Click OK. The New Source dialog box now shows your selected columns:

    Click OK to close the New Source dialog box (we will add settings to the Source in Step 5).

  4. Add Filters: Nett Date and Position Date

    In the Filters grid menu, select New to open the New Filter dialog box. Enter the Nett Date settings as below:

    Click OK to close the New Filter dialog box. Repeat to add the Position Date filter:

    Click OK. The two filters are now shown:

  5. Add Source Settings: Position Date and Nett Date

    In the Sources grid, select BDR Position Test 1. The Edit Source dialog box opens:

    On the Settings tab of the New Source dialog box, click Add to display the Select Settings dialog box:

    Select Nett Date and Position Date and click OK. Your selected settings are now shown:

  6. Edit the settings to use filters

    In the Settings grid, select Nett Date to open the Edit Setting dialog box. Set Input to Filter and set Filter to Nett Date:

    Click OK to close the Edit Setting dialog box. Repeat for Position Date:

    Click OK. Your Settings are now set to their Filters:

    Click OK to close the Edit Source dialog box.

  7. Select a second data source

    In the Sources grid menu, select New to open the New Source dialog box (All Data should be selected as below). In the Source field's Browse window, drill down to (or search on Count, and select) Static Data > Counterparty:

    On the Columns tab, click Add and select columns for the second data source (Counterparty Name and two addresses in this example):

    Click OK. The New Source dialog box now shows your selected columns

  8. Join Analytics' Counterparty to Quantum's and save

    Because there are now multiple sources, a Join tab appears in the New Source dialog box.

    On the Join tab, in the Left Source Browse window, select your first Source BDR Postion Test 1 (Type of Join should be as below):

    Click OK.

    Click Save.

  9. Add Module Security rights for the current user

    (Not applicable if the current User is exempt.)

    In the page menu under MODULE SECURITY, select Add to Current User:

  10. Log off and on to see "Workflow BDR" in the menu

  11. Enter filters and run the query

    NOTE: The first five column values come from the Analytics report, and the two address columns from the matching Quantum Counterparty record (if no match was found, the address columns would be blank).
