CSV

Syntax

<...>
Start Sequence [0..1]
<delimiter> string </delimiter>
Start Choice
<tabSeparated> empty </tabSeparated>
<separator> string </separator>
End Choice
<mode> csvMode </mode>
<reportAndPageSections> sectionExport </reportAndPageSections>
<groupSections> sectionExport </groupSections>
End Sequence
</...>

NOTE: Use the <tabSeparated> tag to define tab separated values.

Example

<CSV>
<delimiter>{None}</delimiter>
<tabSeparated/>
<mode>Standard Mode</mode>
<reportAndPageSections>Do Not Export</reportAndPageSections>
<groupSections>Export and Isolate Sections</groupSections>
</CSV>

delimiter

Syntax

...
<dlimiter>string (length <= 50)</delimiter>
...

TIP: Use {None} to NOT have a delimiter in the CSV file.

tabSeparated

Use this if you want tab separated values.

Syntax

...
<tabSeparated>empty</tabSeparated>
...

Example

...
<tabSeparated/>
...

separator

Syntax

...
<separator>string (length <= 50)</separator>
...

Example

...
<separator>|</separator>
...

reportAndPageSections

Syntax

...
<reportAndPageSections>[Options]</reportAndPageSections>
...

Options

  • Do not Export
  • Export

Example

...
</tabSeparated>
...

groupSections

Syntax

...
<groupSections>[Options]</groupSections>
...

Options

  • Export
  • Export and Isolate Sections
  • Do Not Export

Example

...
<groupSections>Export and Isolate Sections</groupSections>
...

mode

Syntax

...
<mode>[Options]</mode>
...

Options

  • Standard Mode
  • Legacy Mode
