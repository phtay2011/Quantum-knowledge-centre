Holiday Center page (Analytics)

Use this Analytics page to create and maintain named business day records to be applied to specified Currencies, Entities, Counterparties, and Deal Pricing (in the Analytics perspective).

NOTES:

  • If you access a record sourced from Quantum (via the Replication Service), the fields are read-only on this Analytics page.
  • Any changes made here affect only deals saved after those changes.

To add a new record - Click New; then:

  1. In the header area enter a unique Code, enter a text Name, and select the usual non-business days of the week (normally Saturday and Sunday).
  2. In the Holidays tab's grid menu, for each specific holiday: select New; in the dialog box enter the date and text description; then click OK. Repeat as required.
  3. In the Business Days tab's grid menu, for each "non-standard" business day that would otherwise be a non-business day: select New; in the dialog box enter the date and text description; then click OK. Repeat as required.
  4. Save.

To list records - Click to open the Browse list. (Click the Configure link to change list properties.)

To change a record's details - Select it in the Browse list; edit the header fields if required; use each tab's menu to Add, Edit, and Delete individual entries as required; then Save.

NOTE: If you access an existing record sourced from Quantum (via the Replication Service), the fields are read-only on the Analytics page.

To delete a record - Select it in the Browse list and select Delete in the page menu. When prompted, confirm.

For more details of each tab, click on that tab when the Help panel is open. Or follow the link below.
