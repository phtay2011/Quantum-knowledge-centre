Evaluate a Base Template or filters against Fee Base deals

Use the Process Fees page's Evaluate settings and Run Query function to view - within the current Fee Position History table - applicable base deals, total base amount, and total fees when you apply either a specified Fee Base Template or specified Filters values.

NOTES:

  • You must first use the Get Base Details function to populate the Fee Position History database table (for the specified As At Date), as displayed in the Base Deals grid.
  • Running an Evaluation query does not update the database.

To test a Fee Base Template against the Fee base data

  1. In the Process Fees page's Evaluate area, select Use Base Template.
  2. Select the Base Template you want to test.

    TIP: Only Base Templates with an Effective Date range spanning the As At Date for the Fee Position History are available for selection.

  3. Under Filters, enter the Position Date for the test.
  4. Click Run Query.

    The total Fee Amount, Base Amount, and (where applicable) Facility and Facility Notional Amount are displayed above the Base Deals grid, which is refreshed to show the applicable base deals.

To apply deal filters to the Fee base data

  1. In the Process Fees page's Evaluate area, select User Defined.
  2. Under Filters, enter the Position Date for the test and select the deal filters you want to apply (Transaction Type, Instrument, and so on).
  3. Click Run Query.

    The total Fee Amount, Base Amount, and (where applicable) Facility and Facility Notional Amount are displayed above the Base Deals grid, which is refreshed to show the applicable base deals.
