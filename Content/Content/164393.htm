Issuer Instrument (I-I) Margin

Use the Issuer Instrument Margin Rate page to enter the margins that Quantum is to apply to Issuer-Instrument combinations you specify.

This enables you to assign a specific margin curve, incorporating time bands, to those combinations. The underlying curve is shifted by varying margins across the curve. You may define a one-to-one or many-to-one relationship between Issuer/s and Instrument respectively, but not a one-to-many relationship (for example, Issuer = RBA and Instrument = ALL).

You can enter actual margins, as well as the dates when these are to be applied. For FRNs, you can also enter a market margin.

You can then specify default margins for revaluation in Accounting Rules or Management Control. Quantum keeps a dated history of the margins used.

About the I-I Margins grid

Each grid row lists an Issuer-Instrument combination and either a FRN Margin or part of a margin curve for that combination. You can define different parts of the margin curve for the same Issuer-Instrument combination by specifying different time bands and margins in separate rows, using the Date Interval, Date, or Days column to specify when the margins are effective.

To add a new row, select New from the table menu, use the New Margins dialog box to enter the Issuer and Instrument, then the margin details. You can:

  • Link a single Issuer to a single Security Instrument, or ALL issuers to a single Instrument (you cannot link a single Issuer to ALL instruments).

    If you select the Instrument first and an Issuer is assigned to that instrument, only that issuer may be selected. If no Issuer is assigned to the instrument, only ALL is available in the Issuer drop-down list.

    If you select the Issuer first, only instruments that issuer is assigned to may be selected. If instruments are assigned that Issue, only ALL is available in the Instrument drop-down list.

  • Enter an FRN Margin only if an FRN instrument or ALL is selected. If the instrument is set to ALL, this margin is applied to the FRN component only.

    If an FRN Margin is entered, Quantum will automatically apply it when revaluing that FRN. If not, Quantum revalues the FRN using the original trading margin specified on the deal.

  • Enter a margin curve component only if an FRN instrument is NOT selected.
  • Save a row change only if EITHER an FRN Margin OR margin curve component is entered.

Quantum supports standard interest rate time bands, including ACTUAL DATE, which is commonly used for maturity date specification.

The margin curve is processed like an interest rate curve, and linear interpolation provides margins between curve points. A bump against the par curve is created when you save margin points with a value of zero.

TIP: Do not use actual date points for general curve processing. Only use them when seeking a margin for the specified date.

Issuer Instrument Margin Procedures

On the Issuer Instrument Margin page:

To list Margin records

  1. Click the button.
  2. Optionally follow the Configure link to change browse list properties.

To add a new Margin record (from scratch)

  1. Click New.
  2. Select the predefined Rate Type, select the Currency, and enter the effective Date from which the margins will apply.
  3. In the Margins grid menu, select New.
  4. In the New Margins dialog box:
    1. Select an Issuer (ALL is valid) and Instrument combination.
    2. EITHER enter an FRN Margin (if an FRN instrument is selected) OR enter a Margin curve component, using any of the relative time fields (Date Interval, Date, or Days) to specify when the margin applies.
    3. Click OK.

    An entry row is added to the grid.

  5. Repeat Step 4 to complete the Margins grid.

    TIP: For similar entries, use the grid menu's Copy function and edit the copied values.

  6. Save.

To add a new Margin record by copying an existing one

  1. Click to list available records.
  2. Select the Issuer Instrument Margin record you want to copy (use the < and > buttons to page through, or type in the Search box).
  3. In the page menu , select Copy.
  4. Select a different Rate Type and Currency, and enter a new effective Date as required.
  5. Use the Margins grid menu functions to add, edit, delete, and copy entries in the table, as required.
  6. Save your changes.

To edit a Margin record

TIP: Changing the Rate Type field is NOT recommended. Create a new Margin record instead, for the Rate Type you want.

  1. Click to list available records.
  2. Select the record you want to edit (use the < and > buttons to page through, or type in the Search box).
  3. Optionally change the Currency and/or effective Date if required.
  4. Use the Margins grid menu functions to add, edit, delete, and copy entries in the grid, as required.
  5. Save your changes.

To delete a Margin record

  1. Click to list available records.
  2. Select the record you want to delete (use the < and > buttons to page through, or type in the Search box).
  3. In the page menu, select Delete.
  4. When prompted, confirm your action.
