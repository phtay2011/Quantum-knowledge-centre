Replication Status page

Use this page to monitor the status of deal and related data imported into, and exported from, Analytics by Replication Manager. You can also submit selected items for re-import attempts.

Items are listed on tabs corresponding to their current replication status, as follows. A summary giving counts of the number of items currently on each tab is displayed at the top of the page.

TIP: The Errors - In count is of Current items only.

  • The Pending - In tab lists queued items awaiting import. The Type (Deal, Management, Security Identifier, and so on), Time submitted, and Description are listed for each item.

    To submit items for import - Select the items (by clicking the box at the left of the relevant row/s) and in the tab menu, select Process.

  • The Errors - In tab lists failed import items, and has two sub-tabs. All error items initially display on the Current tab, while the Suppressed tab lists error items manually moved there. The Analytics Deal ID, Transaction Type, Instrument, Counterparty, Time submitted, and error Message are listed for each item on each tab.

    To move an item between tabs - From the Current tab, select the items and in the tab menu, select Hide. From the Suppressed tab, select the items and in the tab menu, select Make Current.

    To re-submit items for import - On the Current tab, select the items and in the tab menu, select Retry.

  • The Pending - Out tab lists queued items awaiting export. The Analytics Deal ID, Transaction Type, Instrument, Counterparty, and Time submitted are listed for each item.

    To submit items for export - Select the items (by clicking the box at the left of the relevant row/s) and in the tab menu, select Process.

  • The Errors - Out tab lists failed export items. The Analytics Deal ID, Transaction Type, Instrument, Counterparty, Time submitted, and error Message are listed for each item.

    To re-submit items for export - Select the items (by clicking the box at the left of the relevant row/s) and in the tab menu, select Replicate.
