Discount (Bill) - Periodic Formulas

The standard Discount, or Bill, formula includes modified interest timing that allows for interest to be calculated and paid periodically.

It is very similar to the Yield Periodic formula, however, there are the following differences:

  • Discount Periodic uses the discount formula to calculate interest while Yield Periodic uses the simple interest formula.
  • Interest calculated using Discount Periodic is paid in advance at the beginning of the interest calculation period. With Yield Periodic it is paid in arrears.
  • Interest calculated using Discount Periodic is paid in advance if the Allow Amortization check box is ticked, but not for Yield Periodic.

Formula

Instruments using the Discount Periodic formula calculate interest from:

Interest Amount = Face Value (FV) - Present Value (PV)

Where:

PV

= Standard Discount or Bills formula, which:
= FV/(1+(days in period/basis)*Coupon Rate/100)

Coupon Rate

= Base Rate + Margin
 

NOTE: The change in notional Principal Value is actioned using the Customize Deal form.

Restrictions

  • When using the Discount Periodic formula in a Swap deal, both legs of the deal must use that formula.
  • When customizing a deal that uses the Discount Periodic formula, you can only increase or decrease the notional principal value on an interest calculation date.
  • You cannot unwind a Swap that uses Discount Periodic prior to its start date.
