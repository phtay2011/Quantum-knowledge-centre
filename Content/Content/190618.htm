Common Point in Time Column Items

The following data items are available for selection as Point in Time columns in more than one query type. Items apply to all query types, unless explicitly indicated otherwise.

For items unique to each query type, see the applicable following topic.

Historical Simulation VaR and Monte Carlo VaR queries share the same items (none are unique).

Name

Notes and References

As At Date

Position or Cash Flow queries only.

The date specified in the As At Date field for the Point in Time column.

Bank Account No

Position or Cash Flow queries only.

Bank Balance

Position or Cash Flow queries only.

Base Ccy

Position or Cash Flow queries only.

EITHER the Base Currency as in the Valuation Control Rule OR the Currency Conversion setting (Assumptions tab, Query Configuration).

BP Delta

Position or Cash Flow queries only.

See BP Delta Formulas.

BP Gamma

Position or Cash Flow queries only.

The rate of change of delta. See BP Gamma Formulas.

Convexity

Position or Cash Flow queries only.

Measures curvature of price-yield curve. Defined as the second derivative of the price with respect to yield. See Convexity Formulas.

Deal Date

Position or Cash Flow queries only.

Dealt Rate

Position or Cash Flow queries only.

Discount Factor

Position or Cash Flow queries only.

The transaction currency Discount Factor for the:

  • Expiry or Settlement Date - if a Position query or
  • Cash Flow Date - if a Cash Flow query.

Embedded Option Delta

Position or Cash Flow queries only.

The change in value with respect to a change in the price of the underlying security.

Values are returned only for embedded options.

Embedded Option Gamma

Position or Cash Flow queries only.

The change in Delta with respect to a change in the price of the underlying security.

Values are returned only for embedded options.

Embedded Option Market Value

Position or Cash Flow queries only.

The Market Value of the embedded option.

Values are returned only for embedded options - for example, the Swaption component of Callable/Extendable Swaps or the Bond Option component of Callable/Puttable Bonds.

Embedded Option Theta

Position or Cash Flow queries only.

The rate of change of value with respect to a change in time.

Values are returned only for embedded options.

Embedded Option Vega

Position or Cash Flow queries only.

The change in value with respect to a change in the volatility of the underlying security.

Values are returned only for embedded options.

Expiry Date

Position or Cash Flow queries only.

The Expiry Date or Maturity Date of the transaction.

Face Value

Position or Cash Flow queries only.

Fixed Floating

Position or Cash Flow queries only.

Denotes whether a transaction is fixed rate or floating rate.

Fwd Factor

Position or Cash Flow queries only.

The Forward Rate expressed as a multiplier.

Fwd Rate

Position or Cash Flow queries only.

The Forward Rate to convert the transaction currency to base currency.

FX Diversified VaR

Historical Simulation and Monte Carlo VaR queries only.

Total of the cross correlation risk between different FX rates.

See Diversified and Undiversified VaR Formulas.

FX Swap Face Value

Position or Cash Flow queries only.

The Face Value on the first leg of an FX Swap.

FX Swap Value Date

Position or Cash Flow queries only.

The Value Date on the first leg of an FX Swap.

FX Undiversified VaR

Historical Simulation and Monte Carlo VaR queries only.

Risk of individual risk classes to FX rates.

See Diversified and Undiversified VaR Formulas.

Histogram Bucket

Historical Simulation and Monte Carlo VaR queries only.

A grouping of the numbers of Simulation MV Deviations, enabling the results to be displayed in histogram format (using a Chart view type).

Indexation Ratio

Position or Cash Flow queries only.

The Index adjustment applied to the Principal and Coupon.

Values are returned for Index Linked Bonds only.

Interest Period

Position or Cash Flow queries only.

The number of days in the period.

IRR

Position or Cash Flow queries only.

The Internal Rate of Return for a transaction, expressed as an Annual, Semi-annual, or Quarterly value. See IRR Formulas.

Limit Amount

The amount in the applicable Limits record.

Values are returned when Limits are included in the query.

Limit Currency

The currency in the applicable Limits record.

Values are returned when Limits are included in the query.

Limit Description

A description of the applicable Limits record.

Values are returned when Limits are included in the query.

Macaulay Duration

Position or Cash Flow queries only.

, where:

  • D = Macaulay Duration
  • DMOD = Modified Duration
  • IRRF = Internal rate of return for relevant frequency F
  • F = Rate frequency (annual=1, semi-annual=2, quarterly=4)

Market Margin

Position or Cash Flow queries only.

The Deal Margin, Security Margin, or Yield Margin when specified in the applicable Valuation Control Rule (Assumptions tab).

Returns values when the Include Valuation Margins options is selected (Assumptions tab, Query Configuration).

Market Rate

Position or Cash Flow queries only.

The market rate for a transaction.

Market Value

All except Parametric VaR queries.

The pricing formula used depends on the Transaction Type, Instrument, and Valuation Method used. See Formulas in Analytics.

Modified Duration

Position or Cash Flow queries only.

, where:

  • DMOD = Modified Duration
  • Delta ($) = Delta (Dollar value)
  • MV = Market Value
  • iBP = Basis point shift

Option Delta

Position or Cash Flow queries only.

Call: N(d1)

Put: (N(d1) -1)

Option Gamma

Position or Cash Flow queries only.

Option Status

Position or Cash Flow queries only.

One of: None, Exercised, Lapsed, Knock-In, or Knock-Out.

Values are returned only when Backdated Reporting is enabled.

Other Diversified VaR

Historical Simulation and Monte Carlo VaR queries only.

Total of the cross correlations between the different risk class interest rate risks and cross correlations between FX and interest rate risks.

See Diversified and Undiversified VaR Formulas.

Other Undiversified VaR

Historical Simulation and Monte Carlo VaR queries only.

Risk of individual risk classes to interest rates.

See Diversified and Undiversified VaR Formulas.

Pair Spot Cross Factor

Position or Cash Flow queries only.

For transactions involving two currencies (a currency pair): the spot rate to convert the transaction pair currency to the transaction valuation currency, expressed as a multiplier.

Percentile

Historical Simulation and Monte Carlo VaR queries only.

The Percentile rankings for each Simulation MV Deviation, from the highest profit to worst loss.

Example:

If there are 1000 simulations, the ten highest profits would be assigned to Percentile 1, the next ten-highest profits to Percentile 2, and so on with the 10 worst losses assigned to Percentile 100.

Percentile VaR

Historical Simulation and Monte Carlo VaR queries only.

The VaR figure for each Percentile.

Position Date

Position or Cash Flow queries only.

The date specified in the Position Date field for the Point in Time column.

Principal Outstanding

Position or Cash Flow queries only.

Not updated for Futures or ETO contracts.

Rate Name

Position or Cash Flow queries only.

The YIELD Curve used for valuation as in the applicable Valuation Control Rule (Assumptions tab, Query Configuration).

Rates Date

Position or Cash Flow queries only.

The date specified in the Rates Date field for the Point in Time column.

Repricing Date

Position or Cash Flow queries only.

For Interest Rate Repricing reporting, either:

  • Maturity Date - for fixed rate transactions, or
  • next Rateset Date for floating rate transactions that support Repricing valuation.

Simulation Gain_Loss

Historical Simulation and Monte Carlo VaR queries only.

The Gain/Loss figure for each Percentile.

Simulation Market Value

Historical Simulation and Monte Carlo VaR queries only.

Simulated Market Value based on the query Position Date's position, using simulated rates.

For example, a Simulated Market Value for a EUR/USD Call Option involves the simulated Zero Rates from both EUR and USD, the simulated Spot Rate of EUR/USD, and the simulated Volatility of EUR/USD.

Simulation Spot Factor

Historical Simulation and Monte Carlo VaR queries only.

The Simulated Spot Factor (FX Rate multiplier) used in each simulation.

Simulation Spot Rate

Historical Simulation and Monte Carlo VaR queries only.

The Simulated Spot Rate used in each simulation.

Applies when the instrument has an FX Risk Factor within the valuation.

See Simulation Spot Rate Formula.

Spot Factor

All except Parametric VaR queries.

The Spot Rate expressed as a multiplier (Spot Factor * Transaction Currency gives Base Currency).

Spot Option Delta

Position or Cash Flow queries only.

Call:

Put:

Spot Rate

All except Parametric VaR queries.

The spot rate to convert the transaction currency to base currency.

Theta

Position or Cash Flow queries only.

The rate of change of value with respect to time.

  • MV = Market Value
  • IRRA = Internal rate of return (annual)

Time Profile

Position, Cash Flow, or Parametric VaR queries only.

A description of the Time Profile bucket.

Values are returned when Time Profiles are included in the query.

Underlying Physical

Position or Cash Flow queries only.

The name of the underlying Instrument.

Returns values for Options only.

Units

Position or Cash Flow queries only.

Values are returned only for Equities transactions.

Zero Rate

Position or Cash Flow queries only.
