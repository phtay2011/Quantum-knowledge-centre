TraceLevel

Description

Turns tracing on and sets the level of trace messages that will be output to a log file.

Syntax

TraceLevel =levelNumber

Where:

levelNumber is a single digit from zero through 9.

Default

TraceLevel = 0

Examples

TraceLevel = 0

This inhibits the output of trace messages, ie turns off logging.

TraceLevel = 1

Outputs to the log the same messages that are seen in the Quantum Message Log - informational messages relating to importing bank statements as successful load messages, warnings, or critical errors.

TraceLevel = 2

In addition to the messages output from lower trace level 1, this trace level outputs the option parameter values.

TraceLevel = 3

In addition to the messages output from lower trace levels, this trace level outputs the bank account translation maps messages and the data load process messages.

TraceLevel = 4

At this time there are no messages assigned to level 4; therefore, the output is the same as TraceLevel 3.

TraceLevel = 5

In addition to the messages output from lower trace levels, this trace level outputs the Echos delegation messages.

TraceLevel = 6

At this time there are no messages assigned to level 6; therefore, the output is the same as TraceLevel 5.

TraceLevel = 7

At this time there are no messages assigned to level 7; therefore, the output is the same as TraceLevel 5.

TraceLevel = 8

In addition to the messages output from lower trace levels, this trace level outputs the map configuration messages.

TraceLevel = 9

At this time there are no messages assigned to level 9; therefore, the output is the same as TraceLevel 8.
