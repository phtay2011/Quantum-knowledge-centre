Standard Errors of the estimates

The system also produces statistics about the precision of the estimated Intercept (see Equation 2) and Slope (see Equation 3).

The confidence intervals derived from these statistics enable probabilistic statements to be made. The system sets up a range of values which can be expected to contain a true (but unknown) parameter value, with for example 95% confidence.

Knowledge about the reliability of the estimates makes it possible to predict how likely it is that a different data sample would yield the same results. The system combines regression estimates with knowledge about their reliability when making an assessment of the effectiveness of a Hedge Relationship.
