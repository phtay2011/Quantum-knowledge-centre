Point in Time Columns Unique to Position Queries

In addition to the Common Point in Time Column Items, the following data items are also available for selection as Point in Time columns for Position queries only.

Name

Notes and References

Adhoc Accrual

Adhoc accrual amount on the Position Date.

Values are NOT returned for Futures, ETO, or FRA transactions.

Adhoc Interest Accrual

Adhoc interest accrual amount on the Position Date.

Values are NOT returned for Futures, ETO, or FRA transactions.

Book Value

Book Value of a transaction on the Position Date.

The net dollar value of an asset which is recorded on the balance sheet, as distinct from market value.

Values are NOT returned for Futures, ETO, or FRA transactions.

Capital Accrual

Capital accrual amount on the Position Date.

Values are NOT returned for Futures, ETO, FRA, or Discount Security transactions

Credit Delta

A measure of sensitivity of Market Value to a move in the margin the FRN is currently traded at. (This margin is effectively a credit margin).

Coupon Accrual

Coupon accrual amount on the Position Date.

Values are NOT returned for Futures, ETO, or FRA transactions.

Dispersion

The variance of the times at which cash flows are received using their present values as weights.

Duration Weight

The denominator in the Macaulay Duration formula.

FX Position

Present value of position. For Currency Options, an equivalent physical position - for example Face Value * (Spot or Forward Delta).

FX transactions that are valued using a Forward Rate methodology return the Face Value.

Values are returned for FX transactions only.

Hedge Details

Additional detail relating to Hedge instruments.

Hedge Instrument

A defined hedge instrument - for example, eg. cash, Futures, or Swaps.

Used in Hedge Equivalents reporting.

Hedge PV01

The value of a 1 Basis Point shift for a defined hedge instrument.

Used in Hedge Equivalents reporting.

Intrinsic Value

The difference between the deal Strike Price and the current Market Spot Price:

  • for Options purchased - the in the money amount (which cannot be less than zero), or
  • for Options sold - the out of the money amount (which cannot be greater than zero).

Values are returned for Option transactions only.

Option Premium PV

The present value of a future-dated option premium.

Values are returned for Option transactions only.

Pair Base Rate

For transactions involving a currency pair - the Spot Rate to convert the transaction pair currency to the base currency.

Pair Base Factor

For transactions involving a currency pair - the Pair Base rate expressed as a multiplier.

Pair Ccy

For transactions involving a currency pair - the Pair Currency.

Pair DF

For transactions involving a currency pair - the pair valuation currency Discount Factor for the Expiry/Settlement date of a transaction.

Pair Fwd Rate

For transactions involving a currency pair - the Forward Rate to convert the transaction pair currency to the transaction valuation currency.

Pair Fwd Factor

For transactions involving a currency pair - the Pair Fwd Rate expressed as a multiplier.

Pair Spot Rate

For transactions involving a currency pair - the Spot Rate to convert the transaction pair currency to the transaction valuation currency.

Pair Spot Factor

For transactions involving a currency pair - the Pair Spot Rate expressed as a multiplier.

Period End Date

The Coupon end date.

Period Start Date

The Coupon start date.

Projected Accruals

The Interest Accruals calculated using the Projected Rates.

Projected Balances

The Balances calculated using the Projected Rates.

Projected Daily Accrual

The Daily Accruals calculated using the Projected Rates.

Projected Duration

The Duration calculated using the Projected Rates.

Projected Duration Wt

The Weighted Duration calculated using the Projected Rates.

Projected Interest Cashflows

Projected interest cashflows for Projected Cost of Funds reporting.

Projected Other Cashflows

Projected cashflows(other than Principal and Interest) for Projected Cost of Funds reporting.

Projected Principal Cashflows

Projected principal cashflows for Projected Cost of Funds reporting.

Projected Rates

The forecast rates.

Valuation Ccy

For transactions involving a currency pair - the Valuation Currency.

Vega

The change in the price of an option with a change in the volatility of the underlying security.

Values are returned for Option transactions only.

Volatility

The market Volatility rate.

Values are returned for Option transactions only.
