Select a Query

Select the current query heading or (Quantum only) click the button to open the Select Query or Run browse list and select the query you want.

  • In the breadcrumb trail above the list, click All Queries to list all Query runs.
  • Use the < and > buttons to scroll through the list, if required.
  • A folder icon is displayed next to queries that have previous runs available, indicating that users can drill-down and select the appropriate run for these.

    Select a previous run to re-display that run's results in the Report area, preceded by a summary of the Query parameter values entered for that run.

    TIP: Run results are saved until they expire, as defined by the query's Expiry Strategy settings (where applicable) UNLESS the Keep function was used, when they are kept until they are manually deleted.

When you have selected a query:

  • The default settings for that query are displayed (Filters and Options for most Quantum queries, Reference Dates for Analytics and Adaptiv).
  • (Where available) Click the Advanced Options link to toggle the display of additional options.
  • You can run that query - Enter any specific Filter and/or Option settings and select Run Query (Preview or Update in Processing mode). [more]
  • You can re-configure that Query, create a new Query based on it, or delete it. [more]

    After deleting a Query, the display reverts to the Default Query.
