Cash Flows grid, Cash Forecast page

Use this grid to:

  • View deals or cash flows in a Cash Forecast Deal Set.
  • Optionally change individual Projected cash flows. For example, you can change the Amount of one particular cash flow, while retaining the original amount for all others. In this way, you can individually modify each cash flow.
  • Drill down to access (and optionally edit) full details of individual deals.

To add a new Projected cash flow to the set:

  1. In the grid menu under CASH FLOWS, select New.

    The New Cash Flow dialog box opens.

  2. Enter the required field values and click OK.

    The new Type M cash flow is added as a new row at the end of the grid, with Tx set to PC.

To edit a Projected cash flow in the set (Type M only):

  1. Select the row for the PC cash flow and in the grid menu, select Edit.

    The Edit Cash Flow dialog box opens.

  2. Change the required field values and click OK.

To convert a Projected cash flow to an Actual Cash Flow deal (Type M only):

  1. Select the row for the PC cash flow.
  2. In the grid menu, select Convert to Actual.

    The grid row is removed and an Actual Cash flow deal is created from that Projected cash flow.

To add or change a set of recurring Projected cash flows (Type R only):

  1. Enter or edit the settings on the Template, Recurrence, and Variation tabs as and if required.
  2. Check that Enable recurring cash flows is selected.
  3. Click Recalculate.

    Read-only grid rows of Type R (not M) are added and/or replaced to reflect the Template, Recurrence, and Variation tab settings.

  4. Click Save.

    Deal numbers are added and the grid flows become editable.

To set a Type R Projected cash flow to Manual (preventing recalculation as above):

  1. Select the row for cash flow.
  2. In the grid menu, select Convert toManual.

    The cash flow's Type changes to M.

To remove a cash flow from the set:

  1. Select the row for cash flow and in the grid menu, select Delete.
  2. Click OK to confirm.
