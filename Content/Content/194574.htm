Yield accruals for Indexed Linked Bonds

(First introduced in Release SP 6.7.1, August 2018.)

You can now select the YIELD accrual method when defining an Accounting Rule for Indexed Linked Bonds.

TIP: An indexed bond has its principal linked to a specific price index (for example, a Consumer Price Index). Such an instrument protects investors from changes in the underlying index. Face Value is indexed as well as the coupon's value, because the Coupon Rate applies to the indexed Face Value.

Implementation

To implement yield accruals for Index Linked Bonds, set up the following pages:

  • Rules in the Accounting menu
  • Security Deal

Accounting Rules page

On the Money Market & Securities tab, when creating or editing an Accounting Rules grid entry with an Instrument that has the Indexed Bond formula, you can now select an Accrual Method of YIELD CUM or YIELD EX (in addition to STRAIGHT LINE).

Security Deal page

If the applicable Accounting Rule for an Indexed Bond deal has the YIELD CUM or YIELD EX Accrual Method, the system calculates the amount to accrue by repricing the indexed instrument, measuring the changes in observed parameter values (Capital Accrual, Income Accrual, and Face Value Index Accrual) between the last accrual date and the repricing date.

History

Previously, only the STRAIGHT LINE Accrual Method was supported for Indexed Linked Bonds.

However, if an Accounting Rule included both Indexed Linked Bond instruments and other instruments that supported Yield accruals, you could have selected the YIELD Accrual Method. For the Indexed Linked Bond instruments this would have been updated to the STRAIGHT LINE accrual method on deal save. This override of the Accrual Method setting will no longer occur for Indexed Linked Bonds.
