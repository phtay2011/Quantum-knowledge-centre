Japanese Coupon-Bearing Bond Examples

The conventional Coupon-Bearing Bond issued by the Japanese Government has a fixed maturity date with a fixed annual coupon payable semi-annually in arrears. Coupons are typically paid on the 20th of the appropriate month.

Japanese Government coupon-bearing bonds can be divided into two categories for the calculation of simple yield and price:

  • Those with a residual life of less than one year use the ACTUAL/365 Day Count Convention.
  • Those with a residual life of more than one year use the NL/365 Day Count Convention (as ACTUAL/365 except that Feb 29s are excluded).

Defining the instrument

Use the Securities Instrument page* to define a typical Japanese Government Coupon-Bearing Bond, specifying the following key settings.

Header area:

  • Govt. Bond Template - select Japan - JGB to set the required defaults for a standard bond - see Japan-JGB Govt Bond template.

    NOTE: If you want to define the instrument manually without using the bond template, you must set Formula Timing to BOND PERIODIC and Modifiable to YES (on the Details tab).

Dates tab:

  • Income Day Count - select ACTUAL/365 if the expected residual life is usually less than one year; NL/365 if more (may be changed at deal entry).
  • Dirty Price Day Count - as for Income Day Count.

    TIP: You can achieve the effect of a Ryoha day count convention for the first period of the bond by defining an odd first coupon period using the Dated Date and First Coupon Date fields.

*NOTE: You could use the Money Market Instrument and Dealing pages to specify this type of deal, but those pages have no settings related to tradability - for example, an Issuer, Security Identification, and on on.

Example 1: Ten-year Bond with Residual < 1 year

Security Deal page settings and calculated values for this example are as follows.

Header area:

  • Instrument: your defined Japanese Government coupon-bearing bond instrument

Other Details tab:

  • Income Day Count: ACTUAL/365
  • Dirty Price Day Count: ACTUAL/365

Deal Details tab:

  • Face Value: JPY 1,000,000
  • Bond Price: 100.9160000
  • Coupon: 6.1
  • Deal Date and Settlement Date : 21 Dec 2015
  • Maturity Date: 20 May 2016

Calculated values (Deal Details tab):

  • Term: 151 days ( 21 Dec 2015 to 20 May 2016)
  • Discount / Premium: 9,160
  • Capital (and PPH): 1,009,160 (100.9160000)
  • Income (and PPH): 5,180 (0.5180821)
  • Settlement (and PPH): 1,014,340 (101.4340821)
  • Yield: 3.85, calculated as follows:

    Simple Yield (SY) = { C + [ (F - P )/ (Dk/365) ] / P } * 100

    ...where:

    • F (Face Value at par) = 100
    • P (Bond Price) = 100.916
    • C (Coupon) = 6.1
    • Dk (Term) = 151, including 1 leap day (29 February)

    SY = { 6.1 + [ (100 - 100.916) / (151/365) ] /100.916 } * 100

       = 3.85080303

Example 2: Ten-year Bond with Residual more than > 1 year

Security Deal page settings and calculated values for this example are as follows.

Header area:

  • Instrument: your defined Japanese Government coupon-bearing bond instrument

Other Details tab:

  • Income Day Count: NL/365
  • Dirty Price Day Count: NL/365

Deal Details tab:

  • Face Value: JPY 1,000,000
  • Bond Price: 101.9690000
  • Coupon: 5.0
  • Deal Date and Settlement Date : 28 Dec 2015
  • Maturity Date: 22 Dec 2025

Calculated values (Deal Details tab):

  • Term: 3647 days ( 28 Dec 2015 to 22 Dec 2025)
  • Discount / Premium: 19,690
  • Capital (and PPH): 1,019,690 (101.9690000)
  • Income (and PPH): 821 .0.0821917)
  • Settlement (and PPH): 1,020,511 (102.0511917)
  • Yield: 4.71, calculated as follows:

    Simple Yield (SY) = { C + [ (F - P )/ (Dk'/365) ] / P } * 100

    ...where:

    • F (Face Value at par) = 100
    • P (Bond Price) = 101.969
    • C (Coupon) = 5.0
    • Dk' (adjusted Term) = 3744 (3747 - 3 leap days, excluded with NL/365 Day Count)

    SY = { 6.1 + [ (100 - 101.969) / (3744/365) ] /101.969 } * 100

       = 4.71003520
