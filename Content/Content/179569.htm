Importing Stock Account Transfers

The Stock Account Transfers Deal Importer supports the following QID-format fields.

Mandatory fields

  • deal_dt
  • settle_dt
  • ccy
  • face_value
  • sectype
  • dealer
  • deal_set_id
  • external_ref_id
  • src_bankacc
  • dst_bankacc
  • src_cpdelacc (see Note below)
  • dst_cpdelacc (see Note below)
  • src_payment_category
  • dst_payment_category
  • src_swift_mt_format_id
  • dst_swift_mt_format_id
  • src_pay_method
  • dst_pay_method

Only one of the following is mandatory. If more than one is supplied, only the last one is used to look up the issue details, for populating all three of these Stock Identification fields in Quantum.

  • isin
  • cusip
  • other_ident

NOTE: The src_cpdelacc and dst_cpdelacc fields contain the Counterparty, Currency, Account Number, and Account Name values for the Source and Destination instructions, in that order and separated by commas. These values are equivalent to those specified in the Source and Destination panels on the Settlement Details tab in Stock Account Transfers, respectively.

Optional fields

  • ticket_no
  • src_swift_mt_format_id
  • dst_swift_mt_format_id
  • deal_set_id
  • src_pay_method
  • dst_pay_method
  • external_ref_id
  • comments

Example

(QID format)

EXEName=SAXFER
$DoMenu=mfnew
ticket_no=TEST001
deal_dt=10-Aug-2006
settle_dt=10-Aug-2006
ccy=AUD
face_value=234000000
sectype=ST Instrument A
deal_set_id=0
external_ref_id=0
pri_ident_filter_by=No Filtering
src_bankacc=CGO2[AUD
dst_bankacc=MyAccountABC
src_cpdelacc=ABC Singapore,AUD,987654321,LEGACYDEF1
dst_cpdelacc=CGO,AUD,CGOAUDDELACC,CGO AUD Delivery Account
src_payment_category=Standard Settlement Instruction
dst_payment_category=Standard Settlement Instruction
src_swift_mt_format_id=MT540_543_Supported_Fields
dst_swift_mt_format_id=MT540_543_Mandatory_Fields
src_pay_method=Austraclear
dst_pay_method=SWIFT
$DoMenu=mfInsert
isin=ABC987123123
cusip=ABC987123
other_ident=ABCABCABCABC
comments=Refer to Mr JK Smith at ABC Bank
dealer=IM Porter

(QXT format)

@TransferStock
*DealNumber=5362
$NEW
DealDestinationAccountNumber=Stock - 12390
DealSourceAccountNumber=8005201200
Currency=USD
DealDestinationCustodian=ABC NZ
DealSourceCustodian=ABC AU
SettlementDestinationCustodianSettlementDetail=ABC NZ
DestinationCustodianSettlementDetailKey=58
SettlementSourceCustodianSettlementDetailKey=202
Dealer=Bob
FaceValue=10000000.0000
FilterBy=NO FILTERING
FilterValue=EU
Instrument=Stock Transfer
SettlementDestinationPaymentCategory=STANDARD SETTLEMENT INSTRUCTION
SettlementSourcePaymentCategory=STANDARD SETTLEMENT INSTRUCTION
SettlementDestinationPaymentMethod=SWIFT
SettlementSourcePaymentMethod=SWIFT
SecurityType=SECURITY
SecurityIdentifier=USGB 8.5% 15/02/20
SettlementDestinationSwiftMtFormat=MT541_543_Mandatory_Fields
SettlementSourceSwiftMtFormat=MT540_543_Mandatory_Fields
Entity=Group Treasury
DestinationCounterpartyAccount=ABC NZ
SourceCounterpartyAccount=ABC AU
AccountPrimaryIdentifier=ISIN
$INSERT

  • The src_cpdelacc and dst_cpdelacc fields contain the Counterparty, Currency, Account Number, and Account Name values for the Source and Destination instructions, in that order and separated by commas. These values are equivalent to those specified in the Source and Destination panels on the Settlement Details tab in Stock Account Transfers, respectively.
  • This format uses paragraph marks to separate the input (obtained by pressing Enter at the end of each line).
