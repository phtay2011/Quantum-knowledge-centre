Currency Options Management Rules

Use the Currency Options tab to define the management reporting revaluation rules and methods for currency options.

Matching Criteria (Currency Options Management Rule)
  • Instrument - Defines the instruments to which these rules apply.

    The rules are established by row in the grid, and any number of instruments of this type may be recorded to define the applicable and appropriate treatment.

    TIP: Create a rule for ALL instrument types, and then a rule for each instrument type that requires different treatment.

  • Entity - Defines the entities to which these rules apply.

    TIP: Create a rule for ALL entities for the specified Instrument type, and then a rule for each entity or entity group within the entity structure that requires different treatment.

Settings (Currency Options Management Rule)
  • Perform Revaluation - Select whether revaluations can be performed for the instrument and entity you have selected in the previous fields.

    If you select YES, revaluations can be performed using the settings you define in the remaining columns. The following Option Types will have the relevant models associated with them:

     

    Option Type

    Model

     

    VANILLA

    BARONE-ADESI WHALEY

     

    ASIAN

    LEVY

     

    SINGLE BARRIER

    REINER and RUBINSTEIN

     

    DOUBLE BARRIER

    SIDENIUS

  • Interest Rate - For instrument type/s defined in the Instrument field, the interest rate defines the name of the Interest Rate Table used to source the revaluation rates/prices for the revaluation of a deal meeting the selection parameters.
    The rate type must be known in conjunction with the interest rate to locate an actual rate table and the rates to NPV of any accrual amounts referred to above.
    The rate table to source the FX rates to revalue the currency component is defined in the CCY Translation Rate Type field in the Management Control header. (FX rates do not have a rate type).
    If you are using a User Input Zero Curve or User Input Discount Factor, select the name used for the curve on the Zero Definition form.
    If you select a User Input Zero Curve or User Input Discount Factor for an instrument type that is not compatible with that type of curve, an error message is displayed when you perform a revaluation.
  • Rate Type - Defines the type of rate table used to source the revaluation yield rates for the revaluation of a deal meeting the selection parameters for instrument type/s defined in the instrument column.
    The rate type must be used in conjunction with the interest rate to locate a rate table.
    If you are using a User Input Zero Curve or User Input Discount Factor, select the rate type used on the Yields Rate or Zero Definition form.
  • Revaluation Method - (Only available when Perform Revaluation is set to YES) Select which method will be used to revalue Vanilla European options.

    Select the method by which the deals are to be revalued for Vanilla (or standard) European currency options types defined in the Instrument field for the rule you are defining as follows:

     

    Method

    Description

     

    SCHOLES

    Select this method if you want Quantum to post the Gain or Loss on revaluation.

    This method is a Black-Scholes derived model that values both Time and Intrinsic value of options to determine a theoretical market value.

     

    GARMAN-DUAL

    Select this method if you want Quantum to post the Gain or Loss on revaluation.

    This is the Garman-Kohlhagen modification of the B/S model for Currency Options. This particular version of the model requires the user to specify an interest rate curve for both currencies of the deal.

     

    GARMAN-FWD/DOM

    Select this method if you want Quantum to post the Gain or Loss on revaluation.

    This is the Garman-Kohlhagen modification of the B/S model for Currency Options. This particular version of the model requires the user to specify an interest rate curve for the domestic currency and the forward exchange curve.

    Quantum will use these and other details from the deal to compute the appropriate interest rate for the foreign currency.

     

    GARMAN-FWD/FOR

    Select this method if you want Quantum to post the Gain or Loss on revaluation.

    This is the Garman-Kohlhagen modification of the B/S model for Currency Options. This particular version of the model requires the user to specify an interest rate curve for the foreign currency and the forward exchange curve.

    Quantum will use these and other details from the deal to compute the appropriate interest rate for the domestic currency.

     

    IMPORTED

    Select Imported to use an externally derived revaluation amount for any currency option based on this instrument. When a Revaluation Analysis or position processing is run, Quantum will not revalue the deal, but will calculate profit or loss using an imported figure.

    NOTE: Even if Imported is selected, you will still need to complete the rate fields. The information they provide will be required if the currency option is knocked in or out.

     

    INTRINSIC

    Select this method if you want Quantum to post the Gain or Loss on revaluation.

    The intrinsic value is computed as the maximum of zero and the value of the option if it were exercised immediately.

  • Reval CCY - (Only available when Perform Revaluation is set to YES) Select the currency in which Quantum determines the Unrealized Gain or Loss, for revaluation purposes.

    The options are:

    • Base CCY - Select if you want to recognize the value of the Gain or Loss in the base currency. Quantum will revalue in premium currency, and then convert to base at current spot rate.
    • Non-Base CCY - Select if you want to recognize the value of the Gain or Loss in the currency as set in Currency Control. Quantum will revalue in premium currency, and then convert to dominant currency at current spot rate.
    • Premium CCY - Select this if you want to recognize the value of the Gain or Loss in the premium currency.
  • Volatility Rate - (Only available when Perform Revaluation is set to YES) Select the name of the Volatility Rate table required to revalue the instrument selected in the Instrument field.

    The rate type (selected in the Interest Rate and Volatility Rates fields) must be known in conjunction with the volatility interest rate, to locate an actual rate table.

    The selection you make in the CCY Translation Rate Type field determines what FX rates are displayed in the Rate Names lookup.

  • Risk Free Rates - (Only available when Perform Revaluation is set to YES) Select which type of curve should be used to calculate the risk free rate for the two legs of a currency deal (domestic and foreign). Choose one of the following curve types:
    • YIELD - when selected, Quantum:
      • Converts the rates at the interpolation points to the equivalent annualized rate.
      • Interpolates the annualized rates.
      • Converts the interpolated rate to the equivalent continuously compounded rate.
    • ZERO - when selected, Quantum:
      • Interpolates a rate from the selected zero curve.
      • Converts this rate from the compounding frequency (the compounding frequency of the zero curve) to the equivalent annualized rate.
      • Converts the annualized rate to the equivalent continuously compounding rate.

        NOTE: If there is no Zero curve, or the rate type is not defined. Quantum will not be able to calculate the risk free rate.
