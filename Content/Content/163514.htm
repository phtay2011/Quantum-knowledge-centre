Currency Procedures (Quantum, non-group)

On the Default perspective's Currency page:

To list Currencies (including Groups)

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Currency

  1. Click New.
  2. Enter a unique three-letter Ccy code and Name for the currency.
  3. In Type, select CURRENCY, COMMODITY*, or BOTH.

    NOTE: Commodity Swaps and Exposures use Commoditystatic data records instead of Commodity-type currencies (when your site is licensed for Commodities).

  4. On the Details tab:
    • Select a Days Convention to determine the basis for calculating and accruing interest for the currency - for example, 30/360 or ACTUAL/365.
    • Select an adjustment convention to use when a cash flow in this currency falls on a non-business day.
    • Enter a Days To Spot offset (a number of business days) that will be used to calculate Spot Dates for deals, cash flows, rate records, and settlements in this currency.
    • Optionally enter a Currency Symbol to be displayed for the currency - for example, $, £, €, ¥, and so on.
    • In the Rounding area, specify options and settings for the rounding of amounts and interest rates for deals, cash flows, rate records, and settlements in this currency.
    • In Non Business Days, select the standard non-business days of the week for the currency. Changing these modifies the system calendar for the currency.

      TIPS:

      • If required, you can override the defined business day rules during transaction entry.
      • For information on adding holidays, other non-business days, and "non-standard" business days, see Calendar Maintenance.
  5. (Optional) On the Bank Clearing tab, define the National settlement system code requirements.

    Also Include a CLS (Continuous Linked Settlement) Bank Clearing Field to make a currency CLS-eligible.

  6. (Optional) On the Euro tab, define the details of the Euro currency including convergence dates and translation rules. See Euro Currency/Group Procedures.
  7. Save your new Currency record.
  8. On the Currency Pair page, define the relationship between this and other currencies, and Save that Currency Control record. See Currency Pair Procedures.

    The system uses these settings to convert from one currency to the other, and when cross rates are calculated. Various limit and reporting processes will fail if these relationships are not described for all combinations of currencies.

To edit Currency details

  1. Click to list available records, or use the Search box.
  2. Select the record you want to edit.
  3. Edit the relevant fields.
  4. Save your changes.

To delete a Currency record

  • You cannot delete a record if that affects transaction, settlement, or financial data, and you cannot delete a group if it has members.
  • Deleting removes the record completely from the database. To prevent use of the record but retain it in the database, deactivate the record instead.
  1. Click to list available records.
  2. Select the record you want to delete (use the < and > buttons to page through, or type in the Search box).
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To deactivate a Currency

TIP: You cannot deactivate a Currency in active use.

  1. Click to list available records, or use the Search box.
  2. Select the record you want to deactivate.
  3. In the menu, select Deactivate. This action will not appear if the record is already deactivated.

To reactivate a Currency

  1. Click to list available records, or use the Search box.
  2. Select the record you want to reactivate.
  3. In the menu, select Reactivate. This action will not appear if the record is already activated.

To publish a Currency

TIP: All fully approved new or changed Currency records will be published automatically If the Currency option is selected (on the Operational Settings page under Publish).

  1. Click to list available records (or use the Search box).
  2. Select the record you want to publish.
  3. In the menu, select Publish Selected.

To publish all Currencies (including groups)

  • In the page menu, select Publish All.
