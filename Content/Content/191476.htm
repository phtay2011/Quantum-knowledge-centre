Menu and Navigation

Banner

The banner is available at the top of all application pages. It includes the following.

  • Site menu - enables direct access to application pages by category; displays either Analytics or Quantum items, depending on the current working perspective selected.
  • Notifications - click to display recent notifications to you, the user.
  • User menu - enables access to Log-off, Change Password, and Account Settings pages. The text is the short name (CurrentUserShortName) of the user who is logged-on.
  • Help button - click (or press SHIFT-F1) to open a Help panel within the page; click again to close the panel.

Site Menu

The site menu is available from every page for navigation to different parts of the system. Since it is Module Security-aware, only those menu items with appropriate access permissions are displayed.

Items within the site menu are organized by categories. Each site menu category has been assigned a different color, and pages accessed from within a menu category have the same background color for easy identification.

The site menu provides access to application pages.

  • Use the Default perspective to access Quantum and common pages, including any custom ones you are allowed to access.
  • Use the Analytics perspective to access Analytics pages.
  • Use the Adaptiv perspective to access Adaptiv pages (when Adaptiv is installed).

User Menu

Use the user menu (labeled with your user name) to:

  • configure your account,
  • view site and connectivity information, or...
  • change the working perspective between Quantum, Analytics and (where installed) Adaptiv by selecting Default, Analytics, or Adaptiv, respectively.

Use it to access the following pages:

  • Log Off
  • Change Password
  • User Account Settings

Page Menu

The page menu is located in the upper right-hand corner of all application pages.

To open a menu of actions available for the displayed data - click the page-level menu button .

  • Common ITEM-related actions are: New (to add a new item), Delete, Copy, and Save. For example, to delete an item - select it in the list, click the menu button , and select Delete from the menu.
  • Common VIEW-related actions are New (to add a new view), Configure (to edit a view), Copy, and Delete.
  • Common NOTES-related actions (including attachments) are New, Attach File, and View. These apply to Dealing pages, and to some Cash Management and Static Data pages (in the Default perspective only). For more details, see Notes and Attachments.
  • Common QUERY-related actions are New, Run, Configure, Copy, and Save. (Views also apply to the report results of Query runs.)

Notifications

Reporting and other specific processes (for example, Deal approval/rejection and Analytics Limit Breaches) will put notifications in the notification panel.

Notifications are user-based and can be either in progress or completed. A completed notification has four statuses: success, warning, information, failed.

Click the bell icon to display the list of recent notifications. The number next to the bell indicates the number of notifications that have not yet been dismissed. In this list:

  • Click on a notification to navigate to the page for the relevant process, displaying the output the notification relates to (for example, a report), and dismissing the notification from the list.
  • Dismiss a completed notification by clicking the x to its right.
  • Dismiss all completed notifications using the Dismiss All link.
  • Follow the Older Notifications link to open the Notifications page, which displays notifications that have been dismissed.

Notifications are also sent as e-mails to the address (if any) associated with the user account. The e-mail also contains a link back to the relevant page and/or output the notification relates to, which can be viewed while the email addressee is logged on.

TIP: You can configure email notifications for ALL users using the EmailNotificationsEnabledList key in the Web.AppSettings.config file.

Search

Search is available in the site menu, Browse windows, grids with a dedicated Search box, and in the Knowledge Center.

Basic search

The Search function in the site menu enables you to look for a menu item, data record, Instrument name, or Counterparty name from any page.

When you enter a short string (one or two characters), the Search will try to find items, usually words, starting with that string. If you enter a longer string, the Search will try to find phrases that contain that string. The type of search ("starts with" or "containing") is described at the top of the results.

Partial number search

When you enter a tilde (~) followed by a short string of numbers, or a short string containing numbers and only one decimal point, the Search will complete a numeric wildcard search. This type of search looks through numeric fields, such as the Amount column, but does not search through text fields, such as Bank Account.

If you include any other characters in a partial number search, the Search will instead complete a basic search.

Help Button

Click the help button in the banner to open a Help panel within the application page (or press SHIFT-F1). Repeat the action to hide the help panel again.

Follow any hyperlinks within the help panel to open the Knowledge Center in a new tab, for more detailed help, with navigation and search functions.

TIP: Help panel content may change when you move focus - for example by opening another tab within the page, or moving to a different group of fields.
