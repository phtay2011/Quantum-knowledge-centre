Data Version History

CashXplorer creates a Data Version History for each Data Version, in which it records the status of all data retrieval activity, including incremental data refresh.

Data Version history can be accessed from the History tab in Configure View.

The Data Version History is always available and is deleted only when the Data Version is deleted.

For each Data Version the following information is stored:

  • The Data Retrieval Parameters used to create the Data Version
  • The retrieval messages displayed in CashXplorer
  • The resolved Horizons, Bank Accounts, and Pools.

The Data Version History is read-only information, stored in six tabs that reflect the definition parameters:

  • Details
  • Messages
  • FX Rates
  • Accounts
  • Horizons
  • Pools

TIP: Data Version History is particularly useful if the Data Retrieval parameters have been amended. You can use the Data Version History to see which settings were used for the older Data Versions.
