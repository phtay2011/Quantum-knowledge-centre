Translation area of Euro tab, Currency

(Transaction-level currencies only.)

NOTE: If you are translating bank accounts using the Euro currency, some rounding may take place. Values may be rounded to the largest number of significant figures (14 decimal digits).

Fixed Exchange Rate to the Euro - the rate to be used to convert the IN currency to the Head Currency.

IN Currency Triangulation Rounding - select the number of decimal places to use for Triangulation Rounding. This is the method used during currency translation, based on a set of pre-defined Fixed Translation Rates and rules that round the translated amount to a set number of decimal places.

NOTE: When using Triangulation to convert between IN currencies, the result of the first step (converting the first IN currency to the Euro) may be rounded to not less than three decimal places.

OUT Currency Conversion Method - select a method to convert between OUT currencies and this currency - one of:

  • VIA THE EURO - to first convert the OUT currency to the Euro at the prevailing rate, and then convert the resulting Euro amount to this currency using the Fixed Exchange Rate.
  • CROSS RATE - to convert directly between the OUT currency and this currency, using a derived approximate cross rate.
