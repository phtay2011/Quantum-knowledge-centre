Minimum Value Formulas - Currency Options

Minimum value is typically defined with reference to a theoretical lower bound for option price, derived using some simple arbitrage arguments.

Alternatively, it can be defined as the value that results from applying an appropriate valuation model with an input volatility level of zero.

Quantum provides an appropriate formula for each type of Currency Option entered.

  • For most, minimum value is calculated using an extension of the intrinsic value calculation.
  • For European Style Average Rate Asian Options only, the appropriate valuation model (Levy) can be called with an input volatility level of zero and so the alternative approach is used.

Notation

The following notation is used in the formulas:

MV(c,p)

= Minimum Value of a call or put option

= 1 for a call
= -1 for a put

F

= Forward rate

S

= Spot rate

X

= Exercise or Strike price of the option

r

= Risk-free interest rate for the domestic currency

T

= Time to maturity of the option

R

= Rebate payable at maturity (if any)

European Style Vanilla Options

The minimum value for European vanilla options is determined by discounting the intrinsic value of the option, but with the spot rate replaced by the forward rate.

European Style Single-Barrier Options

The theoretical minimum value for barrier options differs for 'In' options and 'Out' options.

  • For In options, the assumption of zero volatility means that the barrier will never be breached between now and maturity date. The minimum value should therefore equal the present value of the rebate payable at maturity, if any.

  • Using similar logic, one can argue that un-lapsed 'Out' options will never breach the barrier between now and the maturity date, meaning that the minimum value is simply equal to the present value of the expected payoff at maturity. This value can be determined in the same way as we determine the minimum value for vanilla European options.
