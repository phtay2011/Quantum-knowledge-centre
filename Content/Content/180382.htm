CashXplorer Feature Summary

Flexible, User-definable and Multifunctional

  • Fully integrated with Quantum
  • External data import
  • User-definable queries
  • Multiple Views per query
  • Pooling of groups and transactions at any entity level
  • Cash concentration and target balance controls
  • Saved data versions that can be viewed or exported at a later time

Extensive Data Selection

  • Choice of flows:
    • Actual Cash Flows
    • Projected Cash Flows
    • Bank Flows
    • External Cash Flows
  • Choice of opening balances (Start Date and Previous Day, with fallbacks):
    • External Available Balance and Floats
    • External Bank Value Date Balance
    • External Current Available Balance
    • External Ledger Balance
    • Internal Bank Value Date Balance
    • Internal Reconciled Balance
    • Internal Unreconciled Balance
  • Currency translation:
    • Base currency
    • User-definable cross currency
    • Group currency (including Euro)
  • Drill-down on cash flows to review and update the underlying transactions
  • Direct entry of FX, Money Market, Security, Bank Account Transfer, Actual Cash Flow, or Projected Cash Flow deal

    Initial deal settings can be pre-filled based on the selected balance and Instrument (or bank account), as applicable.

  • User-definable totals at grouping level

Customizable Query Display

  • Data filters toggled on and off
  • Concentration flows toggled on and off
  • Multi-currency views
  • Selectable amount rounding methods
  • Collapse and expand rows in the display
  • Increase or decrease font size

Data Flexibility

  • Manual query adjustments
  • One-click Data Version refresh
  • Multiple Data Versions and Views per query
  • Extensive Data Version history
  • Data Version Summary and Refresh Summary

Asynchronous processing

The Run Query, Refresh Version, and Delete Data Version CashXplorer processes run asynchronously.

This means that your query runs to Refresh, Create, and Delete Data Versions do not hold a connection open to the server until they finish, which enables you to perform other CashXplorer tasks while they run. Notifications let you know when runs are completed (see below).

Notifications

Notifications are produced for the Run Query, Refresh Version, and Delete Data Version CashXplorer processes.

For more details, see CashXplorer Notifications.

Application Support

  • Export to Excel, CSV, XML, or PDF
  • Print (via PDF)
  • Full text search
  • Archive, restore, and purge Data Versions
