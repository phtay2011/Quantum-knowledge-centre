Configuring CashXplorer Views

When you have configured a Query, you can configure different Views to apply to the results (Data Versions) obtained from running the query.

Views describe how the results of a query are displayed. You can create three types of View in CashXplorer:

  • Forecast - used for comprehensive cash flow forecasting, they display cash flow information from the Start Date of the first Horizon to the last date of the last Horizon. They can also display system-defined Horizons that cover the period up to and including the Previous Day. Grid columns display Horizons.
  • Cash Position - generally used as a tool for summarizing the current day position, they display a single Horizon only. Grid columns display account-related information.
  • Summary - also used as a tool for summarizing the current day position, they display a single Horizon only and focus on account-related information, including totals and all balances (including Intraday and "below the line" balances). Summary Views also provide grouping of bank account related attributes. Grid columns display account-related information.

    Use a Summary View instead of a Cash Position View if you wish to view summarized account information for a large group of accounts. It has a greater capacity due to its structure: account-related groups are organized into rows, and all flows, balances, and totals are held within columns.

Use the View Configure link to open the Configuration area, and use the tabs as follows.

  • Display tab - to:
    • change the displayed currency and rounding for amounts,
    • display or hide Intraday bank flows, and
    • display or hide cash flows that are Intraday Matched.
  • Attribute Filters tab - to show or hide cash flows depending on data values.
  • Pool Filters tab - to show or hide pools depending on data values.
  • Export tab - to define a header for exported data.
  • Row Groups tab - to add, edit or delete row groups.
  • Columns tab - to define which totals are displayed on the View for each Group you have selected on the Row Groups tab.
  • Captions tab - to define captions for displayed cash flow totals.
  • Drill Down tab - to define data columns to be displayed in the Drill Down Details dialog box.
  • Properties tab - to identify the View.
  • History tab - to view the following details applicable to the Data Version:
    • Query configuration parameters,
    • retrieval messages,
    • FX rate conversions used, and
    • information for each resolved account, horizon, and pool.
