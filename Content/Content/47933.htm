Valuing Simple European Interest Rate Options

This topic outlines the steps required to value a cap and a floor, using an example that is based on the deal details presented in the table below.

The process of valuing caps and floor comprises the following:

  • Determine the caplet and floorlet rate details

    Caps and Floors generally consist of a number of caplets or floorlets, each of which is valued independently. Before calling the Black's model to value the options, the routine must first construct a deal-map to store all of the relevant reset and payment dates, before deriving the appropriate forward rates from the supplied zero curve. This process is broken into two parts:

    • Determining Reset and Payment Dates
    • Determining Discount Factors and Forward Rates
  • Determine the caplet and floorlet values

    All of the required valuation parameters that are not directly provided by the user have been determined in the previous section. Valuing the caps and floors is now just a matter of valuing each of the caplets and floorlets separately, and then adding those values together. See Applying the Blacks formula to Simple European Interest Rate Options for the specific formulas used.

    NOTE: In the implementation of these formulas, the tenor (length in years) of each interest period is based on the number of days in the period calculated using the selected accrual basis. This is not necessarily equal to the actual number of days.
