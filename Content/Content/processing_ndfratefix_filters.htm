Filters - Rate Fix NDFs

All filters are displayed by default and are summarized below. You can change the filters that are displayed. See Create/Change Query Parameters.

Fix Date - the rate fixing date for the Non-Deliverable Forward (NDF) FX deals; a valid business day

Rate Type - the FX Rate type you want to select, or leave blank if no single Rate Type is required

Instrument - a specific instrument (to select only deals based on that instrument) or ALL to select all instruments matching the FX or Commodity selection

Entity - a specific entity (to select only deals for that entity) or ALL

Counterparty - a specific counterparty (to select only deals with that counterparty) or ALL

Dealer - a specific dealer (to select only deals struck by that dealer) or ALL

Relationship - a specific relationship (to select only deals struck under that relationship) or ALL

Strategy - a specific strategy (to select only deals struck as part of that strategy) or ALL

Hold Constant - select which side to hold constant when the system recalculates one side of each currency pair to reflect the new Forward Points or Commodity Forward rate entered for the Extend action (Currency 1 or Currency 2)

Currency 1 - the common currency for all FX deals to be listed (the deals to be extended must have one currency in common)

Currency 2 - one or more contra currencies of the FX deals (if Commodity selected above, select commodity currencies here)

TIP: Both Currency selections are independent of the Buy or Sell side of the FX deals.
