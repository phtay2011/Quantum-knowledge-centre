Cross Currency Swap page

Use this page to create and maintain Cross Currency Swap transactions. A Cross Currency Swap involves an exchange of interest streams in different currencies, of which at least one is at a Floating rate.

TIPS:

  • This should not be confused with the Dual Currency option for a Swap leg, which applies only when, for that leg, the Formula is BOND and the Face Value is to be paid (or received) in one currency, the coupons in another.
  • For swaps involving a single currency, use the Interest Rate Swap page.

The fields on this page can vary, depending on the Formula and Timing of the Instrument you select (and on other key settings). Many field values are defaulted from settings for the Instrument. You can choose whether to keep or change those values.

CAUTION: Do not use Swap Floating legs with the Principal & Interest formula. You are not prevented from specifying Swap instruments and deals with this combination, but deal payment amounts will be incorrect, from the first rate set on.

To add a new deal - Click New, fill in the data fields, and click Save.

To list deals - Click the button to open the Browse list. (Click the Configure link to change list properties.)

NOTE: You cannot edit financial data if actions have been approved on the deal.

To change a deal's details - Select it in the list, edit the relevant data fields on the required tabs, and click Save.

To add, view, or delete Notes and/or attached files - Use the Page menu Notes functions. See Notes and Attachments.

To reset the current deal (CAUTION: May necessitate deal reprocessing; removes any customization) - In the page menu, select Reset; and click Yes to confirm. See Resetting a Deal.

To delete the current deal (CAUTION: Takes immediate effect and cannot be undone - even if subject to Approval/Rejection) - In the page menu, select Delete; and click Yes to confirm. See Deleting a Deal.

To review changes or rollback a deal: In the page menu, select Audit History to view all the actions performed on a deal during its life. You can rollback a deal to a previous deal image from the Audit History dialog box.

To import / export deal data (Subject to Operational and User settings and permissions) - Click the button to open the Browse list. Select the row if exporting. In the Browse window menu, select Import or Export.

To unwind a deal - See Unwinding a Deal.

NOTE: Customizing a deal where you make a change to the Deal Details tab and the Deal Map tab without saving in between will cause an incorrect stamp in the Action Type field in the Audit History. Always save when swapping between these two tabs.

For details of a tab or data item, click on it when the Help panel is displayed. For details of other page menu functions (Deal Allocation, Rollback, and so on), follow the link below.
