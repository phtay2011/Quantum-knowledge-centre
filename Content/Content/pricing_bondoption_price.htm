Price tab

This tab includes the following items.

Position Date - Enter the date at which the deal is to be valued. This determines the time to maturity for a deal. Analytics provides Today as a default, but this may be changed.

Rates Date - Enter the date of the rate curves to be used in the deal valuation calculation. Typically this will be the same as the Position Date. The rate curves may be viewed on the Rates tab once calculation is completed. Analytics provides Today as a default, but this may be changed.

As At Date - Enter the date of the rate image "as at" a particular point in time. Analytics provides today as a default, but you can change this to view and use a rate image as it existed at an earlier date.

Valuation - read-only and set to Black (European) if Style is European; otherwise select the appropriate valuation method or model.

Spot Price, Forward Price (available if Style is European) - Enter the Spot and/or Forward prices. If left blank the value will default from market rates. The associated Cash Price is displayed in the adjacent field in each case.

Discount Rate (available if Style is European) - Enter the Discount Rate (continuous rate). If left blank the value will default from market rates.

Steps per Coupon (available if Style is notEuropean) - Enter the number of tree steps per fixed coupon period in the valuation model.

Volatility - Enter the Volatility. If left blank the value will default from market rates.

Calculate (read-only and set to Price if Style is notEuropean) select:

  • Price - to calculate value,
  • Volatility - to solve for the Volatility given a specified value, or
  • Strike - to solve for the Strike given a specified value.

Value (available only if Calculate is Volatility or Strike) - Enter the value to be used to determine the Calculate variable (Volatility or Strike). Analytics uses an iterative technique, until the value of the deal meets the value entered in this field.

Price button - Use this button to run the selected calculation and populate the grid values.

Bond and Bond Option grids - After you click the Price button, these grids display the calculation result totals for your chosen settings, giving the Position value and either the Per Unit value (for European style) or PPH value (price per hundred - for American and Bermudan styles).
