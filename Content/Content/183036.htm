Supplied (Seed) Confirmation XML Data records

NOTE: Prior to Release 2020.2, this formatting data was available only in separate XML-formatted files (not within the Quantum database).

Quantum provides the following Seed XML Data records (previously Template files) for specifying and building each type of Confirmation message that may be output to the SWIFT, Finastra (Misys), and Concord external matching services - by applying the respective External Method (or the less restrictive Electronic External Method).

NOTES:

  • The Seed XML Data records are available via the XML Data page (new from Release 2020.2).
  • To implement a set of XML Data records for electronic confirmations, you associate the Confirmation's Electronic Confirmation Format Rule with the appropriate XTC Configuration record.
  • The output Confirmation messages produced using these XML Data records are NOT themselves in XML format. They remain in the text format required by the external system. Only the content of the records used to build them are in XML.
  • A separate set of XML Data records is used for manually created Free Format MTn99 messages (for both EFTs and Confirmations) - see below.
