Enhanced Reporting in Hedge Accounting

You can now use standard Quantum Reporting features in Hedge Accounting, enabling you to:

  • create customized queries by adding filters from other tables in the Hedge Accounting Query,
  • include fields from other tables in the Hedge Accounting Views, and
  • configure the View using Calculated Columns.

Hedge Accounting Query

  • The Deal Number and Transaction Detail filters have been replaced with filter lookups that enable you to add fields from other tables.

  • A new check box, Include opening balances, has been added to the Options area.

    When checked, the opening balance record is obtained for all Hedge Relationship deals where:

    • HR Start Date < Reporting Start Date, and
    • HR End Date >= Reporting Start Date
  • Each Query run is stored in the list of Previous Runs.

Hedge Accounting View
