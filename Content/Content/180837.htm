Deleting Adjustments

This topic covers deleting adjustments in either Prior Day or Intraday Matching.

NOTES:

  • In Prior Day or Intraday Matching, you can delete only manually added Balance statements.
  • To delete imported statements (with their associated bank flows) or external cash flows, see Deleting Imported Bank Flows and Cash Flows.
  • To delete other types of adjustment, you must use the applicable Cash Management page (Actual Cash Flow, Projected Cash Flow, Cash Transfer, or Account Adjustment).
  • These functions are not available if Resource Security applies and you are not Exempt or do not have Write access to the Bank Account.

To delete an added statement

  1. On the Overview tab, select an account and in the grid menu, select View Statements.

    TIP: You can also select this function in the Matches grid menu on the Matches tab (the account has already been selected).

    The Bank Statements dialog box is then displayed, showing any statements for the account that have a Statement Date within the active Reconciliation period.

  2. Select the required added statement in the table (added statements do not have an Imported date).
  3. Click Delete.

    A Delete Bank Flow / Statement dialog box is then displayed, showing only a Yes and No button.

    NOTE: If instead the dialog box has Delete Statement, Delete Entire File, and Cancel buttons, you have selected an imported statement. Click Cancel and try again from Step 2.

  4. Click Yes to confirm the deletion.
