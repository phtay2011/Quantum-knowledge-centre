ETO (Exchange Traded Option)

Use the ETO (Exchange Traded Option) Rate page to store inverted yields and prices against Exchange Traded Option contracts. The system links the rate to the relevant contract information.

The combination of Contract/Exchange, Rate Type, Currency, and Date uniquely identifies this type of rate record in the system, and is displayed above the Rate Header.

NOTE:

  • When entering rates using the Offer/Bid format, always include the big figure for both the Offer and the Bid amounts.
  • Always enter rates in Offer/Bid format (not Bid/Offer).
  • You do not have to enter a rate for all contract months and Strike Prices.
  • The price input for Options is always in premium points.

For more information, see Futures closeout formula.

Rates area, Exchange Traded Option Rate

Use this area to enter and/or change the individual Exchange Traded Option rates for set dates.

To add a new interval definition from scratch:

  1. In the grid menu, select New. The New Rate dialog box opens.
  2. Enter the required values (see below) for the new time band and click OK. The new interval definition is added to the grid.

To add a new interval definition by copying an existing one:

  1. Select the row you want to copy and in the grid menu, select Copy. The Copy Rate dialog box opens.
  2. Enter a new Interval, change other values as required (see below), and click OK. The new interval definition is added to the grid.

To edit an existing interval definition:

  1. Select the row you want to edit and in the grid menu, select Edit. The Edit Rate dialog box opens.
  2. Change values as required (see below), and click OK. Your changes will show in the grid.

To remove an interval definition:

  1. Select the row you want to remove and in the list menu, select Delete. The Confirm Delete dialog box opens.
  2. Click OK. The interval definition is removed from the grid.

Field Descriptions

Contract Month (read-only) - the contract months defined in Futures Control for this Contract.

Contract Type (read-only) - CALL or PUT, as defined in Futures Control for each contract month.

Strike Price (read-only) - the strike price for each contract month, as defined in Futures Control.

Rate Input (Offer/Bid) - To enter an interest rate as a spread, enter the Bank's Bid rate, a slash (/), then the Bank's Offer rate. Or enter a single rate without a slash. You can enter the rate as a positive or a negative number (which allows you to use a negative curve in Revaluation Analysis).

Bank Offer (read-only) - the rate you entered to the left of the slash in Rate Input, or the single actual rate.

Bank Bid (read-only) - the rate you entered to the right of the slash in Rate Input, or the single actual rate.

High / Low / Close - the highest, lowest, and closing rates at which the rate is traded on a specific day.
