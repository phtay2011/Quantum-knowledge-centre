Accounting Calendar page

(Available only for Money Market instruments when Accrual Method is Amortized cost.)

Use this page to define a schedule of dates for calculation of amortization amounts in Money Market and Securities deals, when the applicable Accounting Rule has Accrual Method set to Amortized cost and a predefined Accounting Calendar is specified.

TIP: If an Accounting Calendar is NOT specified in the applicable rule, the deal Coupon dates will be used.

To add a new record - Click New, enter a unique calendar Name, add Dates entries as below, and Save.

TIP: This name will populate the Accounting Calendar drop-down list in the Accounting Rules tab.

To list records - Click to open the browse window. (Click the Configure link to change browse list properties.)

To change a record's details - Select it in the browse list, and edit the date detail entries as below:

  • To add a single date entry - In the grid menu, select New, and in the dialog box, enter the date and click OK.
  • To add a schedule of date entry - In the grid menu, select Generate Dates; in the dialog box enter the following settings; then click OK.
    • From and To - Enter the dates between which to generate scheduled dates, according to the Repeat, Using, and Start settings.
    • Repeat - Select how often dates are generated: MONTHLY or 4 4 5 (to divide the year into a series of 4-week, 4-week, and 5-week periods, in that order (making 12 periods per year in that repeating pattern).
    • ACTUAL or MONTH END (available when Repeat is MONTHLY) - Select when you want the cycle to repeat from: ACTUAL to generate dates based on the Start Date, or MONTH END to generate dates based on the end of the From date's calendar month.
    • Start Date (editable only when ACTUAL is selected) - The first date of the cycle - that is, the date of the first accrual calculation.
  • To edit an existing entry - Click an entry value in the grid row; in the Edit window, edit the date as required; then click OK.
  • To delete an existing entry - Select the grid row, and in the list menu, select Delete.
  • To keep your changes - click Save.

To delete an entire record (not recommended; Deactivate instead) - Select the record in the browse list; in the page menu select Delete; and when prompted, confirm.

To deactivate a record - Click , select the record, and in the page menu select DeActivate. You can later reactivate it in the same way.
