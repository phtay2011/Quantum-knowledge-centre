Effectiveness testing and posting values - Currency Options (IFRS 9)

When hedging only the intrinsic value of currency options under IFRS 9, the Hypothetical Derivative method is used to calculate the effectiveness and posting values for the hedge relationship with position processing. The posting values and posting accounts depend upon how the critical terms of the hedge relationship align at inception of the hedge relationship.

Critical terms match exactly at inception

The fair value changes in the time value of the option are accumulated in OCI first and undergo subsequent treatment, depending on the nature of the hedged transaction as follows:

  • For fair value hedges, the time value is separated from the intrinsic value and posted to OCI.
  • For cash flow hedges, the entire value (that is, intrinsic + time) is posted to OCI.
  • For time period related hedges, the time value in OCI is amortized* to P&L on a straight line basis.
  • For transaction related hedges, the time value in OCI is reclassified to P&L; if the hedged transaction results in the recognition of a non-financial item, the amount becomes a basis adjustment.

Critical terms do not match at inception

Under the IFRS 9 accounting standard, if at inception of the hedge relationship, the critical terms of the option and the hedged item do not match, the entity needs to determine how much of the actual time value included in the premium relates to the hedged item. In Quantum the aligned time value is determined using position processing.

NOTE: Where the Actual and Aligned Time Value differs, then a hypothetical derivative should be used to represent the hedged risk; the hypothetical deal represents the aligned time value in such cases.

Total Time Value of the hedges > Total Time Value of the hypotheticals

The Aligned TIme Value = the total Time Value of the Exposures; The account postings from position processing are as follows:

  • The change in the Aligned Time Value is posted to the OCI for the entity.
  • The difference in the change in the time value of the Exposures and the change in the time value of the actual (HEDGE) is treated as an excluded component and remains in P&L.
  • The time value of the Exposures is amortized* to P&L for time period related hedges.

Total Time Value of the Hedges < Total Time Value of the Hypotheticals

The Aligned TIme Value = the total Time Premium of the Hedges; The account postings from position processing are as follows:

  • The lower value of the change in the time value of the Hedges and the change in the time value of the Exposures is posted to OCI Balance (Hedging Costs) for the entity (Posting Rules, BALANCE SHEET tab).
  • Any remainder of the change in fair value of the hedge time value shall be treated as an excluded component and remain in P&L.
  • The time value of the Hedges is amortized* to P&L for time period related hedges.

* Time period related hedges must have the Amortize hedging costs (time value) option selected for the hedge relationship.
