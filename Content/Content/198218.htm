JSON File Format

You can import data for a deal in a JSON-formatted file.

You can export a data file that contains deal information in JSON. An exported JSON-formatted file contains the properties associated with the deal. Find the option "JSON" in the Browse grid menu under Export.... . This opens the Export dialog box and lets you set up the export process.

You can use JSON formatted files to create, update, or delete supported deal types or static data, as well as to invoke supported actions.

TIP: To create a JSON file, either export from another deal in Quantum, use a third-party application, or use the Quantum Web API.

Deal Map items

For some deal types, a JSON-formatted file can be used to update or add items on the Deal Map.

NOTE: A JSON-formatted file cannot be used to create a new deal and customize its deal map items at the same time. The deal needs to be saved first and then customized via an update or add action.

Example

For many clients, the main user case is to be able to create and deactivate users, but also assign and reassign users to groups.
