Final Period Pricing area, Receive Pricing tab, Currency Swap Instrument

(Fields in this area are available only when Formula Timing is BOND PERIODIC or ZERO COUPON DISCOUNT for this Swap leg.)

Enter or select the following default settings for this leg of Swap deals using this instrument.

Method - select a pricing calculation method to apply to the final period for this leg; one of:

  • MONEYMARKET - to calculate the discount as 1/[1+(yield*drem/dbas)], where:

    yield = the Yield percentage divided by 100,
    drem = days remaining, and
    dbas = days basis (the denominator of the Day Count convention - for example, 360 for ACTUAL/360, and so on);

  • MONEYMARKET LINEAR - to calculate the discount as 1/{1+[(yield/cpy)*(drem/dfp)]}, where:

    yield and drem are as above,
    cpy = number of coupons per year, and
    dfp = number of days in final period;

  • COMPOUND YIELD - to calculate the discount using the ISMA Bond formulaOR, where coupons are EXACT,the True Yield formula.

Day Count - select a Day Count Convention for calculation and accrual of interest in the final coupon period for the Swap leg, to determine how days and year-days (the number of days per annum) are handled.

Interest End Date - select the basis for when the interest ends, for the Swap leg; one of:

  • CASHFLOW - to specify the Maturity Date, adjusted if required,by applying the Calculation Date business day convention (Pay/Receive Details tab), or
  • MATURITY - to specify the actual Maturity Date.

No PPH rounding - select or clear this option to (respectively) ignore or apply PPH rounding on Capital and Settlement in the final period.

Start final period early - select this option to start the final period start from the Ex Dividend date of the penultimate coupon; clear the option to start the final period as normal.
