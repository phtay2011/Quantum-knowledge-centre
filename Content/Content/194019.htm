Commodity Exposure Deals

See also: Commodities in Quantum Six

Use the Commodity Exposure Deal page to create and maintain details of your organization's exposure to forecasted purchases and sales of commodities. No trading applies, so no cash flows nor General Ledger entries are generated. Quantum Six enables you to create and maintain two types of Commodity Exposure:

  • Fixed - deals with a fixed forward price for the commodity throughout the trading period, or...
  • Floating - deals with a variable future price within the trading period.

The workflow and functionality for Commodity Exposure deals is similar to that of other non-transactional Dealing modules (those with no cash flows). The Static Data records defining the deal are similar to those for Commodity Swap deals. See Static Data Setup for Commodities.

No accounting nor defaulting rules apply to Commodity Exposure deals, but you can apply Approval and Authorization rules for Deal approvals. Deal approvals are optional, as are Static Data approvals for Commodity Exposure instruments.

While there are no Analytics Pricing pages as such for Commodity deals, they are supported in Analytics Position queries - for example in filters and Columns definitions. See Reporting for Commodity Swaps.

To approve or reject single deals

  1. Open the Commodity Exposure dealing page.
  2. Use the Approve or Reject menu functions.

NOTE: Bulk Deal approval (using the Deal Approvals processing page) is also supported.

To fix the price

  • To manually fix the price of one deal, or one period in a deal, open the Commodity Exposure Deal page, select the deal and period, and in the page menu select Price Fix.
  • To use Analytics to automatically calculate a price, and fix the price for bulk floating exposure deals, open the Processing > Commodity Price Fix page. Follow the instructions on that page.

To create periods / strips

Commodity deals allow you to create strips or "periods" within the deal.

  1. Open Dealing > Commodity Exposure.
  2. Select New to create a new deal.
  3. In the Deal Details tab, find the Periods area and select New.
  4. Add as many periods as you need, entering the Start Date, End Date, Quantity, Fix Date, and Payment Date for each period.

NOTE: In Dealing > Commodity Swaps you can also generate periods with a certain frequency, such as Quarterly.

To replace all data for periods*

  1. Open the deal from the Dealing menu.
  2. Open the Deal Details tab.
  3. In the Periods grid menu, select Bulk > Replace. The Bulk Replace dialog box opens with all data highlighted.

  4. Use standard keyboard actions to replace all or some data:
    • Click in the dialog box – to stop highlighting
    • Ctrl + C - to copy the highlighted data
    • Ctrl + V – to paste new data / replace highlighted data
  5. Use the Quantum format for dates and numbers:
    • 1,000.00 (comma, point)
    • 06/14/2019 (MM/dd/yyyy)
  6. Select Replace to save the new data.
  7. View the updated data. The system sorts the periods chronologically by start date or payment date (fixed exposure).

* This applies to both fixed and floating commodity exposures.
