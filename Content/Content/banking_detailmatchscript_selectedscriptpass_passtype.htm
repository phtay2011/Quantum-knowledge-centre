Pass Type, Detail Match Script page

Select one of the following pass action action types.

  • Match - to match bank flows with cash flows.
  • Bankflow Only - to scan only bank flows and create one automatic adjustment for each individual bank flow.

    TIP: This pass type is a flexible way of creating Quantum deals from imported bank statement details, for reconciliation and other purposes.

  • Bulk Bankflow Only - to scan only bank flows, and create one automatic adjustment for each Bank Value date. The new adjustment is matched with all bank flows contained within its Value date.
  • Zero Balancing - to search for valid Zero Balancing (ZBA) bank flows for the account, and, when one satisfying the pass criteria is found, then search cash flows for a single matching Account Transfer item with the same Value Date and amount (and not already matched), and:
    • If one is found, assign a new Detail-level Match ID and the specified match level to the two items. If (and only if) the Bank Value Date of the matching cash flow differs from that of the ZBA bank flow AND the Update bank value date on cash flow when matched Operational Setting is selected: change the cash flow's date to the bank flow's.
    • If one is not found, generate a Bank Account Transfer (with the Instrument, Ticket Number, Comments, Source Entity, and Destination Entity specified for the pass) and assign a new Detail-level Match ID and the specified match level to the Account Transfer cash flow (for the account being processed) and to the ZBA bank flow.

      The generated Bank Account Transfer's Bank Value Date is also set to that of the ZBA bank flow (for both sides of the transfer), adjusted if necessary according to the Non-Business Day setting for the specified Bank Account Transfer Instrument.

    The account on the other side of the transfer will then have a new unmatched cash flow of the transferred amount.

    TIP: Where you are using Zero Balancing, you would usually run this type of pass first.

Field criteria are more restricted for Bankflow Only and Zero Balancing passes. When changing to these pass types, a message will appear, notifying you that any conflicting criteria will be removed.
