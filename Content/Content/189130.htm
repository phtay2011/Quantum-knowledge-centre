Closeouts Using Straight Line Accounting

If an instrument is set up to use Straight Line accounting, Quantum cannot use a simple difference in price between the open and closed deals to calculate the Gain or Loss on closeout. It must also consider the accruals on the open deal. If calculated on a Straight Line basis, these affect the book value of the open deal.

For these instruments Quantum uses an Adjusted Book Value = Face Value + (-) Premium (Discount) + Incremental Accruals (NOTE: For ILB with amortization Adjusted Book Value = Face Value + Index + (-) Premium (Discount) + Incremental Accruals - Indexed Amortization - Capital Reduction)

where Incremental Accruals consist of:

  • Full Capital Accrual - up to the date when the closeout occurs (Settlement date)

    plus

  • Coupon Accrual for the coupon period in which the closeout occurs - up to the date when the closeout occurs (Settlement date)

When deals that use Straight Line accounting are closed out, Quantum:

  1. Calculates the Adjusted Book Value of the open deal on Settlement date (the date when the closeout occurs).
  2. Calculates the price (book value) of the closed deal.
  3. Calculates the difference in price between these.
