Applying Resource Security to Queries and Reports

When you enable Query resource filtering for Resource Security:

  • you can control user or user group access to public queries and the actions they can perform on associated pages (for example, Run, Configure, Save, Delete, and so on),
  • Module Security and Resource Security work together to allow or restrict actions within queries and reports, and
  • it applies to all Reporting menu queries and the following Processing and Accounting menu queries:
    • Confirmations and Settlements,
    • Deal Approval and Settlement Approval,
    • Account Interest and Account Statements,
    • Accruals,
    • Position,
    • Futures Margin, and
    • Currency Translation.

Resource Security Permissions

The Resource Security permissions work as follows for queries:

  • No Access - The query will not be available for selection.
  • View - The query can be selected and previous runs can be opened and displayed.
  • View/Update - The query can be selected, previous runs can be opened and displayed, and the query can be run.

NOTES:

  • The Resource Security permissions can be applied for a user or user group for ALL public queries in ALL modules (that is, Default access), for ALL public queries in individual modules, or for individual public queries. Access settings at specific levels take precedence over those at a general level.
  • Confirmations, Settlements, Deal Approval, and Settlement Approval only have the No Access and View/Update permissions for selection.

Query Browse List

  • The browse list displays all queries the current user has access to.
  • A folder icon is displayed next to queries that have previous runs available, indicating that users can drill-down and select the appropriate run for these.
  • Reports of scheduled queries, which are set to run for latest, can be selected from the browse list. These reports are prefixed with Latest: in the Summary column of the browse list, and are automatically refreshed with their latest run when it becomes available.
