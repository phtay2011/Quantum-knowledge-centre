How to (CDI)

  • Invest in a CD. See CD Money Market Instrument.
  • Create a Reval curve with Annual Par rates.
  • Set up a Swap Instrument Type for the CD/CDI swap.
  • Set up a basis for the Swap rate.
  • Create a SWAP Deal to swap the CD into the CDI.
  • Use the Rate Setting Deals page to rate set multiple CDI legs at the same time.

Details of these tasks are given in this section.

Create a Reval Curve

  • On the Interest Rates page, set up a Reval curve with nominal annual par rates.

    NOTE: For the purpose of revaluation only, these are converted to monthly rates.

Set up the Swap Instrument

  • On the Interest Rate Swap Instrument page, create an instrument, called for example CD/CDI, with the following settings:

Set up the Basis for the Swap Rate

  1. On the Rate Name page, select the default basis for this Swap Rate.
  2. Select the Rate Usage Basis option.

Create a CD/CDI Swap

Enter a Swap deal as usual, but take note of the following fields.

  1. Set the Day Convention field to ACTUAL/252BD.

    Only business days will then be calculated for the Yield - CDI formula (no interest is earned over holiday or weekends).

  2. Use the Discount field to invest in a percentage of the CDI rate (for example, an investor may invest/borrow 85% of the quoted CDI rate). A 20% withholding tax may apply.
  3. Set the Pay/Receive Frequency fields to AT MATURITY.
  4. Set the Rateset Frequency field to DAILY.

Automate the CDI Updates

You can manually enter the new rate from the Yield Rates page. However, you can automate the process of updating all the CDI deals by setting the new CDI rate for all CDI swaps from the Bulk Renegotiation Selection form.

  1. On the Build Renegotiation Selection form's Preview tab, check the deals that will be updated.
  2. From the Rate Set tab, check that the rate is correct.

TIP: Double-click the Amount column in the Rate Setting form to display the Detail of Rate Set Period form. If you are using the Yield-CDI formula, the Detail of the Rate Set Period form displays the Accumulated Principal that holds the amount to which the rate was applied.
