Calculating Market Values, Swaption Deal

Use the Price tab (inside the Details/Option tab) to calculate market values for the deal, as follows.

  1. Change the Position Date, Rates Date, As At Date, Currency, and/or Exchange rate settings, as required.
  2. If Style is American or Bermudan, select the Valuation model: Black-Derman-Toy 1, Black-Derman-Toy 2, Ho-Lee,Hull-White, or Black-Karasinski.

    (If Style is European, the only option is Black (European).)

  3. In the Calculate field (selectable for new European-style deals only), either:
    • select Value, or
    • select Volatility or Strike to solve that variable for the specified deal value you enter in the adjacent Value field.
  4. If Collar is Yes, select the results View you want: Payer, Receiver, or Collar.
  5. If Style is American or Bermudan:
    • enter the Volatility rate or leave it blank to default the rate according to the Default Valuation Control record,
    • select the Step Type: Per Coupon or Per Year, and
    • enter the number of Steps.
  6. If Style is European, optionally enter the Swap Rate and Volatility rate (if available); or leave either blank to default the rate according to the "Default" Valuation Control record.
  7. Click the Price button.

The Market grid displays the calculation results for your chosen settings, giving the Per Unit and Position values for the market value and Greeks, as summarized below. For details of the calculation formulas used, see Swaption Pricing Formulas.

The Rates tab shows the rate curves used and the Valuation tab displays the calculation results for each valuation cash flow.

The result values are:

  • Value - Market values for the option price.
  • Delta - A measure of the change in the value of an option relative to a change in the price of the underlying.
  • Gamma - A measure of the change of an option's Delta, relative to a change in price of the underlying.
  • Vega - The sensitivity of an Option's value to a one percentage point change in the volatility of the underlying.
  • Theta - The rate of change of the value of an option as its time to maturity decreases, with other terms held constant. (This value is usually negative, as Options with longer periods before maturity have greater time value.)
  • Rho - A measure of the sensitivity of an option value relative to a change in the domestic interest rate.
