Columns

The settings on this tab differ depending on the type of View being configured. For:

  • Cash Position Views - the settings define column grouping, captions, and sort order; and which Group Total rows will show.
  • Forecast Views - the settings in the following areas take effect only when the View configuration option Hide system defined horizons is CLEARED (on the Display tab):
    • System Defined Horizons - these settings define which horizon columns will show, and their captions.
    • Total Prior to Start - these settings define whether a Prior to Start total will show, its caption, and the types of flows to be included in that total.
  • Summary Views - the Columns grid defines which totals will show as columns, their order, and captions.

    Use the grid menu functions to Add, Edit, and Delete columns in the Summary view display.

Flows for the system-defined Horizons will be displayed in the View only if the corresponding Cash Flows were selected when the Data Version was created. See Included Flows.

NOTE: Displayed Columns apply to Forecast Views only.

System Defined Horizons

Select which system-defined Horizons you want to display. By default all the system-defined Horizons are displayed.

System Defined Horizons Caption

Define the caption that will be displayed above the system-defined Horizon columns.

When defining the caption for Previous Day Reconciled Cash Flow you can use the following variables:

  • @Start (to insert the start date of the Horizon).
  • @End (to insert the end date of the Horizon).
  • @NewLine (to insert a line break).

There can be only one @Start and one @End variable per caption, but you can use any number of @NewLine variables.

When defining the caption for the Unreconciled Cash Flows prior to Start and the Projected Cash Flows prior to Start, you can use only the @NewLine variable.

To include an @ symbol in your caption, follow the symbol immediately with a space.

Total Prior to Start

Select the Column Total prior to Start check box to display a totals column on your grid that totals the cash flows in the system-defined Horizons.

When you have selected it, you must select at least one of the following cash flows to include in the total:

  • Add Previous Day Reconciled Cash Flows
    A reconciled cash flow is a cash movement in Quantum that has been matched with an entry in an imported bank statement.
  • Add Unreconciled Cash Flows prior to Start
    An unreconciled cash flow is a cash movement in Quantum that has not yet been matched with an entry in an imported bank statement.
  • Add Unreconciled Bank Flows prior to Start
    An unreconciled bank flow is a cash movement referenced in an imported bank statement that has not yet been matched with a Quantum deal.
  • Add Projected Cash Flows prior to Start
    A projected cash flow is a potential future cash movement.
  • Add Unreconciled External Cash Flows prior to Start
    An unreconciled external cash flow is a cash movement referenced in another system that has not yet been matched with a Quantum deal.

Add Unreconciled Bank Flows prior to Start and Add Unreconciled External Cash Flows prior to Start is available only if your user site is licensed for the Bank Reconciliation module.

By default, none of the check boxes are selected.

NOTE: In some situations the Total Prior to Start may contain duplicate cash flows. For example, Unreconciled Cash Flows Prior to Start and Unreconciled Bank Flows Prior to Start may refer to the same cash flows; therefore if you include both in Total Prior to Start, the total will contain duplicate amounts.

Total Prior to Start Caption

If you select the Column Total prior to Start check box to include a system-defined Totals Horizon on your grid, you can use the Caption field to define a title to appear at the top of the Horizon column.

Within the caption you can use the @NewLine variable as many times as you need to insert new lines.
