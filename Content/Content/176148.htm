Using Expression Builder in Calculated Columns

In Quantum and Adaptiv query Views, you can use the Columns tab's Calculated Columns grid to create and edit an unlimited number and type of customized columns using Expression Builder.

NOTE: This method does NOT apply to Analytics queries, which use a Calculated Columns Query Definition element.

To add one or more columns

  1. Open Configuration > Report and set up the Name, Title etc. for the report.
  2. In the Calculated Columns grid menu select New.
  3. In the New Column dialog box, enter the name that will appear on the column, and whether to display the data as text, dates, etc.
  4. Enter an Expression (mandatory) by hand or click + to display a browse list of available items from which to build the column's expression. The expression can be as simple as using an operator to add two data sources or as complex as setting up joins between data sources or "if" tests. Sources configured in the report will appear in the Expression Builder's browse list, along with all their fields. The browse list may include:
    • Fields - data items applicable to the type of report;
    • Functions - arithmetical (Abs, Round), text-related (Join, Left, and so on), conditional, or date-related functions;
    • Operators - standard arithmetical or logical operators; and
    • literals - text, numeric, or boolean values within double quotes.

      NOTE: Field names are enclosed by square brackets - [...]. Lists of multiple values are enclosed in curly brackets {x,y,z}. Text and Boolean literals are enclosed in double quotes "..". Numeric values have no enclosing delimiters.

  5. If you selected a function, repeat Steps 5 and 6 to replace each of its variables with a selected field, numeric value, text within double quotes, list of values, nested function, and so on.
  6. Repeat Steps 5 through 7 and edit as required, to build the full expression.
  7. Click Validate.

    A message is displayed indicating whether your expression is valid or not. If not, edit as required and try again.

  8. Click Done to exit the Expression Builder dialog box.
  9. Click Apply or Save to save your changes.

To add, edit, or remove columns

  1. Click on a linked item in the Calculated Columns grid.
  2. In the Edit dialog box:
    • To add more columns - click Add Column, enter a new Actual Name, select the Return Type of the calculated result, click the Add Expression link and use the Expression Builder dialog box to define the calculation (as in Steps 3 through 7 of the previous procedure);
    • To delete a column - click the x button at the right of the column row;
    • To change a column's details - (as required) edit the Actual Name field directly, select a different Return Type, and/or click the Expression link and use the Expression Builder dialog box to edit the calculation (as in Steps 3 through 7 of the previous procedure);
  3. Click OK to close the Edit dialog box.
