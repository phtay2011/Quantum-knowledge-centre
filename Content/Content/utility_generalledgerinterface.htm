General Ledger Interface page

Use this page to set up, preview, and run G/L Interface processing, which you can use to preview, update, report on, and export G/L entries for one or more G/L Owners.

WHAT DO YOU WANT TO DO?

  • Select a different Query or previous run - Click on the current Query heading and select the Query or run in the browse window. By default, the system retains run results according to the Expiry Strategy settings for the Query configuration (on the Properties tab) EXCEPT THAT selecting Keep in the Page menu keeps the current run results only.
  • Reconfigure the Query or configure a new one - Click the Query Configure link, make your changes, add expressions, and save them.
  • Select a different result View - Click on the current View heading and select in the browse window.
  • Change the View or create a new one - Click the View Configure link, make your changes, and save them.
  • Run the current Query - Enter specific Filter and Option settings and click Preview to create a results set that you can view but that will not transfer any profit and loss account balances to the Retained Earnings balance sheet account. This enables you to do a trial run and check the results, before committing to the full posting process.

    Click Update when you are satisfied with the preview results and want to save your changes.

    NOTE: If the Query configuration's Advanced Option Write to interface file is selected, a G/L Interface file is output when you run the query in Update mode. The file will be formatted and mapped for interfacing to an external system, based on settings and rules in the applicable G/L Interface Mapping record - EITHER for the specified General Ledger Owners OR as set in the Map Name field.

  • Export current run results to a file (NOT for interfacing to an external system) - In the page menu, select Export; in the dialog box select the export file format (Excel, CSV, XML, or PDF); then click Export.

    The current data is downloaded in the specified format, as indicated by the filename at the bottom left of the page. Use the filename's drop-down menu to open the downloaded file or to locate the file on your computer(Show in folder).

  • Run the query for the Quantum Web API - Enter General Ledger Owners and the source. In the page menu, select Update. The system sends the data to the DTL that you configured for the API associated with the owners.

  • Learn about Filters and Options.

[For details of Filters, Options, and the Configure tabs, click items in those areas when the Help panel is open.]
