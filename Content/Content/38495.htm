Profiling a Deal - Money Market and Securities

Use Deal Profiling to edit deals based on Asset-Backed securities or non-indexed Bond-style instruments - specifically those with the following Formula Timing settings:

  • BOND PERIODIC,
  • CERTIFICATE DEPOSIT PERIODIC, and
  • FRN - CGL PERIODIC.

Deal Profiling:

  • enables you to enter ad hoc principal amortization amounts during the life of the deal, and to specify when amortization is to occur (which must always be on an existing cash flow Effective Date);
  • recalculates the remaining coupon amounts using a simple interest formula (so is not suitable for modifying deals for fixed-equal coupon instruments);
  • where an Asset-Backed Security has both an Effective and a Legal Maturity Date - can be used to modify the Effective Maturity Date;
  • may, for a Securities deal with stock flows, be used only with an Asset-backed FRN deal (that is, where the underlying Instrument Formula is FRN-CGL and Allow asset backed is selected);
  • is implemented via the Deal Map tab's grid menu New function, by selecting any of the following Component types:
    • FACTOR,
    • PAY DOWN %, or
    • PASS THROUGH.

    For details, see Component Types below.

TIP: You can set up your system to ensure any Profiling actions must be approved.

Earlier changes

When you edit the deal profile, the system recalculates amounts from the Effective Date of the edited item to the deal Maturity Date. A record of all previous changes is retained.

Match Deal for Closeout dialog box

If this is relevant, it appears when the deal is saved. See Matched Closeouts.

Editing

You cannot edit coupon amounts and dates using Deal Profiling. Use it only to enter details of amortized principal:

  1. If you save a deal and then realize that some detail of the entered amortization is incorrect, recall the deal and open the Deal Map tab again.
  2. One by one, select each incorrect row that you entered and delete it (there are two for each amortization amount - one that you entered, and one that the system added).
  3. Add the correct details as described above, and save the deal.

Deal Profiling functions are accessed as for other customizing functions. For procedure summaries, see:

  • Customizing Single Deal Map Items
  • Bulk Replace and Insert for Deal Map Items

Component Types

These determine the method for calculating the amount of the principal amortization. Select one of the following:

  • FACTOR (the only selection available for a deal that has stock flows) - to specify the remaining Factor. The amortization amount at the specified date will be the difference between it and the current deal factor.

    TIP: When an Asset-Backed Security is issued or a Standard Bond is dealt, the factor is 1.00.

  • PASS THROUGH - to specify the cash flow amortization Amount applying at the specified date.
  • PAY DOWN % - to specify the Percentage of the Face Value that to be amortized at the specified date.

Effective Maturity Date

(Displayed when an Asset Backed Security is being profiled.)

Change this date if required, to extend (or reduce) the number of events attached to the deal.
