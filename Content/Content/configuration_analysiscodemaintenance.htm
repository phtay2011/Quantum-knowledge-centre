Analysis Code page

Use this page to define custom data items that you can use for categorizing and reporting on key Quantum static data, some instrument types, specific deal types, or across all deal types (depending on the Scope setting).

To list records and select one - Click and select one in the list, using the < and > buttons to page through, or the Search box to reduce the list. (Click the Configure link to change list properties.)

To add a new record - Click New, fill in the data fields as below, and Save.

To change a record's details - Select it in the list, edit the relevant data fields, and Save.

To delete a record - Select it in the list, select Delete in the page menu, and when prompted, confirm.

Data Fields

  • Name - the mandatory unique name for the analysis code.
  • Description - a mandatory text description of the Analysis code; up to 40 characters long.
  • Scope - the type of data the analysis code applies to.
  • Data type - TEXT, NUMBER, AMOUNT, DATE, TIME, or DATETIME.
  • Source of Values - User Input or Predefined Values.
  • Options [for details, click on an option when the Help panel is open.]
  • Predefined Values grid (available only when Data type is TEXT and Source of Values is Predefined Values) - use the grid menu's New, Edit, and Delete functions to create and maintain the list of predefined text values for this Analysis Code.
  • Decimal places (available only when Data type is AMOUNT) - the number of decimal places for amounts entered under this Analysis Code.
  • Maximum and Minimum (available only when Data type is NUMBER or AMOUNT) - optional minimum and maximum limits for values entered under this Analysis Code.

For more details of an item, click on it when the Help panel is open.
