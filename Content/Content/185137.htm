Workflow Example

You set up an Adjust Detail Match Script. Its Pass 1 will set the Match Status to Matched when Bank Flow and Cash Flow Value Dates and Amounts are equal, as below.

Your Adjust Detail Match Script's Pass 2 will set the Match Status to Matched where the Amounts differ (by no more than 7,000,000), PROVIDED THAT the Bank Flow's Generic 5 field is RCP, and create an Actual Cash Flow for each bank flow, as below.

You initiate a Bank Importer run to upload Intraday bank flows, including one for 6,250,000 with and with the Generic 5 field set to RCP. Assuming that Bank Importer does not run* the script and that there is no matching cash flow, when you run an Intraday Match query, the imported bank flow will have a Match Status of Unmatched, as below.

From that page, you initiate a run of the Adjust Detail Match script for that account (for example by selecting Run from the grid menu and specifying that script). Because the bank flow meets the script's Pass 2 criteria, the Actual Cash Flow deal 5202 is created and the Intraday Match Status is set to Matched, as below.

The next day, you use Bank importer to upload the prior day bank statement, and (again assuming that Bank Importer does not run* the script) a Prior Day Match query will show an Intraday Match Status of Matched, and a Prior Day Match Status as Unmatched, as below.

From that page, you initiate a run of the Adjust Detail Match script for that account. Because the two flows now meet the script's Pass 1 criteria and match, the Prior Day Match Status is also set to Matched (and no additional deals are created), as below.

*TIP: Your Bank importer runs can automatically run the script (or chain) linked to an account.
