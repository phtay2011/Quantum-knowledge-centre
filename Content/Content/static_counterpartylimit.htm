Counterparty Limit Rule page

Use this page to define a series of limit profiles and calculation methods for each Counterparty in Quantum. These can be linked to different transaction types and expressed in terms of Deal, Loss, Exposure, Daily, and Settlement risk.

To add a new record - Click New, select a unique combination of Counterparty, Entity, Dealer, Facility, deal Currency, and Limit Currency (ALL is valid in each case), add Limit entries (see below), and Save.

To list records - Click to open the browse list. (Click the Configure link to change list properties.)

To change a record's details - Select it in the browse list, and edit the grid entries (rows) as required:

  • To add a new entry - In the grid menu, select New, and in the dialog box:
    • enter or select Limit Type, Active From date, Calculation Method, Transaction Type, Instrument, and Include in Combine Limit settings to define when the limit applies,
    • enter the Interval (time remaining to Maturity) during which the limit applies, with the Limit amount, and
    • click OK.
  • To edit an existing entry - Click a linked value in the entry's grid row; in the Edit window, edit the values as required; then click OK.
  • To delete an existing entry - Select the grid row and in the grid menu, select Delete.
  • To keep your changes - Click Save.

To delete an entire record - If the record is not currently displayed, select it in the browse list and select Delete in the page menu. When prompted, confirm.
