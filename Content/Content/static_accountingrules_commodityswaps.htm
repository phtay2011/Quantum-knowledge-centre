Commodity Swaps tab (Accounting Rule)

Use this tab to create and maintain accounting rules for Commodity Swap Deals.

NOTE: There must be at least one Accounting Rule entry that matches each type of these deals your organization uses. If more than one, the most specific matching rule entry takes precedence.

To add a new rule entry - In the grid menu, select New1. In the dialog box under Matching Criteria select one or more Instrument Name, Counterparty Name, and Strategy Name values2,3 or ALL (and/or any other customized criteria that may apply); select or enter the required options and settings under Revaluation Settings and Translation Settings as required; and Save. The new rule entry is added to the grid.

To edit an existing rule entry4 - Click an entry value in the grid; in the Edit window, edit the settings as required; then click OK.

To delete an existing entry4 - Select it in the grid; in the grid menu, select Delete; then click OK to confirm.

To select a different grid view - Click on the current view heading and select the View in the lookup.

To change the grid view or create a new one - Click the Configure link, make your changes, then save them.

1 Alternatively, you can copy an existing rule and edit its settings.
2 You cannot select a combination if that entry already exists.
3 Entries for groups take precedence over ALL entries.
4 You cannot add, edit, or delete inherited entries. To override these, create specific rule entries.
