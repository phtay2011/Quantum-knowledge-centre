Rate Set Deals (in bulk)

Use the Rate Set Deals page to:

  • rate-set (in bulk) floating Money Market, FRA, Currency Swap, and Interest Rate Swap deals, or
  • set Forward Rates on floating Interest Rate Swap deals in-arrears, or
  • set Overnight Index Periodic rates for bulk deals.

Set rates for deals, including forward rates

When you set the rate for a deal you can choose an action that uses either Rate Set,In-Arrears, or Average Interest Rate

To run a rate set query

  1. Optionally select a different Query from the drop-down list (next to the Query heading).
  2. Enter your Filter criteria* to define the deals you want to view. First select an action: either Rate Set,In-Arrears, or Average Interest Rate (see below) as required; then select other criteria, including:
    • the optional Rate Set Defaults,
    • the optional Apply rate to subsequent rate sets, which is useful, when deselected, for limiting the number of events by applying the rate only to events within the date range.

    Click the advanced options link to show these.

    TIP: To create new Filter criteria, click Configure, specify filter parameters (on the Defaults tab), name the new Query and set Access Permission to Public or Private (Properties tab), then Save.

  3. Either click Run Query to preview the applicable deals (or select Run Query and Rate Set* to immediately perform the Rate Set action on those deals).

    Deals meeting your filter criteria are listed in the View (if you selected Run Query and Rate Set, the Rate Set has already been actioned on those deals).

  4. (Optional) In the View, select the deal or deals whose rate set parameters you want to change and, in the View menu, select Preview Rates*. In the dialog box, select All Up or Base Rate, optionally change time-band Rate Input values, and select or clear Ignore Settlement Checking as required, then click Rate Set.

    Alternatively, select the deal or deals you want to rate-set without previewing and in the list menu, select Perform Rate Set*.

    In each case, the selected deals are rate set.

  5. (Optional) If you selected a date range in the query (Rate Set only) the View will display the deals within the range. See Rate Setting Deals in a Date Range.
  6. (Optional) To view a summary log - in the site menu under LOG, select View.

NOTE: If you selected a date range in the query (Rate Set only), the Run Query and Rate Set function and the View menu functions Preview Rates and Perform Rate Set work differently. See Rate Setting Deals in a Date Range.

Overnight Index Periodic: Average Interest Rate

From 2019.4 (v6.8.2) Quantum uses Analytics to calculate the overnight index periodic rates for swaps.

To return Analytics data to a Calculated Rates column

  1. In the Processing > Rate Set Deals page, use Action to choose the value Average Interest Rate.
  2. Click Run Query to preview the applicable deals.
  3. Check the rates returned in Calculated Rates are valid.
  4. Under the Calculate Rate column, click the value you are interested in to open the Analytics Average Interest Rate Calculator and see the calculation.
  5. Set bulk (batch) rates for those deals by using Perform Rate Set from the Configure View menu to list menu items.

To override the Average Interest Rate, find the Adjust Rate list menu item and select the rows you want to override.

NOTE:Adjust Rate is not available if, in Processing > Rate Set Deals, the Action is Rate-Set or In-arrears.

To adjust the rate that was returned from Analytics

  1. Open Processing > Rate Set Deals and set the rate set dates and other fields.
  2. Click Run Query to preview the overnight index swap deal you want to trial some rates for.
  3. Highlight at least one row and in the menu select Adjust Rate.
  4. Enter the rate that will override the compound rate that is currently in the highlighted rows.
    • The results do not persist when you leave the page.
    • The results only apply to rows for which you have earlier selected Perform Rate Set from the Configure View menu.

NOTE: Adjust Rate is not available if, in Processing > Rate Set Deals, the Action is Rate-Set or In-arrears.

To rate set an overnight index deal in bulk

Set up the deal, for example a Money Market deal, to use an instrument with Overnight Index enabled, and then use the Rate Set Deals page to run the bulk rate set process.

  1. Open Processing > Rate Set Deals.
  2. In Action, select Average Interest Rate.

  3. In Transaction Type, select Money Market (SE).

    NOTE: With Transaction Type = ALL, the system returns Money Market, Interest Rate Swap, and Cross Currency Swap results.

  4. Set up the rest of the query.
  5. Run the query and the system displays the rate in the "Calculated Rate" column.
  6. Select one or more rows and in the main menu select Rate Set.
  7. The results appears.

On any line of the results, click on any field to open the Average Rate Calculator in a new page. Click on "Deal Number" to go to that particular dealing page.

If both Payment Date Offset and Suspension values are greater than zero in a yield periodic Overnight Index Money Market deal, when calling Analytics for the average rate, suspension is only passed to Analytics for the final rate-set.

Bulk Rate Set for the Arithmetic Average (Calendar) Method

You can run Processing > Rate Set Deals where the average rate returned from Analytics either includes margin based on the calculation method for each deal. When the calculation method is Arithmetic Average (Calendar) Analytics passes margin to the Average Interest Rate Calculator. In this case, the Average Interest Rate Calculator returns the All In rate.

You can manually update the rate for the deals you previously selected by clicking the Adjust Rate.
In the results set, the Calculation Method column displays the calculation method for each deal.
