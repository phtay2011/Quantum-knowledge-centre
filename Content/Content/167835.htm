Included Accounts

This grid lists the bank accounts and broker accounts that have been selected (either directly or indirectly) for this query, and from which cash flows will be retrieved.

NOTE:

  • When the Data Version is created, only those accounts currently in use are included. In Use (IU) is the default setting.
  • Select Include All from the grid menu to display all accounts and include all accounts in the Data Version, whether or not they are currently in use.
  • More accounts can be added using the Add button which will bring up a selection list that allows selection from all available accounts.

TIP: The x at the end of the row removes the account from the selection.

Account Information

Account information is displayed in these adjacent columns:

  • Account Number
  • Account Name
  • Currency
  • Bank
  • Entity
  • Current (Current Opening Balance Type)
  • Opening (Opening Balance Type)
  • Fallback Opening (Fallback Opening Balance Type)

Click on the linked data in a cell to open the Edit dialog box.

Opening Balance Types

Code

Description of Balance type

DEF

Defaulted by Account (the default for new queries)

EA

External Available Balance and Float

The balance calculated by the Bank that reflects all available cash flows and takes into consideration the float.

ECA

External Current Available

The balance calculated by the bank that reflects the amount of funds in your account to which you have immediate access.

EL

External Ledger

The balance calculated by the Bank that reflects all accounting entries, regardless of float.

EV

External Bank Value Date

The balance calculated by the bank that reflects all available cash flows based on the bank-reported value date.

IR

Internal Reconciled

The balance calculated by Quantum that includes all the presented cash flows (or the value of all balance-level and detail-level matched cash flows, and all balance-level and detail-level matched external cash flows).

IU

Internal Unreconciled

The balance calculated by Quantum that includes all cash flows (or all cash flows plus all external cash flows).

IV

Internal Bank Value Date

As the Internal Reconciled Balance (see above), but based on the bank-reported value date (instead of Value Date).


If DEF (Default) is selected, the Current Opening Balance column displays the opening balance as it is currently shown for the bank account. If not, it displays the last-saved opening balance.

When viewing a Data Version, you can change the Opening Balance Type to override the default as specified in Bank Account Rules.
