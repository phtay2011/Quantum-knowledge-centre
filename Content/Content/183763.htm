Configuring Data Importation Services

The following files are available for the configuration of Quantum's data importation services. These services do not apply to Analytics data.

Service

Service Configuration File

Import Bank Balances

AGBankImportService.config

Import Deals

AGDealImportService.config

Import Static Data*

AGStaticDataImportService.config

Process Confirmations

AGConfirmationDeliveryService.config

Match Confirmations

AGConfirmationMatchService.config

*NOTE: The Import Static Data Service also must be running and the Process workflows Operational Settings option selected to enable the import of Workflow data. See Importing Workflow data.

The above files enable you to control a Site Section's access to processing functions, using the following keys:

Key

Description

Default Value

whitelist

Add the Site Section's name to enable processing.

N/A

blacklist

Add the Site Section's name to disable processing.

N/A

NOTE:

  • Multiple entries can be added using semi colon (;) separators.
  • The settings in the Active Services area of Quantum's Operational Settings page override any control settings in the configuration files.

Example

To have the Import Deals service process deal imports for all sections within the Site except the section called TEST, edit the AGDealImportService.config file's appSettings block as follows.

<appSettings>
<add key="whitelist" value="" />
<add key="blacklist" value="TEST" />
</appSettings>

Additional keys for Import Service Configuration files

The AGBankImportService.config file may also include the following optional keys.

Key

Description

Default Value

customFileExtensions*

Custom file name extensions of files to be imported for processing by the Bank Import service.

N/A

customFileExtensionRegEx*

A Regular Expression pattern for dynamic file name extensions of files to be imported for processing by the Bank Import service.

N/A

pollingInterval

The minimum time in seconds between successive starts (sessions) of the matching process for any new imported bank flows.

30

pollingNumberOfBankAccounts

The maximum number of bank accounts that will be processed by the bank flow auto-matching process in one interval session.

50

pollingIntervalForDataTransferLocationQueue

An interval in seconds in which the Bank Importation Service will peek the queue.

If the pollingIntervalForDataTransferLocationQueue key:

  • is not an integer value - the interval is 10 seconds
  • is less than 5 seconds - the interval is 5 seconds

10

*NOTE: To use the Bank Import Status page to import files with custom or dynamic file name extensions, you will also need to specify these in the Web.AppSettings.config file, using either the UploadWhiteList and BankFileImportcustomFileExtensions keys (custom), or the BankFileImportcustomFileExtensionRegex and UploadWhiteListRegex keys (dynamic).

All three ImportService.config files can include an optional additional key to check that incoming file processing is functioning correctly. The File Watching relies on notifications from the File System when new files are created and require processing. This configuration setting acts as a safeguard should the File System not raise alerts to the Services when new files are created.

Key

Description

Default Value

FileWatchResetInterval

The number of seconds between wake-ups, when the system will then read the directory looking for any new files that need processing.

A zero value means no wake up.

500

Example AGBankImportService.config file

<appSettings>
    <add key="whitelist" value="" />
    <add key="blacklist" value="" />
    <add key="customFileExtension" value="bank" />
    <add key="pollingInterval" value="60" />
    <add key="pollingNumberOfBankAccounts" value="40" />
    <add key="FileWatchResetInterval" value="60" />
</appSettings>
