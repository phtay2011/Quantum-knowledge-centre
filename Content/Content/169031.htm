Money Market

Use the Money Market Instrument page to create and maintain instrument details for Money Market deals in Analytics.

The Money Market Pricing function provides a Market Value for various Money Market deals. It takes the cash flow for the instrument used and applies Market Rates to give the Present Value of those cash flows.

Dual Currency Deposits

A Money Market Instrument that has a linked Currency Option Instrument specified provides time deposit default settings for a Dual Currency Deposit deal based on that Money Market Instrument. The specified Currency Option instrument provides the option default settings for the Dual Currency Deposit deal.

Only Currency Option Instruments with the following settings will be available for selection:

  • CO Type is Vanilla European,
  • Buy/Sell is set to Buy if the linked Money Market instrument is a Borrow; Sell if it is a Lend (see below),
  • Call/Put is Call, and
  • Type is Standard.

For more details, see Dual Currency Deposits Pricing.
