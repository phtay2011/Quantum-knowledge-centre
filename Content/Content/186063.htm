AnalysisCodes (Workflow page XML)

The <AnalysisCodes> element holds all analysis codes that are repositioned.

AnalysisCode (Workflow page XML)

The <AnalysisCode> element holds all modifications for a single analysis code.

Attribute

Values

Description

name

text

The name of the analysis code.

fieldId

field id

The id+ of the field the analysis code is associated with.

display

after*

before

When set to after, the analysis code is positioned after the associated field.

When set to before, the analysis code is positioned after the associated field.

mirrorFieldId

field id

The id+ of the field used for mirroring dependencies. By default, the associated field is the mirror field.

mirrorDependecies

false

true*

When set to true, the analysis code hides and disables with the mirror field.

disabled

false*

true

When set to true, the analysis code is disabled. A disabled analysis code displays its contents read-only.

hidden

false*

true

When set to true, the analysis code is hidden. When all analysis codes on the Analysis Code tab are hidden, the tab is hidden.

from

field id

The id+ of the property that holds the analysis codes collection. Use this attribute when the page has two collections of analysis codes.

title

text

The custom title for the analysis code.

defaultValue

text

The default setting for the analysis code.

* Default value.

+ Use your browser's Developer Tools to target field ids.

Example

<AnalysisCodes>
<AnalysisCode name="Mirror" fieldId="OtherCurrency_OtherFaceValue" hidden="false" display="before"/>
<AnalysisCode name="Availability" fieldId="Domain_Counterparty_ExternalCounterpartyName" display="after" mirrorFieldId="Counterparty"/>
<AnalysisCode name="QA PERSON" fieldId="Domain_Location" display="after" mirrorFieldId="Domain_Location"/>
<AnalysisCode name="Review Date" defaultValue="Today"/>
</AnalysisCodes>

Moving Analysis Codes

Analysis Codes can be moved away from the Analysis Codes tab and positioned between regular fields. When all Analysis codes have been moved or hidden, the Analysis Codes tab is also hidden.
