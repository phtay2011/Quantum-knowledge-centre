Assigning Base Templates or Formulas to Fee deal

Use the Fee deal page's Calculation tab to assign Fee Base Templates and/or Formulas to the Fee deal.

You must assign at least one of these, and only one can apply for any one date - that is, if multiple Fee Base Templates and/or Formulas are assigned, each must cover a unique, non-overlapping date range.

The Start Date and End Date entries in the Calculation tab grid define the date ranges for which the Base Template or Formula is effective. The following rules apply.

  • At least one date range must be defined that includes both the Start Date and Maturity Date of the Fee deal.
  • If multiple date ranges are defined, there must be no missing dates between them.
  • No date range may start prior to the Start Date of the Fee deal (in the header area).
  • No date range may end after the Maturity Date.
  • The date range in a grid row is inclusive of both dates: this is different from other date ranges used with Fees, which exclude the end or Maturity Date.
  • The Start Date of the first date range you enter defaults to the Start Date of the Fee deal, and the End Date defaults to the Maturity Date. You can make the End Date earlier if required.
  • The Start Date of subsequent date ranges default to the End Date of the range before it plus one day (provided that End Date is earlier than the Maturity Date of the Fee deal).
  • You cannot create a new date range until you have fully defined the previous one.

Procedure summaries

To add a new grid entry

  1. In the grid menu, select New.
  2. In the New Base Template or Formula dialog box, enter a new Start Date and End Date (subject to the rules above).
  3. Select the Type: Base or Formula.
  4. In the Name lookup, select the Fee Base Template or Fee Formula name, depending on your selected Type.

    If you have selected Base:

    • you may select only a Base Template that is valid for your schedule's specified date range;
    • if the Pay fee cash flow in advance option is selected (Schedule tab), you can select only a Base Template that has a Fee Category of Manual and Formula set to Flat Fee; and
    • if the Base Reset Date Frequency is not set to Daily (Schedule tab), you cannot select a Base Template that has a Fee Category of Position and Base Amount set to Interest Value.
  5. Optionally enter or edit the text Description.
  6. Click OK.

To edit a grid entry

  1. Select the entry in the grid and in the grid menu, select Edit.
  2. In the Edit Base Template or Formula dialog box, edit the field values as required (subject to the rules above).
  3. Click OK.

To delete a grid entry

  1. Select the entry in the grid and in the grid menu, select Delete.
  2. Click OK to confirm.

To go to the Fee Base Template or Fee Formula page

  • In the page menu, select Base Template or Fee Formula , as required.
