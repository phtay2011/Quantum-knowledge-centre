Deal Allocation for Money Market and Security deals

A Money Market and Security Deal Allocation set comprises a master deal - the Source Deal - and one or more additional deals of a similar type. The Source Deal may either be an existing deal, or one specifically created from the Money Market or Securities Dealing page.

The following table shows the relationships between the Source Money Market or Security deal and the Allocated deals.

If you want Quantum to create a reversing deal based on the same Instrument, set up a non-tradeable Money Market or Security instrument with Borrow or Invest set to BOTH. Otherwise, set up the tradeable instrument as either Borrowing or Investment. Quantum creates a reversing deal relative to the Buy/Sell selection on the Dealing page.

To create a Money Market or Security Deal Allocation set

  1. On the relevant Dealing page, either:
    • click New and enter the Source or master deal, or
    • click the Browse button and locate the existing deal you want to be the Source deal.
  2. In the page menu, select Deal Allocation.

    The Deal Allocation dialog box is displayed.

  3. Select the required Deal Allocation Defaulting Rule Template*.

    The Copy options and allocation grid are populated with the default values provided by that template.

    NOTE: If a suitable Money Market template record does not exist, you must create one before you can allocate deals. See Deal Allocation Defaulting Rules.

  4. If required, change the default Copy option settings.

    TIP: Select a Copy option to copy the relevant data (Analysis Codes, Strategy Codes, or Comments) from the Source deal to the other deals in the Allocation set.

  5. To add a new deal to the Allocation set:
    1. In the grid menu, select New (or Copy to copy details from the currently selected row). The New dialog box is displayed.
    2. Select the Role (ALLOCATION or REVERSAL), Counterparty, Entity, and (optional) Relationship.
    3. Enter the Percentage OR the Face Value allocated to this deal (the other field is populated accordingly).
    4. (Optional) Enter the Margin in Basis Points and any text Comment.

      TIP: For bonds, the margin is on the Yield, not on the Coupon.

    5. Click OK.
  6. To edit an existing deal in the Allocation set:

    TIP: You cannot edit the Source deal in the Deal Allocation dialog box.

    1. Click a linked value in the deal row (or select the row and in the grid menu, select Edit). The Edit dialog box is displayed.
    2. Change the settings as required (as in Step 5).
    3. Click OK.
  7. To delete a deal you have added: Select the deal row and in the grid menu, select Delete; then in the Confirm dialog box, click OK.
  8. When you have completed the Allocation, Click OK.

    The Deal Allocation dialog box is closed and the current (Source) deal is saved.

    When any of the deals in the Deal Allocation set are accessed on the Dealing page, the In Deal Allocation badge will appear at the top of the page.
