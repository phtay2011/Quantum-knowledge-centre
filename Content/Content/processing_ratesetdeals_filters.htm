Filters - Rate Set Deals

All filters display by default.

To change the filters displayed, see Create / Change Query Parameters.

Action - select from:

  • Rate Set - to rate-set (in bulk) floating Money Market, FRA, Currency Swap, and Interest Rate Swap deals, or
  • In-arrears - to set Forward Rates on floating Interest Rate Swap deals in-arrears.
  • Compounded Rate - uses the Analytics database to calculate the interest rate near or on the payment date. For more information, see IBOR Replacement 2019.4.

  • Average Interest Rate - uses the Analytics database to calculate the interest rate and rate-set an FRN in bulk. For more information, see IBOR Replacement 2020.2.

Rate Set Start Date (available only when the Rate Set option is selected) - enter the start rate set date of deals you want to include.

Rate Set End Date (available when the Rate Set option is selected) - enter the end rate set date of deals you want to include (defaults to TODAY).

Forward Rate Action Date (available when the In-Arrears option is selected) - enter the forward rate set date of the deals you want to include.

Transaction Type (available for the actions Rate Set or Average Interest Rate) - select a specific deal type (FRA Deals, Currency Swaps, Money Market, or Interest Rate Swaps), or ALL.

Instrument (filtered by Transaction Type) - select a specific instrument or ALL.

Entity - select a specific entity or ALL.

Counterparty - select a specific counterparty or ALL.

Counterparty Type - if your system uses Counterparty Types to group Counterparties, select a specific Counterparty Type or ALL.

Currency (mandatory) - select the applicable currency (you cannot select ALL).

Rate Name - select the name of the rate you want to set.

Rate Type - select the rate type you want to set.

Rate Interval (available when the Rate Set option is selected) - set the interval for rate setting. Select:

  • ALL - to enable rate setting for any time interval,
  • OVERNIGHT - to set that time-specific rate, or
  • a time unit (DAYS, BUSINESS DAYS, WEEKS, MONTHS, or YEARS) and enter the number of those units - to set rates over that time interval. For example, to enter 3 MONTHS, select MONTHS from the lookup, and then type the number 3 in the adjacent field.

Include Previous Rate Set Deals (available when the Rate Set option is selected) - select to include deals that have previously been rate set (these will be indicated by the Already Rate Set column setting in the result list)

Advanced Options - click the link to display these (Rate Set Defaults).
