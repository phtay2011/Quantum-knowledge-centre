Deal Notes in Deal and Settlement Approval

Enhancements have been made to Deal Approval and Settlement Approval which enable you to list deals with notes attached. You can then open and view attached notes before approving or rejecting a deal or settlement action.

Implementation

The following controls are now available when configuring the View in Deal and Settlement Approval.

  • New Deal Note and Has Deal Note columns in the Layout tab.

  • A new Has Deal Note filter in the Filters tab.

Has Deal Note column

In Deal Approval, the Has Deal Note column displays YES for deals which have notes attached; otherwise, it displays NO.

In Settlement Approval, the Has Deal Note column is displayed as follows, for:

  • individual deals - it displays YES for deals which have notes attached, otherwise it displays NO; and for...
  • netted settlements - it displays YES if the netted settlement contains at least one settlement for a deal which has a note attached, otherwise it displays NO.

Deal Note column

In Deal Approval, the Deal Note column displays the Notes badge for deals which have notes attached; otherwise, it is empty.

In Settlement Approval, the Has Deal Note column is displayed as follows, for:

  • individual deals - it displays displays the Notes badge for settlements belonging to deals which have notes attached, otherwise it is empty; and for...
  • netted settlements - it displays the Notes badge if the netted settlement contains at least one settlement belonging to a deal which has a note attached, otherwise it is empty.

Has Deal Note filter

In Deal Approval, when the Has Deal Notes filter is set to YES, only deals with notes attached will be shown in the results of the query; if set to NO, only deals with no notes attached will be shown in the results of the query.

In Settlement Approval, when the Has Deal Notes filter is set to YES, results of the query are displayed as follows, for:

  • individual deals - only settlements belonging to deals which have notes attached will be shown in the results of the query, and if set to NO, only settlements belonging to deals with no notes attached will be shown in the results of the query; and for...
  • netted settlements - only the netted settlement with at least one settlement belonging to a deal which has a note attached will be shown in the results of the query, and if set to NO, only netted settlements with no settlements belonging to deals with notes attached will be shown in the results of the query.

Viewing notes and attachments

For individual settlements in Deal Approval or Settlement Approval, view the attached note by selecting Notes from the quick menu for the item in the View. This opens the Notes dialog box.

For netted settlements in Settlement Approval, view attached notes as follows.

  1. Select Net Details from the from the quick menu for the item in the View.
  2. In the Net Summary dialog box, select a settlement belonging to a deal which has a note attached and then click View Notes (or select View Notes from the grid menu).

    NOTE: View Notes is only enabled for settlements belonging to deals which have a note attached (that is, settlements with a Notes badge in the Deal Note column).
