Cut-Off Time, Rules

Use the Cut-Off Time Rule page to set up and maintain rules for determining cut-off times for:

  • new Actual Cash Flow or Bank Account Transfer deals,
  • FX deals implementing CLS (Continuous Linked Settlement), or
  • settlement flows.

The type of rule is determined by the rule's Scope setting: Deal, CLS, or Settlement, respectively. How each type of rule works is summarized below.

TIP: You can filter the browse view by the Scope setting: All, CLS, Deal, or Settlement.

Deal rules

At Deal entry

When a new Actual Cash Flow or Bank Account Transfer deal is entered on the Dealing page, the system searches for the most-specific matching Cut-Off Rule record with a Scope setting of Deal and the Adjust settlement date option selected.

  • If no matching rule is found - the Settlement Date is not changed.
  • If a matching rule is found, that rule's cut-off time is compared to the rule location's current time, and:
    • if the cut-off time has passed, the Settlement Date is (Today + Cut-Off Days +1), or...
    • if the cut-off time has not passed, the Settlement Date is (Today + Cut-Off Days).

Otherwise, the Settlement Date is not changed (it is outside the range above or the Adjust settlement date option is cleared).

Cut-off days are calculated as Business Days based on the underlying instrument setting.

*TIP: Except for FX deals using CLS rules (see below), you must use Cut-off time Calculated Column functions in page Variants to achieve similar results for other deal type,.

From CashXplorer

If an Actual Cash Flow or Bank Account Transfer deal is created in CashXplorer (for example, for Cash Concentration), the process is as for deal entry EXCEPT THAT when searching for a matching Cut-Off Rule, the Adjust settlement date option setting is ignored.

Settlement rules

Cut-off Time rules with a Scope setting of Settlement will be applied to settlement flows only when they meet rule Matching Criteria values.

When configuring Settlement report results views, you can include Calculated Columns functions that provide cut-off date and time information. See Cut-Off Time functions in Calculated Columns.

CLS rules

Cut-Off Time Rules with a Scope setting of CLS apply only to FX deals using CLS (Continuous Linked Settlement).

The Is CLS value determines which Bank Account and Counterparty Bank Account Defaulting rules apply, and whether an MT300 or MT304 Confirmation message will be output for deal actions.

  • there is a matching Cut-Off Time Rule, and
  • the Is Past CLS Cut Off value is No.
