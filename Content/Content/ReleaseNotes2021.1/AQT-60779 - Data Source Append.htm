Data Source Append

Append the result of one or more data sources to the result of a workflow report. You can now combine rows from various sources into one report.

The Sources grid is split up into 3 grids:

  • Joined Sources – a grid with the main source and the sources that are joined to the main source or other joined sources,

  • Appended Sources – a grid with sources that are appended to the result of the joined sources,

  • Sources – a grid with standalone sources that are used for expressions or filter lookups.

The Appended Sources grid becomes available when there is at least one Joined Source as you cannot add appended sources without at least one joined source. Each column in an appended source corresponds with one of the columns of the joined sources, i.e. have the same display name and data type. The result of each appended source is added to the final result set of the report. Columns defined in the joined sources that don't have a column definition in the appended sources are blank. Appended sources can set columns and settings.

The order of execution is as follows:

  1. Retrieve the joined sources and appended sources (in parallel if possible).

  2. Join the joined sources.

  3. Append the appended source to the result.

  4. Calculate the calculated columns.

The calculated columns expression has a Context field called Source available which contains the name of the source the row of data came from. This can either be the name of the main data source or the name of one of the appended sources. This context field can be used to identify to origin of the data.

UI Changes - Introducing Tabs

We introduced tabs to provide more focus on the different areas of the report definition.

Report– report input, data, and columns (grids for: Joined Sources, Appended Sources, Filters, Calculated Columns)

Actions – actions available to perform on the report (grid for Actions )

Structures– structures defined for the report (grid for Structures )

Sources– standalone sources that are not used as report data but for expressions and lookups of filters.

The sources in existing reports are upgraded as follows:

The first source and any source that has a join condition becomes a Joined Sources. Any source that is not the first source and has no join condition becomes an item in the Source grid.
