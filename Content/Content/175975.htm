Straight-Through Processing

Quantum Straight-Through Processing enables you to automate Confirmations and Settlements through to the payment stage. This is possible only for deal and transaction types that currently support automated matching using an External Matching System. These deal types are:

  • Money Market; Yield - Maturity/Rolling/Statement
  • Money Market; Security (with Stockflows)
  • FX (except partial predeliveries and extensions)
  • Forward Rate Agreements
  • Currency Options (Plain Vanilla Option types only)
  • Interest Rate Swaps
  • Repos

CAUTION: The Action Cashflows option for Importing (Uploading) External Confirmations must be set to achieve Straight-Through Processing.

NOTE: The Process confirmations and Match confirmations Operational Settings must be selected and the corresponding back-end services must be running.
