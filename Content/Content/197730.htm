Expression Builder - New and changed functions

Functions are listed in alphabetical order.

Count

Description

Returns the number of values in the column named

Format

Count(values)

Arguments

values - the values in the (one) named column

Example

Count(Counterparty)

Returns the number of counterparties in the Counterparty column

Description

Returns the number of values of all rows or (optionally) groupings of rows

Format

Count(value, optional groupBy)

Arguments

value - the values in the named column. This is the field/column to aggregate

groupBy - group the results by values from this column. This is the list of fields to group by

Example

Count(Counterparty, {currency})

Choose values from the column named Counterparty and count them, grouping the results by the types of currency as found in the Currency column.

Example

Count([Counterparty] , {[Currency]})

The curly brackets surround the one groupby field, Currency.

Date

Description

Returns a date

TIP: Similar to the Excel date function, invalid day values will return dates in the next month. For example, Date(2020, 9, 31) will return 1 Oct 2020.

Format

Date(year, month, day, optional hour, optional minute, optional second)

Arguments

year - the 4-digit year

month - the 1- or 2-digit month

day - the 1- or 2-digit day value

hour -

minute -

second -

NOTE: All arguments must be integers.

Example

Date(2020,10,31)

The result is 31 Oct 2020.

Description

Returns the date of the relative date described by the text

Format

Date(relativeDate)

Arguments

relativeDate - can be all relative date strings, except "Next Business Day" and "Previous Business Day". For that functionality, use the AddBusinessDays() function.

Example

Date(Yesterday)

The result is the date of the day before today (i.e. if today is 23 Jan 2020, then the result is 22 Jan 2020).

DealLocation

Description

Returns the location of the deal.

Format

DealLocation(dealNo)

Arguments

dealNo - It can be number

NOTE: Argument must be an integer.

Example

DealLocation(17025)

The result populate locations Singapore and United States of a given deal.

DistinctCount

Description

Returns the number of unique items in a column, rather than total number of items

Format

DistinctCount(values)

Arguments

value1 - the values in the named column

Example

DistinctCount[Counterparty)

Returns the number of unique items in the Counterparty column

Description

Returns the distinct number of values of all rows or (optionally) all grouped rows

Format

DistinctCount(values, optional groupBy)

In general: DistinctCount( [Field to Aggregate], { [List of Fields to Group By], ... } )

Arguments

values - the values in the named column

groupBy - group the results by values from this column

Example

DistinctCount([Counterparty], {[Currency]})

Aggregate the number of distinct items in the Counterparty column and group the results by Currency.

Example

DistinctCount([Counterparty] , {[Currency]})

The curly brackets surround the one groupby field, Currency.

LastDayBefore

Description

Returns the date of the last specified day (eg.a Tuesday) inthe week before the specified date, not including the specified date

Format

LastDayBefore(day, date)

Arguments

day – specify the day of the week. Can be a Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, or Sunday

date – the specified date

Example

LastDayBefore("Tuesday", Date(2020, 3, 27))

Returns the date of the last Tuesday that occurs in the week before the 27 March 2020

LastDayBefore( "Friday", LastDayBefore( "Sunday", Date( "Start of Next Month" ) )

Returns the date of the last Friday before the last Sunday of the current month

Mapping

Description

Converts value using the given mapping.

Format

Map (mapping, value)

Arguments

Mapping - entity from the Mapping table header

Value - this is the "from" value located in the Mapping values.

Example

Map('Regis Day Basis','30E+/360')

Returns 30/360

Min

Description

Returns the minimum value of all rows or (optionally) all groups of rows

Format

Min(values, optional groupBy)

In general: Min( [Field to Aggregate], { [List of Fields to Group By], ... } )

Arguments

values - the values in the named column

groupBy - group the results by values from this column

Example

Min([Amount],{[Currency]})

Returns the minimum values for each currency in the Currency column

Example

Min([Amount], {[Currency]})

Max

Description

Returns the maximum value of all rows or (optionally) all groupings of rows

Format

Max(values, optional groupBy)

In general: Max( [Field to Aggregate], { [List of Fields to Group By], ... } )

Arguments

values - the values in the named column

groupBy - group the results by values from this column

Example

Max([Amount])

Returns the maximum value of all values that are in the Amount column

Example

Max([Amount], {[Date]})

Returns the maximum value of the values that are in the Amount column for each date that is in the Date column

Example

Max([Amount], {[Currency],[Bank]})

Looks through the values in the Amount column, then returns the the maximum for each currency, separated into additional rows by the values in the Bank column

Example

Max([Amount], {[Currency]})

Sum

Sum(values)

Description

Returns a sum of the values in the column

Format

Sum(values)

Arguments

value1 - the name of the column with the values to be summed

value2 - the name of the column with the values to be summed

Example

Sum(Amount)

Returns the sum of the values in the Amount column

Example

Sum(Amount)

This returns the sum of the values in the amount column.

Sum(value, optional groupBy)

Description

Returns a sum of the value of all rows or (optionally) grouping of rows.

Format

Sum(value, optional groupBy)

In general: Sum( [Field to Aggregate], { [List of Fields to Group By], ... }

Arguments

value - the values in the named column

groupBy - group the results by values from this column

Example

Sum([Amount], {[Currency]})

Returns the sum of the values in the Amount column, grouped by currency.

Example

Sum([Amount],{[Currency],[Bank]})

The curly brackets surround the two groupby fields, Currency and Bank.

The result includes an aggregated sum of the amounts, grouped by currency and then sub-grouped by bank. That aggregated sum is shown on each row to which it applies.

The number "7,874,900" on the first two purple rows is the aggregated sum of the amounts that are in "USD" and also from "My External Bank". In this case the value in Amount is USD7,875,000 + (-100).

Text

Description

Returns the file name, constructed out of a set of input values

Format

Text(value, format)

Arguments

value - the primitive data type value

format - the format given by the mask string. The accepted formats are the same as the accepted formats in the .NET framework.

Example

Text(Month(Now()),"00")

Returns the file name in the following format: includes the month it is now and the format of the month is 2 digits

Full example:

"INB_" + Year(Now()) + Text(Month(Now()),"00") + Text(Day(Now()),"00") + "_" + Text(Hours(Now()),"00") + Text(Minutes(Now()),"00") + Text(Seconds(Now()),"00") + ".csv"

WAve

Description

Returns the weighted mean of all rows or (optionally) all groups of rows

Format

WAve(value, optional weight, optional groupBy)

In general: WAve( [Field to Aggregate], { [List of Fields to Group By], ... } )

Arguments

value - the values in the named column

weight - the size of the weighting

groupBy - group the results by values from this column

Example

WAve([Amount], 1.0 , {[Transaction Type]})

Returns the weighted average of the amount. If the amount comes from an FX Deals item, then weight by 1.75. If the amount comes from a non-FX Deals item, weight by 1.0. Grouped by currency.

Example

WAve([Amount], if([Transaction Type] = "FX Deals", 1.75, 1), {[Currency]})

The curly brackets surround the groupby field, Currency.

In effect, this carries out the following, depending on the transaction type:-

  • WAve([Amount], 1.75, {[Currency]}), or
  • WAve([Amount], 1, {[Currency]})
