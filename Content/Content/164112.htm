SWIFT-Supported Rate Codes

For SWIFT-formatted confirmations of Cross Currency or Interest Rate Swap deals, you must set up one Rate Name for each SWIFT-supported rate code you will be using. Specify one of the following codes in the Description field for each of these Rate Names.

For each deal, you can then use one of these Rate Names in the Basis field for the Floating leg (on the Instrument Details tab of the Swap Dealing page). Specify a Rate Name with a Description that is valid for the currency of the Floating leg, as indicated below.

The corresponding rate code will then be output in Tag 14F of the SWIFT MT360 or MT361 message.
