Quantum Enterprise Solutions Suite

This section of the release notes covers the features that are new in the Enterprise Solutions Suite since the release of Version 2020.1.

Confirmation Matching

We have implemented the following feature.

Scheduler support

Inward confirmation matching is used to match SWIFT MT300 and MT320 confirmation messages prior to settlement, as a part of the Quantum workflow process. To automate this action, we've added the support for Scheduler to Confirmation Matching. Previously, it was a manual process.

Generic EFT database

The component has been enhanced to support the unicode EFT messages.

Market Data Manager

We have implemented the following two features in the Market Data Manager (MDM):

  • Smile points
  • Reuters Rate ID mapping

To enable the features, please make the changes below in the following two configuration files:

  • SunGard.AvantGard.MDM.API.Windows.exe.config
  • SunGard.AvantGard.MDM.API.Win.exe.config

SunGard Transaction Network

We have enhanced the component to accept and process the Fund Connect Trades from the State Street interface.

Wires

We have implemented the following feature.

Dynamic field caption change on Payment screen

For the Payment screen, there is a new dynamic field caption change feature. Using this, for each Payment template, you can specify a desired caption for the Payment screen fields. When you select a Payment template, all field captions change to those specified in that Payment template.
