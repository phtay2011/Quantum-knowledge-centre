FRA Posting Rules, Profit and Loss tab

  • Settlement Expense - Debit Profit and Loss account. If the Accrue settlement check box in Accounting Rules is:
    • Clear, the system creates this entry at the time the FRA is fixed, and contras it by a bank account entry. The entry date is the FRA settlement date.
    • Selected, the Accruals Posting report creates this entry, and contras it to the Settlement Expense Accrual account.
  • Settlement Income - Credit Profit and Loss account. See Settlement Expense (P/L) above.
  • Gain: Realized - Credit Profit and Loss account. If an instrument is closed out, irrespective of the posting method, this default records the gain on closeout, and is posted at the time of the closeout, but dated the settlement date of the closeout.

    If the applicable Revaluation option in Accounting Rules is:

    • cleared - the system does not post transaction revaluations, so this default is not used; but if...
    • selected - the system posts revaluations - however, this default is used only before closeout if the Posting method is set to Realized. In this event, the Unrealized default is not used.

    If the Posting method is set to Realized, the Revaluations posting report creates this entry, and contras it to the Revaluation Gain: Realized (B/S) account.

    On settlement, there is no unrealized amount to reverse.

  • Gain: Unrealized - Credit Profit and Loss account. If the applicable Revaluation option is:
    • cleared - the system does not post transaction revaluations, so this default is not used; but if...
    • selected - the system posts revaluations.

    If the Posting method is:

    • Realized - this default is not used, but if...
    • Unrealized, -the Revaluations posting report creates this entry, and contras it to the Revaluation Gain: Unrealized (B/S) account.

    On closeout, the system reverses the unrealized amount, and records the realized amount using the Gain: Realized (P/L) account or Loss: Realized (P/L) account.

  • Loss: Realized - Debit Profit and Loss account. See Gain: Realized (P/L) above.
  • Loss: Unrealized - Debit Profit and Loss account. See Gain: Unrealized (P/L) above.
  • CCY Translation Gain: Realized - Credit Profit and Loss account. If the settlement amount is accrued, this records the difference in the base currency amount for non-base currency deals arising as a result of the change in the exchange rates of the accrual entries compared with the Settlement cash flow.
  • CCY Translation Gain: Unrealized - Credit Profit and Loss account. This records the unrealized differences between base currency and natural currency amounts for non-base currency deals, arising as a result of changes in exchange rates.
  • CCY Translation Loss: Realized - Debit Profit and Loss account. See CCY Translation Gain: Realized (P/L) above.
  • CCY Translation Loss: Unrealized - Debit Profit and Loss account. See CCY Translation Gain: Unrealized (P/L). above
