Processor (for WAC)

Use the Weighted Average Closeout Processor page to preview and process portfolio*-based weighted average closeouts (WACs) in Quantum, using the WAC Processor. This provides an alternative to using the standard WEIGHTED AVG Closeout method set up in Money Market & Securities Accounting Rules.

Weighted Average Closeout (WAC) Processing enables accounting using the weighted average price to value a portfolio containing trades of a single stock. When run, the WAC Processor picks up deals entered in the portfolio (as defined by Entity, Instrument, CUSIP, ISIN, and Currency) and generates new deals to reflect the position and price of the portfolio as a whole. These new deals are saved against a specifically created entity. Both the new entity and the dealing entity post into the same group entity.

Weighted Average Price

The weighted average price of the portfolio is calculated as:

Capital value of the portfolio at a particular date / Face value of the portfolio

...where:

  • the portfolio values are the sum of the individual deal values, and
  • if the deals have Capital Accruals set in Accounting Rules, the amortizations are taken into account when calculating the capital value.

Accounting

Both the dealing entity and the WAC entity are part of the same group entity so the General Ledger entries are accounted for in one ledger.

  • The G/L entries of the actual deal are all posted to a clearing account, except for the bank account entries, which remain with the actual deal.
  • The G/L entries of the notional deal are posted to the individual ledger accounts, except for the bank account entries, which are posted to a notional bank account.
  • Revaluations of the portfolio deal use the weighted average price against the market to calculate the gains and losses.
