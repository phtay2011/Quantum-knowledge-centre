Customizing Single Deal Map items

On the Dealing page's Deal Map tab, you can (if you have the required Module Security permission) customize single items on an existing deal, as described in the procedures below. This applies only to the following Deal types:

  • Cross Currency Swap,
  • Interest Rate Option,
  • Interest Rate Swap,
  • Money Market, or
  • Security.

NOTE: When customizing Deal Map items, the Cash Flows and G/L Entries tabs are dynamically updated, enabling you to preview the effects of your customization before saving the customized deal. In this preview state only:

  • the bank accounts shown on the Cash Flows tab are determined by the Bank Account Default rules and DO NOT reflect any previous changes to accounts that may have been made on the Settlements page (Our Instructions). However, such changes are retained for cash flows dated prior to the customization date, and again show once the deal has been saved.
  • the G/L entries are shown as if the deal was a new one, so WILL NOT include process or reversal G/L entries if those exist for a deal previously saved. When the deal is saved after customizing, the G/L entries will be re-calculated.

OTHER POINTS TO NOTE:

  • To bulk insert and replace Deal Map items, for example, by copying and pasting from a spreadsheet, see Bulk Replace and Insert for Deal Map Items.
  • To capitalize (or de-capitalize) Interest items on a Money Market deal, see Capitalizing Interest on Money Market Deals.
  • You cannot edit Deal Map grid rows with no linked cell values.
  • To change the Settlement Amount of priced transactions, change one of the components of the price (rather than the Settlement Amount itself). This ensures the correct accounting treatment will be applied to the transaction. For example, changing bond income will automatically update the bond Settlement Amount.
  • If you amend or delete a Deal Map item that has already been Actioned, or add an item that impacts items that have already been Actioned, you may have to manually reprocess those items. This is indicated by the Flow Lifecycle value, which you can display as a grid column on the Cash Flows tab, in Settlements and Confirmations results grids, and use as a filter in deal or flow queries or result views.

    Also - depending on whether or not Deal Map item changes have a financial effect - the changed deal will be badged as Customized - Major or Customized - Minor, respectively.

To add a Deal Map item (from scratch)

(You must have Customize Deal Module Security permission for the Dealing module.)

  1. Select the grid row you want to insert above.
  2. In the grid menu, do one of the following:
    • select Customize* IF it is present, otherwise
    • select New.

      *TIP:Customize is available only for Money Market and Security deals based on Instruments with Formula Timing of BOND PERIODIC, CERTIFICATE DEPOSIT, or FRN - CGL PERIODIC (New also appears but initially offers only Profiling options - see Other Bond Formulas - Customization Restrictions).

  3. in the New Map Items dialog box, enter or select the following values.
    • Effective Date - the date when the event or cash flow is to occur; the date Interest is calculated to.
    • Action Date - the Action date of the event or cash flow (for example, the date interest is paid or received).
    • Comment (Swaps and Interest Rate Options) - for a Swap, select the leg (mandatory); for an Interest Rate Option, depending on the type and where required, select Cap or Floor, Call or Put, Payer or Receiver as applicable.
    • Component - the type of event or cash flow - for example Rate Set, Interest, Principal, Repay Principal, and so on. For items that are not events, this is simply a descriptive label.
    • Amount/Rate - the amount or rate applicable to the event or cash flow.
  4. Click OK.

    The new Deal Map item row is inserted in the grid. If you have added or modified dates, rows will be re-sorted if required.

  5. Save your change.

NOTE:If your new item impacts items that have already been Actioned, you may have to manually reprocess those items. Also - depending on whether or not Deal Map item changes have a financial effect, the changed deal will be badged as Customized - Major or Customized - Minor, respectively. See the Notes above for more details.

To add a Deal Map item (by copying an existing one)

(You must have Customize Deal Module Security permission for the Dealing module.)

  1. Select the map item grid row you want to copy.
  2. In the grid menu select Copy.
  3. In the New Map Items dialog box, change the field values as required (see above).
  4. Click OK.

    The new Deal Map item row is inserted in the grid. If you have added or modified dates, rows will be re-sorted if required.

  5. Save your change.

NOTE: If your new item impacts items that have already been Actioned, you may have to manually reprocess those items. Also - depending on whether or not Deal Map item changes have a financial effect, the changed deal will be badged as Customized - Major or Customized - Minor, respectively. See the Notes above for more details.

To edit a Deal Map item

(You must have Update Customized Deals Module Security permission for the Dealing module.)

TIP: To capitalize (or de-capitalize) an Interest item on a YIELD PERIODIC Money Market deal, see Capitalizing Interest on Money Market Deals.

  1. Click a linked value in the map item grid row (or select the row and in the grid menu select Edit).
  2. In the Edit Map Items dialog box, change the field values as required (see above).
  3. Click OK.

    The Deal Map item row shows your changes. If you have added or modified dates, rows will be re-sorted if required.

  4. Save your change.

NOTE: If your new item impacts items that have already been Actioned, you may have to manually reprocess those items. Also - depending on whether or not Deal Map item changes have a financial effect, the changed deal will be badged as Customized - Major or Customized - Minor, respectively. See the Notes above for more details.

To delete a Deal Map item

(You must have Update Customized Deals Module Security permission for the Dealing module.)

  1. Select the map item grid row.
  2. In the grid menu, select Delete.

    TIP: You can also delete an item from within the Edit Map Items dialog box.

  3. When prompted to confirm, click OK.

    The selected Deal Map item row is removed from the grid.

  4. Save your change.

NOTE: If your new item impacts items that have already been Actioned, you may have to manually reprocess those items. Also - depending on whether or not Deal Map item changes have a financial effect, the changed deal will be badged as Customized - Major or Customized - Minor, respectively. See the Notes above for more details.
