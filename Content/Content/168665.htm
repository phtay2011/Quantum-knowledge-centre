Zero Curve

Use the Zero Curve page to create interest rate Zero curves that merge existing Cash, Futures (optional), and Swap rates curves. This is useful where you need to adjust and combine published rate feeds to form an interest rate curve that is suitable for valuing cash flows.

The combination of Name, Rate Type, Currency, Date, and Model uniquely identifies this type of rate record in the system, and is displayed above the Header.

There are three possible components used to create a Zero Curve:

  • CASH - User-input Cash rates (which can be up to but not including 12 months) are standardized to 365-day basis, converted to standardized zero coupon rates, and finally converted to annually compounded Discount Factors.
  • FUTURES (optional) - User-input Futures prices are converted to a Yield basis, then standardized to a 365-day basis, then converted to zero coupon rates. The half-yearly rates are interpolated linearly and displayed, together with Discount Factors calculated as for Cash and Swap components.
  • SWAP - User-input Swap rates (which can be from 12 months onwards) are standardized to 365-day basis, then converted to semi-annual basis. These are converted into zero coupon rates semi-annually from 18 months onwards, using a Quadratic bootstrap method. Note that the 12 month rate is calculated using the 6 month rate and so does not use the Quadratic bootstrap method. Finally, these are converted to annually compounded Discount Factors.

For more details, see Quantum Zero Curve.

Defining a User Input Zero Curve

You can define a User Input Zero Curve or User Input Discount Factor, as follows.

  1. On the Yield Rate page, define the Yield Rate record you want to use as the CASH component of your Zero Curve or Discount Factor.
    1. In Input Type, select either Zero Rate or Discount Factor.
    2. Complete the remainder of the header as defined for Yield Rates.
    3. Use the Rates tab to input the curve points.

      TIP: You may want to copy or import these points from another application such as a spreadsheet.

    4. Save your changes.
  2. Repeat Step 1, this time defining the Yield Rate record you want to use as the SWAPS component of your Zero Curve or Discount Factor.
  3. (Required only for Zero Curves with a FUTURES component) On the Futures Rate page, define the Futures Rate record you want to use as the FUTURES component of your Zero Curve or Discount Factor.
  4. On the Zero Curve Definition page, use the Blend / Zero definition table to define the Zero Curve or Discount Factor components. In particular:
    • In the Rate Name field of the CASH row, enter the name of the curve you defined in Step 2.
    • In the Rate Name field of the SWAPS row, enter the name of the curve you defined in Step 3.
    • (Optional) In the Rate Name field of the FUTURES row (if any), enter the name of the curve you defined in Step 4.
  5. Save your changes.

For more details, see Quantum Zero Curve.
