Weighted Average Closeouts: Deletions and Changes

NOTE: You can use the Weighted Average Closeout Processing page as an alternative to selecting the WEIGHTED AVG Closeout Method on the Securities Instrument page.

The Weighted Average calculation is NOT back-valued when changes and deletions of existing deals in a portfolio occur. These are treated as new entries in the portfolio's deal-linking table.

  • A deletion of a deal creates a new reversing entry, which acts as a separate deal with a Face Value and Capital Value equal but opposite to those for the deal it deletes.
  • The Weighted Capital Value for a deletion entry is calculated as for a deal opposite to the one it deletes - that is:
    • as for a closeout deal if it reverses a normal deal (see Example 1 below), and
    • as for a normal deal if it reverses a closeout deal.
  • A change to a deal is treated in two parts: firstly, as a deletion of the old deal value, then as creation of a new value (see Example 2 below).
