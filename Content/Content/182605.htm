Range Accruals

A Range Accrual Note is a Money Market instrument in which a coupon is accrued only while a reference index falls within a predefined range This coupon can be either fixed or tied to an underlying index. Interest is accrued either:

  • at an Enhanced Rate for each period (usually one business day) for which the reference interest rate fixes within a predefined range, OR
  • at a Reduced Rate (often zero or a percentage of the Enhanced Rate) for each period the reference interest rate fixes outside the predefined range.

Any accrued interest is paid on the periodic interest payment dates (typically semi-annually, quarterly, or monthly) or at the end of the investment period.

The payoff structure of a Range Accrual Note can be seen as a series of long digital call options on the lower rate in the range, and short calls on the upper rate, with the settlement of these options deferred to the coupon payment date. Therefore there are two significant components to the valuation:

  • valuing a string of digital caplets, and
  • performing a convexity adjustment on the forward rates used in the above valuation, to account for the deferred settlement.

Key Features:

  • Support for either a fixed or floating Enhanced Rate.
  • A separate Reduced Rate entered as either a fixed rate or a percentage of the Enhanced Rate.
  • Market values and sensitivities provided through the Analytics modules.

Instruments

On the Money Market Instrument page's Details tab, the Range Accrual check box is available when:

  • the Formula Timing combination is YIELD PERIODIC, and
  • Floating is selected.

Select the Range Accrual option to set Range Accrual as the default setting for deals using the instrument. The option is cleared by default.

Dealing

On the Money Market Dealing page, the Apply Range Accrual check box appears on the Deal Details tab when the selected Instrument meets the criteria above, with the default setting as for that instrument.

Select this option to capture the current deal as a Range Accrual Note.

When this option is selected, Range Accrual-related fields appear, as shown below.

Ratesets

Rate setting for Range Accrual deals can be done as for other floating rate deals, using the Dealing page's Rate Set function (one deal at a time) or the Rate Set Deals query (over selected deals), with minor differences due to the nature of Range Accruals.

See Rate Setting for Range Accruals

Deal Revaluation

Deal Revaluation is supported only in Analytics Pricing, Queries, and Deal Analyze. However, the Imported Valuations utility can be used to bring the valuations into Quantum Reporting or Accounting.

Pricing

On the Money Market Pricing page, the new Range Accrual check box and its associated items will appear on the Details tab when, for the selected Instrument, Range Accrual is Yes.

Key Features:

  • Fixed Enhanced Rate or Floating Enhanced Rate option with corresponding rate entry fields.
  • A fixed Reduced Rate or a Reduced Rate as a percentage of the Enhanced Rate.
  • Lower Bound and Upper Bound fields to define the range.
  • Solve for:
    • Value,
    • Rate (to solve for a fixed Enhanced Rate value),
    • Margin (to solve for the margin over a floating Enhanced Rate),
    • Lower Bound, or
    • Upper Bound.

For a Floating Enhanced Rate the system will extract the curve with a Rate date equal to the preceding Interest Payment date (less Days Offset) in order to find the appropriate Enhanced Rate value for the period.

Valuation Rules

Two new options have been made available on the Money Market tab on the Analytics Valuation Rule page in order to price Range Accrual notes.

  • Volatility Rate sets the default Cap/Floor volatility rates for use in the range accrual valuation model, and
  • Volatility Rate Type sets the method for applying volatility - one of the following:
    • Flat Volatility - to interpolate volatility at maturity of a coupon, and apply this to the strip of options that make up the value for that period, OR
    • Spot Volatility - to interpolate a volatility rate at each observation/rateset point.

Analytics Queries

Range Accrual Notes can be included in each of the available Analytics Query types (Position, Cash Flow, Historical Simulation VaR, Monte Carlo Simulation VaR, and Parametric VaR).
