Bank Clearing tab, Currency

(Transaction-level currencies only.)

Use this tab to enter up to five Bank Clearing instruction codes for the currency.

TIPS:

  • You cannot change the preset standard Bank Clearing codes for certain currencies (but you can add new codes if required).
  • For details of currently supported Bank Clearing codes, see About Bank Clearing Codes.

Each Bank Clearing instruction code is numbered (1 through 5) and is displayed as a group three items, as follows.

Bank Clearing Field 1...5

  • Automatically selected and read-only for a standard Bank Clearing code preset for certain currencies - for example, CHAPS for GBP; CHIPS Universal,CHIPS Participant, and FedWire for USD.
  • Select this option to use the corresponding Caption and Length for a Bank Clearing instruction code field. These code fields will then appear, labeled with their Caption values, in the Bank Clearing Fields panel on the Bank Details tab for Counterparty Delivery Instructions and Bank Account Setup when this Currency is selected.

    NOTE: Include a CLS (Continuous Linked Settlement) Bank Clearing Field to make the currency CLS-eligible for Currency Option and FX Deal validation purposes at deal save time.

  • Clear this option where the Bank Clearing instruction code is NOT required.

Caption 1...5

  • Applicable only when the corresponding Bank Clearing Field option is selected and this is not a standard Bank Clearing code preset for the currency.
  • The caption (label) for this Bank Clearing code field, to be displayed in Counterparty Delivery Instructions and Bank Account Setup when this Currency is selected.

Length 1...5

  • Available only when the corresponding Bank Clearing Field option is selected and this is not a standard Bank Clearing code preset for the currency.
  • The maximum length for this Bank Clearing code field to be displayed in Counterparty Delivery Instructions and Bank Account Setup when this Currency is selected.
