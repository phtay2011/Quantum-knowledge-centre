CashXplorer page

Use this page to define and run CashXplorer queries (also known as worksheets) for cash management purposes.

WHAT DO YOU WANT TO DO?

Select a different Query - Click on the current Query heading and select another Query in the drop-down list. If that Query has previously run, its latest run results (Data Version) will be displayed.

Create a new Query with default settings - In the page menu select New; in the dialog box enter a new Name, Short Name, and Description; then click OK. (Use the next procedure for further settings.)

TIP: You can also create a new Query from an existing one using Copy.

Change the current Query - Click the Query Configure link, make your changes, and save them.

View a previous run (Data Version) of the current Query - Click the Previous Runs link and in the browse window, locate and select the previous run you want.

Run the current Query - Click Run Query (or, in the page menu select Run); in the dialog box enter the First Horizon Start Date*, select or clear Calculate Concentration Flows and other run options; then click OK.

*TIP:First Horizon Start Date is based on your User Location's time zone (defaults to the System date/time).

Refresh the current Query results - Click Refresh Version (or, in the page menu select Refresh); select or clear Calculate Concentration Flows and other options; then click OK.

Delete only the current Data Version - In the page menu under VERSION, select Delete and when prompted to confirm, click OK.

TIP: The Query configure Properties tab has options for automatic deletion of older versions.

Delete the current Query and its Data Versions - In the page menu under QUERY, select Delete and when prompted to confirm, click OK.

Select a different result View - Click on the current View heading and select the View in the lookup

Create a new View - In the View menu (depending on the type of View you want) select New Cash Position, New Forecast, or New Summary; on the Configure View tabs enter the View details as required; and click Save.

TIP: You can also create a new View from an existing one using Copy.

Change the current View - Click the View Configure link, make your changes, and click Save.

Delete the current View - In the View menu select Delete; and click OK when prompted to confirm.

Process Cash Concentration transfers for the current Data Version - In the page menu select Concentrate Cash, optionally select Calculate Concentration Flows and other options; optionally review the suggested, bank generated, and Cross Currency transfers; then click either Transfer & Refresh or Create Transfer Only.

View Cash Concentration hierarchy - In the page menu select Concentration hierarchy and browse in the dialog box as required (click Close when finished).

Export a standard report - Select Export in the page menu to export a report to CSV, PDF and XML formats.

Export a Crystal Report - Use the export control in a Crystal View to export a export a report to a variety of different formats.

For details of Query or View Configure parameters (when displayed), click on a tab or item when the Help panel is open.
