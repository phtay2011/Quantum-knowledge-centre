Position Reporting and WAC Processing

Straight-Line or Yield Accrual Methods

For those portfolios where the Accrual Method setting in Accounting Rules is Straight Line, Yield Cum, or Yield Ex, Weighted Average Closeout (WAC) processing will include a Position query (Reporting menu) to identify the Profit and Loss neutral price at which to trade the Notional Closeout Counterparty deal.

For this run, you must save a Position query named WACC, with the following Filters and Options settings.

  • As At Date is TODAY
  • Position Date is TODAY
  • As at End of Day is cleared
  • Deal No selection is included to enable selection of individual deals, rather than processing all deals each time

    TIP: This is not a default Filter parameter. To add it, click the Configure link, click Add Filter, select Money Market Deals then Deal No in the browse window, and click OK.

  • Use Management rules is selected

    This ensures that Premium/Discount Accruals generated for the position date are for the period from deal inception to position date (a setting of Accounting would only see accruals generated from the last posting Accrual Inquiry run).

  • Control Rule Set may be anything (including ALL) provided that the Management Control record selected uses the same accrual conventions as in the applicable Accounting Rules

NOTE: The Position query is run form the Reporting > Position-Query page NOT the Accounting > Position page.

Amortized Cost Accrual Method

For those portfolios where the Accrual Method setting in Accounting Rules is Amortized Cost, WAC processing will also include a Position run (Reporting menu), which will not update the General Ledger.

Save a Reporting > Position run

For this run, you must save a Position run named WACC, with the following Filters and Options settings.

  • As At Date to TODAY
  • Position Date is TODAY
  • As at End of Day is cleared
  • Use accounting rules is selected

    This ensures that only accruals generated from the last posting Accrual reporting run will be included in the results.

Management rules

Management rules do not support the Amortized Cost accrual method. However, with care, it is possible to use Management Control as follows:

  • To use Management Control to calculate the Holding Book Value, check the Accounting Rule for the security has the On Closeout or Unwind option set to Calculate New Amortization Schedule. This is required because Management Control does not support the Apportion Existing Amortization setting.

    It is important that the Accounting Rule and Position Query calculate the book value consistently.

  • To use Accounting to calculate the Holding Book Value, check that, for the Accounting Rule for the security, the deal has Revaluation settings turned on. This is required for Position to identify the record for processing.
