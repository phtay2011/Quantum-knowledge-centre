Unit Holding Deal page

Use this page to view and optionally edit an existing Unit Holding Deal. This includes adding new Unit Deals to, creating Dividend and Split Corporate Actions for, and transferring units from, the holding.

NOTE: You cannot create a new Unit Holding deal from this page. New Unit Holdings are automatically created by the system when you enter a new Unit Deal and no Unit Holding already exists for the Security Identifier, Holding Instrument, and Entity specified for that Unit Deal (If a matching Unit Holding does exist, the new deal is added to that holding).

To list Holding deals - click the button (click the Configure link to change list properties).

To change a Holding deal's details - select it in the list, edit the relevant data fields on the required tabs, and click Save.

To add a new Unit deal to the current Holding - in the page menu (or the Unit Deals tab's grid menu) select New, fill in the data fields in the Unit deal window that opens, and click Save.

To Split (or Reverse Split) the current Holding* - in the page menu select Split; in the Unit Split window that opens, select the Split Type (SPLIT or REVERSE SPLIT), enter the Ratio or For number as required, fill in the remaining data fields, and click Save. Your Split action is applied and displayed on the Corporate Actions tab.

To record a Dividend on the current Holding* - in the page menu select Log Dividend; in the Unit Dividend window that opens, select the Dividend Type, fill in the remaining data fields, and click Save. Your action is applied and displayed on the Corporate Actions tab.

To transfer units to another Holding* - in the page menu select Transfer; in the Unit Transfer window that opens, select the To Entity, Unit number and Price, fill in the remaining data fields as required, and click Save. The transfer is applied and displayed on the Transfers tab.

*TIP: You can delete or edit the last Transfer or Corporate Action (Split or Dividend) from the respective tab.

* NOTE: Corporate Action options are disabled if the Unit deals that make up the Holding have not been fully approved.

For details of tabs and data items, click on the tab or item when the Help panel is displayed.
