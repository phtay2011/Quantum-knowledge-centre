Examples - QXT

NOTE: When using these format examples, separate each line of input with a line break. Separate by pressing Enter at the end of each line.

Import records

You can import records in QXT format.

@CounterpartyContact
$new
Name = Test QXT
Counterparty = ABN Singapore
DealerSettlement = DEALER
Entities = Transactor - Level 2
Instruments = CM FX Forward
TransactionTypes = FX Deals
Currency1 = AUD
Currency2 = CAD
Surname = Surname
FirstName = Name
Title = Mr
Initials = SC
JobTitle = Title
EmailAddress = email@email.com
$$ADD AdditionalContacts#FirstName = Kahu;Surname = Smith;EmailAddress = kahu.smith@email.com;Position = Position1;SendConfirmationEmail=true
$$ADD AdditionalContacts#FirstName = Alice;Surname = Smith;EmailAddress = alice.smith@email.com;Position = Position2;SendConfirmationEmail=false
$Save

Modify records

You can modify records in QXT format.

@CounterpartyContact
$new
Name = Text QXT
Surname = Surname Changed
FirstName = Name Changed
$$Add AdditionalContacts#FirstName = Nick;Surname = Smith;EmailAddress = kahu.smith@email.com;Position = PositionToChange;SendConfirmationEmail=true
$update

Delete records

You can delete records in QXT format.

@CounterpartyContact
$new
Name = Test QXT
$delete

Examples - QID

NOTE: We recommend using QXT from v6.x onwards.

NOTE: If using these QID format examples, separate each line of input with a line break. Separate by pressing Enter at the end of each line.

To import records using QID

(QID format - archive)

@CounterpartyContact
$new
Name = Bank A
Counterparty = Bank A
DealerSettlement = BOTH
Entities = ALL
Instruments = ALL
TransactionTypes = FX Deals
Currency1 = ALL
Currency2 = ALL
Surname = Surname
FirstName = Name
Title = Mr
Initials = SC
JobTitle = Title
EmailAddress = email@email.com
$$ADD AdditionalContacts#FirstName = Bob;Surname = Smith;EmailAddress = bob.smith@email.com;Position = Position
$Save
