Defaults Area Settings (all Interest Rate Option Types)

Use the fields in this area to record default values for Interest Rate Option Deals based on this instrument.

The same fields apply to all Option types, and all are optional, unless explicitly stated here.

Currency

The currency to be used to settle deals based on this Instrument.

For Bond Interest Rate Options:

  • this field is read-only, and
  • it is set to the currency for the selected Settlement Instrument (if none, a currency must be selected at deal entry time).

For Simple Interest Rate Options or Swaptions:

  • this value will populate the Currency field for the Interest Rate Option Deal,
  • if Is modifiable is selected for this instrument, the currency can be changed at deal entry time, and
  • if left blank, a currency must be selected at deal entry time

Dealer

The default dealer for deals based on this Instrument. This default will be applied when non-dealers enter deals based on this instrument, but may be overwritten (Dealers use their own ID).

Facility

Optional. The default formal business relationship (Facility or Dealing Line*) against which deals based on this this Instrument will be dealt.

  • If a Relationship Rule matches this instrument and Relationship Type is set to No Relationship in that rule, this field is initially set to NONE.
  • *The type selected (FACILITY or DEALING LINE) is indicated in gray above the field.

Location

Optional. One or more default locations for deals created using this instrument. The calendars of these default locations will be used to validate business days, as required. See Locations. Any default locations will initially be applied (but may be overwritten).

Eligible for Clearing

Select this option to indicate that flows for deals based on this instrument, by default, can be cleared via a Clearing House.

This default setting may be changed for an individual deal at deal entry time, but even when set, clearing will NOT be required if the Exempt from Clearing option is set for the deal Entity and Counterparty.

Clearing House

(Available only when Eligible for Clearing is selected.)

Select the Clearing House through which flows for deals based on this instrument can be cleared. This default setting may be changed for an individual deal at deal entry time, but clearing will NOT be required if the Exempt from Clearing option is set for the deal Entity and Counterparty.

This drop-down list includes all Counterparties that have the Clearing House category selected.

Report to Trade Repository

Select this option to indicate that deals based on this instrument are, by default, to be reported on to a Trade Repository.

Name of Identifier 1, 2, 3, 4

Use these fields to enter the names of up to four unique trade identifier fields for deals based on this instrument. For each non-blank field value entered here (if any), a field labeled with that name will be displayed on the Other Details tab of the corresponding dealing page when this Instrument is selected.

  • If you enter no names, no trade identifier fields will display on the dealing page when this Instrument is selected.
  • If you enter more than one, enter them in sequence (with no gaps).

See Setting Up Trade Identifiers.
