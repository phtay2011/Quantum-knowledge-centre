Minimum Value Formulas - Interest Rate Options

Minimum value is typically defined with reference to a theoretical lower bound for option price, derived using some simple arbitrage arguments. It can alternatively be defined as the value that results from applying the appropriate valuation model at a volatility level of zero.

Minimum Value for Caps, Floors, and Collars

Each of these options will typically consist of a number of caplets or floorlets, and the calculation of the minimum value will therefore involve applying the appropriate formula to each of these optlets.

The underlying asset for a cap, floor, or collar is a forward interest rate.
