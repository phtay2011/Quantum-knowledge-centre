Management Rule page

Use this page to create and maintain management reporting revaluation rules and methods by type and by rate source. These rules can be used for all reporting of management information (for accounting reports, either Management Control or Accounting Rule defaults may be selected).

To add a new rule record - click New; enter a unique Control Record Name; select the Entity, CCY Translation Rate Type, Reporting CCY, Par Rate Name for Zero Curve and its Type; on each tab enter one or more management reporting revaluation rules for the corresponding transaction type as required (see below), and click Save.

TIP: For each transaction type, create a rule for ALL Instruments and then a line for each specific instrument of that type that requires different treatment.

To list records - Click to open the browse window. (Click the Configure link to change browse list properties.)

To change a record's details - Select it in the browse list, optionally change key values if required; add, edit, and/or delete revaluation rule entries (grid rows) for each transaction type on the corresponding tabs as follows, and click Save.

  • To add a new revaluation rule entry - in the tab's grid menu for the required transaction type, select New; in the dialog box select an Instrument (Contract for Futures) and Entity (ALL is valid in each case); select the Rate Type and specific transaction type-related settings; then click OK. A grid row is added for the new entry.
  • To edit a revaluation rule entry - click a data value in the grid; in the Edit window, make the required changes; then click OK.
  • To delete a revaluation rule entry - select the grid row; in the tab menu select Delete; then confirm.

To delete a rule record in full (not recommended; Deactivate instead) - Select it in the browse list; in the page menu, select Delete; then confirm.

To deactivate a record - Click , select the record, and in the page menu select DeActivate. You can later reactivate it in the same way.
