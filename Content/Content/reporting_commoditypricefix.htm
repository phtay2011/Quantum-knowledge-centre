Commodity Price Fix page

Use this page to set prices for one or more Commodity deals (Swaps and/or Exposures), using either system-calculated prices, or a value you specify.

TIP: You can also Price Fix on the Dealing page (for the current deal only) and - for a Swap deal, from the Settlements page (for a single Price event that is Unactioned).

Filters - Enter Fix Start Date and Fix End Date to set the date range and optionally select Instrument, Commodity, Settlement Currency, Entity, Reference Currency, Rate Name, and/or Rate Type, Deal Type, and Scope values to reduce the result set. Blank settings mean all in each case. In Scope, select Next Fix to filter for the next price fix that will occur, or select Last Fix to filter by the most recent price fix.

WHAT DO YOU WANT TO DO?

  • Select a different Query or previous run - Click on the current Query heading and select the Query or run in the browse window. By default, the system retains run results according to the Expiry Strategy settings for the Query configuration (on the Properties tab) EXCEPT THAT selecting Keep in the Page menu keeps the current run results only.
  • Reconfigure the Query or configure a new one - Click the Query Configure link, make your changes, add expressions, and save them.
  • Select a different result View - Click on the current View heading and select in the browse window.
  • Change the View or create a new one - Click the View Configure link, make your changes, and save them.
  • Run the current Query - Enter specific Filters settings as required and click Run Query.

    TIP: Use relative dates in date fields so you can reuse queries in future runs. The returned data is subject to Resource Security.

  • Open the Deal page for a single item - click on the Deal No value for the item and in the quick menu, select Deal.
  • Price Fix one or more deals - in the results grid, select the deal periods and/or sides you want to price and EITHER:
    • click Price Fix to apply the Average Commodity Price function in each case, OR
    • click Set Price and in the dialog box enter the price to be applied in each case.

    The Calculated Price column shows the results. Errors are indicated by a red bar at the left of the grid row (click the bar to view the error message).

  • Undo the Price Fix of one or more deals - in the results grid, select the the relevant items and click the Undo Price Fix button.

    The Calculated Price column shows zero in each (successful) case. Errors are indicated by a red bar at the left of the grid row (click the bar to view the error message).

  • Export a standard report - Select Export in the page menu to export a report to CSV, PDF and XML formats.
  • Export a Crystal Report - Use the export control in a Crystal View to export a export a report to a variety of different formats.
