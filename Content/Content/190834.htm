Changing the Current Scenarios Definition

Use the following procedures to maintain a Scenarios Definition record.

To add a new Scenario:

  1. In the Scenarios grid menu, select New^ (or click the New button above the grid).

    The New Scenario dialog box appears.

  2. Enter a new Name for this scenario.
  3. Select the Method for specifying scenarios:
    • Shift - to specify shifts to market rates (in absolute or percentage terms), or
    • Rate - to specify outright rates (that is, the shift amount is solved from the specified scenario rate).
  4. Select which Shift Rate is to be shifted: Par or Zero.
  5. Click OK.

    The dialog box closes. Your new scenario is added to the Scenarios grid.

  6. In the Scenarios grid, select your new scenario row.
  7. In the Shifted Rates area, click the Add button above the grid.
  8. In the Select Shifted Rates browse list, select one or more rates as required; then click OK.

    For each rate you selected, a new row is added to the Shifted Rates grid for that rate, showing the Rate Name, Rate Group, Rate Type, and Shift Type.

    TIP: When you select a rate row, its points are displayed in the Rate Points grid on the Rates sub-tab - either by Instrument (for a Security rate) or by Term (for other rate types), and a graphical view is shown on the Graph sub-tab. The Rate Points grid columns show for each point:

    • Market Rate, Shift, and Scenario values;
    • if a Spectrum scenario - the Hedge instrument (the default is the underlying instrument used in the construction of the Yield curve);
    • if a Security rate - the Instrument specified in the Rate record for that point; or
    • for other rate types - the Term specified in the Rate record for that point.
  9. Click Save.

To view and optionally edit individual Rate Points for a Shifted Rate in a Scenario:

  1. In the Scenarios grid, select the row whose rates you want to select from.
  2. In the Shifted Rates grid, select the rate you want to display.

    That rate's points are displayed in the Rate Points grid on the Rates sub-tab.

  3. To edit a rate point's settings, select that point row in the Rate Points grid, and in the grid menu, select Edit (or click on a linked value in the row).

    The Edit Rate Point dialog box appears.

  4. As required, edit the Shift or Scenario value; if a Spectrum definition, also select the Hedge instrument*; then click OK.

    *TIP: Your selected Hedge Instrument may be a Bond, Discount, FRN, Futures, or Futures Option instrument. The default is the underlying instrument used in the construction of the Yield curve.

  5. Repeat Steps 3 and 4 to edit further points if required.

    To see the effect of your changes graphically, open the Graph sub-tab.

To linearly interpolate between two Rate Points:

TIP: Use this procedure to apply linear interpolation between start and end points and shifts, with the intermediate values calculated for you.

  1. In the Scenarios grid, select the row whose rates you want to select from.
  2. In the Shifted Rates grid, select the rate you want to display.

    That rate's points are displayed in the Rate Points grid on the Rates sub-tab.

  3. On the Rates sub-tab, click Interpolate.

    The Interpolate dialog box appears.

  4. Select the Shift Column (Shift or Scenario); select the Start and End rate points and Shift or Scenario values as required; then click OK.

    The Rate Points grid shows the interpolated values and to see the effect of your changes graphically, open the Graph sub-tab.

To edit an existing Scenario's settings:

  1. In the Scenarios grid, select the row you want to edit.
  2. In the grid menu, select Edit (or click on a linked value in the row).

    The Edit Scenario dialog box appears.

  3. Edit the required fields and click OK.

To edit the Shifted Rates for an existing Scenario:

  1. In the Scenarios grid, select the row whose rates you want to edit.
  2. In the Shifted Rates grid:
    • To add more rates - click the Add button above the grid; in the browse list select one or more rates as required; then click OK.
    • To edit an existing rate - select that rate row; in the grid menu select Edit (or click on a linked value in the row); in the Edit dialog box, change the settings as required; then click OK.
    • To delete an existing rate - select that rate row; in the grid menu select Delete; then click OK to confirm.

To delete an existing Scenario:

  1. In the Scenarios grid, select the row you want to delete.
  2. In the grid menu, select Delete.

    The Confirm Delete dialog box appears.

  3. Click OK to confirm.

    The Scenario row is removed from the Scenarios grid.
