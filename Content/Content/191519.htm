Money Market & Securities

Separate dealing pages apply for Money Market and Security deals (prior to Version 6, the Money Market Deal program catered for both).

The following example shows a Money Market deal page.

The following example shows a Security deal page.

The Roll Deal action can be performed via the page menu on the Money Market or Security page, to display the Roll Deal dialog box, as shown in the following example (prior to Version 6, Roll Deal was a separate program).

Customizing a Money Market or Security deal can be done from the Deal Map tab, either by selecting a row on the grid and changing the values in the Edit dialog box, or by using the BULK Insert... or Replace Forward... page menu items (as shown below):

Selecting Replace Forward opens the Bulk Replace dialog box, as shown in the following example. The items displayed will include and be forward of, the row selected in the Deal Map grid.

The data in the dialog box can be copied to and from a spreadsheet using the CTRL-C and CTRL-V keyboard functions.

  • NOTE:
  • The Deal Map will always recalculate using this functionality.
  • The Bulk Insert… and Replace Forward… functionality is also available in the Interest Rate Option, Cross Currency Swap, and Interest Rate Swap pages.
  • For a YIELD PERIODIC Money Market deal only, an additional Capitalize Interest function is also available in the page menu.
