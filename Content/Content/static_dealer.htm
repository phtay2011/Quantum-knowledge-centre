Dealer page

Use this page to store information about users designated as Dealers (on the Default perspective's User page), and to group them for reporting and authorization purposes. Dealer is a required entry field in all Dealing applications.

NOTE: If the Replication Service is running, these records can also be viewed (but not edited) on the Analytics perspective page.

To add a new Dealer group record - click NewGroup, enter a unique Name, text Description, and select a Location; ; click Add and select members of the group; then Save.

TIPS:

  • To make this group a member of another group, access the other group record and add your new group to that one.
  • The breadcrumb trail at the top of the page indicates the group structure of the current record where applicable.
  • You cannot add or delete individual Dealer records.

To list all records - click . (Click the Configure link to change list properties.)

To change a record's details - select it in the browse list, edit the relevant data fields, and Save.

To change group members - select the group in the browse list; then:

  • To add members - click Add and, in the Browse window, select one or more Dealers and/or groups, then click OK. Save your changes.
  • To remove a member from the group - click the x button at the right of that member's grid row.

To move the current record into a group - in the page menu, select Move; in the Select Group to Move To dialog box, select the target group, and Save your change. The current record becomes a member of the selected group and, if previously in another group, is removed from that group.

To remove the current record from its parent group - in the page menu, select Move; in the Select Group to Move To dialog box, select Move to Top, and Save your change. The current record is now "outside" all groups.

To delete a Dealer group in full - select it in the browse list, in the page menu select Delete, and when prompted, confirm.

NOTE: This procedure completely removes the record from the database. To prevent use of the record but retain it in the database, deactivate it instead.

To deactivate an individual Dealer or group - select it in the list and select Deactivate in the page menu. (You can later reactivate it in the same way.)

TIP: A group can be deactivated only if all its members have been deactivated.
