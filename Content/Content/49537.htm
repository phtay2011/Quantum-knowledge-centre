90-Day Bills, Sydney Futures Exchange

Use the standard >Bills formula to price a 90-Day Bank Bill traded on the SFE.

This is the standard Discount or Bill formula.

For a Simple Interest Rate Option - Floor using the Discount formula:

Quantum accounts for the discount amount as Interest in the General Ledger.

Futures contract value

To calculate the contract value of an SFE 90-Day Bank Bill Futures contract, the system uses the Bill formula:

Contract Value = (365 x Face Value)/(365 + (Yield/100 x Days to Maturity))

Futures tick value

To calculate the tick value of a Futures contract at a given price, Quantum compares:

  • Contract Value at the given price (for example 95.00, which equates to 5%)

    -to-

  • Contract Value at a price of 0.01% of yield above this (for example 94.99, which equates to 5.01%)

Each of these Contract Values is rounded to two decimal places, and the difference between them is the Tick Value.

For an SFE 90-Day Bill Option, select ROUND TO 2 PLACES in the Strike Rounding field on the Futures and ETO instrument page.

Option premium

To calculate the value of a single point of premium (for example, 0.01% p.a.), Quantum compares:

  • Contract Value at the Strike Price (100 minus annual yield)

    -to-

  • Contract Value at that strike price less one point (for example, 0.01% of yield above the strike yield)

Each of these Contract Values is rounded to two decimal places. Quantum then multiplies the dollar value of a single point of premium at the strike price by the number of premium points quoted, to derive the Option premium.

This result is rounded according to the currency - for example, to two decimal places in Australian dollars.
