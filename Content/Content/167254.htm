Bond Interest Rate Option Instruments in Quantum

Bond Interest Rate Option instruments are used to provide defaults for option deals based on underlying Securities. A Bond Option deal can be settled in cash or by physical settlement. The default settings for the Bond Security deal created by physical settlement are provided by the Settlement Instrument specified for the Interest Rate Option Instrument. For more details, see Bond Interest Rate Option Deals in Quantum.

Default values for Bond Interest Rate Option deals are provided when an Interest Rate Option deal uses an instrument with the following settings.

Details area

Option Type

Either:

  • Call - to give the buyer the right, but not the obligation, to buy the underlying bond at the exercise price, or
  • Put - to give the buyer the right, but not the obligation, to sell the underlying bond at the exercise price.

Exercise Style

This setting determines when Interest Rate Option deals based on this instrument may be exercised or lapsed; either:

  • EUROPEAN - to exercise or lapse on the deal Expiry Date, or
  • AMERICAN - to exercise or lapse at any time up to the deal Expiry Date.

Settlement Type

The default method of settlement; either:

  • PHYSICAL - where settlement is by entry into a Swap deal, or
  • CASH - where settlement is by cash payment.

NOTE: You must define a Settlement Type before you can exercise a deal using this instrument.

Settlement Instrument (MMkt)

(Mandatory for Bond Options.)

This instrument provides default settings for the Security deal that is created when the Swaption deal is exercised (and Settlement Type is PHYSICAL). This setting is required to ensure the Swaption is valued correctly (even if Settlement Type is CASH).

NOTES:

  • Quantum supports calls or puts on instruments using the BOND calculation formula only.
  • For analysis and valuation purposes, ensure the underlying instrument has a Revaluation Method set in the applicable Accounting Rule record (on the Money Market & Securities tab).

Premium Paid In

This setting gives the default Premium Date value for Interest Rate Option deals based on this instrument. Select:

  • ADVANCE - to use the Deal Date, or
  • ARREARS - to use the Exercise Date.

Pricing Method

This setting determines whether the price of deals based on this instrument is to be entered as a YIELD (interest rate style) rate, INVERTED YIELD, or a PRICE (capital amount).

If PRICE is selected, Strike Price and Reference Price fields appear on the Interest Rate Option deal page; otherwise, Strike Rate and Reference Rate fields appear.

Show broker

Select this option to include the Broker field when entering a deal based on this instrument.

That Broker field can then be used to record a third party or intermediary Counterparty, where one is used to transact the deal.

Broker is required

(Available only when Show Broker is selected.)

Select this option to require users to select a Broker before saving deals based on this instrument.

Defaults area

The same fields apply to all Option types.

For a Bond Interest Rate Option:

  • the Currency field is read-only and set to the currency for the selected Settlement Instrument (if none, a currency must be selected at deal entry time), and
  • all other fields in this area are optional.

For field details, see Defaults Area Settings (all Interest Rate Option Types).
