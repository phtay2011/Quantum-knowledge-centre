Details tab

This tab includes the following fields. Any default values are derived from the selected Instrument.

Buy/Sell - Select Buy, Sell, Borrow, or Lend. (Click in the field for details.)

Face Value - Select the currency and enter the face value of the bond.

Capital Price / Cash Price / Yield (labeled as the selected Price Type) - Enter the capital price OR cash price OR yield for the deal, respectively.

Coupon Rate - Enter the coupon or interest rate, as a percentage.

Dealt Indexation (available if Bond Type is Index Linked) - the index factor used to adjust the Principal and Coupon values of the Index linked Bond.

Deal Date, Settlement Date, Maturity Date - Enter the relevant dates.

Interest Frequency - Select the period interval between interest payments (Daily, Weekly, and so on), or select Maturity (for one compounding interest payment on maturity), or None (if available) (for no compounding). This setting determines the interest cash flow dates, starting from the first payment date.

Previous and Next Coupon - Enter the relevant dates.

Cum/Ex - Select Cum or Ex to specify whether or not the forthcoming coupon value will be included in calculation of the settlement price, respectively.

Cycle on end of month - Select or clear this option to specify whether or not to move the next Payment Date to the last day of the applicable month. For example, if Frequency is Semi Annual and First Payment Date is 30-Sep-2015, the next Payment Date will be either 31-Mar-2016 if this option is selected, or 30-Mar-2016 if not.

Create Call / Put (available if Bond Type is NOT Index Linked) - Select this check box to display items for specifying a Call or Put option on the bond, that option's Style, and its Exercise Date.

Allow amortization (available if Bond Type is NOT Index linked) - Select this option if the principal is to be amortized or accreted; otherwise clear the option. When the option is selected, three related fields are displayed for you to enter the Principal Frequency (of payments), the First principal date, and Residual Amount.

Repo Instrument (available if Buy/Sell is Borrow or Lend) - Optionally select a Repo instrument name and enter the Repo deal's Maturity Date in the adjacent field. This instrument provides Repo-related default settings and analysis codes. See Repo Pricing.

Calculate (read-only and set to Value if Price Type is NOT Yield, OR on a recalled deal) - Select Value to calculate price or Yield to solve for the yield given the Cash Price you specify in the adjacent field.

Settlement grid - This grid displays one or more settlement-related values calculated when the Price button is clicked.

For details of other items, click on them when the Help panel is open.
