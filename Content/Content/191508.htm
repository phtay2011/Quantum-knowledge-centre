Cash Management

Use the Cash Management section to look at actual and projected cash flows and cash forecasts, manage bank reconciliation, and learn how to use CashXplorer.

CashXplorer

CashXplorer is available in the main menu under Cash Management.

The CashXplorer page is split into two sections: the Query section where you can configure and manage your queries/views, and the View section where you can select the various views to display the query results.

When you follow the Configure link in the Query section, a Configure Query panel opens. The following set of diagrams give examples of each of the tabs in the Configure Query panel.

  • Filters tab

  • Horizons tab

  • Pools tab

  • Adjustments tab

  • Properties tab

In the View section, you can select one of three types of view: Cash Position, Forecast, or Summary.

An example of a Cash Position view showing report results is given below.

When you follow the Configure link in the View section, a Configure Pivot panel opens. The following diagram is an example of the Display tab in the Configure Query panel for a Cash Position view.

Actual Cash Flow / Projected Cash Flow

There are separate pages for Actual Cash Flows and Projected Cash Flows. (Prior to Version 6, these were two "modes" within the one program).

Account Adjustment / Interest Adjustment

There are separate pages for Account Adjustment and Interest Adjustment. (Prior to Version 6, these were two "modes" within the one program).

Cash Forecast

There is a separate page for Cash Forecast, available in the main menu under Cash Management.

TIP: Prior to Version 6, this was the Deal Set program in Forecast mode, and the previous Standard mode has been replaced by the Deal Set page, available in the main menu under Dealing.

Bank Reconciliation

Bank Reconciliation functions are provided under three sections of the Cash Management menu: Bank Import, Bank Reconciliation, and Auto Match Setup.

Bank Import section

  • Bank Statement - This page enables you to browse, view, or delete imported bank statements. You can also access this page from the Prior Day Match page by selecting an account, clicking View Statements to display a list of statements for that account, and selecting a statement. An example of the page is given below.

  • Import Balances - An example of this page is given below.

    TIP: Prior to Version 6, this was the Bank Account Balance Upload function within the Account Adjustment program (in Balance Entry mode).

  • Bank Transaction Rule Set - this page enables you to create and maintain rules to manipulate imported bank statement and transaction data before it is loaded into Quantum. An example of this page is given below.

    TIP: Prior to Version 6.6.1, Bank Importer supported data manipulation using INI or TXT files only.
