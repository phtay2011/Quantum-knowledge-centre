Create a Position Query (Spectrum Analysis)

  1. Use the FIS Quantum Analytics Explorer Queries page to navigate to the desired folder and in the page menu, select New Position.

    The Position query page opens with the default query name New Query.

  2. Click the Configure link.

    The Configure Query area opens.

  3. On the Defaults tab, enter the required date values: Position Date, As At Date, Start Date, and Nett Date.
  4. On the Properties tab:
    • enter a Name for your new query,
    • select appropriate Definitions: Assumptions*, Columns*, Filters*, Mapping Profile, Time Profile, and for Scenarios, select a Spectrum type;
    • select or clear the Scenarios-related options Include yield rates, Include FX rates, and Include volatility rates to include or exclude those rates in the Scenarios; and
    • select the required Expiry Strategy options.

      TIP: The starred fields are mandatory and default to the Name value if you do not select an existing Definition in each case (so that a new Definition will be created with the same name as the query).

      In the other Definitions fields, you must EITHER select an existing Definition of that type OR use the page menu's New Definition function to create a new one if required.

  5. On each of the tabs corresponding to each Definition type, specify the required settings, as detailed in the following topics.
    • Specify Assumptions,
    • Specify Columns (including Calculated Columns),
    • Specify Filters,
    • Specify a Mapping Profile,
    • Specify a Time Profile, and
    • Specify Scenario Settings.
  6. Save your query configuration.
