CashXplorer View

The View is the working area of CashXplorer, where the results of a query are displayed.

When you select a Data Version:

  • If you have multiple Views defined, the View that was created last will be displayed.
  • If you have only one View defined, that View will be automatically used when you open the Data Version.

The View Configure area has ten tabs.

  • The Display tab shows options for currency, display filter, rounding, and whether concentration flows and/or Intraday flows are displayed. Use this tab to temporarily change these in the grid.
  • The Attribute Filters and Pool Filters tabs display account attributes and Pools you have selected for this View and allow you to temporarily hide them in the grid.
  • The Export tab displays the default header selected to define the Data Version when you export it to Excel. You can alter these values here, and select Export in the page menu to export the report.
  • The Row Groups, Columns and Captions tabs allow you to control the data hierarchy displayed in the grid.
  • The Drill Down tab enables you to examine the following cash flows:
    • concentration flows
    • total flows
    • total flows and floats
    • the lowest level in the cash flow Attributes/Pools groupings

TIP: Use the Tab key to move around the display easily.

Making Changes to the Grid

You can make changes to the grid to change the way information in the Data Version is shown. You can do this using:

  • tabs in the View
  • menu options

These changes do not affect the information itself and therefore do not make any changes to the Data Version. See Customizing the View.
