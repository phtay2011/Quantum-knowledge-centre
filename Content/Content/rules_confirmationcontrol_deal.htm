Deal tab, Confirmation Rule page

Use the Deal tab to create and maintain, for an Entity, rule record entries for paper (print/e-mail) and/or electronic confirmations of Deal actions. These entries specify which types of Deal Confirmation message are to be created, and how they are to be output, for the different combinations of Instrument, Transaction Type, Counterparty, and so on.

Selection criteria are applied in order (from the top row down), so specify more general and "catch-all" criteria last.

WHAT DO YOU WANT TO DO?

Before you start - Use the Browse list to access the Entity Approval Rule record you want to update, and open the Deal tab.

To automatically submit deal action confirmations for printing or e-mailing (optional) - select Process paper confirmations automatically.

To add a new rule entry (from scratch) - In the grid menu, select New; in the dialog box, define the Matching Criteria to select which deal actions the Deal confirmation rule entry applies to; and:

  1. under Settings - select Require approval if approval is required before a deal action confirmation will be output, set External Method to an applicable Payment Method or to MANUAL or NONE, set Actioning to the type of action to apply to the applicable cash flows following deal action approval and/or confirmation matching, and select the Print Device and, number of Copies;
  2. under Electronic - if External Method is SWIFT Confirmation, specify the Confirmation message file Output Subfolder and Electronic File Extension and, if External Method is Misys (for Finastra), select the Print Paper Action method (ALWAYS, ON FAIL, or N/A if not applicable);
  3. under Paper - select the Print output option/s (NONE, PRINT, EMAIL, and/or FILE) and, if Print is FILE enter the File Subfolder and Paper File Type; then
  4. click OK. A grid row is added for the new entry.

To add a new rule entry (by copying) - In the grid, select the entry you want to copy; in the grid menu, select Copy; in the dialog box, edit the Matching Criteria, general Settings, Electronic, and Paper settings as required (details above); then click OK. A grid row is added for the new entry.

To edit a rule entry - click a data value in the grid; in the Edit window, make the required changes; then click OK.

To delete an entry - select the grid row; in the grid menu select Delete; then in the dialog box, click OK to confirm.
