Confirmations Processing page

NOTE: The Process confirmations and Match confirmations Operational Settings must be selected before you can fully process and match confirmations.

Use this page to manage Quantum deal action confirmation messages. You can selectively view, monitor, and manually initiate actions on electronic and paper confirmations, including matching, rejecting, re-sending, amending, printing, e-mailing, and so on.

Filters (defaults - you can add others) - Enter values and make option selections to reduce the result set.

Options -- Select Include all deal actions to ignore any filtering on Action Type.

WHAT DO YOU WANT TO DO?

Select a different Query - Click on the current Query heading and select in the browse window.

Change the Query or create a new one - Click the Query Configure link, make your changes, and save them.

Select a different result View - Click on the current View heading and select in the browse window.

Change the View or create a new one - Click the View Configure link, make your changes, and save them.

Run the current Query - Enter specific Filter and Option settings and click Run Query.

TIP: Use relative dates in date fields so you can reuse queries in future runs.

Export a standard report - Select Export in the page menu to export a report to CSV, PDF and XML formats.

Export a Crystal Report - Use the export control in a Crystal View to export a export a report to a variety of different formats.

NOTE: The returned data is subject to Resource Security.

IN THE RESULTS GRID, YOU CAN...

  • Action multiple items - select the check box at the left of each item you want to action, and in the grid menu, select the action required or (for some actions) click the relevant button (Match now, Match, Unmatch, or Reject).
  • Action a single item - select the item row in the grid and in the quick menu, select the action required (or use the grid menu or buttons).
  • Drill down on an item to view more details - select the item row in the grid and in the quick menu, select: Open Paper, Open Deal Ticket, or Open Electronic to download a PDF giving the selected details; History to display message and match history; or Deal to open the relevant Dealing page.
  • Go to the Settlement page for an item - select the item row and in the quick menu, select Settlement to open the Settlement Processing page with filters pre-populated for that item.
  • Add a comment to Confirmations (NOT available if Message Status is Not Created, Not Required, or Ready for Creation) - select the item row/s and click Add Comment, enter the Type and text Comments, and click OK. The Message Status and deal Counterparty Reference do not change, nor do Actioned cash flows change to Unactioned.
  • Clear an unwanted item from the processing queue (available only if Message Status is Ready for Creation or Creation Failed) - select the item row and in the grid menu, select Mark as Not required.
