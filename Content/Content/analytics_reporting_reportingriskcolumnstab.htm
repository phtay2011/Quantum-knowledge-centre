Columns tab

Use this mandatory tab to specify default data columns and groupings for a query.

The grids on this tab vary depending on the type of report being created. Their default settings depend on whether or not an existingColumns Definition record is specified on the Properties tab.

  • If so, the defaults are taken from that Definition record.
  • If not, a new Definition record is created with the same Name as the Query (as on the Properties tab) system defaults apply.

In either case, any changes made on this tab are saved in the specified Columns Definition record when the query configuration is saved.

Use each grid as follows:

  • Group By - to specify the report columns determining the level of detail or aggregation of the results,
  • Point in Time - to specify one or more sets of report columns to be presented based on particular dates,
  • Movement (Cash Flow and Position queries only) - to specify one or more sets of report columns to be presented based on data changes made on particular dates,
  • Database (Cash Flow and Position queries only) - to specify one or more sets of database fields to be directly extracted from the database based on particular dates, and
  • Calculated - to add one or more report columns based on specified field calculations.
