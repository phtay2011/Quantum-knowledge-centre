Valuing European Swaptions

This section describes the two parts of the Swaption valuation procedure.

  1. Determining the Forward Swap Rate

    The forward rate is the fixed rate that would be exchanged for LIBOR if a newly issued swap were entered into today for which the first rate set was scheduled for the maturity date of the Swaption. That is, the forward swap rate is the fixed rate that equates the values of the fixed and floating legs of the swap when both are valued today.

    The calculation of the forward swap rate requires valuing both the fixed and floating leg of the swap.

    This process involves the following steps:

    1. Determining the reset and payment dates for the underlying Swap
    2. Projecting cash flows for the fixed leg
    3. Projecting cash flows for the floating leg
    4. Determining the discount factors for the underlying Swap
    5. Valuing both legs of the underlying Swap
    6. Calculating the forward swap rate
       
  2. Valuing the Swaption

    Once the forward swap rate has been determined, the Swaption value can be derived from the Blacks model. The only input to the Blacks model that is not directly specified, or which has not yet been calculated as part of the procedure to this point, is an estimate for the volatility of the forward swap rate.

    This process involves the following steps:

    1. Estimating Swaption Volatility
    2. Estimating Swaption values using the Blacks model

    See: Estimating Swaption Volatility and Values
