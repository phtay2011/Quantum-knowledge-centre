Identifiers tab

Use this tab to record required Clearing House or External Method codes for identification purposes on confirmations or for electronic settlements (EFTs) involving this Counterparty. Initially (depending on Categories tab selections), this tab shows:

  • A BIC Code field (if a Bank category applies) - use this to record a Bank Identifier code that will be used in reporting a bank account reference (in particular, with SWIFT Confirmations and EFTs).
  • A SWIFT Data Source Scheme field (if the Custodian category applies) - use this optional value for a code that will be output in Stock Flow messages when:
    • The Payment Method for the corresponding cash flows is SWIFT.
    • The Stock account has a Unique Identifier value that applies for this code.
    • The Stock account's specified SWIFT MT Format Template indicates that this Data Source Scheme code and the Unique Identifier are to be output.

    For example, enter ECLR for Euroclear.

You can add and record other Clearing House or External Method codes, where required. Identifiers are set up in External Method records, which can have a Method Type of Confirmation, Payment, or Mapping. For example, SWIFT Confirmation and Misys (Finastra) External Method records have the Confirmation method type and are used to format the respective electronic Confirmation output; while the SWIFT External Method record has the Payment method type and is used to format the EFT output.

To add a new Identifier field:

  1. Use the Select Identifier button to open the Browse window, which shows the first ten External Methods that have Method Type set to Payment and Enable Identifiers selected.
  2. Select one or more External (Payment) Method identifiers. Use the Search field at top right to filter the list, and use the < and > buttons to page through the list if required.
  3. Click OK to close the Browse window. Your selected identifiers are displayed.
  4. In the Identifier field for each Pay Method, enter the identifier value for the Counterparty.
  5. Use Save to keep your changes.

NOTE: You cannot edit the identifier field label from the Counterparty page. If it must be changed, you can edit the name for the External Method.

To remove an Identifier:

  1. Use the Select Identifier button to open the Browse window.
  2. Either click the x button to the right of the identifier button at the bottom of the Browse window for the Identifier you want to remove, or clear the selection box for that Identifier in te browse list.
  3. Click OK to close the Browse window. Your selected identifier is no longer displayed.
  4. Use Save to keep your changes.
