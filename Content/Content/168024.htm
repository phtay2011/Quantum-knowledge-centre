Fixing Average Rate

NOTE: The ways in which Exotic currency options are set up and processed are determined by your Quantum license agreement. Contact your Quantum Distributor for more information.

Generally, average rate fixing of an Asian currency option would be done on a business day. However you can also do it for:

  • a past date that may have been a non-business day

    The next available business date is displayed, and you can do a backdated rate fix for that date, without validation.

  • the current day, when it is a non-business day

    The next available business date is displayed, and you can do a future rate fix for that date, without validation.

  • a future date that will be a non-business day

    The next available business date is displayed, and you can do a future rate fix for that date, without validation.

Procedure Summary

To perform an Average Rate Fix

  1. If the current result list does not show the deal you want, check the Filters values and that the selected view is either Average Rate Fixings or Default.

    TIP: Select the Average Rate Fixings view option to show only those (Asian) options this operation is valid for.

  2. If necessary, adjust the Filter values and re-run the query.
  3. In the Result list, locate and select the deal entry - which should have Average Rate Fixings in the Type column and a Status* of Not Fixed.

    *TIP: If the Results list does not have a Status column, use the view's Configure link and the Columns tab to add it.

  4. In the list menu, select Fix Average Rate.

    The Fix Average Rate dialog box opens. The Deal Number, Fixing Date, and Current Average are displayed as read-only values.

  5. In the dialog box, enter the new Fixing Rate* and move to another field.

    The New Average* is calculated (see below) and displayed.

    *TIPS: The default Fixing Rate is based on the FX cross rate defined in Exchange Rates. The new average rate is calculated from the Current Average and your entered Fixing Rate. See the examples below.

  6. Select a Dealer or accept the default, which is either the setting from Currency Option Instrument Types or yourself (the logged-on user - if your User record has the Dealer option selected).
  7. Optionally enter any text Comments for reference purposes.
  8. Click OK.

    The average rate fix operation is performed on the selected Currency Option deal. In the results list, the entry's Status changes to Fixed and a second entry for the deal is added to the list with a Type value of Exercise Lapse and a Status of None.

NOTE: When this action has been performed for a particular date, the action can only be undone by using the Rollback Deal function.

Calculation

The average rate calculation for Asian options is defined as follows:

Average Rate =Sum of all fixed rates up to required date/Sum of all corresponding day weighting values

The day weighting value varies, depending on whether the Day Weighting selected for the deal is Calendar or Business, as follows.

  • Calendar - each average rate fixing has a day weighting value equal to the number of calendar days from the fixing date to the next business day, and:
    • the numerator is the sum of all fixed rates for those calendar days,
    • the rates used for the holidays are the same rate as the last business day, and
    • the denominator is the sum of all day weighting values for those calendar days.

    For example, Friday would normally have a day weighting value of 3 assuming that Saturday and Sunday are defined as non-business days for the Currency.

  • Business - each average rate fixing has a day weighting value of 1, and:
    • the numerator is the sum of all fixed rates for business days only, and
    • the denominator is the sum of the day weighting values for business days only.
