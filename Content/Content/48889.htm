Premium calculation, Futures

Quantum uses the same basic formula to calculate the premium on an Exchange Traded Option, whether the pricing formula used in Futures Contract Types is PER TICK (for fixed tick contracts) or BILLS, BONDS, or SFE ROUNDED BONDS (for variable tick contracts).

To calculate the premium value of an Option, Quantum applies the formula:

This formula assumes that Option prices are entered in Premium points - for example, that a price of 17.5 basis points is entered in decimals as 0.175. However, you can modify the price entry format using the Shift Decimal Place field in the Price Entry panel on the Futures and ETO instrument page. This also affects a decimal Tick Size.

For a fixed tick contract where the price entry is in DECIMAL format, the minimum price movement can differ from the tick movement. You can enter these values in the Price Entry panels on the Futures and ETO instrument page, using different minimum price movements for the Option and its underlying Futures contract.

For a variable tick contract, the value per tick is different at each strike price. It is calculated first, then used in the above formula.

Examples

  1. Set up a Kansas City Board of Trade Wheat Option as follows:
    • Tick Size = 0.0025 (this is 1/4 cent, as it is for the Futures contract)
    • Minimum Price Movement = 0.0001
    • Tick Value = 12.50
    • Shift Decimal Place = -2 (to convert the price entry format)

    A price of 16.35 is a valid multiple of 0.01 (which is 0.0001 shifted by -2 decimal places).

    Premium Value on 50 lots:

    = 150 Lots * 16.35/0.0025 * 12.50 * 10-2

    = $122,625.00

  2. Set up a Sydney Futures Exchange 10-Year Bond Option as follows:
    • Tick Size = 0.01
    • Minimum Price Movement = 0.005
    • Tick Value = 99.2137000 at a strike price of 94.50
    • Shift Decimal Place = 0

    Premium Value on 240 lots:

    = 240 * (0.005 / 0.01 * 99.2137000 rounded to 2 decimal places)

    = $119,056.80

    The Minimum Price Movement is independent of the Tick Size, so you could enter a price such as 0.055. You would enter this as 5.5 if the Shift Decimal Value is set to -2.

    See also Sydney Futures Exchange Calculations and Variable Tick Value - Options Formula.
