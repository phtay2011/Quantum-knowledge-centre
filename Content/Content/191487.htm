Live Updating

You can set Analytics, Reporting, and Processing pages to automatically update, based on system events, time intervals, or execution of a scheduled report run.

Pages that Support Live Updating

Analytics pages:

  • Analytics Query

Reporting pages:

  • Cash Flow Reporting
  • Position Reporting

Processing pages:

  • Settlements
  • Deal Approval
  • Settlement Approval
  • Confirmations
  • FX Cover Requests

For these pages, updates are triggered by certain system events. When such events occur, the data for the live pages will either be partially updated as appropriate for the change, or - for Analytics pages - fully updated. The pages will then reload the updated data.

Deal Monitoring

Changes to all the main deal types are monitored for Live Updates. Also monitored are:

  • Actual and Projected Cash Flows,
  • Account and Interest Adjustments,
  • Cash and Stock Transfers,
  • Hedge Relationships, and
  • Quotes.

NOTE: Deal Sets, Deal Requests, Initial Margin deals, Unit, Unit Holding, and Letter of Credit deals are NOT monitored for Live Updates.

Changes to the view are processed only if the deal and flow changes are already part of the report. New deals are added only if they fall within the selection criteria of the query.

Rate change events are queued for one minute before the full report run is executed on the latest rates, to allow for all rate updates to be completed.

Starting, Pausing, and Stopping Live Updating

To start running a query with live updates

  1. Select Auto-run… from the Run Query button drop-down menu, as shown below.

    The Auto-run Query dialog box appears, prompt you to select a run option, as shown below.

  2. Select the appropriate option and click OK.

    TIPS:

    • Use Update Live when possible, because it is the most efficient approach.
    • Use Auto-run when Update Live is not available or the report depends on data not covered by the change events.
    • Use Update Latest when report results need to be shared by multiple users, or when a query takes a long time to run and is not so time-critical.

To pause and restart live updating for a query

  1. Click the Pause button.

    Live updating is paused and a Resume button is now displayed.

  2. To restart automatic updating, click Resume.

Events that Stop Live Updates

Live updates for a page will stop when you:

  • navigate away from the page,
  • run the query any other way,
  • change the query,
  • create a new query, or
  • select a previous run.
