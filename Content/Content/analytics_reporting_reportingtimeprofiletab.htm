Time Profile tab

(This tab appears only when an existing Time Profile Definition record is specified on the Properties tab.)

Use this tab to define time "buckets" that can be applied across a query, and to see a preview of the actual dates those settings will generate. Time Profiles allow for cash flows or deals to be grouped into user-defined time buckets, which can be based on periods, frequency, or actual dates.

The default settings on this tab are taken from the Time Profile Definition record specified on the Properties tab.

Any changes made on this tab are saved in that Definition record when the query configuration is saved.

On the Details sub-tab:

  • The initial settings are taken from the Time Profile Definition specified on the Properties tab.
  • Use the areas on the Details sub-tab as follows:
    • Fixed Date Options - these settings apply where the time profile is to be generated relative to a fixed recurrent date (not a variable date such as Today).
    • Business Days Convention - Shift Method convention and Holiday Center.
    • Time Profile - the main timing-related settings and one or more Terms entries
    • Hedge Equivalents (available only when Apply hedge equivalents is selected) - one or more sets of Hedge Equivalent parameters corresponding to each Terms entry, for composing an equivalent portfolio of standard Hedge instruments, used for calculating Yield curves: Entity, Instrument, and Currency.

[For more details of the items in an area, click on an item when the Help panel is open.]

On the Preview sub-tab - to list details of the Time Profile buckets your current Details tab settings will generate:

  1. Enter a Start Date (the default is the current date).
  2. Select the applicable Revaluation Timing: Start of Day, Intra Day, or End of Day.
  3. Click Preview.
