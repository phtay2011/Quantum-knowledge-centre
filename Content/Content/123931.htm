Netting by Deal Set

Netting of cash flows by Deal Set is achieved by selecting the Is part of a deal set option in the applicable Counterparty Netting Rule records.

This setting:

  • does not enable netting by Deal Set Instrument (it will net only by Deal Set number and only in accordance with Counterparty Netting Rules),
  • applies only to standard Deal Sets (not to Forecast Deal Sets), and
  • interacts with Settlements Processing options to enable Deal Sets to be netted in accordance with existing cash flow netting requirements.

The following business rules apply when Is part of a deal set is selected:

  • netting will apply only to cash flows within the same Deal Set,
  • if the Deal Set is unlinked from the deal, the appropriate netted cash flow is adjusted,
  • if the Deal Set is linked to a deal (with the correct netting requirements), the appropriate netted cash flow is adjusted,
  • if the Deal Set is changed, the cash flows will remain netted (no validation is necessary),

Clearing the Is part of a deal set option in an applicable Counterparty Netting Rule does not un-net cash flows already netted by Deal Set. To do that, use the results grid Undo Net menu function in Settlements Processing.

In Settlements Processing:

  • no cash flow netting will occur if Cash Flow Netting is not selected in the Counterparty record (on the Transactions tab),
  • if approvals are required, records must be approved before netting can take place,
  • Deal Set Number is not a mandatory Filter parameter, but you can restrict netting to certain deal sets by adding it and specifying one or more values or ranges,
  • netting will be:
    • proposed - if you click Run Query to trigger the creation of Net Proposals that you can review before actually creating the nets (by selecting Net Proposal items in the results grid and clicking Net as Proposed), OR
    • automatic - if you select Run Query and Net from the Run Query button's drop-down menu (netting proposals are skipped).

NOTES:

  • Adding or removing a deal from a Deal Set is not a financial deal change and does not affect any existing nets.
  • Independently of any Counterparty Netting Rules, you can also net cash flows that have the same deal number (in addition to the basic netting criteria). This is controlled by the Net by Same setting in Settlement Processing. See Netting Cash Flows in Settlements Processing.
