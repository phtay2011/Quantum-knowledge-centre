Mark As Reconciled

(Prior Day Matching only.)

Use the Mark As Reconciled function to mark one or more bank accounts as Reconciled up to the specified date. Successfully marked items will then become read-only from a Bank Reconciliation viewpoint (unless you later roll back to a bank account's previous Reconciliation mark).

On deal save, Quantum enforces Bank Reconciliation Match Integrity. Any Match statuses or Reconciliation Marks affected by additional, changed, or deleted flows will be reset or rolled back, to maintain Match Integrity during all phases of the flow life cycle. When a previously Matched flow is reset to Unmatched, all flows that participate in the same match will also be reset at the same time.

The integrity of the Reconciliation mark is maintained. If an account is marked as Reconciled, it cannot have any unmatched flows prior this mark that are not reconciled. This includes bank flows, external cashflows, and cashflows. This means that:

  • If the account was marked as reconciled for a date on or after the Value Date of unmatched bank flows, the Reconciliation Mark is rolled back to the last mark prior to the Value Date of the unmatched items.
  • If a bank flow, cash flow, or external cash flow is created with a Value Date prior to the last Reconciliation mark, the Reconciliation mark is rolled back to the last mark prior to the Value Date of that flow.
  • You may Unmatch only those flows with a Value Date equal to or later than the Last Mark date.
  • The status badge of Fully Reconciled/Marked will change only when you change flows dated prior to the Last Mark Date.
  • Fully Reconciled/Marked is the default status if the account has NO flows in the reconciliation period.

To mark an account as Reconciled

  1. In the Accounts summary, select an account you want to mark. The account should either be Balance level matched or Detail level matched, with:
    • all items fully matched at either level,
    • no Probable or Possible statuses present, and
    • matching totals.
  2. In the grid menu, select Mark As Reconciled.

    An error message is displayed if the account does not meet the above criteria. Otherwise the account is Marked As Reconciled (an indication is briefly displayed).
