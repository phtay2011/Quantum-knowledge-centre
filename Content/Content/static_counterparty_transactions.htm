Transactions tab

Use this tab to specify the following settings for the Counterparty.

(Settings are optional unless otherwise indicated.)

Country - the Counterparty's Country

Location (available when a Country is selected) - select a location within the Country, for use in one of two ways, depending on the associated Use for Cut-off Time only option setting, as below.

  • If Use for Cut-off Time only is cleared - this Location will, by default, populate the Location field for deals with this counterparty and be used to determine non-business days for those deals (this is the default).
  • If Use for Cut-off Time only is selected - this Location will not populate the Location field for deals with this counterparty, but will be used to determine daily cut-off times for those deals (but NOT for Cash Concentration Actual Cash Flow and Cash Transfer deals).

Taxes Applicable - the taxes for which the Counterparty is liable. These are the default taxes for deals with that Counterparty. Select from the drop-down list of Taxes available on your system.

Use Declared Tax Rate % (applies only to RESIDENT WHT tax types) - select this option to use the Declared tax rate specified in the applicable tax record/s; clear it to use the Undeclared rate.

Cash Flow Netting - select this option to enable default netting for the Counterparty. Default netting uses rules specified in Counterparty Netting Rules to automatically net cash flows in Settlement Processing.

  • You must define at least one Net Group in Counterparty Netting Rules before any cash flows can be netted.
  • You can override netting defaults, in part or in full, within Settlement Processing.

Exempt from Clearing - select this option to exclude the counterparty from having any of its trades cleared through a clearing house.
