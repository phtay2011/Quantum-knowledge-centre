Country Procedures (Quantum)

On the Country page:

To list Countries

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Country

NOTE: Countries are linked to currencies, therefore you must first set up your Currencies.

  1. Click New.
  2. Enter a unique Name, then select a Currency and enter a Short Name for the country.
  3. Select the BIC Type to determine the format for displaying bank account information on the Bank Maintenance screen - one of:
    • BIC Code - to display standard Bank Identifier Codes, or...
    • BSB Settle - to display a Bank/State/Branch identifier.
  4. Click Save.

    You have created a new Country record.

    NOTE: The system automatically creates a new Location record for your new country.

TIP: By default, your new country location will use the Business Day Calendar for The World location. To use the holidays of a specific Location group instead, go to the Location page, in the browse window select that Location group, add the new country location to the group, and Save your change.

To edit Country details

NOTE: You cannot change the key Name field. Create a new Country with the new name you want (and optionally delete the wrongly-named Country record).

  1. Click to list available records, or use the Search box.
  2. Select the record you want to edit.
  3. Edit the relevant fields.
  4. Save your changes.

To delete a Country

  • You cannot delete a record if that affects transaction, settlement, or financial data, and you cannot delete a group if it has members.
  • Deleting removes the record completely from the database. To prevent use of the record but retain it in the database, deactivate the record instead.
  1. Click to list available records.
  2. Select the record you want to delete (use the < and > buttons to page through, or type in the Search box).
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To deactivate a Country

TIP: You cannot deactivate a Country specified in active database records.

  1. Click to list available records, or use the Search box.
  2. Select the record you want to deactivate.
  3. In the menu, select Deactivate. This action will not appear if the record is already deactivated.

To reactivate a Country

  1. Click to list available records, or use the Search box.
  2. Select the record you want to reactivate.
  3. In the menu, select Reactivate. This action will not appear if the record is already activated.

To publish a Country

TIP: All fully approved new or changed Country records will be published automatically If the Country option is selected (on the Operational Settings page under Publish).

  1. Click to list available records (or use the Search box).
  2. Select the record you want to publish.
  3. In the menu, select Publish Selected.

To Publish all Countries

  • In the page menu, select Publish All.
