Amounts area, Deal Details tab, Currency Option Deal

TIPS:

  • The relationship between the currency pairs that are dealt must first be defined in a Currency Pair record.
  • The number of decimal places used for a currency is defined in the Rounding area of Currency Setup.
  • Currency Setup and Expiry Location (Dates/Premium area) are also used to validate business days.

Call Amount - Select the currency and enter the face value of the Call leg of the deal. This currency is the foreign currency for a Call, the domestic currency for a Put, and amount rounding is set by the rules recorded in the Currency record.

Strike Rate - Mandatory; the contract rate at which the deal is dealt. You can enter a Strike Rate, or allow the system to calculate it based on the Call Amount and Put Amount values. If Strike/Spot is set to STRIKE, this value may also be used to calculate the Premium Amount.

Put Amount - Select the currency and enter the face value of the Put leg of the deal, or allow the system to calculate the amount when you enter both a Call Amount and Strike Rate. This currency is the foreign currency for a Put, the domestic currency for a Call.

Spot Rate - The market spot rate at the time the currency option is dealt. If Strike/Spot is set to SPOT, this value may also be used to calculate the Premium Amount.

Barrier (Single Barrier options only)

  • With a Down option, set this barrier rate below the Strike Price. If the price of the underlying currency falls below this rate, a Down & In option activates and becomes a normal Call option; a Down & Out option expires.
  • With an Up option, set this barrier rate above the Strike Price. If the price of the underlying currency rises above this rate, an Up & In option activates and becomes a normal Put option; an Up & Out option expires.

Lower Barrier and Upper Barrier (Double Barrier options only) - Set these two rates below and above the Strike Price, respectively. If the price of the underlying currency falls below the Lower barrier OR above the Upper Barrier, a Double In option activates and becomes a normal Call or Put option; a Double Out option expires.
