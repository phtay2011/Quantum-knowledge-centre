XML Data Records (EFTs)

NOTE: XML transformation in Quantum is optional. It is not required, for example, if you are transforming EFT output using only ATS.

If XML transformation is to be performed on the EFT output for an External Method (Payment type), XML Data records are mandatory.

NOTE: Prior to Release 2020.2, this XML formatting data was available only in separate XML-formatted files.

Seed XML Data records are provided in your installed Quantum database for the SWIFT External Methods and are accessible via the XML Data page. See Supplied EFT XML Data records (SWIFT).

You can modify these and/or create new sets of XML Data records for new methods. See XML Data records for other External Methods.

CAUTION: If you edit the system-supplied seed records, your changes will be lost on any upgrade of your database. Always edit copied and renamed seed records .

The XTC Configuration record for each External Method:

  • is specified in the Transformation Configuration File field of the associated External Method Source record, and
  • gives the name of the required Data Template and/or Transformation XML Data record for that External Method.
