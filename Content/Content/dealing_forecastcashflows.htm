Cash Forecast page

Use this page to create and maintain Cash Forecast deal sets* that:

  • link sets of deals for cash flow forecasting and reporting purposes,
  • enable you to enter, manipulate, generate, convert, or analyze a large number of forecasted cash flows, and
  • can be used for scenario-building within CashXplorer, facilitating predictable forecasts.

Typically, you would enter Projected Cash Flows to forecast future cash flows, and later (optionally) change them to Actual Cash Flows.

*TIP: Cash Forecast records are Deal Set records based on Deal Set instruments with the Forecast option selected. They can also be accessed on the Deal Set page if their instrument also has the Standard option selected. The main difference is that this page provides Convert and Recalculate functions for Projected cash flows.

Use the different areas and tabs as follows:

  • Header area - to enter and edit identifying data for the Forecast Deal Set, and to create and recalculate recurring Projected cash flows where applicable.
  • CASH FLOWS grid - to view cash flows in the Forecast deal set, optionally change individual Projected cash flows, convert them to Actual Cash Flow deals, and drill down to individual deal details.
  • Template tab - to define standard details for recurring Projected cash flows in the forecast set.
  • Recurrence tab - to define the recurrence pattern of the forecast Projected cash flows, their life span, and any Non Business Day rule.
  • Variation tab - to enter projected percentage variations to daily and monthly base values of recurring Projected cash flows in the forecast set.
  • PC Analysis Codes tab - to link the Projected cash flows to predefined analysis codes.

To add a new Forecast set - click New, fill in the data fields, and click Save.

To list all Forecast sets - click the button to open the browse window (click the Configure link to change browse list properties.)

To change a set's details - select it in the browse list, edit the relevant data fields, and click Save.

To view the cash flows in a Forecast set - select the set, and browse in the CASH FLOWS grid.

To deactivate a set - select the set, in the page menu select DeActivate, confirm when prompted.

Resource Security can affect the information displayed for deals in a Forecast Deal Set.

For details of data items and tabs, click on an item when the Help panel is open.
