Using the Static Data Transfer Spreadsheet

IMPORTANT: Always use the spreadsheet included with the Quantum release to ensure correct functionality.

NOTE: If using your computer with a non-administrative account, you may receive an error that "Internet explorer has stopped working" when using this spreadsheet.
Ensure that your computer has User Account Settings (UAC) enabled.

Opening the Spreadsheet for the first time

When you open the spreadsheet for the first time, you will be prompted to trust the application that was saved with the spreadsheet. To do this:

  1. Open the spreadsheet and click the Trust this app button.

    The Static Data Transfer panel displays the Log On dialog to Quantum Six, as shown below.

  2. Click Log On.

    The Static Data Transfer panel displays the Upload and Download functions available for the current worksheet, as shown in the following example.

Downloading data

Each worksheet in the spreadsheet can be populated with the static data of that type currently in the database, as follows.

  1. Select the worksheet tab for the type of static data you want to download.

    The Static Data Transfer panel displays the Upload and Download functions available for that worksheet, as shown in the example above.

  2. Click the Download All button.

    If the worksheet already contains data, a warning appeasr stating that all data on the worksheet will be overwritten.

  3. Click OK.

The downloaded data displays in the worksheet and the Status column is set to Downloaded for each row of data.

Downloading Static Data Analysis Codes

NOTE: You can only add analysis codes that have already been defined in Quantum, and only those for Entities, Counterparties, Bank Accounts, and Broker Accounts.

  1. Select a cell located after the key fields (green colored cells) and type in SHIFT+CTRL+A on your keyboard.
  2. Enter the name of the analysis code as defined in Quantum in the dialog.

    This will add a column for the analysis code to the spreadsheet.

  3. Reload the right pane of the spreadsheet using the application menu. See Parts of the Spreadsheet: Right Pane for more information.
  4. Click Download in the right pane to download data for the analysis code from Quantum.

Uploading data

You can upload data that was entered in the worksheet to the database as follows.

To upload all rows

  1. Select the worksheet tab for the type of static data you want to upload.
  2. Click the Upload All button.

    The Status column for all rows changes to Updated or Inserted.

To upload selected rows only

  1. Select one or more cells in one or more rows, as shown in the following example.

    NOTE:Highlighting rows by selecting row headings is not recognized as row selection.

    The Upload button becomes available.

  2. Click the Upload button.

    The system uploads the selected rows and the Status column for those rows changes to Updated or Inserted.

Deleting data from the database

To mark data in a worksheet for deletion from the database

  1. Select the worksheet tab for the type of data you want to delete.
  2. Select one or more cells in one or more rows.

    NOTE:Highlighting rows by selecting row headings is not recognized as row selection.

  3. Click the Delete button.

    The Status column for the selected rows changes to Deleted.

Reviewing and fixing rows in error

When rows cannot be uploaded or deleted:

  • the Status column will indicate Update Error, Insert Error, or Delete Error, and
  • the Message column at the end of the grid will explain why the error occurred.

Data validation

The spreadsheet performs some basic data validation before it uploads the data. The system indicates validation problems automatically while editing the data, as follows.

  • Where appropriate, cells have drop-down lists showing the available values for that cell.
  • Cells will validate against data from other worksheets when appropriate (for example, Currency).
  • Required cells will highlight if left blank.
  • Rows with errors will highlight.

The final data validation occurs during the upload process.

Clearing all data from a worksheet

To clear all data from the current worksheet

  1. Press Ctrl+Shift+C.

    This removes data from the worksheet only.

    Alternatively, to delete data from the database, use the Delete button.

Exporting data to a QXT file

To export a selection of data from a worksheet to a QXT file

  1. Select the worksheet tab for the data you want to export.
  2. Select one or more rows to export.
  3. Press Ctrl+Shift+E to export the data on the current tab.

    An Open dialog is displayed.

    Enter or browse to the location, and enter the file name of the export file.
