Changing Counterparty Instructions from Dealing Pages (Existing Deals)

(First introduced in Release 6.5.1 SP, September 2017.)

From most Dealing pages with a Cash Flows tab, you can now view and change the Counterparty Delivery Instructions for a selected cash flow for saved deals (subject to applicable Module Security and Resource Security rights).

For deal types that can have large numbers of cash flows, you can include forward-dated cash flows when you apply your Counterparty Instruction change. That change will then apply to cash flows with the same Counterparty and in the same currency.

Changes cannot be applied to intercompany cash flows.

Implementation

On an applicable Dealing page's Cash Flows tab, the tab's grid menu now includes a COUNTERPARTY INSTRUCTIONS section when you select a cash flow (subject to the restrictions below). That menu section includes:

  • EITHER a Change function if the EFT has not yet been created for that flow, OR a View function if it has, and
  • a Change All Forward function (in addition to Change) when the deal type can potentially have a large number of cash flows (principal repayments, interest coupons, and so on). Currently, these are Money Market, Security, Interest Rate Swap, and Currency Swap deals.

When you select View, the Counterparty Instructions page is displayed in a new window (as when the same page is invoked from Settlements Processing) with all fields read-only.

When you select Change, the Counterparty Instructions page is displayed in a new window, and you can optionally:

  • select a different Counterparty Account, or
  • select either Change to Ad Hoc Instruction or Change to Third Party from the page menu.

Click Save to change the instruction, which will result in the Payment Instruction setting for the cash flow changing to either Modified SSI or Ad Hoc.

A different Payment Instruction may affect the Settlement Approval Defaulting Rule that will be applied when the cash flow is Actioned in Settlements Processing.

When you select Change All Forward, the Counterparty Instructions page is displayed and the same points apply (as for Change) EXCEPT THAT:

  • the title on the Counterparty Instructions page has the words All Forward appended, and
  • any saved changes apply to the selected cash flow PLUS any later-dated cash flows* that:
    • have the same Counterparty and Currency,
    • are not part of a Net or Proposed Net,
    • are not intercompany cash flows,
    • are not in the EFT Completion phase, and
    • allow the type of change - namely, from:
      • NONE to SSI or Adhoc/3rdParty,
      • SSI to Modified SSI or Adhoc/3rdParty, or
      • Adhoc/3rdParty to Adhoc/3rdParty only (cannot revert to NONE, SSI, or Modified o SSI),

*TIP: The Settlement Date is used for finding forward-dated cash flows.

You will be notified how many cash flows were changed and if any were not updated.

NOTE: If any View or Search filters are filtering out cash flows meeting the criteria above, those cash flows are still updated.

Audit Log

Details of all cash flow Counterparty Instruction changes are recorded in the Audit log and can be viewed using the dealing page's Audit History function.

Counterparty Instructions matching previous deal images are shown as read only.
