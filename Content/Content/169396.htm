Facility Fee Template Requirements

This topic summarizes specific Base Template settings required when the Fee Category is Facility.

TIPS: A Facility fee applies:

  • to Money Market and Securities deals only,
  • when the Fee Base Amount is based on a Facility position - for example, Notional Value, Utilized Value, or Unutilized Value.

Fee Category - select Facility.

Fee Base Amount - select the amount the fee will be based on:

  • Notional Facility - the Exposure Limit for applicable deals using the specified Facility or Dealing Line, as specified on the Limits tab on the Quantum Facility or Deal Line page (Unlimited (zero) values are not supported).
  • Utilized Facility - the amount of the Facility or Dealing Line that has been drawn, determined by the Net Modifier method. This method offsets buy and sell deals by adding positive values and deducting negative values. The final figure is presented as an absolute value.
  • Unutilized Facility - the Notional Facility value less the Utilized Facility value for applicable deals.

% Adj - no special requirements.

Base Computation (read-only and set to Sum) - the Fee Base Amount is the sum of the value of all qualifying deals (those satisfying the Selection criteria) as at the Base Reset date.

TIP: Base Reset dates are defined on the Schedule tab of the Fee deal definition page.

Formula - select the method for calculating the fee:

  • Percent per Annum - on each Calculation date and for each qualifying deal (satisfying the Selection Criteria), the fee amount is the Unit Fee Rate (%) value multiplied by the calculated Fee Base Amounts on Base Reset dates within the Calculation period, adjusted for the number of days in that period
  • Simple Percentage - on each Calculation date and for each qualifying deal, the fee amount is the sum of each calculated Fee Base Amount on Base Reset dates within the Calculation period multiplied by the Unit Fee Rate (%) value (with no adjustment for the number of days in the Calculation period).
  • Flat Fee - a flat fee is calculated, and must be paid, if the absolute value of the Fee Base Amount is greater than zero on a Base Reset date.

    On each Calculation date and for each qualifying deal, the fee amount is the sum of the Flat Fees that were calculated on each Base Reset date within the Calculation period.

    The Flat Fee amount is defined in the Unit Fee Amount ($) field.

Other Definition area items - no special requirements.

Selection Criteria area items:

  • Transaction Type - read-only and set to Money Market Deals (which also covers Securities deals) and
  • Facility - a specific Facility or Dealing Line, or ALL.

Tiers area items - no special requirements.
