Bank Account (Crystal Report Formulas)

Parameters

Four different Crystal reports can be run from the Bank Account Inquiry form, depending on the settings for the Statement Run and Flow Type options, as follows.

Crystal report

Statement Run

Flow Type

Bank Account Balance (BNKACRPT)

Cleared

Broker Flows or Cashflows

Stock Account Balance (STKACRPT)

Cleared

Stockflows

Bank Account Statement (STMTRUN)

Selected

Cashflows

Stock Account Statement (STMTRNST)

Selected

Stockflows

Formulas

Formula

Description

thename

Short name of the user who ran the report

rpttime

Date and time the report result set was created

params1 - params10

Selection Parameters

startdate

Start date

enddate

End date

asatdate

As at date

runmode

(Statement runs for Cashflows or Stockflows only)

The mode the inquiry was run under; one of the following:

  • REALTIME - normal output,
  • CREATESAVE - save the result set, or
  • USESAVE - use the latest saved result set.

equivratetype

(Optional for Account Balance runs for Cashflows or Broker flows only)

CCY Translation Rate Type

balcalc_baseon

(Optional for Account Balance runs only)

The date calculation of balances will be based on - one of the following:

  • Account Setting
    Use the date specified for the account (either Value Date or Bank Value Date)
  • Value Date
    Use the Value Dates of the cashflows
  • Bank Value Date
    Use the Bank Value Dates of the cashflows
