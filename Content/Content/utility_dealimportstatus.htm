Deal Import Status page

NOTE: The Import deals Operational Setting must be selected before the Deal Import Service will run.

Use this page to:

  • monitor the status of deal data files imported into Quantum (automatically or manually) using the Deal Import Service, and
  • manually initiate the upload of a specific file (select Upload File in the page menu).

    TIP: Files must be uploaded from the configured AppDataPath\DealImportation\Import location under your site root folder on the Application Server. For more details, refer to the Quantum Installation Instructions or your site administrator.

The total numbers of files of each status are shown, and files are listed under the grid for their current status, as follows:

  • Pending - queued files awaiting processing,
  • Processing - files currently being processed,
  • In Error - files with errors detected in some records, and therefore moved to the Errors folder), and
  • Processed - files processed by the import service, and therefore moved to the Processed folder.

In each grid, the file Name, current Folder, and the Arrived time are listed for each file.

Grid menus* provide relevant functions in each case, as follows:

  • Download a selected file (from the Application Server),
  • Delete a selected file from the grid,
  • Retry an In Error or Processed file (by moving it to the Pending grid), and
  • Replace an In Error file with another file, removing the selected item from the In Error grid and uploading a specified file (to the Application Server).

*TIP: The Processing grid has no menu because the system is working on these files.
