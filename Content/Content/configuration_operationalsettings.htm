Operational Settings page

CAUTION: It is strongly recommended that access to this page is restricted to System Administrators only.

Use this page to specify and review system-wide application-related settings to apply throughout Quantum.

Security Change Approval

This applies when the Operational Settings Data Security option is selected and changes are made on this page.

In that case, and until the changes are approved (or rejected):

  • the page is read-only and still shows the current Before settings,
  • a Pending Changes button appears,
  • clicking the Pending Changes button opens the Pending Changes dialog for users to review and possibly approve or reject the changes,
  • the dialog shows both old and new values,
  • Approve and Reject buttons are available only for Security Approvers with the applicable Approve option selected (except that no-one may approve their own changes),
  • clicking Approve (when available) applies the changes and returns the page to its normal state, and
  • clicking Reject (when available) ignores the changes and returns the page to its normal state.

If Security Approval DOES NOT apply, changes are immediately accepted.

In either case, changes are recorded in the audit log. Use the Audit-Query reporting page to list details of changes, including the user who made the change, the approver/rejector (where applicable), and the dates and times of each event.
