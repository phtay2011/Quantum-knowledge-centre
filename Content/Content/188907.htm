FX Posting, Profit and Loss tab

  • Forward Points Expense - The contra entry is Forward Points Payable (B/S).
  • Forward Points Income - The contra entry is Forward Points Receivable (B/S).
  • Base Gain: Realized - This account is credited when a P&L Gain is calculated, and debited with a decrease in a P&L Gain, the contra account being Base Reval Gain: Realized (B/S). See Relationship between Accounting Rules & G/L Components for the Accounting Rule settings that use this G/L Component.
  • Base Gain: Unrealized - This account is credited when a P&L Gain is calculated, and debited with a decrease in a P&L Gain, the contra account being Base Reval Gain: Unrealized (B/S). See Relationship between Accounting Rules & G/L Components for the Accounting Rule settings that use this G/L Component.
  • Base Loss: Realized - See Base Gain: Realized above (reverse the direction of the accounting entries where a P&L Loss is calculated).
  • Base Loss: Unrealized - See Base Gain: Unrealized (P/L) above (reverse the direction of the accounting entries where a P&L Loss is calculated).
  • Gain: Realized - This account is credited when a P&L Gain is calculated, and debited with a decrease in a P&L Gain, the contra account being Revaluation Gain: Realized (B/S). See Relationship between Accounting Rules & G/L Components for the Accounting Rule settings that use this G/L Component.

    NOTE: If an instrument is closed out, irrespective of the posting method, this default records the gain on closeout, and is posted at the time of the closeout, but dated the value date of the closeout.

    If the applicable Revaluation option on the Accounting Rules tab is:

    • cleared - the system does not post transaction revaluations, so this default is not used at this point; but if...
    • selected - the system posts revaluations. However, this default is only used before closeout if the Posting method is Realized. In this event, the Unrealized default is not used.

    On the Value Date, there is no unrealized amount to reverse.

  • Gain: Unrealized - This account is credited when a P&L Gain is calculated, and debited with a decrease in a P&L Gain, the contra account being Revaluation Gain: Unrealized (B/S). See Relationship between Accounting Rules & G/L Components to indicate the Accounting Rule settings that use this G/L Component.

    NOTE: On closeout, Quantum reverses the unrealized amount and records the realized amount using the Gain: Realized (P/L) or Loss: Realized (P/L) account.

  • Loss: Realized - See Gain: Realized above (reverse the direction of the accounting entries where a P&L Loss is calculated).
  • Loss: Unrealized - See Gain: Unrealized above (reverse the direction of the accounting entries where a P&L Loss is calculated).
  • Separate Buy Gain: Realized - This account is credited when a P&L Gain is calculated, and debited with a decrease in a P&L Gain, the contra account being Reval Separate Buy Gain: Realized (B/S). See Relationship between Accounting Rules & G/L Components to indicate the Accounting Rule settings that use this G/L Component.

    NOTE: The system posts the entries in the currency of the leg of the deal the revaluation relates to.

  • Separate Buy Gain: Unrealized - See Separate Buy Gain: Realized above; the contra account being Reval Separate Buy Gain: Unrealized (B/S).
  • Separate Buy Loss: Realized - This account is debited when a P&L Loss is calculated, and credited with a decrease in a P&L Loss, the contra account being Reval Separate Buy Loss: Realized (B/S). See Relationship between Accounting Rules & G/L Components to indicate the Accounting Rule settings that use this G/L Component.

    NOTE: The system posts the entries in the currency of the leg of the deal the revaluation relates to.

  • Separate Buy Loss: Unrealized - See Separate Buy Loss: Realized above; the contra account being Reval Separate Buy Loss: Unrealized (B/S).
  • Separate Sell Gain: Realized - See Separate Buy Gain: Realized above. This relates to the revaluation of the Sell leg of the deal. The contra account being Reval Separate Sell Gain: Realized (B/S).
  • Separate Sell Gain: Unrealized - See Separate Buy Gain: Realized above. This relates to the revaluation of the Sell leg of the deal. The contra account being Reval Separate Sell Gain: Unrealized (B/S).
  • Separate Sell Loss: Realized - See Separate Buy Loss: Realized above. This relates to the revaluation of the Sell leg of the deal. The contra account being Reval Separate Sell Loss: Realized (B/S).
  • Separate Sell Loss: Unrealized - See Separate Buy Loss: Realized above. This relates to the revaluation of the Sell leg of the deal. The contra account being Reval Separate Sell Loss: Unrealized (B/S).
  • CCY Trans Gain FWD Pts: Real - This account is credited when a P&L Gain is calculated by CCY Translation on/after the Settlement date for which the forward points apply.

    NOTE: This is applicable only if the Accounting Rule option Accrue forward points is selected and Posting Date is Maturity.

    If amortizing the forward points, CCY Translation will be completed at the account level.

  • CCY Trans Gain FWD Pts: Unreal - See CCY Trans Gain Fwd Pts: Real above. This is calculated for a CCY Translation gain that occurs at a date prior to the Settlement date for which forward points apply.
  • CCY Trans Loss FWD Pts: Real - This account is debited when a P&L Loss is calculated by CCY Translation on/after the Settlement date for which the forward points apply.

    NOTE: This is only applicable if the Accounting Rule option Accrue forward points is selected and Posting Date is Maturity.

    If amortizing the forward points, CCY Translation will be completed at the account level.

  • CCY Trans Loss FWD Pts: Unreal - See CCY Trans Loss Fwd Pts: Real above. This is calculated for a CCY Translation loss that occurs at a date prior to the Settlement date for which forward points apply.
  • Exch Principal Gain: Realized - This account is credited with the P&L Gain calculated by CCY Translation on the exchanged Principal on/after Settlement date.
  • Exch Principal Loss: Realized - This account is debited with the P&L Loss calculated by CCY Translation on the exchanged Principal on/after Settlement date.
