Price Fixing Commodity deals in bulk

Use the Commodity Price Fix page to set prices for one or more Commodity deals (Swaps and/or Exposures), using either system-calculated prices, or a value you specify.

You can filter for a group of similar commodity deals, have Analytics automatically calculate the market price, and fix that price, OR manually set a price, for multiple deals.

TIP: You can also Price Fix on the Dealing page (for the current deal only) and, for a Swap deal, by navigating from the Settlements page (for a single Price event that is Unactioned).

To fix prices for selected deals

  1. Enter the Fix Start Date and Fix End Date values to specify the date range for the deals you want to include.

    TIP:: Use relative dates in date fields so you can reuse queries in future runs.

  2. Optionally select a single Instrument, Commodity, Settlement Currency, Entity, Reference Currency, Rate Name, Rate Type, Deal Type, and/or Scope to reduce the result set; or leave a field blank to select all in each case.

    • To include only Swap or Exposure deals, set Deal Type to Deal or Exposure, respectively (or select an Instrument of either type).
    • In Scope, selecting Next Fix will filter for the next price fixes that will occur; selecting Last Fix will filter for the most recent price fixes that have occurred.

  3. Click Run Query.

    The results view lists the applicable deal items, where:

    • An Exposure or Fixed for Floating Swap deal has an entry row for each period.

    • A Floating for Floating Swap deal has two entry rows: one for each side.

    • You can open the Dealing page for a listed item by clicking on its Deal No value and selecting Deal in the quick menu.

    • The returned data is subject to Resource Security.

  4. In the results grid, select the deal periods and/or sides you want to price.

    TIP:: For Swap deals, the corresponding Price events (as shown on the Settlements page) must have a Flow Status of Unactioned.

  5. Do one of the following:

    • click Price Fix to apply the Average Commodity Price function in each case, OR

    • click Set Price and in the popup window enter the price to be applied in each case.

The Calculated Price and Settlement Amount columns reflect the results. Errors are indicated by a red bar at the left of the grid row (click the bar to view the error message).

TIP:: For Swap deals, the Flow Status of the corresponding flows (as shown on the Settlements page) changes to Actioned after the Price is successfully fixed.

To undo Price Fixes for selected deals

  1. Enter Filters values (as above).

  2. Click Run Query.

    The results view lists the applicable deal items.

  3. In the results grid, select the items whose Price Fixes you want to undo.

    TIP:: For Swap deals, the corresponding flows must have a Flow Status of Actioned (as shown on the Settlements page for Price events).

  4. Click Undo Price Fix.

The Calculated Price column shows zero in each (successful) case. Errors are indicated by a red bar at the left of the grid row (click the bar to view the error message).

TIP:: For Swap deals, the Flow Status of the corresponding flows (as shown on the Settlements page) changes back to Unactioned after the Price Fix is successfully undone.
