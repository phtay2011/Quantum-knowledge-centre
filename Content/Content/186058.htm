Targeting Workflow Page elements

Page elements like fields, radio buttons, drop-down lists, and menu options can be targeted for modification by obtaining their element ids.

In general, to determine the id of a page element in the Google Chrome browser, do the following:

  1. Navigate to the core page that requires modification.
  2. Right-click the element and select Inspect.
  3. The Developer Tools grid is displayed with the HTML code for the element in focus.
  4. Locate the id tag. This is usually within an <input> element.

    The part that follows Domain_ is the field id.

Targeting Combined fields

Combined fields are two fields that are displayed together in the space of a single field. These combined fields have their own id.

TIPS:

  • The element in focus is usually an <input> element (the first field of the combined fields).
  • Go up the HTML hierarchy to the div element to locate the id attribute.

In the example below, Currency_FaceValue is the field id.

Targeting radio buttons

A group of radio buttons is internally handled as one field, where each button is a separate option.

To change the titles of the radio buttons, identify the field and find the values behind each radio button.

TIPS:

  • The element in focus is usually an <input> element (the first radio button of the group).
  • The value attribute contains the value that represents that radio button selection.

NOTE: To change the titles of the radio buttons, set the optionTitles attribute of the <Field> element as a series of value and title pairs. For example, optionTitles="1=Outbound;2=Inbound".

In the example below, Payment has a value of 1 and Receipt has a value of 2. The field id is PaymentReceive.

Targeting drop-down lists

Besides the title above a drop-down field, the titles of the options in the drop-down can also be modified.

To change the titles of the drop-down options, identify the field and find the values behind each option.

TIPS:

  • The element in focus is usually an <select> element.
  • The value attribute on each of the option elements contains the value that represents that option.

NOTE: To change the titles of the options, set the optionTitles attribute on the <Field> element as a series of value and title pairs. For example, optionTitles="10090001=First;10090005=Second".

In the example below, Deal has a value of 10090001 and Hypothetical has a value of 10090005. The field id is DealType.

Targeting menu options

The options in menus can be modified using the <Action> attributes.

To identify the action's id in the core page, examine the menu option using your browser's Developer Tools.

TIPS:

  • The element in focus is usually an <a> element.
  • The part before Action is the id for the action.
