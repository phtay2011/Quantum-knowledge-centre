WAC Terminology

Deals

Actual Deal

An actual deal is a deal that is transacted with an actual Counterparty and a True Transactional Entity (TTE).

Notional Deal

A notional deal is a deal that is created by the Quantum Weighted Average Closeout (WAC) Processorprogram with a Notional Counterparty (e.g. Z-WAC) and a Notional Transactional Entity (e.g. Z-TE).

Notional deals are not settled and are filtered out of all Settlements reports. They ensure correct net accounting in weighted average closeout portfolios.

For more see Deal processing and the WAC Processor

Business Structure (Entities/Portfolios)

True Transactional Entity (TTE)

A True Transactional Entity (TTE) is an Quantum Transactional Entity that weighted average closeout accounting is applied to using the Weighted Average Closeout (WAC) Processorprogram. The TTE is sometimes referred to as the actual transactional entity.

NOTE: For each True Transactional Entity (TTE) that is to be processed by the WAC Processor,there must exist a Notional Transaction Entity (e.g. Z-TE).

Notional Transactional Entity (e.g. Z-TE)

A Notional Transactional Entity (e.g. Z-TE) is a Transaction Entity specifically set up for use by the Quantum Weighted Average Closeout (WAC) Processor program to capture the notional deals created by this program.

NOTE: .A Notional Transaction Entity (e.g. Z-TE) must be set up for each True Transactional Entity (TTE) that WAC processing is applied to.

For more see Business Structure and WAC Processing

Counterparties

Notional Counterparty (e.g. Z-WAC)

A Notional Counterparty (e.g. Z-WAC) is a Counterparty specifically set up to be the counterparty in notional deals that create, increase, or reduce the position in the Notional Transaction Entity (e.g. Z-TE).

Notional Closeout Counterparty (e.g. Z-WACC)

A Notional Closeout Counterparty (e.g. Z-WACC) is a Counterparty created to be the counterparty in the notional deals that nullify the existing position in the Notional Transactional Entity (e.g. Z-TE).

Notional Bank (e.g. Z-WACO)

A Notional Bank (e.g. Z-WACO) is a Counterparty (Bank) specifically set up for use when setting up the notional Bank Accounts that capture notional cashflows associated with the Notional Counterparty (e.g. Z-WAC) and the Notional Closeout Counterparty (e.g. Z-WACC) deals.

Notional Custodian (e.g. Z-CUST)

A Notional Custodian (e.g. WACOC) set up in order to create the notional Custodian Account to capture notional stockflows associated with the Notional Counterparty (e.g. Z-WAC) and the Notional Closeout Counterparty (e.g. Z-WACC) deals.

NOTE: If your site is not licensed for Stockflows, then this notional counterparty is not required.

For more see Counterparties and WAC Processing

Bank/Stock Accounts

Notional Bank Account

A Notional Bank Account is a bank account that has been set up for use by the Quantum Weighted Average Closeout (WAC) Processor program.During this process the notional deals created must post to a notional bank account.

Bank Account Defaults should also be set up for the Notional Bank Account.

Notional Stock Account

A Notional Stock Account is a stock account that has been set up for use by the Quantum Weighted Average Closeout (WAC) Processor program. During this process the notional deals created must post to a notional stock account.

  • Stock Account Defaults should also be set up for the Notional Stock Account.
  • If your site is not licensed for Stockflows, then this Notional Stock Account is not required.

For more see Bank/Stock Accounts and WAC Processing

General Ledger Accounts

General G/L Account

A general G/L account specifically set up for use as a netting account for the accounting entries of the actual deal (except the Bank Account with its related Currency Translation entries), and the Bank Account and related Currency Translation entries of the notional deal.

This definition applies only when this term is used with reference to weighted average closeout accounting using the Weighted Average Closeout (WAC) Processorprogram.

For more see General Ledger and WAC Processing
