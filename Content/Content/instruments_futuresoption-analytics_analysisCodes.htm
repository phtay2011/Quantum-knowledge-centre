Analysis Codes tab, Futures Option Instrument

IMPORTANT NOTE: Analysis Codes appear in Analytics Instrument pages only after they are replicated from the corresponding Quantum Instrument pages, by running the Analysis Code Replication wizard.

Use this tab to specify any Analysis Code values for this instrument.

TIP: These Analysis Codes are NOT carried through to any deals based on this instrument. They are used only for reporting on instrument records themselves.

This tab appears when one or more Analysis Codes have been set up for this application page, as determined by the Scope setting in Analysis Code records.The fields are labeled with the applicable Analysis Code names. See Setting Up Analysis Codes.

Use these fields to select the required Analysis Code values for the current data record.

Some of the Analysis Codes for your application may be mandatory, and you must have specific access rights to change their settings.

  • On Quantum Dealing pages when the current deal matches the selection criteria in an Analysis Code Defaulting rule record, the Analysis Code fields will default to the values specified in that Defaulting rule record.
  • On static data pages, no defaults apply.
  • On Analytics Pricing or static data pages, the Analysis Codes will be those replicated from the corresponding Quantum Dealing or static data pages when the Analysis Code Replication wizard was last run.

Example Using Analysis Codes

To be able to group and report on Counterparties by a surveyed Sustainability rating (say), you could apply an Analysis Code as follows.

  1. In Analysis Code Setup, add a new code named Sustainability Rating with Scope set to Counterparty. Select the appropriate Options and add Predefined Values such as Poor, Moderate, Good, and Excellent.
  2. On the Analysis Codes tab on the Counterparty page, select the Sustainability Rating code value, assigning one of the predefined values to each Counterparty.

    Your new analysis code will then become available in Counterparty queries as data, as a group, or as a sort parameter.
