Viewing Deal Audit History and Imaging

To provide each deal's audit history, they are all imaged in the Quantum database. The first image for a deal is a full copy of the deal map (the events and cash flows that comprise the deal) and the deal and sub-deal tables. Every subsequent action completed on the deal (including edits) results in a new image of the deal that records sub-deal tables and deal map entries that have changed.

NOTE: The image represents the state of the deal before the action is applied.

When you request a rollback of a deal to a previous image, the system builds the required deal map by modifying the first image with each set of imaged deal map entries, up to and including the required image.

To view the history of a selected deal in Quantum

  1. In the relevant dealing program, enter the deal number or browse to find the deal.
  2. In the page menu, select Audit History.

    The read-only Audit History dialog box lists events that have taken place for the deal, and specific changes that have resulted in each case.

To view a previous deal image (before a selected action)

  1. In the Audit history dialog box, select an Action row.
  2. Click OK.

    The deal page changes to display:

    • full details of the deal before that selected action was applied, and
    • Image and the Image number in the title (if you have selected a previous deal image).
  3. To restore the current image: in the page menu, select Audit History again, select the latest image and click OK.
