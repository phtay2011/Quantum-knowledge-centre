Importing Broker Accounts

NOTE: You must have at least one Broker Account group set up (even a Dummy one) to enable importing of this data. See About Bank Account Groups.

The Broker Account Importer supports the following fields.

Mandatory fields (new record)

Field ID

Field description (GUI label)

bankacc_mode

Bank Account mode (Options menu) - must be Broker

acc_name

Account Name/Group Name

acc_no

Account Number/Group Reference

ccy

Currency

cparty

Broker

entity

Owner

thelevel

Level In Structure (Transaction or Group)

tx_entity

Default Transaction Entity (to be used for for Bank Account Adjustments)

Optional fields

Field ID

Field description (GUI label)

accr_exp

Interest Expense Accrual

accr_inc

Interest Income Accrual

analyse01 to analyse20

Analysis Code (up to 20)

ba_csh_con

Cash Concentration (check box)

ba_tbal

Target Balance (check box)

bal_glcode

Balance Adjustment

calc_int

On

coa_code

G/L Account Code

con_ba

Account Number

con_bank

Bank (for Cash Concentration account)

con_conc_cparty

Default Concentration Counterparty

con_orig_cparty

Default Origin Counterparty

con_phase

Cash Concentration (Processing Phase)

con_sectyp

Default Instrument

con_trf

Suggested Concentration Transfer

cp_owner

Broker

Fund Deficit Threshold Amount (check box)

Fund Deficit Threshold Amount

Observed Non Business Days

Mandatory fields (DeActivate and ReActivate)

Only cparty and acc_no are mandatory with the mfDeActivate and mfActivate commands.

Examples

NOTE: When using these format examples, separate each line of input with a line break. Separate by pressing Enter at the end of each line.

(QXT format)

@BrokerAccount
$new
AccountNumber = 100900800700
Bank = NZ Bank
AccountName = Broker account EUR
Currency = EUR
ParentGroup = NONE
GroupTransactor = Transaction
Entity = Hedges
Facility = NONE
DefaultTransactionEntity = Derivatives
GLOwner = Group Treasury
GLAccountCode = 100100 - EUR
GLAccountBalanceAdjustment = 100930 - EUR
InitialMargin = 89099
InitialMarginContra = 89900
InterestIncome = NONE
InterestExpense = NONE
InterestIncomeAccrual = NONE
InterestExpenseAccrual = NONE
TheInterest.EffectiveDate = 27-Aug-2014
TheInterest.CalculateInterest = YES
TheInterest.InterestCalculationBasedOn = Bank Value Date
TheInterest.On = TIERED RATES
TheInterest.Using = DAILY CLOSING BALANCE
TheInterest.RateType = Accounting Rate
TheInterest.GrossInterest = N
TheInterest.DaysBasis = ACTUAL/ACTUAL
TheInterest.Frequency = DAILY
TheInterest.DayOfMonthForEnds = 0
TheInterest.DayOfMonthForDue = 0
TheInterest.CashflowDue = SAME DAY
TheInterest.AllowCashflowsOnNonBusinessDays = Y
TheInterest.IncludeZeroBalance = N
