Counterparty Mapping Files for SWIFT EFTs

If you want to output different SWIFT message formats (MT101, MT103 or MT103+) for different counterparties, you must set up a comma-separated text file to map these. This file specifies which format is required for specific counterparties. One line is required for each counterparty, in the following format:

CounterpartyName , Mtnumber

Where:

  • CounterpartyName is a valid Quantum counterparty name,
  • the comma between the values is mandatory,
  • MTnumber is MT101, MT103, or MT103+, indicating the type of Payment record format for the counterparty,
  • the default record format will be the Non-Financial Transfer setting in the SWIFTExternal Method Source record, and
  • the file path is given by the Counterparty Mapping setting in the SWIFT External Method Source record.

NOTE: If you will be using the same SWIFT output message format for all counterparties, you do not need this file. The format is given by the Non-Financial Transfer setting in the SWIFT External Method Source record.
