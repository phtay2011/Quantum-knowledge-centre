Cash Flows tab, Actual Cash Flow

Use this tab to:

  • view details of the cash flows for the entire life of an existing or new deal (even before a new deal is saved), and to
  • view and optionally change the Counterparty Instructions for a selected cash flow.

To change the grid view:

  1. Click the Configure link.
  2. On the Layout tab:
    • To reposition a column in the grid, drag-and-drop that column name in the Columns box.
    • To add an additional column, select it from the drop-down list.
    • To remove a column from the grid, click the x button for the column.
  3. On the Filters tab, optionally add one or more data items with specified values for filtering (in or out of) the viewed results.
  4. On the Columns tab, optionally change columns names and add calculated columns.
  5. On the Colors tab, optionally add colors to columns and/or rows.
  6. (Pivot Views only) On the Actions tab, optionally apply actions on deals or deal requests where applicable (Create, Open, Delete, and Update; also Open Analytics Deal Analyse).
  7. On the Properties tab:
    • Name the view (required before you can save it).
    • Optionally set this view as the default for the current query. This will be the first view you'll see when you select the current query.
    • Optionally change the way amounts appear, the type of rounding, and what decimal format to use.
    • Set the Access Permission for the view (Public or Private).
  8. To save your changes for later re-use, click Save.

Fields available for display and filtering are: Cash Flow Date, Settlement Date, Currency, Amount, Comments, Bank, Account Name, Account Number, Intercompany, Beneficiary, Counterparty Account Name, Counterparty Account Number, Counterparty Bank, Flow Lifecycle, Flow Lifecycle State, Flow Lifecycle Phase, and/or Payment Instructions.

To view or change the Counterparty Instructions for a cash flow:

TIP: Restrictions apply. See Changing Counterparty Instructions from Dealing Pages.

  1. Select the required cash flow and in the grid menu select either View or Change. (depending on whether or not the EFT has been created for that flow). The Counterparty Instructions page is displayed in a new window.
  2. (As and where allowed) Optionally change the Counterparty Instructions mode and specific settings and click Save.
