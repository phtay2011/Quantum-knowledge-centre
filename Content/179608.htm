Effectiveness tab fields, Imported Hedge Relationships

The Hedge Relationships Deal Importer supports QID-format fields corresponding to those fields on the Effectiveness tab.

Method

The Prospective Effectiveness method you select determines the fields available for you to enter data into.

Field code

Field name

Example and Notes

shortcut

Short-Cut Effectiveness (assume 100% effective)

shortcut=Y

If set to Y*, this disables all other effectiveness fields.

ef_retro

Retrospective Test Method

ef_retro=PER.DOLLAR OFFSET

ef_prospec

Prospective Test Method

ef_prospec=CUM. DOLLAR OFFSET

retro_interval

Interval - for retrospective regression calculations

retro_interval=1 MONTH

ef_range

Range - for Cumulative Dollar Offset prospective effectiveness calculations.

ef_range=1 MONTH(S)

This field combines the settings for the two Range/Interval fields (number, and unit of time) that appear on the Effectiveness tab.

std_interv

Interval - for Regression calculations

std_interv=1 MONTH(S)

This field combines the settings for the two Range/Interval fields (number, and unit of time) that appear on the Effectiveness tab.

low_limit

Dollar Offset: Lower

low_limit=80

A setting is required for both Prospective effectiveness methods.

up_limit

Dollar Offset: Upper

up_limit=110

A setting is required for both Prospective effectiveness methods.

s_method

Sample Style
- for Regression

s_method=ROLLING

s_size

Initial Sample Size
- for Regression

s_size=35

retro_r_squared

Minimum R-squared
- for Retrospective Regression

retro_r_squared=0.855

retro_slope_lower

Slope Lower
- for Retrospective Regression

retro_slope_lower=1.25

retro_slope_upper

Slope Upper
- for Retrospective Regression

(As above)

retro_intercept_lower

Intercept(CCY) Lower
-for Retrospective Regression

retro_intercept_lower=1.25

retro_intercept_upper

Intercept(CCY) Upper
-for Retrospective Regression

(As above)

retro_con_interval

Confidence Interval
-for Retrospective Regression

retro_con_interval=95% (Student's t)

r_squared

Minimum R-squared
- for Prospective Regression

r_squared=0.855

t_slope

Slope Lower
- for Prospective Regression

t_slope=1.25

t_slope_upper

Slope Upper
- for Prospective Regression

(As above)

tintercept

Intercept (CCY Units) Lower
- for Prospective Regression

tintercept=0

(This field code has no underscore.)

tintercept_upper

Intercept (CCY Units) Upper
- for Prospective Regression

(As above)

con_interval

Confidence Interval
- for Prospective Regression

con_interval=95% (Student's t)

* A setting of Y indicates that the corresponding check box in Hedge Relationships would be selected (on). N indicates that it would be cleared (off).

Results

You cannot import Effectiveness results themselves, as these are calculated by Quantum.
