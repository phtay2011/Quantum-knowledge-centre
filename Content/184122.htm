Configuring a Chart View

When creating a report, use the Layout tab and Properties tab to change the Chart view. Click Apply (or Save) to see the effect of your changes.

A Chart view presents a:

  • Bar Chart - for Number, Reference, or Text fields,
  • Column Chart* - for Number, Reference, or Text fields,
  • Line Chart - for Amount, Date, or Number fields,
  • Combo Chart - for Reference, or Text fields, combining Line and Column charts, and
  • Pie Chart - for Amount or Number fields.

Procedure Summaries

To configure a Chart view

  1. Do one of the following:
    • for an existing Table view - select the name of the Table view in the name lookup and click the Configure link, or
    • for a new view - in the grid menu or the Configure View menu, select New Table.
  2. In the Layout tab's Group By multi-select lookup, select one or more data fields, where each value or combination of values will become a main axis point (Y axis for Column charts, X axis for others).
  3. In the Display As field, select the chart type: Bar Chart, Column Chart, Line Chart, Combo Chart, or Pie Chart.
  4. In the Sort By multi-select lookup, select one or more data fields, to sort the results by. These field values also display as data points, grouped according to your Group By selection/s. This means that in some cases more data points will be displayed when a sort order is added.
  5. (Optional for Bar or Column chart only) Select Split 'Group By' and select one or more Then By items if you want to specify secondary grouping on those items. See Secondary Grouping in Bar and Column Charts.
  6. In the Values - aggregate on multi-select field, select one or more* Amount or Number fields to summarize the grouped data.

    *TIP: If you have selected a Pie Chart OR more than one Group By or Then By item, you may select only one value to aggregate on.

  7. (Bar or Column chart only) If you selected more than one item in the previous step, then in the adjacent Bars or Columns field, select Clustered, Stacked, or 100% Stacked to display the multiple aggregated values as side-by-side bars/columns or as stacked segments (where 100% Stacked shows segments based on their percentage of the total rather than actual value).
  8. For each Values - aggregate on item, select the type of aggregation in the Aggregate By lookup: Count, Distinct Count, Sum, Max, Min, Mean, and so on.
  9. (Optional) On the Properties tab, enter a new Name for the View, set the Access Permission (Public or Private), and Save your View changes. The current report results are redisplayed in that view.

TIP: Click Apply to see the effect of View changes without having to save them.
