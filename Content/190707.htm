Changing Counterparty Instructions

This topic covers the restrictions on, Module Security requirements for, and the resulting effects of, changing Counterparty Instructions.

Restrictions

You CANNOT change Counterparty Instructions:

  • if you access them from the Settlement Approvals page;
  • if you do not have the required Resource Security and Module Security rights (see below);
  • for an intercompany cash flow;
  • for a flow currently in a netted group or Proposed for Netting (add Type to the View to identify such flow types);
  • for linked flows (cash and stock) that have a Payment Sent status of YES (Deliver/Receive against Payment);
  • for a flow in the EFT Creation Lifecycle phase (see EFT Lifecycle Phase and Flow Status below);
  • to target a different Counterparty account if Continuous Linked Settlement (CLS) is mandatory and the deal is CLS-eligible, but that change would make the deal non-eligible; or
  • to an Adhoc Instruction if the Enable Adhoc Instructions option is NOT selected (on the Operational Settings page).

Also, you may only change the Ordering Customer when in Third Party mode.

Changing Forward Cash Flows

If the Instructions have been accessed from a Dealing page's Cash Flows tab via the Change All Forward menu function:

  • the words All Forward are appended to the page title,
  • any changes will be applied to the selected cash flow AND similar* forward-dated flows IF the EFT has NOT been created.

* Restrictions apply. For more details, see Changing Counterparty Instructions from Dealing Pages.

Module Security and Operational Settings

The following Module Security permissions apply for the Counterparty Delivery Instruction Change module (Module Security tab, User or User Group page).

  • To change instruction data - Modify SSI permission is required.
  • To only view the instructions - Inquire permission is sufficient.
  • To select Adhoc or Third-Party mode* - Adhoc Instructions permission is required.

*NOTE: To enable selection of Adhoc or Third-Party mode, the system-wide Operational Settings option Enable Adhoc Instructions must also be selected (under Bank Accounts).

Payment Instructions setting (Type of Change)

The Payment Instructions setting for the applicable cash flow indicates whether or not the Counterparty Instructions have been changed and the type of change, and has the following values:

  • SSI - all details are as defaulted (using Counterparty Account defaulting Rules);
  • Modified SSI - only the Counterparty account, Delivery Narrative, Bank to Bank Instructions, or Charges details are changed; or
  • Adhoc - any other details are changed.

This setting is appended to the window title IF the instructions apply to:

  • a single, un-netted cash flow, OR
  • a Netted group PROVIDED THAT the setting is the same for all cash flows in the Net (if not, no value is shown).

NOTES:

  • The words All Forward are appended to the page title IF the Instructions have been accessed from a Dealing page's Cash Flows tab via the Change All Forward menu option.
  • When the Details of Charges setting is changed, it is considered a Modified SSI, even if it is subsequently changed back to its original value.
  • If Payment Instructions is Adhoc when your changes are saved, the setting cannot be change it back to Modified SSI.

Approvals

Changing the Counterparty Instructions can change the Approval requirements, because Payment Instructions is one of the Matching Criteria for Settlement Approval rules. You can therefore:

  • specify different Settlement Approval rule entries for the different types of change (SSI, Modified SSI, or Adhoc as above) or
  • select ALL to apply that rule whether or not the Instructions change.

External (Payment) Methods

Changing the Counterparty account resets the External Method to the default for that account, but you may subsequently change the External Method.

EFT payment methods are affected by Adhoc delivery instructions.

EFT Lifecycle Phase and Flow Status

NOTE: The current Flow Lifecycle text value is displayed in a colored badge at top right of the Counterparty Instructions page, where the color represents the Flow Lifecycle Phase*.

You cannot change instructions for a flow in the EFT Completion Flow Lifecycle phase*.

However, you can change instructions for a flow in the EFT Preparation Flow Lifecycle phase* but will first receive a validation message that this action will reset the Flow Lifecycle and Flow Status, as shown below.

If you click OK to confirm:

  • Flow Lifecycle is reset to Awaiting Settlement Status for the flow, and
  • Flow Status is reset to UNACTIONED.

* For a full list of Flow Lifecycles and Phases, see Flow Lifecycles, States, and Phases.
