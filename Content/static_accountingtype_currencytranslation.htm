Currency Translation area, Accounting Type page

Use this area to specify the default currency translation settings for General Ledgers using this Accounting Type.

Apply currency translation - select to enable, by default, currency translation for ledgers using this Accounting Type. Currency Translation will then be applied to ledger entries, with deal-level and account-level currency translation based on the settings in Accounting Rules (Deal Level) (Currency Translation levels, Apply Deal Level Currency Translation to a Ledger) and G/L Accounts Templates (Account Level), respectively.

Translation Method - select the methodology for Bank Account currency translation and FX Deals from the drop-down list.

NOTE: A methodology must be defined regardless of whether the Entity is using currency translation or not. The currency translation methodology defined here dictates the rate used on FX Deals (where one leg is base) to translate the foreign amount to the base currency.

Unrealized Rate Type - select the unrealized rate type (FX) for all currency translation gain/loss calculations for the ledger from the browse list.

NOTE: When Use Transaction rate type is selected, this field is disabled and defaults to the Transaction Rate Type (under Deal Booking Rate).

Realized Rate Type - select the realized rate type (FX) for all currency translation gain/loss calculations for the Ledger from the browse list.

NOTE: When Use Transaction rate type is selected, this field is disabled and defaults to the Transaction Rate Type (under Deal Booking Rate).
