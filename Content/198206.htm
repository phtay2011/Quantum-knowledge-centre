Interest Rate Swap enhancements

The following Interest Rate Swap enhancements relate to this version.

Quantum Interest Rate Swaps

With the demise of LIBOR as a benchmark, the various jurisdictions have begun using their preferred benchmark rates. This second phase is to enhance current Overnight Index Swap (OIS) functionality for Interest Rate Swap (IRS) deals. In this release we implement the following in Quantum:

  • The ability to view the rates used in the average rate calculation by selecting a hyperlink in the Calculated Rate column of the Rate Set Deals query page. This then opens up the Average Rate Calculator page in Analytics where the user can view the rates used
  • The ability to enter a lag to calculate the average rate. This is available on the instrument and deal pages.

NOTE: You must have Analytics installed for the functionality to work correctly.

Analytics Interest Rate Swaps

In this release we have further extended the Overnight Index Swap implementation in Interest Rate Swaps with the following changes in Analytics:

  • Added Rate & Weight to the Lag field for Pricing > Interest Rate Swap / Cross Currency Swap deals that use the new Overnight Index type.
  • Added Rate & Weight to the Lag field for Instrument > Interest Rate Swap / Cross Currency Swap instruments that use the new Overnight Index type.

NOTE: From the results grid of a Processing > Rate Set Deal page, click on a value in the row to find out which calculator the system used for the calculations. Click the dialog box to open the associated Analytics page and find out more.
