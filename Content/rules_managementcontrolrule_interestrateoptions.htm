Interest Rate Options tab (Management Rule)

Use this tab to define the management reporting revaluation rules and methods for Interest Rate Options.

To add a new Management RuleSet - Click New1 and in the Browse window, select Entity2, give the Rule Set a name, and then complete the Common Settings. Select the tab/s for the transaction type/s you want to define, add rule detail entries in each case (see below), and Save.

To list records - Click . (Click the Configure link to change browse list properties.)

To change a record's details - Select it in the browse list, select the tab/s for the transaction type/s you want to change, and edit rule detail entries as below:

To add a new entry - In the grid menu, select New1. In the dialog box, enter the Matching Criteria and Settings, and click Save.

To edit an existing entry3 - Click an entry value in the grid; in the Edit dialog box, edit the values as required; then click OK.

To delete an existing entry3 - Select the entry row in the grid; in the grid menu, select Delete; click OK to confirm.

To keep your changes - click Save (for the page).

TIPS:
1 Alternatively, you can copy an existing rule set or rule and edit its settings.
2 Entries for groups take precedence over ALL entries.
3 You cannot add, edit, or delete inherited entries. To override these, create specific rule entries.
