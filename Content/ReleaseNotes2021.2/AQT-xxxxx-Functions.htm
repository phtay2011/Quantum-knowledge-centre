Expression builder - New and changed functions

The following are new functions used in Expression Builder.

Ceiling

Description

Returns the value rounded up to the nearest multiple of significance

Format

Ceiling(value,optional significance)

Arguments

value - the number value that you want to round

optional significance - the multiple to which you want to round

Example

Ceiling(13,2)

Returns the number value rounded up to the nearest multiple of significance which in this case is 14

Floor

Description

Returns the value rounded down to the nearest multiple of significance

Format

Floor(value,optional significance)

Arguments

value - the number value that you want to round

optional significance - the multiple to which you want to round

Example

Floor(13,2)

Returns the number value rounded down to the nearest multiple of significance which in this case is 12

RoundDown

Description

Returns the value rounded down to a given number of decimal places

Format

RoundDown(number,decimalPlaces)

Arguments

number - the value you want to round down

decimalPlaces - decimal number place to which you want to round down

Example

RoundDown([5.7899],1)

If number value is:

  • 5.7899 - the result is 5.7

RoundUp

Description

Returns the value rounded up to a given number of decimal places

Format

RoundUp(number,decimalPlaces)

Arguments

number - the value you want to round up

decimalPlaces - decimal number place to which you want to round up

Example

RoundUp([5.1242],1)

If number value is:

  • 5.1242 - the result is 5.2

Trim

Description

Returns the text by removing leading and trailing space or optional characters specified.

Format

Trim(Text,optional characters)

Arguments

text - this is a text from which we want to eliminate the leading and trailing characters, special symbols or text pattern

optional characters - This is a text which needs to be removed if it appears as prefix or suffix to Text. This can be single character or a pattern.

Example

Trim(Sample Text,"123")

The result is Sample Text.
