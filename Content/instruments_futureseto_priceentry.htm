Price Entry, Futures and ETO Instrument

Use the fields in this area to indicate how prices for this Futures contract are quoted in the market. The system uses these settings to interpret prices.

Entry Type - if Pricing Formula is PER TICK, select either DECIMAL or FRACTIONAL; otherwise this is read-only and set to DECIMAL. This setting determines which fields appear in the rest of the Price Entry area.

Tick Move (available if Entry Type is DECIMAL) - for both fixed and variable tick contracts, enter the price increment the tick value is based on - that is, how many percentage points of yield should be added to the deal price to calculate the variable tick value.

Minimum Price Move (available if Entry Type is DECIMAL) - the smallest amount of change allowed in the price of the contract. Enter a positive number of up to one whole number with up to 10 decimal places.

Tick Basis (available if Entry Type is FRACTIONAL) - if a price is quoted as a whole number and a fraction, enter the denominator (basis) of the fraction here. When you enter a price, the number after the "point" (delimiter) must be between 0 and this number. For example, if the contract trades in 1/32nds, enter 32; If it trades in 1/100ths, enter 100.

Value Per Tick - if Pricing Formula is PER TICK, enter the monetary value of one tick in the contract currency; otherwise this is read-only and set to VARIABLE. The system rounds this value according to the rounding method specified for the contract currency.

Shift Decimal Place (available if Entry Type is DECIMAL) - the number of places an entered value's decimal point should be shifted by to give the actual value; enter a positive or negative number to shift the decimal point to the right or left, respectively. The default is 0 or no shift.

Rate Rounding - select UP, DOWN, or NEAREST. The system validates mid-rates and interpolated rates calculated in the Interest Rates application against the Minimum Price Movement for the contract. If a rate fails validation, the rounding rule specified here is applied
