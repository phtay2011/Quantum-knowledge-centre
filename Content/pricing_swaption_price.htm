Price tab

Use this tab to enter pricing parameters and to calculate pricing valuations. The tab includes the following items.

Position Date - Enter the date at which the deal is to be valued. This determines the time to maturity for a deal. Analytics provides Today as a default, but this may be changed.

Rates Date - Enter the date of the rate curves to be used in the deal valuation calculation. Typically this will be the same as the Position Date. The rate curves may be viewed on the Rates tab once calculation is completed. Analytics provides Today as a default, but this may be changed.

As At Date - Enter the date of the rate image "as at" a particular point in time. Analytics provides today as a default, but you can change this to view and use a rate image as it existed at an earlier date.

Calculate (selectable for new European-style deals only) - Either select Value, or select Volatility or Strike to solve that variable for the specified value you enter in the adjacent Value field.

Value (available when Calculate is Volatility or Strike) - Enter the value to be used to determine the Calculate variable (Volatility or Strike). Analytics uses an iterative technique, until the value of the deal meets the value entered in this field.

Valuation - read-only and set to Black (European) if Style is European; otherwise select the appropriate valuation model.

View (available if Collar is Yes) - select the results view you want: Payer, Receiver, or Collar.

Swap Rate (European-style only) - Enter the market Swap Rate (as a percentage) or alternatively, if left blank, this rate will default according to the Default Valuation Rule record.

Volatility (read-only if Calculate is Volatility) - Enter the Volatility rate (as a percentage) or alternatively, if left blank, this rate will default according to the Default Valuation Rule record.

Step Type / Steps (American- or Bermudan-style) - select Per Coupon or Per Year, and enter the number of steps.

Price button - Use this button to run the selected calculation and populate the grid values.

Market grid - After you click the Price button, this grid displays the calculation result totals for your chosen settings.

TIP: The Rates tab shows the rate curves used and the Valuation tab displays the calculation results for each valuation cash flow.
