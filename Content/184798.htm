Editing Job Schedule Settings

Use the Job page's Schedule tab to view and optionally edit the schedule settings for the currently selected FIS Scheduler job.

NOTE: The initial settings are those defined for the Job in Scheduler.

To edit the current Job's Schedule settings

  1. Open the Schedule tab.
  2. Edit the field values as required (see Field descriptions below).
  3. Save your changes.

Field descriptions

  • Frequency - select how often you want to run the Job: Once, Daily, Weekly, or Monthly.
  • On Date (available only if Frequency is Once) - the date you want to run the Job.
  • At (Hour, Minute) (available if Frequency is Once OR the Once option is selected) - the time of day you want to run the Job.
  • Every...in days (available only if Frequency is Daily) - the number of days between runs of the Job.
  • Every...in weeks (available only if Frequency is Weekly) - the number of weeks between runs of the Job.
  • On Days (available only if Frequency is Weekly) - select one or more days of the week when you want to run the Job.
  • Actual / Relative (available only if Frequency is Monthly) - select whether to run the job on a numbered or relative day of the month, respectively - for example, the 20th (Actual) or the last weekday (Relative).
  • Day (available only if Frequency is Monthly) - EITHER:
    • if Actual is selected - enter a day number, OR
    • if Relative is selected - select First, Second, Third, Fourth, or Last (and enter which day/s in Unit (below)).
  • Unit (available only if Frequency is Monthly AND Relative is selected) - select the day of the week or Day, or Weekday, or Weekend.
  • Every...in months (available only if Frequency is Monthly) - enter the number of months between runs of the Job.
  • Once / Every (unavailable if Frequency is Once) - select whether to run the job only once on the scheduled dates, or repeatedly (as specified in the associated fields), respectively.
  • Interval (available only if the Every option is selected) - enter the number of time units and in the drop-down list, select the unit: Minutes or Hours.
  • From (Hour, Minute) (available only if the Every option is selected) - the time of day from when you want to start running the Job.
  • To (Hour, Minute) (available only if the Every option is selected) - the time of day after which you no longer want to run the Job.
  • Start Date (unavailable if Frequency is Once) - the date from which you want to start running the Job.
  • End Date (unavailable if Frequency is Once) - optional; the date after which you no longer want to run the Job. A blank value means "never".
