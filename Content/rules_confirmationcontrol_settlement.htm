Settlement tab, Confirmation Rule page

Use the Settlement tab to create and maintain, for an Entity Confirmation Rule record, one or more entries for paper (print/e-mail) confirmations of cash flow actions. These entries specify which types of Settlement Confirmation message are to be created, and how they are to be output, for the different combinations of Instrument*, Transaction Type*, Counterparty, and so on.

Selection criteria are applied in order (from the top row down), so specify more general and "catch-all" criteria last.

*NOTE: A netted group of cash flows can potentially contain different values for Entity, Transaction Type, and/or Instrument. The ALL Entity record is used where the Entity differs; the applicable ALL rule Matching Criteria values apply where the other values differ.

WHAT DO YOU WANT TO DO?

Before you start - Use the Browse list to access the Entity Approval Rule record you want to update, and open the Settlement tab.

To automatically submit Settlement confirmations for printing or e-mailing (optional) -select Process paper confirmations automatically.

To add a new rule entry (from scratch) - In the grid menu, select New; in the dialog box, define the Matching Criteria* to select which deal actions the Settlement confirmation rule entry applies to; and:

  1. under Settings - select Require approval if approval is required before a Settlement confirmation will be output, and select the Print Device and number of Copies;
  2. under Paper - select the Print output option/s (NONE, PRINT, EMAIL, and/or FILE) and, if Print is FILE, enter the File Subfolder and Paper File Type; then
  3. click OK. A grid row is added for the new entry.

To add a new rule entry (by copying) - In the grid, select the entry you want to copy; in the grid menu, select Copy; in the dialog box, edit the Matching Criteria*, general Settings, and Paper settings as required (details above); then click OK. A grid row is added for the new entry.

To edit a rule entry - click a data value in the grid; in the Edit window, make the required changes; then click OK.

To delete a rule entry - select the grid row; in the grid menu select Delete; then in the dialog box, click OK to confirm.
