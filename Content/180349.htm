Sorting Grid Items

Sorting is optional. You can set none, one, or more default sort columns when configuring the grid view, and you can override these on the displayed grid (see below).

When you first display a grid, items are listed in the default sort column sequence, if any (if not, rows are displayed in the order the data is returned by the application).

The current sort columns (if any) have column headings with either an up or down chevron or (for ascending or descending sequence, respectively). The following example Confirmations page grid shows how the configured view's display Columns and default Sort columns are applied.

Where default sorting applies:

  • To change to another sort order (on any one column only) - Click the required column heading. That heading (and only that heading) will be highlighted in blue and marked to indicate the list is now sorted on that column, in ascending sequence. Click the column heading again to sort in descending order - the heading will now be marked .
  • To revert tothe default sorting - Click the current sort column heading once or twice until the default sort columns are restored.

Where no default sorting applies:

  • To sort on any one column - Click the required column heading. That heading will be highlighted in blue and marked to indicate the list is sorted on that column, in ascending sequence. Click the column heading again to sort in descending order - the heading will now be marked .
  • To revert to an unsorted list - Click the sort column heading once or twice until it is no longer highlighted and no sort indicator shows.

TIP: To change the default sort columns or their order of precedence, see Configuring Grid Views.
