Search on partial numbers

You can now search on partial numbers by using a wildcard tilde (~) character.

NOTE: When used in a Search box above a grid view, this looks in numeric fields such as Deal Number and Amount, but does not look in other types of field, such as Bank Account.

Search is available in the site menu, Browse windows, grids with a dedicated Search box, and in the Knowledge Center.

Basic search

The Search function in the site menu enables you to look for a menu item, data record, Instrument name, or Counterparty name from any page.

When you enter a short string (one or two characters), the Search will try to find items, usually words, starting with that string. If you enter a longer string, the Search will try to find phrases that contain that string. The type of search ("starts with" or "containing") is described at the top of the results.

Partial number search

When you enter a tilde (~) followed by a short string of numbers, or a short string containing numbers and only one decimal point, the Search will complete a numeric wildcard search. This type of search looks through numeric fields, such as the Amount column, but does not search through text fields, such as Bank Account.

If you include any other characters in a partial number search, the Search will instead complete a basic search.

Strips out / removes the thousands separator, which in this example is a comma. The thousands separator can also be a point.
