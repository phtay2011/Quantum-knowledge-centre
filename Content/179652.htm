Multi-line text fields

Multi-line addresses can be added for counterparties. To do this from the Importer, separate each line of the address with a pipe character ( | ). For example:

street_add = 10 Bowhill Lane|Chisolm|Hillsville

Payment Method Identifiers

Counterparty records can use optional fields to specify, among other things, the Clearing Houses to be used with different payment methods. Up to ten such fields (Identifiers) can be added. You can use the Importer to add some fields and change some Identifiers (Clearing Houses). These fields are also available from the Counterparty Setup form.

Use the following syntax:

clrhouse = PayMethod|Identifier

Where:

  • Clrhouse is the Identifier
  • PayMethod is the payment method for which you want to specify an Identifier ("clearing house").
  • Identifier is the Identifier you want to import.

For example:

clrhouse = Cedel|test2
clrhouse = Concord|test

NOTE: If PayMethod is already present for the counterparty, the clearing house will change to ClearingHouse. If PayMethod is not yet present, and less than ten payment methods have already been specified, it is added.
