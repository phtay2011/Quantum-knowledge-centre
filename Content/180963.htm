Customizing the View

The data displayed in the View can be customized using the tabs in Configure View.

Customizing the VIew simply changes the way you look at the Data Version. It does not change the information stored in the Data Version.

(To change the information in an existing Data Version, change the Query's Query Configuration parameters.)

Tabs and controls

Tabs in the Configure View area allow you to make temporary changes to the current View. When you change the View, these changes are not saved.

Except for changes you make with the Filters tabs, to refresh the display to show the changes you have made, select Refresh Version.

To apply the filters you have selected from the Attribute Filters tab or Pool Filters tab, from the View menu select Apply.

If a row caption in the grid has a + or - sign next to it, the row can be collapsed or expanded to hide or reveal lower level rows. The opening setting is as defined in Concentration Hierarchy.
