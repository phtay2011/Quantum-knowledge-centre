Importing Cash Management Data

Use the Bank Import Service to import the following non-Quantum data before you run Bank Reconciliation, Intraday Matching, and/or CashXplorer for a particular period:

  • External cash flows - bank account transactions not directly related to Quantum deals, but handled by other organizations or systems (for example, a retail banking operation).

    NOTE: You cannot use this method to load new deal cash flows into Quantum.

  • Bank flows - bank statement balances and (optionally) individual bank statement transactions for the relevant bank accounts, as recorded by the bank.

    NOTE: A Bank Account balance entry may also be imported from a .QID file using Bank Account Adjustment import.

  • Intraday bank flows' - unconfirmed bank flows imported for use by Intraday Matching. These are ignored by Bank Reconciliation.

See Importing Bank Files for more information.

After importing, you can then:

  • Attempt to reconcile each relevant account by matching its bank flow balances with cash flows (Quantum and external). You can do this by running Auto Match scripts or chains (either automatically* after successful data import, or manually), or by individually matching selected items on the Bank Reconciliation or Intraday Matching.

    *NOTE: An automatically initiated Match script will not run and a message will be logged to the Message Log if Resource Security applies and the user running the import is not Exempt or does not have Write access to the underlying Bank Account/s.

    and/or...

  • Start a CashXplorer data retrieval, either automatically after successful data import and Auto matching (if run), or manually.

About the Bank Import Service

The Quantum Bank Import Service offers options and flexibility for importing bank statement and/or external cash flow files into Quantum.

The BkImport.ini Configuration file can specify many processing options for this service, including data manipulation and filtering. For installation-related settings, including the optional FileWatchResetInterval key, see the AGBankImportService.config file details under Configuring Data Importation Services.

NOTE: If you want to import Bank transaction data from ECHOS, you must set up the Bank Import Configuration file to enable this.
