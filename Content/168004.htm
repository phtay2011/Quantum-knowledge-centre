Currency Option Deal Statuses

Option deal statuses indicate what actions have been, and now can be, performed for a deal.

TIP: If the Results list does not have a Status column, use the view's Configure link and the Columns tab to add it.

Different deal statuses apply to each option category (Type column in the Results list).

Asian and Barrier option deals that are In the Money will have a second listing with a Type value of Exercise Lapse if they are:

  • an Average Rate Fixing (Asian) option that has been rate fixed,
  • a Single or Double Barrier IN option that has been knocked in, or
  • a Single or Double Barrier OUT option that has not yet been knocked out.
