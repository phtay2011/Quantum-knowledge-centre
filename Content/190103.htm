Pivot Drill-Down summary and Examples

When the Pivot view configuration has any Group by rows, Group by columns, and/or aggregated Values with a Level set greater than zero, users viewing the report results in that Pivot View will be able to drill down on such data values to go to the next level of detail.

  • By default, a pivot displays the Level 0 fields.
  • When there is a next level configured, the pivot will display hyperlinks that can be clicked to drill down to the next level.
  • Navigating to the next level will make all fields up to that level visible.
  • A breadcrumb trail appears at the top of the page from Level 1 onwards to show the context of the drill-down.
  • Use the breadcrumb trail to navigate back up the levels. Close the breadcrumb to return to level 0.

The following example illustrates drilling down into a Pivot view with rows grouped on Instrument (Level 2) within Entity (Level 1). Clicking on the CASHMAN2 Entity in the initial view (top) drills down to show the results for that Entity (center); then, clicking on the Annuity Vanilla Borrow Instrument drills down further to show the results for that Instrument within the CASHMAN2 Entity (bottom).

In Pivot View results displaying a Deal Number or Deal ID column (which could possibly be renamed), you can click on one of those values to drill down, either:

  • in Quantum - to the Dealing page for that Deal Number, or...
  • in Analytics - to the Deal Analyze page for that Deal ID.

Adaptiv has no equivalent function.

NOTES:

  • A Pivot View can be split into one or more groups.
  • You can also configure drill-down on aggregated values to a Table view by selecting the Drill down to table view option and configuring the Table View Columns and Sort fields. See Drill Down to Table View in Pivots.
  • Additional Public or Private user-defined Pivot Views can be added by selecting New Pivot in the View menu.
