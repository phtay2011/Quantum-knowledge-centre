Alerts tab

(This tab appears only when an Alerts definition is specified on the Properties tab.)

Use this tab to configure any alerts for the current query. Alerts generate and send messages to specified Recipients whenever message criteria match those in the query result sets.

The default settings are taken from the Alerts Definition specified on the Properties tab. Any changes made on this tab are saved in that Definition record when the query configuration is saved.

When a row in the Alerts grid is selected, the Messages grid lists the individual messages defined for that Alert object.

To add a new Alert object to the current Definition - Click the New button above the Alerts grid; in the New Alerts dialog box, enter a new Name for the alert, select a Column; click Add Recipient and enter a recipient Name and E-Mail Address (repeat to add more recipients); then click OK. Add message text and criteria for the alert as below.

To add a new message to a selected Alert object - Select the row in the Alerts grid to display the Messages grid for that alert; click the bottomNew button (above the Messages grid); in the New Message dialog box, enter a new Name for the message and the Resend Timeout in minutes; click the Add Message link to open the Expression Builder window; type in your message text and use the + button and Browse window to select any data items to include, then click Done; click Add Criterion and enter a Criteria Name, Column, Column Type, Operator, Value, and Tolerance (repeat to add more criteria or rules); then click OK.

To edit an existing Alert object - Select the row in the Alerts grid and in that grid menu, select Edit; in the dialog box change the settings as required (click Add Recipient to add a new recipient, click the x button to remove one); then click OK. Add, edit, or remove messages using the procedures above and below.

To edit a message for a selected Alert object - Select the Messages grid row and in that grid menu, select Edit; in the dialog box change the settings as required (click Add Criterion to add a new rule, click the x button to remove one); then click OK.

To remove a message from a selected Alert object - select the Messages grid row and in that grid menu, select Delete; then click OK to confirm.

To delete an existing Alert object in full - Select the row in the Alerts grid and in that grid menu, select Delete; then click OK to confirm.
