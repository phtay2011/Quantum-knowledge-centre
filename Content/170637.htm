Maintain Session Entities

A Netting participant is an entity that is permitted to participate in a netting session. The entities in the netting session are available for the counterparty when maintaining netting invoices or offers and requests.

Use the Netting Session page's Entities tab to view, add, edit, or delete the entities permitted to take part in the session.

NOTE: When participants are set up with additional settlement currencies:

  • Participants invoiced in an alternate currency will settle those invoices in that currency.
  • Invoices in other currencies will be settled in their local currency.
  • Offers and Requests will always be settled in their local currency.

To add an Entity to a Session

  1. On the Netting Session page, select the Pool and Session from the respective lists.

    TIP: You can use the search box in the list pages to quickly find the record you need.

    The current data for the session is displayed.

  2. In the Setup area, open the Entities tab.

    The Pool's Top Entity is always listed at the top (at Level 0). Any other entities already participating in the session are also listed.

  3. In the tab menu, select New; in the dialog box select the Entity; then click OK.

    The entity Code, name, and Local Currency (if any) is listed.

  4. (Optional) To provide the ability for the participant entity to settle in currencies other than their local currency (if they have one):
    1. Select the Entity row and in the tab menu, select Alt. Settle CCs.

      The Alternate Settlement Currencies page opens.

    2. In the Currencies tab menu on that page, select New; in the dialog box select the Alt Settle Curr code; then click OK.
    3. To add more alternate currencies, repeat the previous step.
    4. On the Alternate Settlement Currencies page, click Save.
  5. On the Netting Session page, click Save.

TIP: To define a hierarchy of multiple entities, see Maintain Session Entity Hierarchy.

To edit Entity details for a Session

  1. On the Netting Session page, select the Pool and Session from the respective lists.
  2. In the Setup area, open the Entities tab.

    Entities currently participating in the session are listed.

  3. Select an entity row and in the tab menu, select Edit (or click a data value in the entity row); in the dialog box edit the entity details; then click OK.
  4. To edit the entity's alternate settlement currencies (if they have a local currency):
    1. Select the Entity row and in the tab menu, select Alt. Settle CCs.

      The Alternate Settlement Currencies page opens.

    2. In the Currencies tab menu on that page, select New; in the dialog box select the Alt Settle Curr code; then click OK.
    3. To add more alternate currencies, repeat the previous step.
    4. On the Alternate Settlement Currencies page, click Save.
  5. On the Netting Session page, click Save.

To remove an Entity from a Session

(Not available if there are any invoices, offers, or requests still tied to this entity and session)

NOTE: Removing an entity from a session clears the current entity hierarchy, and you will need to rebuild it, if required. See Maintain Session Entity Hierarchy.

  1. On the Netting Session page, select the Pool and Session from the respective lists.
  2. In the Setup area, open the Entities tab.

    Entities currently participating in the session are listed.

  3. Select an entity row and in the tab menu, select Delete; then confirm.
