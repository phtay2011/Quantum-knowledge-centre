Deleting Imported Bank Flows and Cash Flows

From the Prior Day or Intraday Matching page, after you have run a query and selected an account, you can use the procedures below to delete imported bank data.

NOTE: When Resource Security applies, you may delete imported statements and flows for bank accounts you have Write access to. Therefore a warning message displays if you do not have Write access to all Bank Accounts for all items imported in the same file as the item/s you want to delete.

You must then choose whether to continue with the Delete action or not.


To delete external cash flows

  1. On the Flows tab, select an external cash flow (in the Cash Flows grid).
  2. In the Cash Flows grid menu, select Delete.

    The Delete External Cash Flow dialog box is displayed.

  3. Click either of the following:
    • Delete flow - to delete only the selected cash flow, or
    • Delete entire file - to delete all cash flows imported from the same file as the selected cash flow.

See also the Note on Resource Security above.


To delete an imported bank statement and all its bank flows

NOTES:

  • You can delete complete statements only (not individual bank flows).
  • An imported statement need not have any associated bank flows.
  • Deleting bank flows may affect multiple bank accounts. All matches and adjustments associated with the imported bank flows will also be deleted - including Quantum adjustment deals generated by an Auto Match process specified for the import.
  • You can also delete a statement from the Overview tab.
  1. On the Flows tab, select a bank flow (in the Bank Flows grid).
  2. In the Bank Flows grid menu, select Delete.

    One of two Delete Bank Flow / Statement dialog boxes appears, as follows:

    • if the statement has been imported from a file containing other statements - Delete statement, Delete entire file, and Cancel buttons appear; otherwise,
    • only a Yes and No button appear.
  3. Click one of the following, as relevant:
    • Delete statement or Yes - to delete the statement and its bank flows (if any), or
    • Delete entire file - to delete all statements imported from the same file as the selected bank flow.

See also the Note on Resource Security above.
