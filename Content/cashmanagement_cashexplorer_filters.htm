Configure Query - Filters tab

Use this tab to specify or change the following filter settings for the current Query.

Included Accounts grid - lists the bank accounts and broker accounts from which cash flows are to be retrieved. In the grid:

  • To include all accounts, or all accounts of one type - in the grid menu, select Include All, Include All Bank Accounts, or Include All Broker Accounts, as required.
  • To exclude all accounts, or all accounts of one type - in the grid menu, select Exclude All, Exclude All Bank Accounts, or Exclude All Broker Accounts, as required.
  • To add more accounts and/or groups - click Add and select them in the dialog box.
  • To change which Opening Balance will apply for an account - select the grid row for the account; in the grid or popup menu, select Edit; in the Edit dialog box select the Opening balance type, and click OK.
  • To remove an individual account - click x at the right of the account grid row (or select the row and in the grid menu, select Exclude).
  • To remove an account group - select the grid row for any account in the group and in the grid or popup menu, select Exclude Group. All group members are removed.

Included Flows area - contains the following sets of options:

  • Prior to Start of First Horizon - select or clear these options to respectively include or exclude the specific flows prior to the first user-defined Horizon.
  • Within Horizons - select or clear these options to respectively include or exclude the specific flows and/or additional balances*within user-defined Horizons.

    *NOTE: You select whether or not to retrieve additional balances each time you run the query.

  • Previous Day - select or clear this option to respectively include or exclude prior-day reconciled cash flows. This option is selected by default.
