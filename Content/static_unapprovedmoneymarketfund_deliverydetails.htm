Delivery Details tab

Use the fields on this tab to provide default delivery instruction details. For example, if the delivery instruction is for Fund Manager A and Fund ID B, set up a (Counterparty) delivery instruction for Fund Manager A with an account number of Fund ID B, then, on this tab, link to that instruction.

NOTE: You can use this same delivery instruction for many funds.

Counterparty -Select the internal Counterparty to be used for transfers to this fund account.

Account Number / Identifier - Select the appropriate Fund ID (or Account ID) for the selected Fund Transfer Delivery Counterparty and Fund Currency (when populated, the bank name is displayed below this field).

Bank Name (read-only) - When a transfer Account Number is selected, this field displays the name of that fund.

Narrative - Enter any special delivery details into this free-format text box. Depending on the selected Fund Details option, this text will either be added to, or replace, the default Narrative text.

Fund Details options - Select either:

  • Add to account details - to add any Narrative text specified here to the standard delivery instructions, or...
  • Replace account details - to replace the standard delivery instructions with the Narrative text specified here (if any).

Allowable Payment Methods - Select one or more payment methods for settlement from this fund.

Default Payment Method - Select the default payment method that should be used for settlements from this fund. This default can be overridden in Settlements Processing.

Balance Import Source - Select the External Method Source that applies to this fund and payment method/s. This is required for operating EFT interfaces, imports, and exports. Only funds with a Balance Import Source can have details imported into them.

Charges area - Enter any Senders Charges and Receivers Charges (currency and amount in each case), and - if either of these currencies differs from the FundCurrency - select the Translation Rate Type (an FX Rate).

[For more details of a Charges field, click in it when the Help panel is open.]
