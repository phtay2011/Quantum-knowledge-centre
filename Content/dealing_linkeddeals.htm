Deal Set page

Use this page to view, edit, and create deal sets, which group related deals for supported Transaction Types. You can then run queries on these deal sets, or perform reporting and scenario-building within CashXplorer to generate predictable forecasts.

TIP: Cash Forecast records based on Deal Set instruments with both the Forecast and Standard options selected may also be accessed on this page, and Deal Set records based on similar instruments (with both options selected) may also be accessed on the Cash Forecast page. So additions and changes made to a recordon either page are reflected on the other. The main difference is that the Forecast Cash Flow page provides Convert and Recalculate functions for Projected cash flows.

On this page, use the different areas and tabs as follows:

  • Header area - to enter and edit identifying data for the Deal Set;
  • Details area - to specify the mandatory Deal Set Instrument and Deal Set Rule, and an optional Master Deal Reference and/or text Comments; and the
  • Deals grid - to view deals for which this Deal Set is selected. While you cannot add, edit, or delete any grid rows, you can click on a Deal No value to drill down to the applicable Dealing page, where you may be able to make any required changes (including to the Deal Set field).

To add a new set - click New, fill in the data fields, and click Save.

To list all sets - click the button to open the browse window (click the Configure link to change browse list properties.)

To change a set's details - select it in the browse list, edit the relevant data fields, and click Save.

To view the deals in a set - select the set, and browse in the Deals grid.

To deactivate a set - select the set, in the page menu select DeActivate, confirm when prompted.

To delete the current Deal Set - in the page menu, select Delete and confirm when prompted. Delete is available only when there are no live deals linked to the Deal Set and no links to deal images.

  • Removing a live deals link can only be done on the Dealing page.
  • A new deal image is created when, for example, a deal is recalled and the amount changed. If the deal is linked to a Deal Set, a link to the deal image exists. If a link exists, then the Deal Set can never be deleted, even if the deal itself is deleted or the Deal Set link removed.

TIP: If you cannot delete a Deal Set, consider deactivating it instead.

Resource Security can affect the information displayed for deals in a Deal Set.

For details of data items, click on an item when the Help panel is open.
