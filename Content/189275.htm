Interest Rate Options Management Rules

Use the Interest Rate Options tab to define the management reporting revaluation rules and methods for Interest Rate Options.

Matching Criteria (Interest Rate Option Management Rule)
  • Instrument - Defines the instruments to which these rules apply.

    The rules are established by row in the grid, and any number of instruments of this type may be recorded to define the applicable and appropriate treatment.

    TIP: Create a rule for ALL instrument types, and then a rule for each instrument type that requires different treatment.

  • Entity - Defines the entities to which these rules apply.

    TIP: Create a rule for ALL entities for the specified Instrument type, and then a rule for each entity or entity group within the entity structure that requires different treatment.

Settings (Interest Rate Option Management Rule)
  • Interest Rate - For instrument type/s defined in the Instrument field, the interest rate defines the name of the Interest Rate Table used to source the revaluation rates/prices for the revaluation of a deal meeting the selection parameters.
    The rate type must be known in conjunction with the interest rate to locate an actual rate table and the rates to NPV of any accrual amounts referred to above.
    The rate table to source the FX rates to revalue the currency component is defined in the CCY Translation Rate Type field in the Management Control header. (FX rates do not have a rate type).
    If you are using a User Input Zero Curve or User Input Discount Factor, select the name used for the curve on the Zero Definition form.
    If you select a User Input Zero Curve or User Input Discount Factor for an instrument type that is not compatible with that type of curve, an error message is displayed when you perform a revaluation.
  • Rate Type - Defines the type of rate table used to source the revaluation yield rates for the revaluation of a deal meeting the selection parameters for instrument type/s defined in the instrument column.
    The rate type must be used in conjunction with the interest rate to locate a rate table.
    If you are using a User Input Zero Curve or User Input Discount Factor, select the rate type used on the Yields Rate or Zero Definition form.
  • Revaluation Method - Specify whether to revalue and post accounting entries for deals of this type. Use the browse list to select from the following methods:
    • NONE - Select to prevent deals of this type from being revalued.
    • BLACKS
    • BLACK-SCHOLES
    • BLACK-SCHOLES SA
    • INTRINSIC
    • INTRINSIC SA
    • IMPORTED - Select to use an imported market value to value deals using these instruments.

    For information about the other selections available in the browse list, see Revaluation Method in Accounting Rules.

  • Volatility Rate - The volatility rate defines the name of the rate table for the volatility component of the revaluation of a deal meeting the selection parameters for instrument type/s defined in the Instrumentfield.

    The rate type (common for the interest and volatility rates) must be known in conjunction with the volatility interest rate to locate an actual rate table and the rates to NPV of any accrual amounts referred to above.

    The rate table to source the FX rates is defined in the CCY Translation Rate Type field in the Management Control header (FX rates do not have a rate type).
