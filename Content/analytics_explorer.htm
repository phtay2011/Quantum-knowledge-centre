Queries page (Analytics Explorer)

Use this explorer-style page to drill down through the Analytics folder structure of folders and saved queries. The breadcrumb trail at the top of the page indicates your drill-down path (click on a higher level in the trail to go back to that level).

The icon indicates a folder - click on the folder name to drill down, showing sub-folders and queries within that folder.

No icon indicates a query - click on the query name to open the Query page, showing that query's configuration and view settings. On that page you can change the query configuration, run the query, view report results, and so on.

A folder or query may appear at one level only. Folders may be nested within higher-level folders.

To create a new folder at the current level - click New* and in the drop down list select Folder; enter the folder Name and any Comments; then click Save. Add items as below.

To add a folder or query into the current folder - Under Folder Contents click Add; browse to and select existing† folders and/or queries to include in the new folder; then click OK.

†CAUTION: Items you add will be removed from their current location.

To create a new query at the current level - click New* and in the drop down list select the type of query (Position, Cash Flow, Historical Simulation VaR, Monte Carlo Var, or Parametric Var); in the New Query page, configure the query parameter Defaults and Properties including a unique Name; then click Save.

*TIP: These functions are also available in the page menu of this page or any Query page.

To move the current folder‡ - In the page menu, select Move; in the dialog box, browse to and select the new parent folder.

To delete the current folder‡ (which must be empty) - In the page menu, select Delete and click OK to confirm.

‡ You can remove a Query from a folder only by adding it into another folder.

To export a report definition file for the current level - In the page menu, select Export. A file download button appears at the bottom right of the page, showing the XML file, named as the current folder. Click the button to view the file; use the button menu to locate the file in File Explorer.

To import a report definition file at the current level - In the page menu, select Import and in the dialog box, browse to and select the file; then click Import.
