Discount Security Deal Pricing page

Use this page to create and maintain Discount Security deals in Analytics.

To add a new deal - Click New, fill in the data fields, and click Save.

To list deals - Click the button. (Click the Configure link to change list properties.)

To change a deal's details - Select it in the list, edit the relevant data fields on the required tabs, and click Save.

NOTE: You cannot edit financial data if actions have been approved on the deal.

To reset the current deal (CAUTION: May necessitate deal reprocessing; removes any customization) - In the page menu, select Reset; and click Yes to confirm. See Resetting a Deal.

To delete the current deal (CAUTION: Takes immediate effect and cannot be undone - even if subject to Approval/Rejection) - In the page menu, select Delete; and click Yes to confirm.

To clone an existing deal - Recall an existing deal and, in the page menu, select Copy.

To clone an existing deal for calculation - Recall an existing deal and in the page menu, select Copy for Calculation. The cloned deal's Details and Schedule tabs retain the original deal values, including deal customizations, and new calculations are performed.

To clone an existing deal with schedule - Recall an existing deal and in the page menu, select Copy with Schedule. The cloned deal's Deal, Details and Schedule tabs retain the original deal values, including deal customizations. This cloned deal can then be saved as an actual deal in Analytics.

TIP: Actions completed on a saved deal will not be removed from the Schedule by the Clone Complete Schedule function, which could lead to invalid deals. You should always review the deal schedule fully to ensure it is correct.

To perform a calculation on a deal - Recall an existing deal and in the page menu, select Price.

To run a Limit Check on a deal - Recall an existing deal and in the page menu, select Check Limit. See Limit Check.

To change a deal's valuation settings - Recall an existing deal and in the page menu, select Change Valuation Settings.

For details of data items, click on an item when the Help panel is open.
