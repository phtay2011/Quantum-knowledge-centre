Adjustments

A Worksheet Adjustment is a minor correction or change to a cash flow displayed in a View. For example, you might create a Worksheet Adjustment in the following circumstances:

  • You know that a cash flow will occur but it has not yet been entered into the system. You do not want to register it in Quantum, but would like to see its effect on your cash position query.
  • You think a cash flow will occur but you do not want to use a Projected Cash Flow as you do not want to register it with Quantum. You would like to see the impact of the cash flow on the cash position query.
  • You know that a value on the View grid is incorrect and you want to correct the value and see the impact it has on the query.

You can create a Worksheet Adjustment at any time while defining the Worksheet or when you have already created Data Versions.

When you define and save a Worksheet Adjustment, no adjustment will be made to the View until you apply the adjustment to the Data Version.

A Worksheet Adjustment is always stored within the Data Retrieval Parameters. You can use the same Worksheet Adjustments in one or more Data Versions.

To create a new Worksheet Adjustment, select New from the grid menu and complete the New Adjustment Form.

When you have created a Worksheet Adjustment it is added to the Worksheet Adjustments grid. You can use the Worksheet Adjustments grid to create, edit and delete Worksheet Adjustments.

Value Date

Mandatory. Enter the value date to which the adjustment will be applied.

TIP: This date is based on your User Location's time zone (which defaults to the System date/time.

Amount

Mandatory. Enter the amount for the Worksheet Adjustment. The value can be positive or negative but not zero.

Transaction Type

Mandatory. From the lookup select the Transaction Type for the Worksheet Adjustment.

Instrument

From the lookup select the instrument type for the Worksheet Adjustment. The instrument type must be a valid instrument belonging to the Transaction Type you have specified.

Bank

From the lookup select the bank for the Worksheet Adjustment. The bank must have accounts in the currency you have specified.

Account Number

Mandatory. From the lookup select the account number for the Worksheet Adjustment. The account number must be a valid account at the bank you have specified and in the currency you have specified.
