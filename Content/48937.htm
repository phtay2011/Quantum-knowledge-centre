Overview of the Purge Utility

As part of your regular maintenance of the system, your System Administrator will need to use the Purge utility to delete data that is of temporary nature and can be safely removed from the Quantum database. This will help ensure that system performance is maintained.

NOTE: The Purge utility is installed on your application server (in the same location as other system programs and executables).

Command line switches and parameters

As with any command line tool, there are a number of different switches and parameters that you must define when deleting the data. The syntax of these are detailed in the Purge Utility's command line help text (see below).

The switches and parameters deal with how the connection is established to the database, and what needs to be deleted.

What type of data can be purged?

The following type of data can be deleted from the Quantum database:

  • any archived data,
  • Cash Management Worksheet data versions (from archive or directly from the database),
  • unprocessed (usually legacy) deal action confirmations,
  • data in the Alerts log,
  • data in the Message Log table,
  • data in the Log-ins log table,
  • data in the Monitoring table,
  • data in the temporary report tables, and
  • message data received from ECHOS (as viewable in the ECHOS Messages page).

Specifying the Age of Data to be Purged

When purging data, you can explicitly specify the oldest data you want to retain (in terms of the number of days old) or accept the default value of 30 days. Data older than that will be purged.

This applies to all types of data except the temporary reporting tables (see /REPORT).

To optionally specify the number of days:

  • after the /CASHMGMT or /MSS parameter - type /DAYS, a space, and the number of days; or
  • after the /ALERT, /MESSAGE, /LOGIN, or /RTU parameter - type a space and the number of days.

Accessing the Purge tool

The Purge utility is installed on the application server, with other system programs and executables, in the following location:

C:\Program Files (x86)\SunGard\AvantGard Quantum\Bin

Command Line Help

To display Command Line syntax for the Purge utility

  • From the Command prompt, enter:

purge /?

-or-

purge /help

-or, to copy the help to a text file:

purge /? > file_pathname

For example:

purge /? > c:\temp\ArchvHlp.txt
