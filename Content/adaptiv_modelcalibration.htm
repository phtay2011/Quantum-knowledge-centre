Model Calibration page (Adaptiv)

Use this page to configure and run the Model Calibration utility. You can create multiple configuration records with different sets of parameter values.

When you run the utility, it calculates correlations between Price Factors (FX Rate and Interest Rate) using the specified configurations (Market Data Archive data, Statistics, Calibration, and Models).

To display an existing record - click the browse button and in the browse window, locate the record name you want and select it. (Click the Configure link to change browse list properties.)

To add a new record - click New, enter a new Code and descriptive Name, select or enter the required parameter values (see below), then click Save.

TIP: You can also add a new record by copying the current one - select Copy in the menu, change the data, and click OK.

To run the utility using the current record's settings - click Run; when prompted to confirm, click OK.

TIP: After a run, you can manually add, edit, or delete rows in the results grids, using the standard grid menu functions in each case.

To change a record's details - select it in the browse window, edit the relevant data values, and Save.

To deactivate the current record (retaining it in the database) - in the page menu, select Deactivate; then click OK to confirm. The record will be badged as deactivated (to reactivate, select Reactivate).

To delete the current record completely - in the page menu, select Delete; then click OK to confirm.

Data descriptions

  • Code - the unique identifier for the Model Calibration record.
  • Name - a descriptive name for the Model Calibration record.
  • MDA Configuration - select an Adaptiv Market Data Archive record.
  • Statistics - select an Adaptiv Statistics configuration record.
  • Calibration Configuration - select an Adaptiv Calibration Parameters record.
  • Price Models - select an Adaptiv Price Model configuration record.
  • Keep Calibration Artifacts - select this option to specify, when the utility is run, the folder where calibration files (ADA, AAS, and AAC) will be saved. Clear the option to temporarily save calibration files in the TempFolderPath specified in your site's AAI.AAIS.exe.config file.
  • Calibrated Models grid - populated with run results, but may be edited.
  • Risk Factor Correlations grid - populated with run results IF the specified Calibration Configuration record has a SparseMatrix grid row, but may be edited.

For details of the data in the grids, you may need to refer to the Adaptiv Analytics Interfacing Guide or to your FIS Consultant.
