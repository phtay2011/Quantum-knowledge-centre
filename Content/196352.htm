Calculation-based Fee schedule

This section describes how to define a calculation-based Fee deal's schedule and how to use the Manual Schedule field.

Defining a calculation-based Fee deal's schedule

On the Fee deal page's Schedule tab:

  1. Select the Calculation option (under Base).
  2. Select the Calculation row (the first) and in the grid menu select Edit (or click on any value in the row). In the Edit Schedule dialog box, do the following.
    1. Select a Frequency for fee cash flow calculation - one of the following:
      • Once (On the dealâ€™s Maturity Date, unless Pay fee cashflow in advance is selected, when the fee is calculated on the Start Date),
      • Daily (overnight),
      • Weekly,
      • Monthly,
      • Quarterly,
      • Semi Annually,
      • Annually, or
      • Manual Schedule (sets Frequency in the Payment and Invoice rows to Manual Schedule also, and displays a Calculation Schedule grid, enabling you to create virtually any Calculation and Payment schedule).
    2. Edit the First Date setting if required. This is the first Fee Calculation date, initially based on Start Date and the selected Frequency. Changing this date changes the calculation cycle from the 1st calculation date onwards, based on this date.

      TIP: The last calculation period automatically ends on the specified Maturity Date.

    3. If you selected a Frequency of Manual Schedule above, optionally select a different Business Day Convention (the rule to apply when the relevant date falls on a non-business day).
    4. Click OK to close the Edit window.

    Your Calculation date settings are shown in the grid.

  3. If you selected a Frequency of Manual Schedule above, skip to Step 6 (because the Schedule grid's Payment and Invoice rows are read-only). Otherwise proceed from the next step.
  4. Select the Payment row (the second) and in the grid menu select Edit (or click on any value in the row). In the Edit Schedule dialog box, do the following.
    1. Select a Frequency to specify the offset between Calculation Date and Payment Date - one of the following:
      • Same Day (no offset),
      • Next Day,
      • N Days*,
      • N Business Days*,
      • Nth of Month* (day of the month; next occurrence after Calculation date),
      • Nth of Following Month* (day of month, in the month following the Calculation date's month),
      • End of Month (last day of Calculation date's month),
      • End of Following Month (last day of the month following the Calculation date's month), or
      • Manual Schedule
    2. *When the Offset field is available (with N or Nth frequency settings above), enter the number of days or day of the month for the Payment Date offset, as required.
    3. If you selected a Frequency of Manual Schedule in Step 4a, optionally select a different Business Day Convention (the rule to apply when the relevant date falls on a non-business day).
    4. Click OK to close the Edit window.

    Your Payment date settings are shown in the grid.

  5. If you are not applying True Ups, ensure that the Invoice row's Frequency is Same Day (or blank). If you are applying True-Ups, follow the procedure under Set Fee Invoice Date Options (for True Ups) to edit the Invoice date settings.
  6. Follow the procedure under Set Base Reset Options for Fees to edit the Base Reset date settings.
  7. If you selected a Frequency of Manual Schedule for the Calculation row in Step 2a, follow the procedure under Manually Schedule Fee Calculation / Payment / Invoice Dates to specify your Calculation Schedule grid dates.
  8. If you selected a Frequency of Manual Schedule for the Base Reset row in Step 6, follow the procedure below to specify your Base Reset Schedule grid dates.

Now:

  • Check your other Schedule grid settings.
  • Use the View Schedule grid menu function to view and check the actual dates being scheduled.
  • Save your settings.

More about the "Manual Schedule" field

(Applicable only to calculation-based schedules when the Schedule grid's Calculation row has Frequency set to Manual Schedule.)

Use the Calculation Schedule grid to manually schedule almost any Calculation, Payment, and Invoice dates for the current Fee deal schedule.

NOTE: The set of Calculation, Payment, and Invoice dates is independent from the set of Base Reset dates. You can manually schedule either set of dates, both, or neither.

To enter Calculation schedule dates

  1. In the Calculation grid menu, select Generate.

    The Generate Calculation Dates dialog box opens.

  2. Select the Calculation Frequency for the range:
    • Days,
    • Weeks,
    • Months,
    • Years,
    • Daily,
    • Weekly,
    • Monthly,
    • Quarterly,
    • Semi-Annually, or
    • Annually;
  3. If you selected Days, Weeks, Months, or Years, then in the adjacent field, enter the number of those time units between Calculation dates.
  4. Set the Start Period Date and End Period Date to define the range, which must be within the Fee deal's effective date range (Start Date to Maturity Date).
  5. Select the Payment Frequency for the range:
    • Same Day,
    • Next Day,
    • N Days*,
    • N Business Days*,
    • Nth of Month*,
    • Nth of Following Month*,
    • End of Month, or
    • End of Following Month.
  6. *If you selected an N or Nth value, then in the adjacent field, enter the number of following days or the day of the month, as applicable.
  7. Select the Invoice Frequency for the range:
    • Same Day,
    • Previous Day,
    • N Preceding Days*, or
    • N Prec Business Days*,.
  8. If you selected an N value, then in the adjacent field, enter the number of days or business days, as applicable.
  9. Click OK.

    The schedule of dates is displayed.

  10. Repeat the procedure to define further dates (or use the Bulk Insert or Bulk Replace functions if preferred - see below).

    Any new ranges may supersede existing ranges or parts of a range, or may be distinct.

To change a single date in the schedule

  1. In the Calculation Schedule grid, select the date row.
  2. In the grid menu, select Edit.

    In the Edit Calculation Schedule dialog box, edit the Start, End, Invoice, and/or Payment dates as required.

  3. Click OK.

To remove a single date from the schedule

  1. In the Calculation Schedule grid, select the date row.
  2. In the grid menu, select Delete.
  3. Click OK to confirm.

To use Bulk Replace to overwrite the current entries

TIP: With the Bulk Replace and Insert functions, you can use a spreadsheet application (for example, Excel) to create and maintain multiple schedule entries. You can also edit directly in the Bulk Replace and Bulk Insert dialog boxes.

  1. In the Calculation Schedule grid menu under BULK, select Replace.

    The Bulk Replace dialog box is displayed, showing the column headings and all schedule rows, as in the following example.

  2. Select all rows and press CTRL-C.

    Your selection is copied to the clipboard.

  3. In your spreadsheet application:
    1. Open a new workbook.
    2. Press CTRL-V (or select the Paste function).

      Your selection is pasted into the equivalent spreadsheet rows and columns.

    3. Add, edit, and/or delete entry rows using spreadsheet functions as required. Ensure that the date formats match those used in Quantum.
    4. Select all rows and columns you want to take back to Quantum.
    5. Press CTRL-C (or select the Copy function).
  4. In the Bulk Replace dialog box, press CTRL-V.

    Your spreadsheet selection is copied into the Bulk Replace dialog box, replacing the current selection.

    TIP: You can also edit directly in the Bulk Replace dialog box.

  5. Click Replace.

    The dialog box closes and the Calculation Schedule grid now shows the entries from the Bulk Replace dialog box.

    Error messages will appear if any entries are incorrect or incomplete.

  6. Fix or delete incorrect entries as necessary.

To use Bulk Insert to add new entries

  1. In the Calculation Schedule grid menu under BULK, select Insert.

    The Bulk Replace dialog box is displayed, showing ONLY the column headings.

  2. Select the columns and press CTRL-C.

    Your selection is copied to the clipboard.

  3. In your spreadsheet application:
    1. Open a new workbook.
    2. Press CTRL-V (or select the Paste function).

      Your selection is pasted into the equivalent spreadsheet rows and columns.

    3. Add, edit, and/or delete entry rows using spreadsheet functions as required. Ensure that the date formats match those used in Quantum.
    4. Select all rows and columns you want to take back to Quantum.
    5. Press CTRL-C (or select the Copy function).
  4. In the Bulk Insert dialog box, press CTRL-V.

    Your spreadsheet selection is copied into the Bulk Insert dialog box.

    TIP: You can also edit directly in the Bulk Insert dialog box.

  5. Click Insert.

    The dialog box closes and the Calculation Schedule grid now shows the entries from the Bulk Insert dialog box.

    Error messages will appear if any entries are incorrect or incomplete.

  6. Fix or delete incorrect entries as necessary.
