Structured Address Migration page

Use this page to migrate selected sets of unstructured addresses to a default structure.

  • WHAT DO YOU WANT TO DO?
  • Select a different Query or previous run - Click on the current Query heading and select the Query or run in the browse window.
  • Filter on Source - In the Source drop-down list, select one of Bank Account, Entity, Futures Exchange, Counterparty, or Cparty Delivery Account.
  • Reconfigure the Query or configure a new one - Click the Query Configure link, make your changes, and save them.
  • Select a different result View - Click on the current View heading and select in the browse window.
  • Change the View or create a new one - Click the View Configure link, make your changes, and save them.

    Run the current Query - Click Run Query. The results view is displayed.

  • Edit or enter a structured address - Click on a value in the results grid row you want to change, and in the quick menu select Edit Address. In the Edit Address dialog box, fill in the structured address lines as required, using Copy and Paste, then click OK.
  • Update selected items with any changes - Click the selection box at the left of the items you want to update; or click the check mark in the column heading row to select all (click again to deselect). Then click Save Structured Address.
