Exporting data from Quantum

You can export data from the Quantum database as follows:

  • Export most types of data record held in the database in any of these formats: CSV, Excel, JSON, QXT, PDF, and XML where valid. This includes deals, static data, rules, rates, cash management data, workflow -related data, and so on.

    You can also export some grid data in those formats, for example, deal cash flows and General Ledger entries, query configurations, report results, processing grids, and so on. See Exporting Grid Data.

  • Download Static Data and Rates to Data Transfer spreadsheets and optionally export in QXT file format.
  • Export deal data files in supported formats (CSV, Excel, PDF, QXT, or XML) from Dealing pages. For some pages you can also export JSON.
  • Export some deals for which you are using the Quantum Web API.
  • Replicate Deals, Static Data, User details, Rates, and Resource Security settings added or updated in Quantum in the Analytics database - either automatically (if Real time Update is running), or manually using Replication Manager, as set in Analytics Service Manager Configuration.
  • Replicate Analysis Codes in the Analytics database by using the Analysis Code Replication wizard.
  • Archive the following data, in proprietary format, using the Archive utility (see Archiving Quantum Data):
    • deal and cash flow audit information,
    • General Ledger entries for matured deals,
    • Cash Management Worksheet data versions, and
    • message data received from ECHOS.

  • Export Adaptiv Market Data Archive (MDA) results in CSV format or in the proprietary ADA format.
