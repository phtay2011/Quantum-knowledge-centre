Linking Hypothetical deals

Hypothetical deals are linked to their corresponding live exposure and derivative hedge deals in a hedge relationship.

NOTE: The instructions below also apply to linking non-hypothetical deals in a hedge relationship.

Once you have entered your Hypothetical Derivative deal, link it into a hedge relationship as follows.

  1. On the Hedge Relationship page's Details tab, enter the hedge relationship details.
  2. On the Linked Deals tab, select the Exposure, Hedge, and deals you want to link into the relationship, using any of the following tab menu functions (depending on what you know about the deals):
    • Add By > Deal... - select one or more deal numbers, separated by commas, and click OK.
    • Add By > Projected Cash Flow... (live deals only) - select one or more cash flow numbers, separated by commas, and click OK.
    • Add By > Query... - to view available deals, select the Entity, enter a From Date and To Date, and optionally select Include Unapproved Deals. Optionally filter these using the Search field. In the result list, select the deals you want and click OK.

      Note: Auto Allocate is not valid when a Hypothetical deal is linked.

    Your selected deals are displayed in the Linked Deals list.

  3. Ensure that for the Hypothetical deal, the Revaluation Method is the same as that for the Hedge deal. If not, click any value in that deal row (or select the Hypothetical deal and in the tab menu, select Edit); change the Revaluation Method as required, and click OK.
  4. Ensure that for a Swap or Currency Option Hypothetical deal, the Pay/Receive legs are equal and opposite. If not, change the required settings on the dealing page for the Hypothetical deal and save your changes. Return to the Hedge Relationship page and refresh the Linked Deals tab.
  5. Save the Hedge Relationship.

Maintaining a Hypothetical Derivative deal

Changes to a Hypothetical FX deal

When you make changes to a Hypothetical FX deal that is part of a Hedge Relationship (including Pre-delivery and Extension actions), a dialog box appears for you to enter the effective date for that change.

Either accept the default (today's date) or select a different date if required, and click OK. This creates an image with that effective date, for use when a revaluation is run on the deal.

TIP:

  • Internally, a date and time are recorded so that multiple changes in the same day are recorded in sequence.
  • You can view an audit history of changes if required (see below).

To view an audit history

  1. On the FX Deal page, recall the FX deal.
  2. In the page menu, select Audit History.

    The Audit History for Deal window displays a list of actions that have been performed on the deal. Use this information to identify previous images and their corresponding change dates.

Changes to the live Exposure deal

For any actions on, or changes to, the live exposure deal in a Hedge Relationship, the corresponding actions must also be performed on the Hypothetical deal.

For example, a Hedge Relationship might consist of a Money Market deal hedged by a Swap deal, then tested against a Hypothetical Swap deal. When the Money Market deal is rateset, the Hypothetical Swap deal requires the same action to maintain its status of the "perfect" hedge.

When you change any details of the Exposure deal, the system generates a warning message alerting you that parallel changes must also be performed on the Hypothetical deal (on the appropriate Dealing page). In particular, changes to the original exposure deal face value, dates, or price values must be also changed on the Hypothetical deal to ensure the accuracy of the effectiveness calculation.
