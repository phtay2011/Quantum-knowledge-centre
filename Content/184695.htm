Scheduled Jobs

Use the Scheduled Jobs page to:

  • view executed and scheduled Scheduler jobs,
  • re-run an executed job, using its current settings,
  • terminate a job in progress,
  • immediately run a scheduled job, and
  • view the message log for executed jobs.

NOTES:

  • The Job data is held in the Scheduler database, and Scheduler must be integrated with Quantum Six before you can use this page.
  • You can view and run only those jobs covered by a predefined Custom Module record for their Scheduler category, and for which you have the relevant Module Security permissions.
  • You must still use Scheduler to initially create and schedule jobs and tasks. See Batch Processing.
  • You can also use the Job page to run, view details, run history, and logs for, a single job.

On the Scheduled Jobs page:

  • the EXECUTED JOBS grid lists the Result status and details of completed Jobs in Job Categories you have access to, and
  • the SCHEDULED JOBS grid lists the previousResult status and details of Jobs that are scheduled to run and have not yet started or are still in progress.

Scheduled Jobs procedures

This topic covers the following Scheduled Jobs page procedures:

  • running a scheduled job immediately,
  • re-running a completed Job,
  • terminating a job being re-run, and
  • viewing the message logs for a completed Job.

To run a scheduled job immediately

TIP: You can also run a Job from the Job page.

  1. In the SCHEDULED JOBS grid, select the job row.
  2. Click Run. (or select Run from the grid menu).

    All Tasks defined in that Job will start, in Run ID order (as defined on the Job page).

    The job's Result status will show as In Progress until it finishes, then as either Success or Failure.

To re-run a completed Job run

  1. In the EXECUTED JOBS grid, select the job run row.
  2. Click Rerun (or select Rerun from the grid menu).

    All Tasks defined in that Job will start, in Run ID order (as defined on the Job page).

    The job's Result status will show as In Progress until it finishes, then as either Success or Failure.

To terminate a job currently being rerun

  1. In the EXECUTED JOBS grid, select the job run row (which must have a Result status of In Progress).
  2. In the grid menu, select Terminate.

    The Job re-run will stop and the job's Result status will show as Failure.

To view the message logs for a completed Job

  1. In the EXECUTED JOBS grid, select the job run row.
  2. Click View Log. (or select View Log from the grid menu).

    The Log dialog box displays the logged messages for the run. The types of message displayed depend on the Include Messages of Type setting/s for the Job.

TIP: You can also view a past run's logs from the Job page's History tab.
