Exercising Currency Options

You can exercise a Currency Option deal when it is in the money* and is:

  • a Vanilla European option expiring on Effective Date,
  • a Vanilla American option expiring on or before Effective Date,
  • a Barrier OUT option not yet knocked out,
  • a Barrier IN option that has been knocked in, or
  • an Asian option that has been Average Rate fixed

Such an option will have two entries in the Results view: one with Average Rate Fixings in the Type column and a Status of Fixed; the other with Exercise Lapse in the Type column and a Status of None. The Exercise action is available only on the latter entry.

TIP: If the Results list does not have a Status column, use the view's Configure link and the Columns tab to add it.

*NOTE: An option is said to be in the-money if its Strike Price is favorable compared to the current price of the underlying security - that is, when the Strike Price is:

  • below the current price for a Call option, or
  • above the current price for a Put option.

To Exercise a Currency Option option deal

  1. If the current result list does not show the deal you want, check the Filters values and that the selected view is either Exercise/Lapse or Default.

    TIP: Select the Exercise/Lapse view option to show only the options an Exercise is currently valid for.

  2. If necessary, adjust the Filter values and re-run the query.
  3. In the Result list, locate and select the deal entry.
  4. In the list menu, select Exercise.

    Either the Cash Exercise or Physical Exercise dialog box opens, depending on the Settlement Type setting for the deal. The Deal Number is displayed as a read-only value (and so is the Settlement Currency if it is a Cash Exercise).

  5. In the dialog box:
    • if a Cash Exercise, edit the Cash Exercise Date, and/or Amount* if required, or
    • if a Physical Exercise, edit the deal values as required - that is, Ticket Number, Instrument, NBD Settlement, Counterparty, Entity, Relationship, Broker (when applicable), and Strategy - also edit the Exercise Date if required, and
    • in both cases, enter or edit the common settings as required - that is, Dealer, Current Market Rate*, Value Date, Comments, and Location.

      *TIP: For a Cash Exercise, the Amount value is automatically calculated by the system based on the displayed Current Market Rate value (however, if you change the Amount value, the Current Market Rate is not changed).

  6. Optionally enter any text Comments for reference purposes.
  7. Click OK.

    The exercise operation is performed on the selected Currency Option deal. In the results list, the entry's Status changes to Exercised, and:

    • if a Cash Exercise, a cash flow is created, or
    • if a Physical Exercise, an FX deal is created.

NOTE: When this action has been performed for a particular date, the action can only be undone by using the Rollback Deal function.
