Mapping Profile Definition page

Use this page to create Mapping Profile Definition records and folders.

A Definition record's settings will be applied when a query configuration of the corresponding type specifies that record's name (Mapping Profile field, Properties tab, Query Configure). A Mapping Profile is mandatory for Monte Carlo and Parametric VaR queries, and if you do not select an existing Definition record for such queries, a new Definition record will be created (with the sam Name as the query).

Any changes made on the Mapping Profile tab will be saved in the specified Definition record when the query configuration is saved.

To display an existing folder or record - Click , navigate in the browse list to the item you want (or use the Search tool), and select it. The folder contents or record settings are displayed and the breadcrumb trail indicates the current location.

To add a new record (at the current location) - Click the New button and select the profile type (General or VaR); enter a new Name for the Definition record; optionally enter text Comments; select or enter the required settings*; then click Save.

TIP: You can also create a new record by copying the current one. In the page menu select Copy; enter a new Name; change the required settings*; then click Save.

To add a new folder at the current location - Click the New button and select Folder in the drop-down menu; enter a new Name for the folder; then click Save.

To change a Definition record's details - Locate the record as above, change settings* as required; then click Save.

To move a record or folder to a different location - Locate the record or folder and in the page menu select Move. In the Select Group to Move To dialog box, locate the target folder and select it. The new location becomes the current location.

To delete a record or folder not in use - Locate the record or folder and in the page menu select Delete. When prompted, click OK to confirm.

To view changes made to the Definition record - In the page menu select Audit History.

* Each custom grouping has a set of Matching Criteria values and the appropriate Settings for those criteria - namely, an optional override for the Yield Curve and the following values, depending on the type of Mapping Profile. For a:

  • General type (all other query types) - Time Profile and Risk Type; or for a
  • VaR type (Monte Carlo VaR and Parametric VaR queries only) - Asset Class, Probability, Correlation Data File, and Volatility Data File.
