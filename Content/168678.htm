Account Interest

Use the Account Interest Rate page to record interest rate tables that can be used to calculate Bank/Fund Account interest.

TIP: The 'all-in' Bank Interest Rate may be positive or negative, depending on the Base Rate and Margin.

The combination of Account Type, Bank Account, Rate Type, and Date uniquely identifies this type of rate record in the system, and is displayed above the header fields.

Rates area, Account Interest Rates

The following rules apply when working in this area.

  • You cannot save the rates table if it contains blank rows. To delete rows, select the row then select Delete Row from the Edit menu.
  • For each balance category (O/D or In Funds), you must define one, and only one, Over row.
  • For O/D rows, Over means further into overdraft. For example, an Over $5.00 O/D row would include balances that are $6.00 O/D, but not balances that are $4.00 O/D.
  • Up To means Up to and including. Over means Over and excluding.
  • You can enter rows in any order. When you save the record, Quantum sorts the rows based on the amount column, with O/D rows at the bottom of the spreadsheet, and In Funds rows at the top.

To add a new rate entry from scratch:

  1. In the Actions menu, select New. The New Rate dialog box appears.
  2. Select the Balance type, enter the other values (see below) and click OK. The new entry is added to the table.

To add a new rate entry by copying an existing one:

  1. Select the row you want to copy. In the Actions menu, select Copy. The Copy Rate dialog box appears.
  2. Select a new Balance type, change other values as required (see below), and click OK. The new entry is added to the table.

To edit an existing rate entry:

  1. Select the row you want to edit. In the Actions menu, select Edit. The Edit Rate dialog box appears.
  2. Change values as required (see below), and click OK. Your changes are shown in the table.

To remove a rate entry:

  1. Select the row you want to remove. In the Actions menu, select Delete. The Confirm Delete dialog box appears.
  2. Click OK. The entry is removed from the table.

Field Descriptions

Balance - Select In Funds or O/D to define the conditions the rate applies to. Double-click to toggle between the options.

Type - Select Up To or Over to indicate whether the rate is for a marginal balance band, or up to a balance amount. Click to toggle between the options:

Amount - The amount at which the rate applies.

Base Rate - The base rate from which the actual bank Rate is automatically calculated by the system (positive or negative), as follows:

Rate = Base Rate ± Margin, adjusted if necessary so that:

Floor Rate <= Actual Rate <= Cap Rate

Margin - Optional. Enter any margin that the system will apply to the Base Rate when calculating the actual bank Rate, You can enter a positive or negative value.

Cap Rate - Optional. Enter any cap value that the system will apply when calculating the bank Rate. You may enter a negative value if negative rates apply.

Floor Rate - Optional. Enter any floor value that the system will apply when calculating the bank Rate, You may enter a negative value if negative rates apply.

Rate (Read-only; automatically calculated by the system) - The rate (positive or negative) calculated as:

Rate = Base Rate ±- Margin, adjusted if necessary so that:
