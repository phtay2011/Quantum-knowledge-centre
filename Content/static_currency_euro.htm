Euro tab, Currency

(Transaction-level currencies only.)

This tab applies when you are adding a new currency to the Euro currency group, or when you select a currency that is a member of that group.

Use this tab to set the role of the currency within the group, the conversion period, and the translation rules for conversion to the Euro Currency (the combination of Fixed Translation Rates and Triangulation Rounding Methods used for the conversion). See also: About Currency Groups and About the Euro.

NOTE: If you are translating bank accounts using the Euro currency, some rounding may take place. Values may be rounded to the largest number of significant figures (14 decimal digits).

Euro Group Role - one of the following:

  • Head - if you are setting up the Head Currency (the Euro Group can have only one Head Currency)
  • IN Currency - If the selected currency is not the Head Currency (an IN currency is a member of the group of currencies that form the EMU)

TIP: Head currency applies to the Euro only. It is a currency that has been set as the head of the basket of IN currencies (including the Head currency). For example, EUR could be the Head currency of the Euro Group. When setting up the Head currency, you set up the translation rules used to convert the IN currencies to the Head Currency.

Start of Convergence - the date from which the Translation Rules apply

End of Convergence - the date after which the Translation Rules no longer apply
