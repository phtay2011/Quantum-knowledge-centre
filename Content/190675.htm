Audit Reporting Enhancements

(First introduced in Release 6.5.1 SP, September 2017.)

Audit Reporting has been updated to record additional rates and changes to message sources, Deal Notes, Counterparty Delivery Instructions, and Netted cash flows.

Rates

Audit Reporting is now supported for the following additional* rates:

  • Futures Rates
  • ETO Rates
  • Account Interest Rates
  • Index Rates
  • Yield Rates
  • Issuer Instrument Margin Rates
  • Issuer Instrument Rates
  • Swaptions Volatility Rates
  • ETO Volatility Rates

*NOTE: Auditing of Rates was first introduced in Release 6.5.0 for FX, FX Volatility, and Deal Margin rates.

External Method

In addition to standard actions (Add, Delete, Activate, Deactivate), changes to all fields and options concerning the selected Method Type are saved to Audit Reporting.

External Method Source

In addition to standard actions (Add, Delete, Activate, Deactivate), the following changes to a record in the Data Transfer Location page are saved to Audit Reporting:

  • Bank Balance Adjustment Import area (Format, File Name, and Source Folder fields).
  • EFT Processing area (Type Export, Output File Name, File Destination, XML Configuration File, and CRC Program fields).
  • EFT Message Defaults area (Non-Financial Transfer, Counterparty Mapping, Target2 Directory, and Target2 Address Source fields).
  • Digital Signature area (Detach digital signature, Signature Type, Certificate Store, Certificate Criteria Type, and Certificate Criteria fields).

Data Transfer Location

In addition to standard actions (Add and Delete), the following changes to a record in the Data Transfer Location page are saved to Audit Reporting:

  • Details area (Location Type, Purpose, Location, and Transformation fields).
  • Digital Signature area (Signature Type, Certificate Store, Certificate Criteria, and Certificate Criteria Type fields).
  • Encryption area (Encryption Type, Certificate Store, Certificate Criteria, and Certificate Criteria Type fields).

Deal Notes

For both Deal Notes and Deal Notes Attachments, you can now review all related changes (adding, deleting, editing) in the Audit report.

The Audit report (subject to any filters), displays the following data for Deal Notes.

  • Table and table field affected by the change. The field affected could be:
    • Note Text - if the text of note is added, updated, or deleted, or
    • Note Text Format - if the format of the note text is changed (text bolded for example).
  • Type of change - Add, Update, or Delete.
  • Deal number to which the note is related.
  • The value of the field before and after the related change.

    The value is formatted and displayed as readable text except when the text format is changed, when HTML is displayed. If the text of the note is long, you can scroll to the right to see the complete note.

  • Date and time of the change.
  • User who made the change.

Counterparty Delivery Instructions

In Audit Reporting:

  • when you access a Deal Image, you can view the Counterparty Delivery Instructions that applied to any cash flow in that deal image;
  • changes to Counterparty Delivery Instructions are now recorded for each individual Cash Flow in a netted group; and
  • audit entries of Counterparty Delivery Instruction changes now also record changes to:
    • Details of Charges,
    • Sender's and Receiver's Charges, and
    • Charges Comments.

Netted Cash Flows

Changes to Counterparty Delivery Instructions for each individual Cash Flow in a netted group are now saved to Audit Reporting.
