Mapping Profile Definition Procedures

Use the following procedures to create and maintain Mapping Profile Definition records and folders.

Managing Records and Folders

To display an existing folder or record - Click , navigate in the browse list to the item you want (or use the Search tool), and select it. The folder contents or record settings are displayed and the breadcrumb trail indicates the current location.

To add a new record (at the current location) - Click the New button and select the Mapping Profile type (General or VaR);; enter a new Name for the Definition record; optionally enter text Comments; select or enter the required settings*; then click Save.

TIP: You can also create a new record by copying the current one. In the page menu select Copy; enter a new Name; change the required settings*; then click Save.

To add a new folder at the current location - Click the New button and select Folder in the drop-down menu; enter a new Name for the folder; then click Save.

To change a Definition record's details - Locate the record as above, change settings* as required; then click Save.

To move a record or folder to a different location - Locate the record or folder and in the page menu select Move. In the Select Group to Move To dialog box, locate the target folder and select it. The new location becomes the current location.

To delete a record or folder not in use - Locate the record or folder and in the page menu select Delete. When prompted, click OK to confirm.

To view changes made to the Definition record - In the page menu select Audit History.

* Changing the current Definition record

To add a new set of Matching Criteria and Settings:

  1. Click the New button^ and in the drop-down menu, select the type of mapping Profile: either General or Var, as required.

    TIPS: The default is General (which applies if you select New from the Matching Criteria grid menu). VaR applies only to Monte Carlo VaR or Parametric VaR queries.

    The New Matching Criteria dialog box appears.

  2. Under Settings, select a Mapping Code.

    The other Settings values show the corresponding defaults (to change these, see Step 6).

  3. Under Matching Criteria, select specific values or All for Currency, Entity, Transaction Type, Instrument, Counterparty, Issuer, Curve, and Unit Issue.
  4. Under Matching Order, enter the Order Number that specifies the priority order for applying this criteria (in relation to other criteria).
  5. Click OK.

    A new row is added to the Matching Criteria grid, showing the new values, and - only if no Settings grid row already exists for the selected Mapping Code - a new row is also added to that grid.

  6. In the Settings grid, select the row for the new Mapping Code and in the grid menu, select Edit.

    The Edit Settings dialog box appears.

  7. If required, select a different Yield Curve to be used in the calculation.

    TIP: This Yield Curve will override the one specified in the applicable Valuation Rule.

  8. If the Mapping Profile type is:
    • General - select a Time Profile and Risk Type, or
    • VaR - select an appropriate Asset Class and enter a Probability(%) between 1 and 100.
  9. Click OK.

    The Edit Settings dialog box closes.

  10. (VaR Mapping Profile only) Under VaR Files, enter a Correlation Data File and/or Volatility Data File.

^TIP: You can also create a new set by selecting an existing one and selecting Copy. Enter a new name and edit the Matching Criteria and Settings values as required.

To edit existing Matching Criteria:

  1. In the Matching Criteria grid, select the row you want to edit.
  2. In the grid menu, select Edit.

    The Edit Mapping Profile dialog box appears.

  3. Edit the required fields and click OK.

To edit the Settings for existing Matching Criteria:

  1. In the Settings grid, click on the Mapping Code for the criteria.

    The Edit Settings dialog box appears.

  2. If required, select a different Yield curve to be used in the calculation.

    TIP: This Yield curve will override the one specified in the applicable Valuation Rule.

  3. If the Mapping Profile type is:
    • General - select a different Time Profile and/or Risk Type if required, or
    • VaR - select a different Asset Class and enter a new Probability(%) between 1 and 100 if required.
  4. Click OK.

    The Edit Settings dialog box closes.

To change the priority order of a Matching Criteria set:

  1. In the Matching Criteria grid, select the row you want to move.
  2. In the grid menu, select Edit.

    The Edit Mapping Profile dialog box appears.

  3. Under Matching order, change the Order Number as required, and click OK.
  4. The rows will reorder accordingly.

To change the order of the Matching Criteria Columns:

TIP: This order determines which column takes precedence when more than one set matches.

  1. In the Matching Criteria grid menu, select Reorder Columns.
  2. in the dialog box, drag and drop the column buttons as required.
  3. Click OK.

To delete an existing set of Matching Criteria and Settings:

  1. In the Matching Criteria grid, select the row you want to delete.
  2. In the grid menu, select Delete.

    A confirmation message appears.

  3. Click OK to confirm.

    The Matching Criteria row and its corresponding Settings row are removed from the respective grids.
