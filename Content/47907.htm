Passing through Zero

NOTE: You can use the Weighted Average Closeout Processing page as an alternative to selecting the WEIGHTED AVG Closeout Method on the Securities Instrument page.

If a new deal in a portfolio will cause the portfolio's total Face Value to pass through zero, whether from a long or short position, the new deal is split into two entries. These follow the FIFO closeout methodology.

  • The first entry has an equal but opposite Face Value and Weighted Average Capital Value to those currently for the Portfolio, to bring the new Portfolio Face Value and Weighted Average Capital Value to zero.
  • The second entry has the remaining Face Value and is treated similarly to the first deal in a portfolio.
