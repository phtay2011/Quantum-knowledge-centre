/DB

/DB type server [database] ...

Used with the ARCHIVE or PURGE command to specify the database to connect to, where:

  • type specifies the type of database: SQLServer or Oracle
  • server specifies the database server name
  • database specifies the database name (not required for Oracle)

The /DB parameter must be used in conjunction with at least one of the following parameters that specify the type of data concerned.

With ARCHIVE

With PURGE

/CASHMGMT

/DEALAUDIT

/GL

/CASHMGMT

/ALERT

/MESSAGE

/REPORT

/RTU

Other optional parameters also apply.

For more details, see Archive Command Line Parameters and PURGE Command Line Parameters.

Example 1

/DB SQLSERVER OURSERVR\SQL2005 AGQMAINDBS ...

Connects to an SQLServer database named AGQMAINDBS on the server OURSERVR\SQL2005.

Example 2

/DB ORACLE SERVRORA11 ...

Connects to the Oracle database on the server SERVRORA11.
