Deleting a deal

A DELETE DEAL ACTION TAKES IMMEDIATE EFFECT AND CANNOT BE UNDONE - EVEN IF SUBJECT TO APPROVAL OR REJECTION.

The approval or rejection simply records whether the action was approved or not - rather than controlling whether the action can actually take place.

Methods

To delete a deal using the menu item

  1. Select Delete in the Dealing page menu and the following window appears:

  2. Check the list of cashflows before proceeding.
  3. Add notes, if required.

NOTE:

  • Deleting a deal affects cashflows associated with the deal.
  • Deleting a linked deal triggers deletion of all linked Cash Flow deals.

 

TIP:To reduce the risk of inadvertent deletions, System Administrators can restrict the specific Delete Module Security permission for each Dealing module to a few selected users.

To delete a deal using QXT functionality

You can delete single or multiple deals using QXT functionality.

However, the system applies some limits when you delete FX Swap deals that are linked to FX deals. When deleting an FX Swap deal, a cashflow warning message appears that describes which flows will be impacted. If you go ahead and delete the FX Swap deal, the system records the delete action in the audit history. Open the Audit History from the page menu item. You cannot delete an FX deal when the FX deal is linked to an FX Swap deal.

You can delete an FX Swap or Index Rate Swap using the QXT importation functionality, but you cannot delete an FX deal that is linked to an FX swap using the QXT importation functionality. To delete an FX deal:

  1. Open a QXT Formal file.
  2. In the data record, enter the following command for an FX Swap:

    @FXSwapManager
    DealNumber=<one of the deal's numbers, near or far>
    $DELETE

  3. In the data record, enter the following command for an Interest Rate Swap. The first is a single delete, the second is multi-deal delete:

    @InterestRateSwapDeal
    $NEW
    DealNumber=12
    $DELETE

    @InterestRateSwapDeal
    $NEW
    DealNumber=replace_DealNo2
    $DELETE
    $NEW
    DealNumber=replace_DealNo3
    $DELETE
