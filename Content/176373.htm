Targets used in Regression

Target measurement values on the Hedge Relationship's Effectiveness tab default from the corresponding Hedge Strategy page settings. They may be changed only for new Hedge Relationships (not after one has been saved).

Minimum R-squared

(Mandatory when the associated Test Method is REGRESSION.)

Enter a minimum R2 value between 0 (no correlation) and 1 (perfect correlation), or accept the default of 0.8. Up to four decimal places will be accepted.

A hedge relationship will be deemed as effective only when the calculated R2 value is equal to or higher than this specified value. The calculation is based on the change in total Hedge value and the change in total Exposure value from one valuation date to the next.

TIP: The correlation coefficient R2 represents the overall variance from the line of best fit for the Ordinary Least Squares (OLS) calculation. It indicates the proportion of variation in the value of the Exposure that is explained by the variation in the Hedge position. It is always used as a criterion in determining whether the hedge relationship is effective or not.

Slope (Lower/Upper)

(Mandatory when the associated Test Method is REGRESSION.)

Enter a slope coefficient. Valid values are from -999.9999 to 999.9999 and the Lower value cannot exceed the Upper value.

TIP: A value of 1 indicates perfect effectiveness (the Regression line has a 45-degree slope).

Enter both Lower and Upper values to define an acceptable range for the slope. A deal will be deemed as effective only when the calculated slope is within this range.

The slope indicates the angle of the line of best fit (the Regression line). It is always used as a criterion in determining whether the deal is effective or not.

Intercept (CCY Units)

(Optional when the associated Test Method is REGRESSION; applies only if the Include intercept option is selected.)

Enter an intercept limit, or accept the default of 0 (perfect effectiveness). Valid values are from -999.9999 to 999.9999 and the Lower value cannot exceed the Upper value.

Enter both Lower and Upper values to define an acceptable range for the intercept value. When Include intercept in effectiveness assessment is selected, a deal will be deemed as effective only when the calculated intercept is within this range.

TIP: The Y intercept indicates the origin of the line of best fit (the Regression line). It determines where results should appear on the Y axis. Points along this axis are plotted in currency units. Where there is perfect correlation, the line passes through the origin of both the X and Y axes.

Confidence Interval

(Optional when the associated Test Method is REGRESSION.; applies only if the Include slope confidence interval option is selected.)

Select the method for calculating the statistic that describes the precision of the Intercept and Slope estimates used in the regression calculations, and the level of confidence required for a:

  • Student's t distribution - choose a percentage confidence level, or for a
  • Standard Error calculation - choose a target of 1, 2, or 3 Standard Deviations from the mean.

This value:

  • may be left blank for the hedge strategy, but if so, must be entered for the hedge relationship,
  • is system-derived and is displayed in the Slope and Intercept OLS Range columns of the Effectiveness result grid for the deal,
  • Is used to determine the effectiveness of a deal if Include slope confidence interval in effectiveness assessment is selected (a deal will be deemed as effective only when movements in the value of the Hedge and Effectiveness position lie within the allowable deviation from the line of best fit, as determined by this Confidence level), and
  • Is used only for information purposes if the Include slope confidence interval option is cleared.

Slope T Statistic

(Optional when the associated Test Method is REGRESSION; applies only if the Include slope t statistic option is selected.)

Select the t statistic percentage confidence level.

TIP: The t statistic for the X coefficient evaluates the probability that the slope is zero. A slope of zero indicates that there is no relationship between the X and Y variables. A high t statistic, positive or negative, is generally a good indicator of correlation.

This value is used:

  • to determine the effectiveness of a deal if Include slope t statistic in effectiveness assessment is selected (a deal will be deemed as effective only when the significance level of the calculated t statistic is at the specified level or higher), but
  • only for information purposes if the Include slope t statistic option is cleared.

Include Intercept in effectiveness assessment

(Optional when Prospective Test Method is REGRESSION.)

Select this option to include the Intercept result as a criterion in determining Prospective effectiveness for hedge relationships based on this hedge strategy. A hedge relationship will be deemed effective only when the calculated Intercept is between the associated Intercept Lower and Upper values.

Clear this option to exclude the Intercept result as a criterion in determining Prospective effectiveness. Any associated Intercept Lower and Upper values will be used only for information purposes.

Include Slope Confidence Interval in effectiveness assessment

(Optional when Prospective Test Method is REGRESSION.)

Select this option to include the Slope Confidence Interval result as a criterion in determining Prospective effectiveness for hedge relationships based on this hedge strategy. A hedge relationship will be deemed effective only when the calculated Slope Confidence Interval is between the associated Slope Lower and Upper values.

Clear this option to exclude the Slope Confidence Interval result as a criterion in determining Prospective effectiveness. Any associated Slope Lower and Upper values will be used only for information purposes.

Include Slope t Statistic in effectiveness assessment

(Optional when Prospective Test Method is REGRESSION.)

Select this option to include the Slope t statistic result as a criterion in determining Prospective effectiveness for hedge relationships based on this hedge strategy. A hedge relationship will be deemed effective only when the significance level of the calculated t statistic is at or higher than the associated Slope T Statistic significance level.

Clear this option to exclude the Slope t statistic result as a criterion in determining Prospective effectiveness. Any associated Slope T Statistic value will be used only for information purposes.
