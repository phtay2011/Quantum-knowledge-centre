Account Selection Clause

  • This Account Selection clause syntax also applies when Iron Python Scripting is used for data manipulation.
  • Within a Bank Transaction Rule Set entry, the Bank Account multi-select field corresponds to this Selection clause, and if an existing Rule is uploaded from a file, that field is populated from the Selection clause, EXCEPT THAT specifying a contiguous range of accounts using a TO operator is not supported (you will have to select each account).

Description

Specifies the account or range of accounts to which the data manipulation will be applied.

Syntax

[FOR ( ACCOUNT number | ACCOUNTS range) | ALL ACCOUNTS]

Where:

  • number is an account number string. The account number can have hyphens, brackets and other similar characters in it.

    If the account number has spaces, commas or semi-colons in it, then the account number must be enclosed in quotes.

    For example, FOR ACCOUNTS "123 456" SG78, AQT-910; will be read as three accounts:

    • 123 456
    • SG78
    • AQT-910
  • range specifies the first and last account number in a range, in the following format:

    account number1 TO account number2

Examples

FOR ALL ACCOUNTS
FOR ACCOUNTS 1234 TO 2000
FOR ACCOUNT 2468
