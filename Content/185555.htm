Matching Criteria, Settlement Approval Rules

These items define which cash flows or stock flows the Settlement Approval rule entry applies to.

  • Flow Type - select CASH, STOCK, or NET*.
  • Transaction Type* - select one transaction type or ALL*.
  • Instrument* (filtered by Transaction Type) - select one instrument or ALL*.
  • Flow Ccy - select the flow currency, a currency group, or ALL.
  • Payment Method - select one External Method, or ALL.
  • Matched Status* - select ALL* or a specific Deal Confirmation Message Status: Auto Matched, Manual Matched, or Unmatched.

    This value refers to the current Message Status of the flow's Confirmation message, as shown on the Confirmations Processing page. It enables you to define different Settlement Approval rule entries for the different ways that the related Deal Confirmation is actually matched at run time.

  • Counterparty - select one counterparty or ALL.
  • Flow Direction - select Payments, Receipts, or Both.
  • Account (filtered by Flow Type) - select ALL, one account, or one account group.
  • Payment Instructions* - select Ad Hoc, Modified SSI†, SSI†, or ALL*.

    This value refers to the change status of the flow's Counterparty Instructions. It enables you to define different Settlement Approval rule entries depending on whether and how these Instructions get changed, where:

    • SSI (Standard Settlement Instructions) means all details are unchanged from the defaults supplied by the applicable Counterparty Account defaulting Rule;
    • Modified SSI means only the Counterparty Account, Delivery Narrative, Bank to Bank Instructions, or Charges details have been changed from the defaults;
    • Adhoc means any other details are changed so that a one-off ad hoc set of instructions apply; and
    • ALL means the rule applies whether or not changes are made.
  • Limit Ccy - select the currency of the Limit Amount OR NONE for non-tiered approvals (no lower limit).

    For Tiered Approvals, you must specify the same currency in entries with identical Matching criteria except forLimit Amount.

  • Limit Amount (not applicable for Unit stock flows) - enter the minimum threshold amount for flows you want this Approval Rule entry to apply to. Limit amounts apply to the cash flow or stock flow amount, or, for a NET, to the Net total.

    For no lower limit, set Limit CCY to NONE and Limit Amount to zero. Approval will then be required for flows of any amount.

    For Tiered Approvals, the lowest Limit Amount equaled or exceeded by that deal amount determines the applicable rule entry (the approval tier).

*Netting and Settlement Approval

Netted groups of cash flows can potentially contain a mixture of Entities, Transaction Types, Instruments, Match Status, and/or Payment Instructions. The ALL Entity record is used where the Entity differs; the applicable ALL rule Matching Criteria values apply where the other values differ. See Cash Flow Netting and Settlement Approval Rules.
