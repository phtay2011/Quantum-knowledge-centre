Calendar Procedures

NOTE:

  • Holidays and non-standard Business days can be defined at either a currency or location level, therefore you must first define your currencies and locations.
  • Calendar changes affect only deals saved after the changes are saved.

On the Business Day Calendar page:

To list Calendar records

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Calendar record

  1. Click New.
  2. Select either the Location or Currency that you want to define holidays and/or non-standard Business days for.
  3. On the Holidays tab's menu, select New.
  4. In the New Holiday dialog box, enter the new Holiday Name, use the remaining fields to set those rules necessary to define it, and click OK. See Specifying Holidays.
  5. Repeat Steps 3 and 4 for any further holidays you want to specify for the Location or Currency.
  6. (Optional; more likely in Asia) On the Business Days tab's menu, select New.
  7. In the New Business Day dialog box, enter the new Holiday Name and the Fixed Date you want to change from a standard Non Business day to a non-standard Business day, select the required Observable setting, and click OK. See Specifying Non-Standard Business Days.
  8. Repeat Steps 6 and 7 for any further dates of this type for the Location or Currency.
  9. Save your new record.

You have defined holidays and/or non-standard Business days that transactions will reference based on Product-specific rules.

To edit a Calendar record

NOTE: You should not change the key Location and Currency fields. Create a new record instead.

  1. Click to list available records (or use the Search box).
  2. Select the record you want to edit.
  3. Use the Holidays tab to maintain fixed holidays for the Currency or Location, as follows.
    • To add a new holiday, select New from the tab's menu. In the New Holiday dialog box, enter the new Holiday Name, use the remaining fields to set those rules necessary to define the holiday, and click OK.
    • To edit an existing holiday, select the holiday row you want to edit and select Edit from the tab's menu. In the Edit Holiday dialog box, edit the data fields as required, and click OK.
    • To remove an existing holiday, select the holiday row you want to remove and select Delete from the tab's menu. In the Confirm Delete dialog box, click OK.
  4. (Optional; more likely in Asia) Use the Business Days tab to maintain non-standard Business days for the Currency or Location (specific dates otherwise non-business days), as follows.
    • To add a new non-standard Business day, select New from the tab's menu. In the New Business Day dialog box, enter a new Name and Fixed Date, select the required Observable setting, and click OK.
    • To edit an existing Business day, select the day row you want to edit and select Edit from the tab's menu. In the Edit Business Day dialog box, edit the Fixed Date as required, and click OK.
    • To remove an existing Business day, select the day row you want to remove, and select Delete from the tab's menu. In the Confirm Delete dialog box, click OK.
  5. Save your changes.

To list holidays and non-standard business days over a period

  1. In the page menu, select Test Holidays and Business Days.

    The Test Holidays and Business Days dialog box is displayed.

  2. Select a From Date and To Date;.
  3. Select a From Date and To Date to specify the date range you want.
  4. (Optional) Select a Currency and/or Location (individual or group in each case).
  5. Click Test.

    The Dates grid is populated with the applicable holidays and non-standard business days.

To delete a Calendar record

  • You cannot delete a record if that affects transaction, settlement, or financial data, and you cannot delete a group if it has members.
  • Deleting removes the record completely from the database. To prevent use of the record but retain it in the database, deactivate the record instead.
  1. Click to list available records.
  2. Select the record you want to delete (use the < and > buttons to page through, or type in the Search box).
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To deactivate a Calendar record

  1. Click to list available records, or use the Search box.
  2. Select the record you want to deactivate.
  3. In the menu, select Deactivate. This action will not appear if the record is already deactivated.

To reactivate a Calendar record

  1. Click to list available records, or use the Search box.
  2. Select the record you want to reactivate.
  3. In the menu, select Reactivate. This action will not appear if the record is already activated.
