About Layout XML Data records (Confirmations)

The Layout XML Data record for each message type and Confirmation External Method:

  • defines output message structure and content,
  • contains XML code giving a variety of rules relating to the structure and formatting of the message,
  • enables users to construct almost any message desired from the data defined in the Data Template XML Data record for the message type,
  • contains links referring to common structures and elements in an Includes XML Data record, and
  • can use, or be based on, a Seed record provided in Quantum for that message type and named SeedXMLLayout_MTnnn (for SWIFT) or SeedXMLLayout_MTnnn_Misys (for Finastra) where nnn is the message type - for example, SeedXMLDataTemp_MT300, SeedXMLLayout_MT304_Misys, and so on.

Elements and attributes

These are common to both Confirmations and EFTs.

See Layout file elements and attributes.
