Counterparty Instructions function (Cash Flow)

(Quick menu function on a single Cash Flow item or on a Netted group OR Cash Flow tab grid menu function from applicable Dealing pages.)

For a cash flow, the Counterparty ("their") delivery instructions are those concerning the counterparty's account, which is the bank account you are paying into or receiving cash from, EXCEPT THAT:

...if an Adhoc instruction currently exists for the cash flow, the default instructions are obtained from the applicable Counterparty Bank Account Defaulting Rule.

NOTE: When a Clearing Broker is specified for the deal, the default instructions are those for that Broker Counterparty.

You can view and possibly change these instructions from the Settlements page results grid by clicking a linked value in the appropriate Cash flow grid row and selecting Counterparty Instructions from the Quick menu. A separate Counterparty Instructions window opens for the purpose.

You can also access Counterparty Instructions from these pages:

  • Actual Cash Flows and other Dealing pages - for viewing and changing if required, and
  • Settlement Approvals - for viewing only.

TIP: The words All Forward are appended to the page title IF the Instructions have been accessed from a Dealing page's Cash Flows tab via the Change All Forward menu option. For more details, see Changing Counterparty Instructions from Dealing Pages.

Subject to any restrictions, three modes of operation are available (for a cash flow) when this page is accessed fromthe Settlements or a Dealing page's Cash Flows tab. Each mode allows different types of change and can affect the Payment Instructions setting, which (if unique) is displayed after the window title (see Changing Counterparty Instructions).

The three modes are:

  • Account mode (the default and the only mode available when this page is accessed from the Settlement Approvals page),
  • Adhoc Instruction mode, and
  • Third Party* mode.

TIP: Only Third Party mode allows you to change the Ordering Customer.

More details of each mode are given below.

Account mode

This is the default mode when no Adhoc instructions already exist for the cash flow.

Use this mode to view and optionally change the following default values.

  • Bank to Bank Instruction,
  • Delivery Narrative,
  • Charges, and/or
  • the Counterparty Bank Account (which changes the instructions to those for that account).

    NOTE:

    • The list of accounts either shows all Counterparty accounts or only accounts for the selected Counterparty and Currency, depending on whether or not the system-wide Operational Settings option Display Bank Accounts for all Counterparties is selected (under Bank Accounts).
    • Changing the bank account resets the External Method to the default for that account, but you may change it.

Making any of the changes above changes the Payment Instructions setting from SSI to Modified SSI, which may affect Approvals and EFT output. See Changing Counterparty Instructions.

In this mode the following tabs apply (see also Header).

  • Beneficiary Details tab
  • Bank Details tab
  • Intermediary Bank tab
  • Identifiers tab

Adhoc Instruction mode

To access this mode: in the page menu, select Change to Adhoc Instruction.

Use Ad Hoc Instruction mode to:

  • view any existing Adhoc Delivery Instructions for the selected cash flow (if any apply), and/or
  • create new ad hoc (one-off) Delivery Instructions for payment of the selected cash flow, replacing the default Counterparty Bank Account Delivery Instructions. This creates a separate payment record.

This is the default mode when an Adhoc instruction currently exists for the cash flow.

NOTE:

  • You can either use the existing Beneficiary Name or enter a new one in free format.
  • Creating an Adhoc instruction sets the Payment Instructions value to Adhoc instructions, which may affect Approvals and EFT output. See Changing Counterparty Instructions.

In this mode, no defaults are displayed, and the following tabs apply (see also Header).

  • Beneficiary Details tab
  • Bank Details tab
  • Intermediary Bank tab
  • Identifiers tab

Third Party Payments mode

To access this mode: in the page menu, select Change to Third Party.

NOTE: This is the only mode that allows you to change the Ordering Customer. It is effectively the Adhoc Instruction mode with the added ability to process Third Party Payments, and has the same restrictions as the Adhoc mode.

Use Third Party mode to:

  • view existing Adhoc Delivery Instructions for the selected cash flow on a third party payment (if any such instructions apply), and
  • create new, ad hoc (one-off) Delivery Instructions for payment of the selected cash flow on a third party payment, replacing the default Counterparty Bank Account Delivery Instructions. This creates a separate payment record, and sets the Payment Instructions value to Adhoc instructions, which may affect Approvals and EFT output. See Changing Counterparty Instructions.

In this mode, no defaults are displayed, and the following tabs apply (see also Header):

  • Beneficiary Details tab
  • Bank Details tab
  • Intermediary Bank tab
  • Ordering Customer tab
  • Identifiers tab
