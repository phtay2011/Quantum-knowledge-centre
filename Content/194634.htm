Set up Structures to capture tabular data

(First introduced in Release SP 6.7.2, December 2018.)

Reports created by the Enterprise Workflow now support capture of tabular data. This data can be embedded in the report definition or entered by a user through bulk insert.

Implementation

On the Custom Report page, a Structures grid now appears, as in the following example showing two data structures - both included in the Sources grid and joined to each other (as highlighted).

Each structure has:

  • a mandatory display Name,
  • a list of one or more mandatory Columns, each with a display Name, and Type (Text, Number, Date, Yes/No),
  • optional Default Value settings (in tab-delimited form), and
  • either the Hidden or Bulk Insert option set, where:
    • Hidden indicates set data that users will not be able to change, while...
    • Bulk Insert provides a link enabling users to use a spreadsheet application (for example, Excel) to create and maintain the tabular data (including the Column headings).

      TIP: This is similar to the method used to Bulk Replace and Insert Deal Map Items.

Once a structure has been defined, it is available for selection as a data source.
