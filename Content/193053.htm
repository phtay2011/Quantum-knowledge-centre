Boolean literals

You can also use the Boolean literals true and false in any Boolean expression in calculated columns.

The literal values are case-insensitive and, as for all literals, must be within double quotes.

Null values

The operators that compare expressions handle blank values in the same way as SQL filters. This includes:

  • null = null - Not applicable because null can't be entered in a filter
  • null > 0 - False, but
  • null < 0 - False

And when you use a null value in a calculation, the system converts it to 0 (zero):

  • null + 1 converts to 0 + 1
  • 1 / null converts to 1 / 0
