Utilization Dates, Rules

Use the Limit Utilization Date page to set up and maintain Utilization Date records for combinations of Transaction Type and Instrument. These records control the way exposure is allocated between the deal counterparty and the underlying instrument's acceptor or issuer.The settings in these records will be used by Limit Query when calculating limit amounts.

By default, exposure is allocated to the deal counterparty from deal date until settlement date, and to the acceptor or issuer from settlement date onwards.

Quantum enables you to specify and monitor different Utilization Dates at a series of levels, including ALL Transaction Types, specific Transaction Types, or specific Instrument Types.

You can define Utilization Dates for transaction types and instrument groups, as well as for single instruments.

For example, you can set the same Utilized From date for all Securities instruments, by specifying a date only for the SE Transaction Type and ALL Instrument combination. The setting will apply throughout, unless overridden by a different setting for the SE Transaction Type and a specific Instrument.

About the Utilized From date

Use this setting to control the point in the life of the deal from which, for the selected Transaction Type and Instrument, exposure will be allocated to the Issuer/Acceptor or acceptor, according to the following rules.

Utilized From

Exposure is with...

From...

SETTLEMENT DATE
(the default)
  • Deal counterparty
  • Issuer/Acceptor
  • Deal date to settlement date
  • Settlement date onwards

DEAL DATE
  • Issuer/Acceptor
  • Deal date to settlement date and from settlement date onwards

NOTES:

  • For applicable Securities deals with an Issuing and Paying Agent and the All cash flows option selected on the underlying Instrument, the Utilized From setting is overridden (Limit Exposure is with the Issuer/Acceptor throughout).
  • You must save each record before selecting a new Transaction Type.
