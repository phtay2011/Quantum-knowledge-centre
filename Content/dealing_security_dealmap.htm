Deal Map tab, Security Deal

Use this tab to view the chronological record of the key cash flows and events in the life of the current deal, and to optionally customize the deal by adding, editing, or deleting Deal Map items, as follows.

  • To add a new item - in the grid menu select New; in the New Map Items dialog box, enter or select the Effective Date, Action Date, Component, and Amount; then click OK. (You can also add a new item by copying an existing one and change the field values as required.)
  • To delete an item - select the map item grid row; in the grid menu select Delete; then click OK to confirm.
  • To edit or insert items in bulk - use the Replace Forward or Insert function and edit directly or via a spreadsheet application - see Bulk Replace and Insert for Deal Map Items.
  • To capitalize Interest items (Money Market YIELD deals only) - use the Capitalize Interest function - see Capitalizing Interest on Money Market Deals.

NOTE: If you amend a Deal Map item that has already been Actioned, or add an item that impacts items that have already been Actioned:

  • you may have to manually reprocess those items. This is indicated by the Flow Lifecycle value, which you can display as a grid column on the Cash Flows tab, in Settlements and Confirmations results grids, and use as a filter in deal or flow queries or result views (for more details, see Flow Lifecycles, States, and Phases); also...
  • depending on whether or not Deal Map item changes have a financial effect, the changed deal will be badged as Customized - Major or Customized - Minor, respectively.

To select a different grid view - Click on the current view heading and select the View in the lookup.

To change the grid view or create a new one - Click the Configure link, make your changes, and save them.

Fields available for display

  • Effective Date - the Effective date of the event or cash flow
  • Action Date - the Action date of the event or cash flow
  • Component - the type of event or cash flow
    • You can preview and edit Put Exercise and Call Exercise amounts for Bermudan Callable Bonds before the deal has been approved and locked down
  • Amount/Rate - the amount or rate applicable to the event or cash flow
  • Principal Balance (where applicable) - the principal balance after the event or cash flow
  • Comment - any comment text manually or automatically entered with the event
  • Linked Deal Number - the number of the deal linked to the main deal
