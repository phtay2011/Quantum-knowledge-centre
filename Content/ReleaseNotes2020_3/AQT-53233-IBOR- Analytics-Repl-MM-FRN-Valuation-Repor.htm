Analytics Money Market and FRN - New options

With the demise of LIBOR as a benchmark, the various jurisdictions have started recommending their preferred benchmark rates. This phase enhances Analytics FRN functionality to meet client expectations.

The Analytics FRN solution can now calculate the market-expected cash flows and valuations.

New parameter - "last_int_period_accrual_date"

last_int_period_accrual_date - This new parameter is used by the Analytics > Rates > Average Interest Rate Calculator . The parameter is the last day of the whole coupon period that the interest is accrued. If not present, its value is 0; otherwise, its value should be equal to

Coupon End Date – 1 Calendar Day

This date, together with the end_date parameter, dictates how we use Suspension to calculate the Average Rate.

For more information, search for the Knowledge Center topic "Average Interest Rate Formulas".

Average Interest Rate Calculator now in Default perspective

Previously, the Average Interest Rate Calculator was available in the Analytics perspective. In this release we have shifted the Average Interest Rate Calculator to the Quantum "Default" perspective.

Quantum > Rates > Average Interest Rate Calculator

Average Interest Rate calculation method changes

We now calculate the average throughout the averaging calculation period and you can choose when to round - at each day or at the end. Previously, we calculated the arithmetic average only at the end of the averaging calculation.

To set up the system to round at each day or at the end:

  1. Open the relevant Analytics > Pricing screen.
  2. Select the calculation method.
  3. Either:
    • Set Intermediate Rounding = None to round only at the end of the averaging calculation. Set the type of rounding and number of decimal places by using Final Rate Rounding. The default is no rounding.
    • Set Standard = <an integer> to set the number of days for which to use the same rate.
  4. Use Final Rate Rounding (was "Rate Rounding") to round the final value. For example, set the system to round to 7 decimal places at all intermediate steps of the averaging calculation, and then to round the final value to 5 decimal places.

For more information, search for the Knowledge Center topic "Average Interest Rate Formula".

Instruments - New Intermediate Rounding field

There is a new, field Intermediate Rounding, with options None and Standard. This is a combination of a drop down menu and an input box that accepts the integers 0-12. It defines the number of decimal digits used to round the intermediate rates when using standard (5/4) rounding.

This applies to:

  • Analytics > Instrument > Money Market
  • Analytics > Instrument > FRN
