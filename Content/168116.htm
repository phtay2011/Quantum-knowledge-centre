Rolling Back an Interest Rate Option Deal

The system saves a deal image record of a deal as it was before processing each action in the life of that deal. Deal images are "snapshots" of earlier deal states used primarily to support Audit History tracking. When you roll a deal back to an earlier image you effectively undo all changes that occurred after the time the image was saved. See Results below.

Exercise and Lapse actions may be rolled back.

NOTE: You cannot roll back a deal that has been deleted or changed, and you cannot reverse a rollback function.

To roll back to an earlier version of an Option deal

  1. If the current result list does not show the deal you want, check that the selected view is Default.

    TIP: Select another view option to show only options of the selected type.

  2. If necessary, run a new query: select New from the page menu, enter the Filters values required, then click Run Query.
  3. In the Result list menu, select the deal and in the list menu, select Rollback Deal.

    The Deal image Rollback dialog box opens, listing, for each previous deal image, the Image Number, Image Date, Action Date, Market Rate, Image Reason (typically an earlier action), and Rolled To value.

  4. In the dialog box, select the deal image you want to roll back to, and click OK.

Results:

  • The deal reverts to the state it was in before the selected action.
  • Any actions that were currently awaiting approval no longer apply and are therefore ignored.
  • All deal, cash flow, general ledger, and bank account information reverts to the state it was in when the selected deal image was saved.
  • The result list is refreshed to reflect that state of the deal.
  • If you have rolled back to before a Physical Exercise action (as opposed to a Cash Exercise), the associated Security or Swap deal will be deleted, and a Delete action will then be shown on the Deal Approval page for that deal.
  • If deal approvals apply, the Rollback action awh6>s approval.

    NOTE: The Delete action on the associated deal will not be automatically approved when the Rollback action is approved.
