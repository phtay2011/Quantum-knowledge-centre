Cash Flow Filters, Bank Account Conversion

(This area appears only when the Cash Flow Movement option is selected.)

Use these filters to narrow down the cash flows that will be returned by your query run.

TIP: The available selections in each case can be reduced based on other filter selections.

Transaction Type - Either select one Transaction Type or select ALL (the default).

Instrument - Either select one instrument or leave blank for no filtering on instrument (effectively selecting ALL).

Deal Entity - Either select one deal entity or leave blank for no filtering on entity (effectively selecting ALL).

Counterparty - Either select one deal counterparty or leave blank for no filtering on counterparty (effectively selecting ALL).

Relationship - Either select one Facility or Dealing Line, or leave blank for no filtering on relationship (effectively selecting ALL).

Cash Flow Type - Either select one type of cash flow or leave blank for no filtering on this (effectively selecting ALL).

Deal Numbers - Either enter one or more specific deal numbers (separated by commas) or leave blank for no filtering on Deal Number (effectively selecting ALL).

End Date - Enter an End Date to update future flows up to and including that date (which must not precede the Effective Date). Or leave blank to update all future flows.
