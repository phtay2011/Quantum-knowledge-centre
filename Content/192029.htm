'Our' Stock Account and Accounting Defaults

You can create a Stock Account and specify Stock Account Defaulting Rules for specific combinations of Entity, Instrument, Transaction Type, Counterparty, and so on.

To set up a Stock account

Create a Stock Account static data record with the following specific settings:

  • In the Custodian lookup field, select the Counterparty.

    NOTE: You can select only from Counterparties specified as Custodians.

    The BIC Code and SWIFT DSS read-only fields then show the BIC Code and SWIFT Data Source Scheme values specified for that counterparty.

  • Specify a Unique Identifier for your Stock account, as supplied by the Custodian (optional). Change other existing or default values as required.

    You must specify values for Custodian, Account Number, Account Name, and Currency.

  • In the Allowable SWIFT MT Formats lookup field, select the SWIFT output format templates found in the AdvicePath location. If there are more than one, then from the Default SWIFT MT Format list, select the default template for this Stock account.

To set up or change defaults for Stock accounts

Create a Stock Account Defaulting Rule with the following specific settings:

  • In the Custodian lookup field, select the Counterparty.
  • In the Account Number lookup field, select the Stock account for the custodian. You can select only from Stock accounts set up for that custodian.

    The Account Name and Currency read-only fields then show the values for that custodian.

  • In the Entity Identifier lookup field, select a default entity identifier value for these default instructions.

    TIP: You can select only from Entities valid for the Payment method specified.
