Standard Cap/Floor Risk Statistics (Greeks)

This section describes Standard Cap/Floor Risk Statistics (Greeks) formulas that are used both in the Black, and the Bachelier model.

Greeks under the Black Model

Delta

Delta is a measure of the sensitivity of the option value to a small positive change in the forward interest rate. The per unit (or raw) delta is calculated as:

CDpu = DF(0,ti+1) * N(d1)              for a Cap

FDpu = DF(0,ti+1) * [ N(d1) - 1 ]     for a Floor

...where:

CDpu = Cap Delta per unit

FDpu = Floor Delta per unit

DF(0,ti+1) = Discount Factor for the period between time 0 and time ti+1

                 = e-rT

e = 2.71828182... (the base of the natural logarithm)

N(d1)= Normal Distribution function for d1

s = Volatility

The position Delta approximates the change in the overall value of the option given some pre-specified change in the underlying. For caps and floors, the position delta is computed assuming that the forward interest rate has increased by 1 basis point. That is:

CDpos = CDpu * FV * t * 0.0001

FDpos = FDpu * FV * t * 0.0001

CDpos = Cap Delta Position

FDpos = Floor Delta Position

FV = Face Value of the deal

     = P/(1 + Ft)

P = Principal

F = current Forward Rate (value of the reference floating rate at reset date i)

t = Cap/Floor term in years; the time between the ith reset date and payment date

Gamma

Gamma is a measure of the sensitivity of Delta to a small (positive) change in the underlying variable. The per unit (or raw) gamma for a Cap is the same as that for a Floor.

For each of the i = 1,2,...n optlets, Gamma per Unit Gpu is computed as:

Gpu = [ n(d1) * DF(0,ti+1) ] / [ F * s * v(ti) ]

...where:

n(d1) = Standard normal probability density function for d1

s = Volatility

(other values as above)

The position Gamma Gpos approximates the change in the position Delta of the option for a basis point change in the forward interest rate:

Gpos = Gpu * FV * t * (0.0001)2

Vega

Vega is a measure of the sensitivity of the option value to a small (positive) change in the Forward Rate volatility. The per unit (or raw) Vega for caplets and floorlets Vpu is derived from the same formula:

Vpu = F * v(ti) * n(d1) * DF(0,ti+1)

The position Vega Vpos approximates the change in the overall value of the option given a percentage point change in the volatility of the underlying asset:

Vpos = Vpu * FV t * 0.01

Theta

Theta is a measure of the sensitivity of the option value to a small (positive) change in time (that is, a decrease in time to maturity). Assuming that the change in time has no impact on any of the discount factors or forward rates, then the solutions for Theta per Unit (Thcall and Thput) are as follows:

Thcall = rf * F * DF(0,T*) * N(d1) - F * DF(0,T*) * n(d1) * s /(2vT) -
               rf * X * DF(0,T*) * N(d2)

Thput = rf * F * DF(0,T*) * N(d1) - F * DF(0,T*) * n(d1) * s /(2vT) -
               rf * X * DF(0,T*) * N(d2)

...where T* and T are the payment date and option maturity date, respectively.

The position Theta Thpos approximates the change in the position value of the option given that time has advanced by 1 day:

Thpos = Thraw * FV * (1/365) *t

...where Thraw is Thcall or Thput as required.

Rho

Rho is a measure of the sensitivity of the option value to a small (positive) change in the riskless rate. Assuming that the change in the riskless rate has no impact on the forward rate, then the solution for rho is simple. For both calls and puts, the raw Rho Rraw is computed as follows:

Rraw = -T* • OV

...where T* is the payment date and OV is the raw option value.

The position Rho Rpos approximates the change in the position value of the option given a 0.0001 increase in the riskless rate:

Rpos = Rraw • FV • 0.001

Greeks Under the Normal Bachelier Model

Delta

Delta is a measure of the sensitivity of the option value to a small positive change in the forward interest rate. The per unit (raw) delta is calculated as:

CDpu = DF(0,ti+1) * N(d) for a Cap

FDpu = -DF(0,ti+1) * [N(-d)] for a Floor

...where:

CDpu = Cap Delta per unit

FDpu = Floor Delta per unit

DF(0,ti+1) = Discount Factor for the period between time 0 and time ti+1

= e-rT

e = 2.71828182... (the base of the natural logarithm)

N(d) = Normal Distribution function for d

si = Volatility

The position Delta approximates the change in the overall value of the option given some pre-specified change in the underlying. For caps and floors, the position delta is computed assuming that the forward interest rate has increased by 1 basis point. That is:

CDpos = CDpu * FV * t * 0.0001

FDpos = FDpu * FV * t * 0.0001

CDpos = Cap Delta Position

FDpos = Floor Delta Position

FV = Face Value of the deal

= P/(1 + Ft)

P = Principal

F = current Forward Rate (value of the reference floating rate at reset date i)

t = Cap/Floor term in years; the time between the ith reset date and payment date

Gamma

Gamma is a measure of the sensitivity of Delta to a small (positive) change in the underlying variable. The per unit (or raw) gamma for a Cap is the same as that for a Floor. For each of the i = 1,2,...n optlets, Gamma per Unit Gpu is computed as:

Gpu= [ n(d) * DF(0,ti+1) ] / [ s * v(ti) ]

...where:

n(d) = Standard normal probability density function for d

si = Volatility

(other values as above)

The position Gamma Gposapproximates the change in the position Delta of the option for a basis point change in the forward interest rate:

Gpos= Gpu* FV *t * (0.0001)2

Vega

Vega is a measure of the sensitivity of the option value to a small (positive) change in the Forward Rate volatility. The per unit (or raw) Vega for caplets and floorlets Vpu is derived from the same formula:

Vpu= v(ti) * n(d) * DF(0,ti+1)

The position Vega Vpos approximates the change in the overall value of the option given a percentage point change in the volatility of the underlying asset:

Vpos= Vpu* FV t * 0.01

Theta

Theta is a measure of the sensitivity of the option value to a small (positive) change in time (that is, a decrease in time to maturity). Assuming that the change in time has no impact on any of the discount factors or forward rates, then the per Unit (or raw) Theta for a Cap is the same as that for a Floor:

Thpu= [2vT* rf * OV – DF (0,T*) * sDF (DF )] / (2v T)

...where T* and T are the payment date and option maturity date, respectively and OV is the raw option value

The position Theta Thpos approximates the change in the position value of the option given that time has advanced by 1 day:

Thpos = Thpu * FV * (1/365) *t

Rraw= -T* • OV

...where T* is the payment date and OV is the raw option value.

The position Rho Rpos approximates the change in the position value of the option given a 0.0001 increase in the riskless rate:

Rpos= Rraw• FV • 0.001
