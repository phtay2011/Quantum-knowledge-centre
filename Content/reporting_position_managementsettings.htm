Management Settings, Position Reporting

(Click the Advanced Options link to toggle the display of this area.)

NOTE: These options apply only when Use management rules is selected)

Calculate market value - select this option to calculate market values for deals.

Calculate duration and modified duration - select this option to calculate duration and modified duration figures for deals.

Sensitivity Shift Factor - Enter a percentage value to assess the effect of a change in the revaluation par curve on the value of the portfolio. The system calculates how much the NPV of the portfolio would change if the whole par curve was to shift by the amount you specify. For example, if you enter 1, the system shifts the entire par curve by 1 percent. The sensitivity is calculated for both positive and negative shifts in the par curve.

Calculate hedge effectiveness - select this option to preview the processing for Hedge Accounting. Hedge Reclassification can be run at the same time as a standard revaluation or independently of it. Effectiveness calculations are always run at the same time as a Hedge Reclassification.

Include Margin - select this option if you want to use an active margin rate name.

Margin Type (available when Include Margin is selected) - select a record from the list of active Rate Names.
