Schedule tab, Fee Deal

Use this tab to define the frequency of the scheduled dates for fee calculation, payment, invoicing, and base amount resets.

A Maturity Date must be entered.

Base - read-only and set to Calculation if Pay fee cash flow in advance is selected, otherwise select either Calculation or Payment to specify whether your schedule is to be based on Calculation dates or Payment dates respectively (as indicated by which of the two dates is first in the Schedule grid).

Schedule grid - each row shows one of the four date entries, with columns as below.

  • Name - Calculation, Payment, Invoice, and Base Reset
  • Frequency (available settings vary depending on the Base and which grid row):
    • 1st (base) and 4th (Base Reset) rows - select a period (Daily, Weekly, and so on), Manual Schedule* (where applicable - adds a separate Calculation Schedule or Base Reset Schedule grid), or Once (on Maturity Date, unless Pay fee cashflow in advance is selected, when Start Date applies);
    • 2nd (non-base) row - if this is the:
      • Payment row - select Same Day, Next Day, N Days, N Business Days, Nth of Month, Nth of Following Month, End of Month, End of Following Month, or Manual Schedule;
      • Calculation row - select Same Day, Previous Day, N Preceding Days, or End of Previous Month;
    • 3rd (Invoice) row - select Same Day, Previous Day, N Preceding Days, N Preceding Business Days, or Manual Schedule (if Base is Calculation only).
  • Offset (available only when Frequency requires an offset) - the number of preceding or following days, or the day of the month, as applicable.
  • First Date (read-only on 2nd and 4th rows) - the 1st fee calculation, payment, or base reset date, as applicable. Changing this date changes the relevant cycle of dates from this date onwards.
  • Business Day Convention (available only when Frequency is Manual Schedule, Previous Day, or requires an offset) - the rule to apply when the relevant date falls on a non-business day.

To edit a grid entry - Select the entry in the grid and in the grid menu, select Edit. Edit the field values as required, and click OK.

TIP: You cannot add or delete grid entries, as they are fixed.

To view the Calculation schedule - In the grid menu, select View Schedule. The scheduled dates are listed below the Schedule grid.

(For details of the options on this tab, click on them when the Help panel is open.)
