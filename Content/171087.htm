Configuring Grid Views

To toggle display of the Configure panel - click the Configure link when the selection list is showing.

To save a view for later re-use (where allowed) - on the Properties tab, name the view and click Save.

Use the tabs as follows.

Layout tab

Use this tab to change the columns displayed, their sequence from left to right, and (optionally) the default sort columns.

To reposition a column in the grid - drag-and-drop that column name in the Columns box.

To add an additional column - select it from the list (start typing to find columns quickly).

To remove a column from the grid - in the Columns box, click the x button for the column.

To apply one or more default sort columns - in the Sort box, select the columns, in order of precedence. Drag a column to change this order. Click a column in the box to reverse its sort order ( or for ascending or descending, respectively). To cancel sorting on a column, click the x button for that column.

TIP: The default sorting can be overridden by clicking a column heading in the list.

Filters tab

Use this tab to configure filters on the data that comes back from the query. This enables you to further extract data from the query without having to re-run it.

NOTE: You must save the view before the grid will update to reflect any changes.

To add a filter field to the view

  1. Click the Add Filter button and select the field you want from the list of those available (start typing to find your filter quickly).
  2. In the field label's drop-down list, select an applicable logical operator to apply filter values, depending on the type of field.
  3. In the field itself, select one or more filter values to apply.

To remove a filter field from the view - In the drop-down menu on the filter field's caption, select Remove Filter.

Columns tab

Use the Columns tab to choose the headings for the columns in the results grid. Add extra columns called "calculated columns" that show the calculated values that are the results of an expression associated with the column. This tab uses Expression Builder to create the expressions.

Colors tab

Use the Colors tab to add colors and conditions to columns and rows in the results grid.

Properties tab

Use the Properties tab to:

  • Enter a new Name for the view (required before you can save it).
  • Optionally set this view as the default for the current query. This will be the first view you'll see when you select the current query.
  • Set the Access Permission for the view (Public or Private).
  • Change the way any amounts appear (columns), the type of rounding, whether the amounts are in thousands, millions etc., and what decimal format to use.
  • Embed the units in the results table.

To save a view for later re-use - Make your required changes (on all tabs) and click Save.

To alter how amounts appear - in the Rounding area, select Columns for this to apply to, enter a rounding method (5/4, Down, Up), a unit (called a Round Amount and in Whole Amounts, Thousands, Millions, Billions or with the unit Embedded) and how many decimal places.

Once saved, Views for queries like this one - that can be accessed in either Reporting or Processing mode - are available for the query in both modes.

The export to Excel uses the raw numbers, but other exports use what is in the View.
