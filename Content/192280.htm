Stress Test File Upload (Adaptiv)

System Administrators can use the Stress Test File page to upload a Stress Test file, and to save (download) or remove previously uploaded files of this type.

An uploaded Stress Test file can be optionally selected in an Adaptiv Base Valuation, Cash Flow, or Market Sensitivity query configuration (in the Stress Test field on the Assumptions tab). This enables you to run a stress test on the query - for example, to test new queries before any "live" runs.

For more details, refer to the Adaptiv Analytics Interfacing Guide or to your FIS Consultant.

TIP: Uploaded files are stored in the location specified in the MarketSensitivityConfigFilePath key in your site's AAI.AAIS.exe.config file.

When a query is run with the stress test file selected, the format of the raw report results table changes.
