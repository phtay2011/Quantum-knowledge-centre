Details tab, Unit Holding Deal

Use this tab to enter, edit, or review the main details for the Unit Holding, as follows.

Entity (read-only) - The entity owning the holding (the shareholder organization); defaulted from the underlying Holding Instrument.

Security ID Group (read-only; optional) - The Group name (if any) from the specified Security ID record.

Security ID (read-only; mandatory) - The Security Identifier for the Unit Holding, as specified in the underlying Holding Instrument. The Primary Identifier value specified for that Security Identifier record is displayed in the adjacent read-only field, labeled according to the specified Primary Identifier type (CUSIP, ISIN, or Other Identification).

ISINorCUSIPorOther Identification (read-only ) - labeled with the Primary Identifier setting for the selected Security ID, this value gives that identifier value for the specified Security ID record.

Exchange (read-only; applicable when Trading Style is Exchange Traded on the Holding Instrument) - the Exchange that trades this Unit Holding (a Counterparty with the Exchange Category selected).

Strategy (optional) - select a Strategy group in which to include this holding.

Taxes (optional) - a tax to be applied to Dividends paid on this holding.

Industry Group (read-only; optional) - the Industry group (if any) for the Issuer of the holding; obtained from the Industry setting in the Counterparty record for the Issuer specified in the Security ID record; blank if an Industry is not specified for the Issuer or if the Industry does not belong to a group.

Industry (read-only; optional) - the industry classification (if any) for the Issuer of the holding; obtained from the Industry setting in the Counterparty record for the Issuer (if any) in the specified Security ID record; blank if an Industry is not specified for the Issuer.

Unit Deal Default Counterparty (available when Trading Style is Over the Counter on the Holding Instrument) - the default Counterparty selling or buying Unit deals in this Holding (a Counterparty with the Deal ExternalorDeal Internal Category).

Unit Deal Default Broker (available when Trading Style is Exchange Traded on the Holding Instrument) - the default Broker handling deals in this Unit Holding (a Counterparty with both Clearing Broker and Deal External Categories).

Trading Information (optional; available only when Trading Style is Over the Counter on the Holding Instrument) - Enter/edit meaningful text to provide any further trading information.
