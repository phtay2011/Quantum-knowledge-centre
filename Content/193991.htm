Rules menu (Default perspective)

This section covers core modules accessed via the Rules menu in the Default perspective (shown below).

NOTE: The Valuation Rule module is also accessible via the Static & Rules menu in the Analytics perspective.

MENU GROUP/
Module

Use to...

Unique ID (Area/Controller)

DEFAULTS

Analysis Code Defaulting Rule

Maintain rules for providing default Analysis Code values for deal entry

rules/analysis-codes-defaulting-rule

Bank Account Defaulting Rule

Maintain rules for applying bank account instructions to deal cash flows

rules/bank-account-rule

Broker Account Defaulting Rule

Maintain rules for applying broker account instructions to deal cash flows

rules/broker-account-rule

Counterparty Bank Account Defaulting Rule

Maintain rules for applying settlement instructions to counterparty cash flows

rules/counterparty-cash-account-rule

Counterparty Stock Account Defaulting Rule

Maintain rules for applying settlement instructions to counterparty stock flows

rules/counterparty-stock-account-rule

Deal Allocation Defaulting Rule

Maintain rules linking a master deal to additional deals of a similar type

rules/deal-allocation-defaulting-rule

Inter Company Bank Account Defaulting Rule

Maintain rules linking internal bank accounts, used to create inter-company cash flows and G/L entries

rules/inter-company-bank-account

Stock Account Defaulting Rule

Maintain rules for applying stock account settlement instructions to deal stock flows

rules/stock-account-rule

Unit Fee Defaulting Rule

Maintain rules for applying fees to equities transactions

rules/unit-fee-defaults

RULES

Approval and Authorization Rule

Maintain rules for applying approval requirements to deal actions and settlement of cash flows (or stock flows)

rules/approval-control

Business Day Calendar

Maintain non-business days and non-standard business days for a currency or location

static/business-day-calendar

Confirmation Rule

Maintain rules for specifying Confirmation types and output methods

rules/confirmation-control

Counterparty Contact Rule

Maintain contact information for Counterparties, including details for sending Confirmations

static/counterparty-contact

Counterparty Limit Rule

Maintain a series of limit profiles and calculation methods for each Counterparty

static/counterparty-limit

Cut-Off Time Rule

Maintain rules for determining cut-off times when saving deals and processing flows

rules/cut-off-time

Counterparty Netting Rule

Maintain rules for netting settlement cash flows by Counterparty

rules/counterparty-netting

Deal Set Rule

Maintain rules specifying types of deal in deal sets (and if deals may be updated)

instruments/deal-set-rule

Electronic Confirmation Format Rule

Maintain rules for specifying XML Confirmation template files (to format electronic Confirmation messages for deal actions)

rules/electronic-confirmation-
format-rule-mode

FX Cover Rule Set

Maintain FX Cover Rule criteria and defaulting values

rules/fx-cover-rule-set

Management Rule

Maintain Management reporting revaluation rules and methods by type and rate source

rules/management-control-rule

Paper Confirmation Format Rule

Maintain Paper Confirmation template files for formatting print/email/fax Confirmation messages for deal actions

rules/paper-confirmation-
format-rule-mode

Relationship Rule

Maintain rules for associating relationships to new deals (Facility/Dealing Line/both/none)

rules/relationship

SWIFT Local Authentication Rule

Apply SWIFT SA-2 or HMAC-SHA256 Local Authentication to Confirmation and EFT output

rules/swiftlocalauthentication

Utilization Date Rule

Maintain Utilization Date records for combinations of Transaction Type and Instrument (to control exposure allocation)

static/utilization-date

Valuation Rule

View and maintain Analytics Pricing Valuation rules

static/valuation-control-analytics
