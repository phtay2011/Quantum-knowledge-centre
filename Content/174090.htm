Time Options, FX Deals

A Time Option is a type of FX deal that permits multiple deliveries.

Each FX deal created using a Time Option instrument has a defined Start Date. From this Start Date onwards, it is possible to make draw-downs against the FX Deal. You can draw down as often as necessary, either until the deal value date, or until the original deal amount has been fully drawn down.

Quantum uses the Value Date to calculate the forward rate.

Any balance outstanding on the value date of a Time Option deal is settled in the same way as the balance on any other FX deal.

TIP: For FX Time Options using the IMPORTED Revaluation Method, the postable Gain/Loss is calculated by applying the Revaluation Currency amount in the Imported Mark to Market database table against the aggregate of the Revaluation Currency Face Value leg amounts on the deal when one or more drawdowns (as well as the residual balance) are open at Position Date. This applies particularly to deals imported from Analytics, and also affects Pre-deliveries and Extensions with one or more open "tranches" at Position Date.

To draw down a Time Option

  1. Open the the Time Option deal (on the FX Deal page).

    TIP: You can also draw down a Time Option in Settlement Processing (when you select a single cash flow for a suitable FX Time Option deal).

  2. In the page menu, select Drawdown.
  3. In the Time Option Drawdown dialog box (shown below):
    1. Specify the date, and either the Buy Amount or Sell Amount for the drawdown.
    2. Optionally edit the New Value Date and Rate if required.

      Quantum calculates the unspecified amount, using the Rate value.

      The remaining balance on the deal is shown in the Residual section at the bottom of the dialog box.

    3. Click OK.
  4. Click Save.

Example of the dialog box

TIP: If the instrument is set up to display the forward rate on a commodity, the Commodity Forward value appears instead of Forward Points.

Changing the Residual Amount

The Residual on a Time Option deal is calculated automatically.

The Residual uses the Rate at which the original deal was struck, and the amount remaining on the original deal after subtracting any drawdowns that have been made.

It is sometimes necessary to change the residual - for example to reflect changes in market conditions.

To change the Residual

  1. In the Time Option Drawdown dialog box:
    • Change one of the Amount fields and the Rate, or
    • both Amount fields.

    Quantum calculates the unspecified amount or rate.

  2. Click OK.
