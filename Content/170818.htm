Viewing Settlement Cash Flows

On the Netting session page, after performing a Generate Settlements and/or Create Cash Flows action for the session, use the Settlements area tabs to view the results and optionally action selected records when necessary, as follows.

Tab

Settlement records listed

Actions available*

All

All generated for the session

Create Cash Flow

Unprocessed

Those not yet passed to Quantum for processing

Create Cash Flow

Created

Those passed to Quantum for which cash flow deals have been successfully created (Deal number in Message column)

[None]

Failed

Those passed to Quantum but which contain errors (see Message text and Note below)

Reset Cash Flow
Create Cash Flow

Pending

Those being processed when an interruption to system operation occurred

[None]

NOTE:

  • To select all records on the tab, select the check box at top left (by the Entity column heading).
  • Create Cash Flow will process only settlements with a status of Unsent.
  • Reset Cash Flow changes the status of selected settlement records from Failed to Unsent.

The following values are shown for each settlement record on each tab (all values are read-only):

  • Entity - the participating entity to which the settlement relates
  • Currency - the currency of the settlement flow
  • Amount - the cash flow amount
  • Counterparty - the counterparty
  • Message - information about the settlement record
  • Status - one of the following (usually corresponding to the tab listing the record):
    • Unprocessed - the settlement record has NOT been passed to Quantum
    • Processed - the settlement record has been successfully passed to Quantum
    • Failed - the settlement record has been passed to Quantum, but a cash flow record could not be created
    • Pending -the settlement record has been sent to Quantum but remains unacknowledged by Quantum

The administrator has two ways in which they can create Actual Cash Flows.

  • If the administrator would like to choose the settlements they would like to create actual cashflows for, they would go to the Settlements tab and select the HSS Settlement flows and then click the Process button.
  • If the administrator would like to create Actual Cash Flows for all the HSS Settlements then they need to click the check box next to the Entity column (this selects all settlements), then click and select Create Cash Flows.
