Settings grid, Mapping Profile

Common fields

Mapping Code (read-only) - The code identifying the applicable set of Matching Criteria and Settings (as entered in the Matching Criteria grid row).

The Matching Criteria fields define which report result items the Mapping Code's Settings will be applied to (when Mapping Profile is selected in a query or results View configuration.

Yield Curve - Select the Yield curve that will be used for the valuation of the transaction mapping. This will override the Yield curve specified in the applicable Valuation Rule.

General Mapping Profile fields

Time Profile - Select the Time Profile to be used for the transaction mapping. The default is the Default Time Profile.

Risk Type - Select one of the following:

  • At Maturity of Generic instruments
  • At Next Repricing or Maturity
  • Yield Curve Analysis
  • Pro Rata Delta

VaR Mapping Profile fields

Asset Class - Select one of:

  • Money Market,
  • Swaps,
  • Government Bonds,
  • Commodities, or
  • Equities.

Probability(%) - Enter a percentage probability from 1 to 100.
