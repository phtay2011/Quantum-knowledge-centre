Options and Files area, Assumptions tab (Adaptiv Query)

Use this area to specify the following options and settings for the query. Unless explicitly stated below, these apply to all Adaptiv report types.

Export Market Prices - Select this option to enable market prices (par rates) to be seen in the Market Price tab of AA Studio.

Keep Job File - Select this option to save the .aaj file when the query is run.

Stress Test File (Base Valuation, Cash Flow, and Market Sensitivity)- Select a file name previously set up using the Adaptiv Stress Test File Utility page.

Market Sensitivity Configuration File (Market Sensitivity) - Select a file, previously uploaded using the Adaptiv Market Sensitivity Configuration File Utility page, that specifies the shifting rules for the Market Sensitivity calculation.

Calculate Decomposition (CVA and PFE) - Select this option to enable users to see the CVA decomposition results in AA Studio (in a pivot table).

Netting / Collateral (CVA and PFE) - Select one of the following Netting options:

  • Netting Pools - Collateral Opening Balance - apply complex netting based on counterparty hierarchy and collateral accounts, with bank account balance mapped as opening balance property of netting set
  • Netting Pools - Cash Collateral - apply complex netting based on counterparty hierarchy and collateral accounts, with bank account balance mapped as cash collateral asset
  • Net All - No Collateral - apply simple netting to all subgroups under the selected Counterparty Group, ignoring collateral accounts.
  • No Netting or Collateral - apply no netting

Calculate CDS Sensitivity (CVA) - Select this option to calculate CVA sensitivity based on the CDS rate shift.

Counterparty Group (CVA and PFE reports only) - Optionally select a Counterparty Group to include only Counterparties in that group. The default is a blank value for all groups.

Counterparty Group CDS Rate (CVA and PFE) - Optionally select a rate (Yield curve) representing the probability to default.

Entity Group (CVA and PFE) - Optionally select an Entity Group to include only Entitles in that group. The default is a blank value for all groups.

Entity Group CDS Rate (CVA and PFE) - Optionally select a rate (Yield curve).

TIP: For CVA and PFE reports, the default setting of the AA Studio Calculate dialog value for First_To_Default_Adjustment is None, so that no adjustment will be applied. If you want to apply the Bilateral setting instead, you must include the following property setting in theAAI.AAIS.exe.config file:

<add key="FirstToDefaultAdjustment" value="Bilateral" />

Other settings are also valid. For more details, refer to the AA Studio documentation.
