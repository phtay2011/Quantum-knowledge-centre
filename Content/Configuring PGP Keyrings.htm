Configuring PGP Keyrings

Configure individual PGP keyrings for the encryption, decryption, signing, and verification of the digital signatures of incoming and outgoing messages.

  1. On the Configure PGP Keyring Settings page, select the site Section Name to use PGP Keyring encryption.

  1. Click Next.

  2. In the PGP Password dialog enter a passphrase, and then click OK.

NOTE: Do not forget this password as it is required when sending or opening messages which require or have PGP Keyring encryption.

  1. The status is updated for the selected site section on the Configure PGP Keyring Settings page.

  1. Click Next.

  2. Check you have an appropriate key management tool on the computer, for example, the Gpg4win installer package (https://www.gpg4win.org/).

  3. Create PGP private and PGP public keys, as required. Use the file extension asc.

  4. Export the keys to a folder you create called PGPKeyrings, for example, trunk/site/DB/PGPKeyrings.

Consider the following:

  • The password for a private key must be unique.

  • The password for a private key is the same as the password defined in the PGP Password dialog box in Quantum Configuration.

  • The PGPKeyrings folder name is hard-coded so you cannot change or modify it.

  • The key names and key numbers in this folder are not limited.
