Confirmation Match Reply Page

You can now use the new Confirmation Match Reply page to configure the location (file or queue) and format of the incoming confirmation replies to be matched by the Confirmation Match Service. Additionally, the Confirmation Match Service now supports both tab-separated and pipe-separated formats for Finastra Match Replies with Financial Status, Instruction Status, or both as their matching criteria.

Implementation

A new Confirmation Match Reply page has been added to the Configuration menu.

In the Location lookup, select the folder or queue that the Confirmation Match Service monitors for incoming confirmation replies. For confirmation replies that are in file format, this location maps to any sub-folders created under Confirmation\Import in your Site's AppData folder; for confirmation replies that are transported via queue, the required message queue must be set up in Data Transfer Location.

In the Matching Format drop-down list, select the external matching system's (SWIFT or Finastra) message format. For Finastra, the following options are available:

  • Finastra Pipe Status - Finastra Status messages in pipe- and space-delimited format - that is, with a pipe character followed by a space "| " between values).

    NOTE:No delimiter is needed at the end, but if present, the same character pair must be used (pipe character followed by a space).

  • Finastra Tab FinStatus - Finastra messages in tab-delimited format, where Financial Status is the matching criteria.
  • Finastra Tab InstStatus - Finastra messages in tab-delimited format, where Instruction Status is the matching criteria.
  • Finastra Tab FinStatus & InstStatus - Finastra messages in tab-delimited format, where Financial Status and Instruction Status are the matching criteria.

NOTE: Finastra was formerly known as Misys.

History

Previously, the Confirmation Match Service only monitored folders with specific names for incoming confirmation replies, and Finastra confirmation replies had to be in tab-separated format for successful processing.
