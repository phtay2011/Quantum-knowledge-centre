Default Account Settings

Use this area to define the default settings for the G/L Accounts that will be generated from this template. When G/L accounts have been generated, based on this template, the G/L accounts and their settings may be viewed and modified.

NOTES:

  • Some settings are not available with some Type settings (as detailed below).
  • Changing these default settings on the template will not affect accounts previously created from that template.

    To change the settings for a generated account code: select the template in the grid; in the grid menu, select View G/L Accounts to display the General Ledger Accounts list; click in the relevant account row to open the Edit dialog box; make your changes; then click Save.

Create accounts when posted to for first time

(Not available if Type is SYSTEM SUSPENSE.)

Select this option to:

  • automatically create the relevant G/L accounts when the the ledger is posted to for the first time, and to...
  • disable the Generate Accounts grid menu item when this Account Template is selected in the grid.

Clear this option to allow the Generate Accounts grid menu item for this Account Template.

Exclude from consolidated G/L position reports

(Not available if Type is SYSTEM SUSPENSE.)

Select this option to exclude an account from reporting when a consolidated General Ledger is required.

This is not an accounting elimination - it does not affect posting.

Disallow use in general ledger journals

(Not available if Type is SYSTEM SUSPENSE.)

Select this option to prevent users posting manual entries to G/L accounts created from this account template.

Apply account level currency translation

(Available only if Type is BALANCE SHEET, USER SUSPENSE, or SYSTEM SUSPENSE.)

Use this option to define whether or not currency translation is to be applied at an Account level for G/L accounts created from this template.

TIP: If you intend to apply Currency Translation to your accounts, we recommend that you set up the Currency Translation Gain/Loss accounts, before setting up Balance Sheet Account Templates.

When you select this option, the associated Translation Gain Template and Translation Loss Template fields are displayed and must be defined (see below).

See also: Currency Translation Reporting, Currency Translation Processing.

Translation Gain Template

(Available only when Apply account level currency translation is selected.)

Select the Account Template for the account the system will post the credit entry to when a revaluation results in a gain (the debit entry will be posted to the account being revalued).

TIP: SYSTEM SUSPENSE and CCY POSITION Account Templates are not available for selection here.

Translation Loss Template

(Available only when Apply account level currency translation is selected.)

Select the Account Template for the account the system will post the debit entry to when a revaluation results in a loss (the credit entry will be posted to the account being revalued).

TIP: SYSTEM SUSPENSE and CCY POSITION Account Templates are not available for selection here.

Account Name

(Read-only if Type is SYSTEM SUSPENSE.)

The default descriptive name for G/L accounts created from this template; mainly used as a description for reporting purposes. The default is the Template Name, but changing that value does not affect the Account Name.

TIP: Adopt a meaningful naming convention for your Account and Template names. Template names are used in Posting Rules, and if too different from Account names, will make it unclear which accounts are posted to.

The account name will initially be the same for each G/L account created from the template. Changing it here (on the template) will not affect those accounts.

To change the Account Name for a generated account code

  1. Select the template in the grid.
  2. In the grid menu, select View G/L Accounts.

    The General Ledger Accounts list is displayed.

  3. Click in the relevant account row.

    The Edit dialog box opens.

  4. Make your changes, then click Save.
