EFT Layout file - Enhancements

The Layout file (EFTs) now includes formatting of the MT200 message.

Additionally:

  • We removed the REC code from the MT200 in the XML-formatted message to adhere to SWIFT 2019 Standards.
  • We simplified the Layout file by using paymentDetails and we removed all logic to check for adhocDeliveryInstructions and use as a preference over deliveryInstructions. This has drastically reduced the amount of logic required to format fields containing beneficiary information.
  • In the XML Framework, it is now possible to use the bypass attribute for Combination and use fieldItems in SelectCase. In the following example, the XML for Tag 57A in the MT202 code has been reduced by 40 lines.

If upgrading Quantum, decide whether you want to use your existing Layout file or the new file. If you want to use the new file, then look at the new file and plan how to migrate across.
