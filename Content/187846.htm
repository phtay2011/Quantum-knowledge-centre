Implementing a Tiered Fee Structure

You can set up a Base Template with a tiered fee structure for any Fee Category and Formula type, to apply different fee rates for different 'tiers' (levels) of the base amount.

  1. Open the Base Template record you want to set up.
  2. Select the Allow fee to be tiered option.

    The Apply to options, Tiers grid, and grid menu, are displayed for you to define your tiered fee structure.

    TIP: The Unit Fee (Rate or Amount) Definition field no longer applies, so is disabled.

  3. Select one of the two Apply to options, as follows:
    • Intersected tier only - to apply one rate only to the Fee Base Amount, namely the rate in the highest row whose Tier Start value is less than or equal to the Fee Base Amount; OR
    • All tiers - to apply one or more rates to the Fee Base Amount, for those parts of the Fee Base Amount falling within each applicable tier, accumulatively. The rates applied are all those in rows whose Tier Start values are less than or equal to the Fee Base Amount.

    For more details, see Examples below.

  4. To add a new tier: in the grid menu, select New; in the dialog box enter the Tier Start and Rate cells for the corresponding row; then click OK.

    The new tier row is added in the correct sequence.

  5. To edit tier values: click a cell value in the tier row; in the dialog box edit the Tier Start and/or Rate cells as required; then click OK.
  6. To delete a tier: select the tier row and in the grid menu, select Delete; then click OK. to confirm.
  7. When you have finished, Save your changes.
