Strategy - Individual procedures

On the Strategy page:

To list Strategies

This applies to groups or individual items.

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Strategy

  1. In the New menu, select Item.
  2. Enter a unique Name for the strategy, then enter a suitable Short Name and formal External Name.
  3. Optionally enter a relevant text Comment.
  4. Save your new Strategy record.

To edit Strategy details

NOTE: You cannot change the key Name field. Create a new Strategy with the new name you want (and optionally delete the wrongly-named record).

  1. Click to list available records, or use the Search box.
  2. Select the record you want to edit.
  3. Edit the relevant fields.
  4. Save your changes.

To delete a Strategy record

  • You cannot delete a record if that affects transaction, settlement, or financial data, and you cannot delete a group if it has members.
  • Deleting removes the record completely from the database. To prevent use of the record but retain it in the database, deactivate the record instead.
  1. Click to list available records.
  2. Select the record you want to delete (use the < and > buttons to page through, or type in the Search box).
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To deactivate a Strategy

  1. Click to list available records, or use the Search box.
  2. Select the record you want to deactivate.
  3. In the menu, select Deactivate. This action will not appear if the record is already deactivated.

To reactivate a Strategy

  1. Click to list available records, or use the Search box.
  2. Select the record you want to reactivate.
  3. In the menu, select Reactivate. This action will not appear if the record is already activated.

To add the current Strategy to a group (or move to another group)

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select the target group.
  3. Save your change.

    The current record becomes a member of the selected group, and if previously in another group, is removed from that group.

To remove the current Strategy from a group ("un-grouping")

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select Move to Top.
  3. Save your change.

    The current record is now "outside" all groups.

TIP: You can also add and remove members within a group when that group is displayed.
