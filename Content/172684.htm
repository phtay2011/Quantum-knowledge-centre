Defining a Callable Bond Instrument

Use the Securities Instrument page to define Callable Bond instruments. Instrument definition is consistent with defining a standard bond, with the following additional characteristics.

Details tab

  • Formula Timing is BOND PERIODIC
  • Include embedded options is selected
  • Option Type is either:
    • CALL - to give the bond issuer the right to redeem the bond early, or
    • PUT - to give the bond holder the right to redeem the bond early.
  • Exercise Style is either:
    • BERMUDAN - to allow options to be exercised on any date between the First Exercise Date (or the Settlement Date, if it occurs after the First Exercise Date) and the Last Exercise Date; the exercise dates do not have to coincide with a coupon date or a notional coupon date*, or
    • EUROPEAN - to allow options to be exercised on any date that is between the Settlement Date and the Exercise Date.

*TIP: A notional coupon date is a date that would be a coupon date if the Coupon Frequency was the same as the Exercise Frequency of the embedded option.

Dates tab

  • Maturity Basis is either:
    • ACTUAL DATE - if there is a known maturity date for the instrument, or
    • an OFFSET setting (DAYS, MONTHS, or YEARS) - to define a time period from the Settlement Date to the Maturity Date in the relevant time units.
  • Exercise Frequency

    If the option Exercise Style is Bermudan:

    • Enter a frequency for the exercise date schedule (this can be more or less than the Coupon Frequency for the instrument).
    • A First Exercise Date and Last exercise Date must be entered on the Dealing page.

    If the option Exercise style is European:

    • An Exercise Frequency field does not appear.
    • The single Exercise Date must be entered on the Dealing page.
  • First Exercise Offset
    • If Maturity Basis is an OFFSET setting (a number of DAYS, WEEKS, or YEARS) - enter the number of coupons that will be paid before the first Exercise Date can occur, to define any required lockout period during which the bond cannot be called, or
    • If Maturity Basis is ACTUAL DATE - this field does not appear.

Pricing tab

  • Option Strike Price is the default redemption or strike price for the option.

    For Bermudan style options, this price is the default Strike Price for every exercise date. Changes to one or more strike prices can be made at deal entry time using the Deal Map tab on the Security Deal page.

Issuer tab

  • Security Identifier - mandatory for the instrument if Modifiable is NO (on the Details tab).

    TIP: The listed Security Identifiers are filtered by Issuer, Issue Date, Currency, Maturity Date, and/or Coupon, when these are specified for the instrument.

  • Display
    • Read-only and selected if Allow stockflows is selected for the Instrument (on the Details tab).
    • Otherwise, select this option if you want to display a Security Identifier field for deals specifying this underlying Instrument (Security deal, Bond Interest Rate Option deal, or Repo deal Security leg).
  • Required (available only if Display is selected)
    • Read-only and selected if Allow stockflows is selected for the Instrument (on the Details tab).
    • Otherwise, select this option if you want to make selection of a Security Identifier mandatory for deals specifying this underlying Instrument.
  • Use for Defaulting (available only if Display is selected) - select this option if, for deals specifying this underlying Instrument, you want to use default values from the selected Security Identifier record where applicable, overriding the defaults specified on the Instrument. This applies only to the Issuer, Issue Date, Maturity Date, Coupon (%), and Currency default settings.
  • Issuer - this field is not mandatory for the instrument, but one must be recorded before you can exercise a Callable Bond deal.
