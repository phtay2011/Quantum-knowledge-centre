Setting Combined Counterparty Limits

You can combine existing limits to produce aggregate limit totals for a Counterparty across different Transaction Types and Calculation Methods. The system will calculate the combined limit utilization by adding the utilizations of all included limits.

To create a Combined Limit entry for a Counterparty Limit Rule record:

  1. On the Counterparty Limit Rule page, open the required Rule record.
  2. In the grid menu, select New.
  3. In the New Counterparty Limit dialog box:
    1. select the Limit Type and enter the Active From date, as applicable;
    2. for Calculation Method, select Combined Limit;
    3. select the Transaction Type and Instrument;
    4. set the Interval: in the lookup (right side) select either Total Limit, or (to specify a time band) a time unit (Days, Weeks, and so on) and enter a number of those units (left side);
    5. enter the Limit amount;
    6. ensure that Include in Combine Limit is cleared; then
    7. click OK.
  4. Save your changes.

To include existing* Limits in the Combined total:

  1. Click a linked value in the existing entry's grid row (or select the row and in the grid menu, select Edit).
  2. In the Edit Counterparty Limit dialog box window:
    1. check that the Limit Type is the same as in the Combined Limit,
    2. ensure that the Active From date is appropriate,
    3. ensure that Calculation Method is NOTCombined Limit,
    4. select Include in Combine Limit, then
    5. click OK.
  3. Repeat Steps 1 and 2 for all other Limits to be included in the Combined Limit total.
  4. Save your changes.

*TIP: When later adding any new Limit entries, select or clear Include in Combine Limit to include or exclude that Limit in the applicable Combined one, as required.
