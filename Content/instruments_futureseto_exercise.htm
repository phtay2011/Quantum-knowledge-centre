Options Exercise area, Futures and ETOs Instrument

Use this area to enter settings Quantum will use when exercising the Futures Option contract.

Type - select when the Option may be exercised:

  • American - any time after its deal date and before its expiry date, or
  • European - only on its expiry date.

Settlement Offset (available if Type is American and the PremiumType is CLOSEOUT) - optionally specify an offset for the settlement of the premium when the option is exercised prior to expiry. Enter the number of days between the exercise date and when the premium actually settles for this contract type.

NOTE: If the Option is exercised on expiry date, the Options Settlement Day Rule is applied instead.

Allow Exercise Choices - select or clear this option to allow or disallow manual changes to the exercise choices for the Options contract. See also the Settlement Method field (Contract Dates and Settlement area, Options Contract Dates tab).

Strike Rounding (available only if the contract's Pricing Formula is BILLS, BONDS, or SFE ROUNDED BONDS - select the decimal places to round to when calculating the variable Tick Value that the Option Premium is based on. The default is the currency rounding for the contract type.

Rounding is applied to the Notional settlement value at the Strike Price and at the basis point shift (as defined in the Tick Move field on the Futures Contract Details tab).

  • For an SFE (Sydney Futures Exchange) Bill Option, select ROUND TO 2 PLACES.
  • For an SFE Bond Option, select NO ROUNDING. See Premium rounding, SFE Bond Option Contracts.
