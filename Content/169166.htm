Group Currency Procedures (Analytics)

On the Analytics perspective's Currency page:

To list Currencies and Currency groups

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Currency group

  1. In the New menu, select Group.
  2. Enter a unique three-letter Code and Name for the currency group.
  3. To add currencies to this group, click Add, select a currency from the Browse dialog box, and click OK.

    TIP: To include this new group within another group, open that group record and add this group as a member.

  4. Save your new currency group record.

To edit Currency group details

  1. Click to list available currencies, including groups.
  2. Select the group you want to edit.
  3. Edit the Name if required.

    NOTE: You cannot change the key Code field. Instead, create a new currency group with the new name you want (and optionally delete the wrongly-named record).

  4. To add new currencies to the group, click Add, select a currency from the Browse dialog box, and click OK.
  5. To remove a currency from the group, click the x button to the right of that currency in the list of members.
  6. Click Save.

To delete a Currency group

TIP: Deleting a group does not delete its members.

  1. Click to list available records, including groups (or use the Search box).
  2. Select the group you want to delete.
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To add the current group to another group (or move it to another group)

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select the target group.
  3. Save your change.

    The current record becomes a member of the selected group, and if previously in another group, is removed from that group.

To remove the current group from a parent group

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select Move to Top.
  3. Save your change.

    The current record is now "outside" all groups.

TIP: You can also add and remove members within a group when that group is displayed.
