Maintaining Netting Session Data

Use the Netting Session Transaction list page to view, edit, and delete transaction data for a participating entity in a netting pool session.

  1. In the main menu, select Processing and (in the Processing sub-menu) under NETTING INVOICES, select Transactions.
  2. Select the Pool, Session, and Entity.

    The pool session's current status is shown, and the relevant data details (filtered according to your security profile) are listed on the following tabs.

    • Invoices tab - lists transactions (to edit and delete these, use the respective tab menu items) - see Maintaining Netting Session Transactions
    • Pay/Recv Flows tab- lists cash flows projected for the session, based on the current saved transactions and current session rates - see Viewing Netting Session Cash Flows
    • Import Files tab - lists any import files used in the session, and enables removal of imported transactions - see Maintaining Netting Import Files

      TIP: To back out an import run, select the import file and in the tab menu, select Backout Import.
