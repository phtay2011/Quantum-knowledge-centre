Common Movement Columns

The following data items are available for selection as Movement columns in both Position and Cash Flow queries.

For items unique to Position queries, see the following topic. Cash Flow queries have no unique items.

NOTE: Movement columns do not apply for any VaR query types.

Name

Notes and References

Action Type

A status relating to a deal action or event that occurred between the Start Date and End Date of the Movement period - one of the following:

  • Input,
  • Rateset,
  • Missing Rateset - an expected rateset for the period was not captured or not actioned,
  • Closed,
  • Cancelled,
  • Modified,
  • None - the deal was not modified in the period, or
  • Multiple - more than one of the above actions occurred.

NOTE:

  • Any events occurring on the Start date are ignored with the End-of-Day Revaluation Timing.
  • The most recent image captured on or before the Start Date or End Date will be used in each case. The two images are then compared to determine the Action Type. For example, where a Rateset due on the End Date is captured the day after it is due, the Action Type will be Missing Rateset, because the image used will be the one prior to the Rateset image. Similarly, if a modification is effective on the End Date is captured the day after it is effective, the Action Type will be None.
  • Action Type was originally designed to analyze Time Periods that end on the current or previous day, and over short durations

As At Date

The As At Date value specified for the Point in Time column

Backdated Principal

Backdated principal cash flows actioned in the defined date range with an Effective Date prior to the start of that range.

Backdated Interest

Backdated interest cash flows actioned in the defined date range with an Effective Date prior to the start of the range.

Backdated Other

Backdated other cash flows actioned in the defined date range with an Effective Date prior to the start of the defined date range.

CF Spot Factor

The spot rate on a cash flow date expressed as a multiplier.

CF Spot Rate

The Spot Rate on a cash flow date.

Dealt Rate

The rate at which the deal was dealt.

End Date

The end date of the movement period.

End Spot Rate

The Spot Rate at the end of the defined date range.

End Spot Factor

The Spot Rate at the end of the defined date range expressed as a multiplier.

Interest Cashflows

Interest cash flows in the defined date range.

Interest Period

The number of days in the period.

Modifications

A Boolean value indicating whether any modifications to the deal or deal side have been made in the period, where:

  • 0 = False
  • 1 = True

Movement Limit Amount

The Limit Amount from the appropriate Limit record.

Values are returned when a Limits Definition is included in the query.

Movement Limit Currency

The Limit Currency from the appropriate Limit record.

Values are returned when a Limits Definition is included in the query.

Movement Limit Description

The Limit Description from the appropriate Limit record.

Values are returned when a Limits Definition is included in the query.

Other Cashflows

Other cash flows in the defined date range (not Principal nor Interest).

Principal Cashflows

Principal cash flows in the defined date range.

Principal Outstanding

The principal that is still outstanding.

Start Date

The start date of the movement period.

Start Spot Factor

The Spot Rate at the start of the defined date range expressed as a multiplier.

Start Spot Rate

The Spot Rate at the start of the defined date range to convert the transaction currency to base currency.

Time Weighted Cashflows

, where:

  • CF = Cash flow amount
  • D1 = Query end date
  • D0 = Cash flow date

Units

Values are returned only for Equities transactions.
