Options Premium settings, Futures and ETOs Instrument

Use the Premium area of the Options Contact details tab to enter settings Quantum will use when calculating the premium on a Futures Option contract. Define the Premium Type and Premium Settlement Offset, and specify whether Variation Margin postings are required.

An Option premium is based on the following formula.

The tick value is:

  • variable for contracts that use the BILLS, BONDS, or SFE ROUNDED BONDS formula,
  • fixed for PER TICK contracts.

TIP: For contracts that have a variable tick value, the Strike Rounding setting (in the Exercise area) determines the rounding convention used to calculate the tick value.

For DECIMAL Option contracts, the Shift Decimal Place and Minimum Price Move Option settings may differ from those for the Futures contract.

Complete the items in this area as described below.

Type

Select:

  • DEAL DATE - if the premium on the Option contract is to be settled:
    • On the deal date - the premium on the closing deal is calculated on a gross basis, and is posted independently of the premium on the opening deal.
    • A number of days offset from the deal date - for example, the premium on LIFFE FTSE 100 Options settles one day after deal date. See Settlement Offset.
  • CLOSEOUT - if the premium on the Option contract is to be settled on closeout (on a net basis), exercise (gross basis), or at expiry (gross basis).

    If a premium settles at expiry for example, the premium cash flows are dated according to the Option settlement date rule. See Exercising Exchange Traded Options.

    NOTE: Although a CLOSEOUT premium is not settled at deal date, the Option is marked to market from that date. This generates positive and negative Variation Margin flows.

Settlement Offset

Enter the number of days offset from the premium settlement date generated by the rule in the Type field. This is the number of days that elapse between the date the premium would be expected to settle (in terms of its premium type) and when the premium actually settles for this contract type.

A negative settlement offset is not possible. If an offset results in a default settlement date that is greater than the contract settlement date, Quantum uses the contract settlement date.

NOTE: This setting does not apply when the Option is exercised prior to expiry, or when it expires (the Settlement Offset field in the Exercise area applies in those cases). When the Option is exercised at expiry or when it lapses, the Options Settlement Day Rule is applied.

Shift Decimal Place

(Available if Entry Type on the Futures Contract Details tab is set to DECIMAL.)

Specify how users will enter and view values that include decimals. This is the number of places an entered value's decimal point should be shifted by to give the actual value, and it applies to the Futures and ETO Deals and Interest Rates pages, and to the Revaluation Analysis report.

Enter a positive or negative number to shift the decimal point to the right or left, respectively. The default is 0 or no shift.

TIP: If you prefer to enter prices as percentage points (without entering a decimal point), set this to -2. You can then enter 50 points when the price of the Option is 0.5 (0.5% p.a.).

Minimum Price Move

(Available if Entry Type on the Futures Contract Details tab is set to DECIMAL.)

Enter the Minimum Price Movement for this Option contract. This is the smallest shift in price that is recognized for this contract on this Exchange.

TIP: The Minimum Price Movement for an Option contract may differ from the Tick Size and Minimum Price Movement specified for the underlying Futures contract.

Post Variation Margin

If you select this option, Variation Margin will be posted for this contract.

If you clear it, Variation Margin is not posted; however, calculation figures will still be displayed in Futures Margin queries.
