View generated Fees and Base amounts

To view daily Fee and Base amounts applying to a selected Fee Deal and calculation period:

  1. On the Fee deals tab, select the grid row for the Fee Deal.
  2. Click Base Amount....

    The Fee Deal - Base Amounts dialog box opens, listing the Fee Amount and Base Amount values for each applicable From/To Date range, as shown in the following example.

    TIP: This example shows Fee and Base Amounts for an Unutilized Facility fee, where the last row (highlighted) indicates that on 15 May 2018, the Facility was drawn on, halving the unutilized amount (and therefore the fee).

  3. If necessary, scroll to the latest calculation Period to see the most recent amounts generated for the Fee Deal.

Notes on the data

The Period is the calculation period for which the fee Amount and Base Amount corresponding fee entries are generated. For example, the entry for Period 1 is used to calculate the fee cash flow for the first calculation period - which is from the Fee Deal's Start Date to its calculation First Date (as specified on the Schedule tab of the Fee Deal page).

The Base Amount is calculated from the applicable deal data in the Fee Position history table, adjusted by the % Adj value specified on the applicable Base Template/s for the Fee Deal.
