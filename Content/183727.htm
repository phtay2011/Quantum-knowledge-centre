Tab Summary, Deal Details page

Only the Details tab is affected when any 2nd Deal is specified. All other tabs only show details for the selected Deal Number and Image Number.

Details tab

This tab displays the core deal information, as stored in the database, where:

  • if only one deal is specified (2nd Deal is blank or zero) - all data is displayed for the selected Deal Number and Image Number, or...
  • if two deals (or images) are specified, only those fields whose values differ are displayed.

Cash Flows tab

The grid on this tab lists full details of the cash flows for the selected Deal Number and Image Number, as stored in the cflows database table with the column headings showing the field ids.

  • Select or clear Show Intercompany cash flows to include or exclude intercompany flows.
  • For the LIVE deal image, the grid items will correspond to those on the Cash Flows tab of the relevant Dealing page when the same deal is selected.

G/Ls tab

The grid on this tab lists full details of the General Ledger entries for the selected Deal Number and Image Number, as stored in the gl_entry database table (with the column headings showing the field ids).

  • The G/L entries include the cash flow-based entries to deal maturity date, any Contingent entries, and accrual and revaluation entries, as applicable.
  • Select the available options (Include reversalsInclude accruals, and so on) as required, to include specific deal-related G/L entries.
  • For the LIVE deal image, the grid items will correspond to those on the G/L Entries tab of the relevant Dealing page when the same deal is selected.

Map tab

The grid on this tab lists the main details of the key cash flows and events for the selected Deal Number and Image Number, as stored in the database.

For the LIVE deal image, the grid items will correspond to those on the Deal Map tab of the relevant Dealing page when the same deal is selected.

Events tab

The grid on this tab lists full details of the key cash flows and events for the selected Deal Number and Image Number, as stored in the events database table (with the column headings showing the field ids).

  • Examples of events are Ratesets, Rolls, and Indexations.
  • For the LIVE deal image, the grid items will correspond to those on the Deal Map tab of the relevant Dealing page when the same deal is selected.

Actions tab

The grid on this tab lists full details of the deal's approval points (including the approval status at each point), for the selected Deal Number and Image Number, as stored in the actions database table (with the column headings showing the field ids).

Flows tab

The grid on this tab lists full details of any associated stock flows for the selected Deal Number and Image Number, as stored in the flows database table, with the column headings showing the field ids.

Acrl Items tab

This tab applies to deals with accruals of interest or coupons.

The grid lists full details of any accrual cash flows for the selected Deal Number and Image Number, as stored in the acrl_items database table. The column headings show the field ids.

For the LIVE deal image, the grid items will correspond to the accrual entries on the Cash Flows tab of the relevant Dealing page when the same deal is selected.

Acrl SubItems tab

This tab applies to deals with accruals of interest or coupons.

The grid lists full details of any accrual sub-items for the selected Deal Number and Image Number, as stored in the acrl_subitems database table. The column headings show the field ids.

TIP: Accrual sub-items relate to events that occur within an interest period - for example, ratesets or unwinds - when interest is being accrued.

Acrl Items Posting tab

This tab applies to deals with accruals of interest or coupons.

The grid lists accrual postings to the General Ledger for the selected Deal Number and Image Number, as stored in the acrl_item_post database table. The column headings show the field ids.

For the LIVE deal image, the grid items will correspond to the accrual entries on the G/L Entries tab of the relevant Dealing page when the same deal is selected.

HR Link Event tab

This tab applies to Hedge Relationship (HR) deals only.

The grid lists full details of any OCI Release events following maturity of the selected HR Deal Number and Image Number, as stored in the hrlinkevent database table. The column headings showing the field ids.
