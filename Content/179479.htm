Special Analysis Codes for Currency Option Deals

Some Analysis Codes are mandatory.

For example, the following Analysis Code values are required for Exotic Currency Option Confirmations that are to be electronically matched using the SWIFT external matching system, and their set values will populate required SWIFT MT306 message tags. If they do not appear on the Analysis Codes tab, they must be added using Analysis Code Setup.

NOTE: Analysis code values can be set by default - see Defining Analysis Code defaults for deals.

Analysis Codes required for SWIFT MT306 messages

Additional Conditions

This Analysis Code is required only for Exotic Currency Option Confirmations that are to be electronically matched using SWIFT.

This Analysis Code value:

  • is mandatory only if the Type of Agreement Analysis Code value is OTHER,
  • contains additional information concerning the terms and conditions of the underlying agreement,
  • may be up to 25 characters long, and
  • is output in Tag 77D of the SWIFT MT306 message for the Deal Confirmation.

For a list of acceptable code values, refer to your SWIFT User Handbook.

Settlement Rate Source

This Analysis Code is required only for Exotic Currency Option Confirmations that are to be electronically matched using SWIFT.

This Analysis Code value:

  • is mandatory,
  • specifies the rate source for the settlement of the non-deliverable option,
  • contains three characters followed by two digits, and
  • is output in Tag 14S in Mandatory Sequence H of the SWIFT MT306 message for the Deal Confirmation.

For a list of acceptable code values, see Annex A of the 1998 FX and Currency Option Definitions published by ISDA, EMTA and the US FX Committee.

Trigger Hit Location

This Analysis Code is required only for Exotic Currency Option Confirmations that are to be electronically matched using SWIFT.

This Analysis Code value:

  • is mandatory,
  • specifies the financial center of the trigger event,
  • must begin with a supported four-character value (see the list below),
  • may be up to 25 characters long (but only the first four characters are output), and
  • is (in part) output in Tag 29H of the SWIFT MT306 message for the Deal Confirmation (first four characters only).

Type of Agreement

This Analysis Code is required only for Exotic Currency Option Confirmations that are to be electronically matched using SWIFT.

This Analysis Code value:

  • is mandatory,
  • specifies the type of the agreement covering the transaction, optionally followed by the date on which the agreement was signed, and the version of the agreement,
  • must begin with a supported value up to six four-characters long (see the list below),
  • is output in Tag 77H of the SWIFT MT306 message for the Deal Confirmation, and
  • may optionally (see the examples below) have appended:
    • a forward slash followed by the eight-digit agreement date value in YYYYMMDD format and/or
    • two forward slashes followed by the four-digit agreement version value.
