Group Counterparty Procedures (Quantum)

On the Counterparty page:

To list Counterparties and Counterparty Groups

  1. Click the button.
  2. Optionally follow the Configure link to change list properties - see Configure Browse list views.

To add a new Counterparty Group

  1. In the New menu, select Group.
  2. Enter a unique Name for the group.
  3. To add counterparties to the group, click Add, select a counterparty from the Browse dialog box, and click OK.

    TIP: To include this new group within another group, open that group record and add this group as a member.

  4. (Optional) On the Analysis Codes tab, link the Counterparty group to predefined Analysis Codes for reporting, as applicable.
  5. Save your new Counterparty group record.

To edit Counterparty Group details

  1. Click to list available counterparties, including groups.
  2. Select the group you want to edit.
  3. Edit the Short Name and/or External Name,as and if required.NOTE:

    NOTE: You cannot change the key Name field. Create a new counterparty group with the new name you want (and optionally delete the wrongly-named record).

  4. To add new counterparties to the group, click Add, select a counterparty from the Browse dialog box, and click OK.
  5. To remove a counterparty from the group, click the x button to the right of that counterparty in the Group Contents grid.
  6. (Optional) On the Analysis Codes tab, add or remove Analysis Codes for the Counterparty group if required.
  7. Click Save.

To delete a Counterparty group

NOTE:

  • Deleting a group does not delete its members.
  • This procedure deletes the record completely from the database. To keep a record of the group, deactivate it instead.
  1. Click to list available records, including groups, or use the Search box.
  2. Select the group you want to delete.
  3. In the menu, select Delete.
  4. When prompted, confirm your action.

To add the current group to another group (or move it to another group)

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select the target group.
  3. Save your change.

    The current record becomes a member of the selected group, and if previously in another group, is removed from that group.

To remove the current group from a parent group

  1. In the page menu, select Move.
  2. In the Select Group to Move To dialog box, select Move to Top.
  3. Save your change.

    The current record is now "outside" all groups.

TIP: You can also add and remove members within a group when that group is displayed.

To deactivate a Counterparty group

  1. Click to list available records, including groups, or use the Search box.
  2. Select the group you want to deactivate.
  3. In the menu, select Deactivate. This action will not appear if the group is already deactivated.

To reactivate a Counterparty group

  1. Click to list available records, including groups, or use the Search box.
  2. Select the group you want to reactivate.
  3. In the menu, select Reactivate. This action will not appear if the group is already activated.

To publish a Counterparty group

TIP: All fully approved new or changed Counterparty records will be published automatically If the Counterparty option is selected (on the Operational Settings page under Publish).

  1. Click to list available records (or use the Search box).
  2. Select the group record you want to publish.
  3. In the menu, select Publish Selected.

To publish all Counterparties (including groups)

  • In the page menu, select Publish All.
