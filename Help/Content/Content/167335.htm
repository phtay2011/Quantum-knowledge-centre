European

Analytics

The Black (European) model applies.

Value

Payer:

  • Per Unit:   Ppupay = e-rT1 * [ F * N(d1) - K * N(d2) ]
  • Position:   Ppospay = Ppupay * P * ( d / dpy )

Receiver:

  • Per Unit:   Ppurcv = e-rT1 * [ K * N(-d2) - K * N(-d1) ]
  • Position:   Pposrcv = Ppurcv * P * ( d / dpy )

Where:

F = Current Forward rate

T = Time to expiration (years)

T1 = Time to cash flow (years)

K = Strike Price

r = continuously compounding risk-free rate (to expiry)

N = Normal distribution function

e = 2.718281828

P = Principal

d = number of cash flow days

dpy = cash flow days per year

ln = natural logarithm (to base e)

s = Standard Deviation of asset returns

Delta

Payer:

  • Per Unit:   DPpupay = e-rT1 * N(d1)
  • Position:   DPpospay = DPpupay * P * ( d / dpy ) * 0.0001

Receiver:

  • Per Unit:   DPpurcv = e-rT1 * [ N(d1) - 1 ]
  • Position:   DPposrcv = DPpurcv * P * ( d / dpy ) * 0.0001

Gamma

Payer or Receiver:

  • Per Unit:   Gpu = [ e-rT1 * N'(d1) ] / [ F * s * ÖT ]
  • Position:   Gpos = Gpu * P * ( d / dpy ) * 0.00012

Vega

Payer or Receiver:

  • Per Unit:   Vpu = e-rT1 * F * ÖT * N'(d1)
  • Position:   Vpos = Vpu * P * ( d / dpy ) * 0.0001

Theta

Payer:

  • Per Unit:   Qpucall = - [ ( e-rT1 * F * s * N'(d1) )/ (2 * ÖT) ] + r * e-rT1 * [ F * N(d1) - K * N(d2) ]
  • Position:   Qposcall = ( Qpucall * P ) * ( d / dpy ) / 365

Receiver:

  • Per Unit:   Qpuput = - [ ( e-rT1 * F * s * N'(-d1) )/ (2 * ÖT) ] + r * e-rT1 * [ F * N(d1) + K * N(-d2) ]
  • Position:   Qposput = ( Qpuput * P ) * ( d / dpy ) / 365

Rho

Payer:

  • Per Unit:   Rpucall = e-rT1 * [ K * T * N(d2) - [ F *T * N(d1) ]
  • Position:   Rposcall = ( Rpucall * P ) * ( d / dpy ) / 365

Receiver:

  • Per Unit:   Rpuput = - e-rT1 * [ F * T * N(-d1) - [ K *T * N(-d2) ]
  • Position:   Rposput = ( Rpuput * P ) * ( d / dpy ) / 365
