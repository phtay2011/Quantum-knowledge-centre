About Exercising Exchange Traded Options (ETOs)

In Quantum, ETO deals can be:

  • partially or fully exercised / assigned,
  • partially or fully closed out, or
  • deleted

NOTE: Quantum supports Exercise and Assign for Hypothetical Exchange Traded Options (ETO). However, these must be mirrored in the equivalent ETO Live deal.

You can exercise or assign an ETO whether it is cash-settled or deliverable, European or American. The system will calculate the correct Variation Margin, regardless of when the premium is settled (the premium on a LIFFE Short Sterling contract, for example, is paid the day after exercise).

When a buyer exercises an Option, this is an exercise to the buyer and an assignment to the seller. The system distinguishes between assignments and exercises.

NOTE:

  • The system does not automatically generate a Futures deal when you exercise a deliverable ETO. You need to input both the Options exercise and the Futures deal manually.
  • You can set up Approval Control to restrict which users can exercise Options, and to require approval of an exercise - see Defining Approval Control Records. If you set the Allow Deals option in Approval Control to YES, you can exercise the deal before the deal save is approved.

Multiple Exercises or Assignments

You cannot enter multiple exercises or assignments on a single day for a deal, but you can modify the entry for a particular day, and change the number of lots exercised or assigned that day. See Modifying an Exercise.

Viewing Exercise details

Details entered on the Exercise window appear on the Exercise/Assignment tab.

Updating the General Ledger

General Ledger entries that relate to an exercise or assignment are updated when you save the exercise or assignment. These entries relate to:

  • Premium for the exercise / assignment,
  • reversed contingent entries for lots exercised / assigned, and
  • gain or loss related to cash-settled exercises or assignments.

Dates on these entries depend on settings in the Contingent accounting settings in the Accounting Rules of Accounting Static Data Setup, and the Option Contract Details tab in Futures Control.

NOTE: Variation Margin on American Options that are exercised prior to expiry is reversed when you run Futures Margin processing.

Modifying an Exercise

If you modify or delete an exercise or assignment, the system recalculates all values affected by that change.

ETOs that are not exercised are assumed to have lapsed on expiry of the contract.

The number of lots lapsed is the number of lots remaining as at the end of day on the expiry date. Off Balance Sheet entries are reversed, premium is transferred (if the Premium Type is CLOSEOUT), and any outstanding Variation Margin is reversed. Deal Details shows how many lots are lapsed.

Exercising an Exchange Traded Option (ETO)

  1. Open the Exercise window from either the Deal, or from Settlements, as follows.
    • In Futures and ETO Deals, browse for and select the deal, then select Exercise in the page menu.

    -or-

    • On the Settlements processing page (assuming you have specified and run suitable parameters), select the ETO deal you want to action, and select Exercise from the page menu.
  2. In the Exercise dialog box, enter the following details.
    • Exercise Type (read-only) - AMERICAN or EUROPEAN, as entered on the Options Contract Details tab in Futures Control
    • Settlement Method (read-only) - CASH SETTLED or DELIVERABLE, as entered on the
    • Exercise Date (for purchases) or Assignment Date (for sales)
      • For a European Option, this field is read-only and shows the Option Expiry Day Rule set on the Options Contract Details tab in Futures Control (European Options can only be exercised on their expiry date).
      • For an American Option, this defaults to Today's date, but you may can change it to any date from the Deal Date to the Expiry Date or Settlement Date, depending on the rules set on the Options Contract Details tab in Futures Control.
    • Settlement Date - the day the premium is settled
      • For a European Option, this field is read-only and shows the Option Settlement Day Rule set on the Options Contract Details tab in Futures Control.
      • For an American Option, this defaults to Today's date, adjusted by the Exercise Settlement Offset. You can change this to any date from the specified Deal Date to the Expiry Date or Settlement Date, depending on the rules set on the Options Contract Details tab in Futures Control.

      If a premium is settled on exercise, the Option Premium Type field is set to CLOSEOUT on the Options Contract Details tab. Premium cash flows are dated the exercise Settlement Date.

    • Lots to Exercise - defaults to the available lots remaining on this contract, but you may change it for a partial exercise of the Options contract
    • Exercise Price (for purchases) or Assignment Price (for sales) (displayed only if Settlement Method is CASH SETTLED) - enter the Exchange Delivery Settlement Price (EDSP) at which a CASH SETTLED European Option is exercised

      NOTE: Position the decimal point according to the settings on the Price Entry panel of the Futures Contract Details tab in Futures Control.

Modifying ETO Exercises or Assignments

You can only change or delete an exercise or assignment after it has taken place if the Modifiable Exercises option is selected on the Options Contract Details tab in Futures Control.

  1. Browse to and select the deal.
  2. In the page menu, select Modify Exercises Assignments.

    NOTE: For Options bought, field names on the form include the word Exercise. For Options sold, this is replaced by Assign or Assignment.

  3. In the dialog box, select the fields that you want to modify.
    • For European Options, you can modify only the number of lots (Quantity) to be exercised.
    • For American Options, you can modify the Exercise Date, Settlement Date, and Quantity.
    • If the Settlement Method is CASH SETTLED, you can modify only the Exercise Price (this field is not displayed if the Option is DELIVERABLE).
  4. Save your changes.

    Quantum replaces the relevant accounting entries to reflect these changes when you run Futures Margin processing.

NOTE: Details of exercises against an Option appear on the Exercise / Assignment tab.

Deleting an ETO Exercise or Assignment

  1. Browse to, and select the deal.
  2. In the page menu, select Modify Exercises Assignments.
  3. Reduce the number of lots for the day by the amount of the exercise or assignment (set the lots for this deal to zero). See Modifying an Exercise.
  4. Save your changes.
