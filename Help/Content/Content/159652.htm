About Currency Options

Data Options

Option Type - Both Vanilla and Exotic options are supported. See Currency Option Types

Option style - See Currency Option Styles

Instrument setup - Before you can record a Currency Option deal, the default for the Currency Option type and Style need to be defined for the underlying Currency Option Instrument.

CLS (Continuous Linked Settlement) - This optional process for Currency Options (and FX deals) enables cross-border currency transactions to be settled intra-day, eliminates settlement risk, and substantially reduces operational risk in FX markets. The Instrument's CLS Settlement option determines if CLS is OPTIONAL, REQUIRED, or NOT APPLICABLE. See CLS in Quantum and Implementing CLS for Currency Options.

Strategy - This provides a powerful ad hoc grouping mechanism that you can use to report structured transactions where the structure is a composite of several deals. Using Strategies does not impact on the business structure as it is independent and may be used to group any number of deals across the business structure. The Strategy is selected on the Dealing page's Other Details tab.

NOTE: This Strategy should not be confused with a Hedge Strategy used for Hedge Accounting.

External Reference numbers and text - these will appear as read-only values on the Deal Details tab, but only when:

  • the record data has been imported or input via an API, and
  • the External Reference fields or properties are populated.

Deals in Hedge Relationships

TIP:

  • To add and maintain any deals in a Hedge Relationship, use the Linked Deals tab on the Hedge Relationship page.
  • The Dealing page shows a Hedge Relationships tab if the current deal is a Linked Deal within one or more Hedge Relationships. If so, you can quickly access the Hedge Relationship from this tab by clicking on the HR Deal Number in the Allocations grid.

Hypothetical mode

If you are licensed for Hedge Accounting, you can use this form to create a Hypothetical Currency Option deal to measure the effectiveness of your live deal - by setting Deal Type to Hypothetical. See Hypothetical Derivative deals for more information.
