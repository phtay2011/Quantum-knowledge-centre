Query Page Procedure Summaries

To select a different Query or a previous run - Click on the current Query heading and select the Query or run in the browse window.

TIP: Query run results are retained according to the Expiry Strategy settings for the Query configuration (on the Properties tab). EXCEPT THAT selecting Keep in the Page menu keeps the current run results (until they are manually deleted).

To create a new Quantum Query from scratch - In the page menu, select New; click the Query Configure link, make your changes, and save them.

TIP: Use relative dates in date fields so you can reuse queries in future runs.

To create a new Analytics or Adaptiv Query from scratch - In the Query Explorer, navigate to the folder you want to hold the Query; in the page menu, select the applicable New function depending on the Query type; click the Query Configure link, make your changes, and save them.

To configure a new Query by copying the current one - In the page menu, select Copy; click the Query Configure link, make your changes, enter a new Query Name (on the Properties tab), and Save.

To reconfigure the current Query - Click the Query Configure link, make your changes, optionally change the Query Name (on the Properties tab), and Save.

To select a different result view - Click on the current name of the view and select the view in the browse window.

To change the view or create a new one - Click the Configure link, make your changes, and save them.

To run the current Query

  1. If the settings are not currently displayed (because previous run results are), click a value in the parameter summary row to display them, or (Quantum only) select Change Parameters in the Run Query button's drop-down menu.
  2. Enter specific Filter and Option settings and click Run Query* or select a run option in the Run Query button's drop-down menu: Continue on error, Stop on error, or Auto-run (see next procedure).

    NOTE: Some Processing Queries (for example Accruals, Bank Statements, and Position) have Preview and Update buttons instead (with no run options).

To run the current Query with automatic updating of results - In the Run Query button's drop-down menu, select Auto-run...; in the dialog box, select a run option and click OK. An Auto-run badge is displayed next to the query name and, if it is updating the results automatically (Live, Latest, or Auto-run), a Pause button is displayed next to the Run Query button. For more details, see Auto-Update a Query.

To pause and resume automatic updating - Click Pause to pause the automatic update; click Resume to have the system automatically update the query results again.

To export a standard report to Excel™ - Click the button in the name of the view in the results grid to export the full results table to Microsoft Excel. A button at the bottom of the view area shows the generated Excel file name (click to open that file in Excel).

To export a standard report to a file - Select Export in the page menu to export the full results table to Excel, CSV, PDF, and XML formats.

To export a Crystal Report - Use the export control in a Crystal view to export report results to a variety of different formats.

To publish the report (Quantum only) - Select Publish in the page menu to make the current query run results available to other users.

NOTE: When Quantum query run results are published, they will not expire as per the Expiry Strategy configured for the query.

To display previous run results - EITHER click the Previous Runs link (if present) OR click the Query heading, and select the previous run from the browse window or drop-down list.

NOTE: The returned data is subject to Resource Security.
