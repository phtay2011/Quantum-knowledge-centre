About the Face Value Hedge Ratio

Once a hedge relationship includes both a Hedge and an Exposure, Quantum will attempt a simple calculation to show how effective that hedge relationship is, based on Face Values only.

These results are displayed on the Linked Deals tab and updated as deals are added to the hedge relationship. The percentage is always the most current: that is, the Face Value Ratio as at today.

NOTE: The Face Value Ratio is a simple ratio of Hedge to Exposure, expressed as a percentage. Calculations are only possible if the deals (or deal legs) being linked are in the same currency. If no ratio can be calculated, the effectiveness shown will be zero percent (0%). This does not necessarily mean that it is not an effective hedge relationship. Full Effectiveness calculations by Position Query will provide the necessary details.

Intended ratio

While Automatic Allocation is active, Quantum will attempt to achieve the Target Face Value Ratio specified in the Operational Settings, at each point of change for a hedge relationship. This includes at designation, dedesignation, and at any date where activity affects the value of a sub deal.

Gains and losses

Effectiveness percentages apply only if Hedge and Exposure deals offset one another - that is, if one shows a gain and one a loss.

A hedge relationship is not effective if both deals show a gain, for example. The effectiveness of the hedge relationship in this case would be zero percent (0%).

Short-Cut method

If this method is applied, 100% effectiveness is assumed and 100% is displayed as the Face Value Hedge Ratio. See Short-Cut criteria in Hedge Accounting.

NOTE: If deals that use the Short-Cut effectiveness method are set up incorrectly within a hedge relationship, then the Face Value Hedge Ratio is set to 0% and the relationship is deemed ineffective.
