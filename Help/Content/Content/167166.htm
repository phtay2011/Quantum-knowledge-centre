Index Ratio Formulas

This topic gives formulas for the Index Ratio that will be applied to cash flow amounts for deals based on an Instrument where Calculation Formula = INDEXED BOND, to adjust for inflation.

The Indexed Bond Instrument specifies the Index Rate Series and the user-definable Index Calculator Definition to be applied (on the Indexation tab). The Index Calculator Definition specifies the two offsets (Ref_CPIM and Ref_CPIN) giving the data points in the Index Rate Series table for calculation of the Index Ratio () for a specific date.

About Effective Dates

Effective Dates in an Index Rate Series table are not used in any calculations. They control the dates from which CPI index values are observable. On dates before the Effective Date of an index entry, Quantum treats that entry as if it did not exist. This is especially important when back-dated deals are entered.

The Effective Date for an Indexed Bond deal is calculated on deal save, using a combination of the available Index Rates and the Index Calculator Definition associated with the Instrument setup.
