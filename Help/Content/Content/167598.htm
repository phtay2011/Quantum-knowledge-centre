Recording a Quote

Open the Quote page and record details of the new Quote Request on the Deal Details tab.

The only fields required to save a Quote Request are Quote Type and Instrument.

NOTE: The Entity field must also be completed if you want to create and save deals directly from a quote.

When you have selected a Quote Type, the displayed fields will change to reflect the appropriate instrument.

  1. Use the Counterparties field to record one or more counterparties you are requesting Quotes from.

    You may select the same counterparty more than once. Use this to record quote requests to different sections of the same counterparty.

  2. Record any other information you have about the Quote Request in the remaining fields on the Deal Detail and Other Detail tabs.

    NOTE: The Comments field on the Other Details tab provides space to make text notes about the Quote Request. This text is stored only with the Quote record. To save Comments with the deals that result from the quote request, record them in the Comments field on the Header Details tab.

  3. Save your changes.

    The system will assign a deal number to the Quote Request when it is saved.

  4. Make a note of the deal number, so that you can retrieve the request and use it to record the bids as they come in.

    See Recording Quotes Received.

TIP: You can also save basic Quote Requests from supporting deal entry programs. See Recording a Quote Request at Deal Entry.
