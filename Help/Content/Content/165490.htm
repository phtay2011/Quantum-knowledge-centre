Calculating Bond Option Pricing Values

Use the Price tab (inside the Details tab) to calculate market values for the deal, as follows.

  1. Change the Position Date, Rates Date, As At Date, and/or Volatility settings, as required.
  2. If Style is European:
    1. Enter the Spot Price and/or Forward Price. If left blank the value will default from market rates. The associated Cash Price is displayed in the adjacent field in each case.
    2. In the Calculate field (selectable for new deals only), either select Value, or select a variable to solve for a specified value (Volatility or Strike).
    3. If Calculate is Volatility or Strike, enter the specific Value to solve for.
    4. Go to Step 4.
  3. If Style is notEuropean, select the Valuation method and Steps per Coupon period.
  4. Click the Price button.

For a European style option, the Bond Option grid displays the calculation result totals for your chosen settings, giving the Position and Per Unit values for the Price and Greeks, as shown below.

For an American or Bermudan style option:

  • The Bond Option grid displays the calculated results for your chosen settings, giving the Position and PPH (price per hundred) values for the Price and Greeks.
  • The separate Bond grid displays the calculated Capital Price, Accrued Interest, and Cash Price result for your chosen settings, giving the Per Unit and Position values in each case.

In each case the Rates tab shows the rate curves used and the Valuation tab displays the calculation results for each valuation cash flow.

The result values are summarized below. For details of the calculation formulas used, see Bond Option Formulas.

  • Price - Values for the option price.
  • Delta - A measure of the change in the value of an option relative to a change in the price of the underlying.
  • Gamma - A measure of the change of an option's Delta, relative to a change in price of the underlying.
  • Vega - The sensitivity of an Option's value to a one percentage point change in the volatility of the underlying.
  • Theta - The rate of change of the value of an option as its time to maturity decreases, with other terms held constant. (This value is usually negative, as Options with longer periods before maturity have greater time value.)
  • Rho - A measure of the sensitivity of an option value relative to a change in the domestic interest rate.
