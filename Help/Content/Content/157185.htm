Query Page Menu Functions

Click the button at top right to open the page-level menu for the relevant Query page.

The page menu functions depend on the current perspective and the page function.

TIP: The Configure View panel has a separate menu with functions to create new views or to reconfigure, copy, save, and delete the current view.

Common Page Menu Functions

QUERY section (functions related to Query configuration and running):

  • New (multiple functions in Analytics and Adaptiv Explorer - see below)
  • Run (replaced by Preview and Update in some Quantum Processing queries)
  • Configure
  • Save
  • Copy

REPORT section (functions related to Query run results):

  • Print
  • Export
  • Keep (bypasses the Expiry Strategy setting for the current Query run results only)
  • Publish (Quantum only)
  • Delete

Analytics Explorer - Query level Page Menu
