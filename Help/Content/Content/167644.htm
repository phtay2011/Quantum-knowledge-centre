Balances Definition

NOTE: All Balances Definitions are shared by Analytics and Adaptiv.

Use the Balances definition page to configure filters on the balances considered for the query run. For example, you may wish to only report on balances for specific Bank Accounts, Account Owners, Currencies, and so on.

A specific record's settings will be applied when a query configuration specifies that record's name in the optional Balances field (Properties tab, Query Configure).

You can create balances at one of two levels:

  • Bank Balance - the balances of the selected Bank Accounts for the Position Date, or
  • Entity - the sum of the past or settled cash flows up to the Position Date for the selected entities and currencies.

To display an existing folder or record - Click , navigate in the browse list to the item you want (or use the Search tool), and select it. The folder contents or record settings are displayed and the breadcrumb trail indicates the current location.

To add a new record (at the current location) - Click the New button and select the type (Bank Balance or Entity) in the drop-down menu; enter a new Name for the Definition record; optionally enter text Comments; select or enter the required settings*; then click Save.

TIP: You can also create a new record by copying the current one. In the page menu select Copy; enter a new Name; change the required settings*; then click Save.

To add a new folder at the current location - Click the New button and select Folder in the drop-down menu; enter a new Name for the folder; then click Save.

To change a Definition record's details - Locate the record as above, change settings* as required; then click Save.

To move a record or folder to a different location - Locate the record or folder and in the page menu select Move. In the Select Group to Move To dialog box, locate the target folder and select it. The new location becomes the current location.

To delete a record or folder not in use - Locate the record or folder and in the page menu select Delete. When prompted, click OK to confirm.

To view changes made to the Definition record - In the page menu select Audit History.

* A filter is based on logical operators and field-level elements. You can create multiple filters with one or more selection criteria under a "branch", include other saved Balances definition filters (by using the Add Filter Link function), and nest branches. A branch must be set to either Match All (Logical AND) or Match Any (Logical OR), and there is always one overall branch.

To add a branch - Select the "parent" branch, and in the Add Filter button's drop-down list, select Add Branch. A new branch is added under that parent. To change the branch type, select Match All or Match Any in the branch's drop-down list as required.

To remove a branch - Select the branch, and in its drop-down list, select Remove Branch.

To add a new filter row within a branch - Select the required branch, click the Add Filter button, and select the Bank Account filter field from the drop-down list (start typing to find your filter quickly). Specify the field's filtering criteria as follows:

  • Drill down on a link to show the fields within that data folder -Bank Account, then (within that) Bank, Ccy, and so on.
  • For a multi-select field - in the field's drop-down list, select one or more values. If required, click one of to change to not one of, and vice versa
  • For an alphanumeric field - type in one or more values, separated by commas. If required, click one of to change to not one of.
  • For a numeric field - in the operator drop-down list above the filter field, select the arithmetic operator (equal to, less than, between, and so on), and enter one or two values as required.
  • For a date field - in the operator drop-down list above the filter field, select the comparison operator (equal to, between, and so on), and use the calendar tool to select one or two dates as required.

To include an existing Balances Definition within a branch:

  1. Select the required branch and select Add Filter Link in the Add Filter button's drop-down list.
  2. In the browse window, select the Balances Definition record to also be applied at that point in the logical structure.
  3. (Optional) To negate the logic in the linked Balances Definition, change the equal to caption above the Filter Link field to not equal to.

To remove a filter row - In the drop-down menu on the filter field caption,select Remove Filter.
