Freezing OCI or MTM in Hedge Relationships

A Freeze can only occur between the Inception Date and Derived Maturity Date of a hedge relationship. These dates are calculated and displayed on the Events tab.

No Freeze event

If you run Hedge Reclassification and there are no Freeze events for the selected hedge relationship during the period since the last revaluation, Quantum will perform any of the following that are due:

  • MTM/OCI revaluation
  • Dedesignation
  • Amortization
  • Release
  • Transfer
  • Release Transfer

Freeze event

If you run Hedge Reclassification and there is a Freeze event at any time during the period since the last revaluation, Quantum will still perform the following without interruption if they are due to take place:

  • Amortization
  • Release
  • Transfer
  • Release Transfer

These calculations continue as if no Freeze is in place.

While a Freeze is effective, however, the following processes are suspended and no related postings take place:

  • MTM/OCI revaluation
  • Dedesignation

There can be more than one Freeze between successive revaluations. Quantum will suspend these calculations while each Freeze is in effect, resuming them in between.

NOTE: Effectiveness calculations for a hedge relationship remain unchanged from when a Freeze occurs until the first revaluation after that Freeze ends. All revaluations performed while the Freeze is in effect will show the same Effectiveness results that were calculated when the Freeze occurred.

Revaluation on Actual Freeze Date

A Freeze is only effective from the day after the Freeze Date onwards. Revaluation and related postings still take place on the Freeze date if a revaluation is run then.

Expiry of a Freeze

Quantum prioritizes events by effective date order. A Freeze expires on the effective date of the next event in the life of the hedge relationship. This might be a Designation, for example, or a Dedesignation. On the date of that event, full processing is resumed.

If no event occurs before then, MTM/OCI is automatically unfrozen on the Derived Maturity Date of the hedge relationship.
