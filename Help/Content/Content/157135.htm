Select a Result View

The Default Table View showing the query's raw results is always available and initially applies. It shows data for all columns currently specified in the Query configuration (Columns tab).

To apply another saved Table View (if any), click on the name of the view and select from the Select View lookup. The current report results are then redisplayed in that view.

The standard Quantum Six views are Table, Pivot and Chart. These can be created from the Configure View menu.

Crystal views are also available where:

  • in Quantum - an existing Crystal Reportâ„¢ has been mapped to the query and the required Crystal export formats have been configured, or
  • in Analytics or Adaptiv - a Pivot view is associated with a user-defined output data table specifically for an existing Crystal Report to use.

TIPS: In the Select View lookup:

  • the type of each view is shown (Pivot, Table, Chart, or Crystal);
  • the view that was selected when the query was saved is marked with Default in the Comment column;
  • use the Search box to quickly find a view;
  • use the > and < buttons to go to the next or previous page of views; and
  • (when more than two pages) click the lookup index button and select the page or click the or button to go to the first or last page, respectively.
